「啊啊啊啊啊... ... !」
不经意间吐出的深深的叹息消失在迷茫的森林里。
浴室的温度是40度，感觉不错。一个经过相当严格训练的浴缸保持不漏水。
如果你把火堆放在炽热的火焰中，它会保持温暖一段时间。
「啊... ... 好。」
「 ... ... 呃... ...」
」我是不是因为做爱太多而发疯了?」
我已经好几个月没洗澡了，两个不知道洗澡水的人看着我，眼神里流露出奇怪的神情。
「怎么，两个人都不知道洗澡吗?」
「浴室... ... 是做饭还是什么?」
」我没有兴趣在热水里煮」
真是太可惜了。精灵们似乎没有沐浴在热水中来缓解疲劳的文化。
这里不是火山区，而且地热似乎没有地方可以烧水。
「进去就知道了，这个好处。」
凡事都是经验。为了不弄脏热水，让赤裸的区域进行净化，让脚进入。
「啊，... ... 啊啊，这，这是啊啊... ...」
这个区域因为热量而痛苦不堪，全身颤抖。站着扭来扭去总觉得有意思。
「你看，不要翻来覆去地洗澡。」
「是、是的。」
热水溢出的程度足以让这个区域进入热水中。当然，我们已经设法不泄露给火箭。
「啊啊啊... ...」
「是的... ...」
对进入腰部就已开始被浴池魔力所束缚的地区表示怀疑的特蕾西亚。虽然也想加入特蕾西亚，但不巧没有三个人可以进去的空隙。
「主、主人... ... 这、这个，太厉害了。」
「对吧? 因为洗澡不能获得这种感觉。」
「…………」
经过一段时间的微妙空气后，似乎吐得很爽的茴芹和利巴也会随之而来。
「主人，我刚才失礼了。 ... ... 那是什么?」
「好像是叫浴室的锅哦。」
「又不是锅子... ... 嘛，还有用锅子洗澡的浴缸，还能说什么呢?」
「利巴也要进来了哦!」
等等，等等，等等
喊叫的利巴没有弄掉泥土就要跳进浴缸里，但他还是抓住了泥土，阻止了他。
「等一下，我来接手!」
结果在天气变暖之前被拿出来，之后去掉泥土和其他东西的利巴也被温暖浴池的魔力所迷惑，变得温顺了。
「虽然没想到要泡在热水里，但是这么大的块头我不能泡。」
试探性的茴芹遗憾地这么说。
「大体上水是怎么来的? 床上的水不会积到这个地步吧?」
「啊，被海耶夫吃掉的那种带状魔法，竟然变成了有用的程度。」
即使没有使用的时候，谜能力也随意地不断加强和改良，现在已经进化到可以把半个浴缸的水量保持成球状的程度。
也就是说，把带子编成球状，在里面放水后滚来滚去。只不过，在滚动数百米的过程中，仍然保留了三成的魔力。
「还不足以吊起一个人。 ... ... 不，应该是绞在一起做成绳子那样的。」
「你的能力根本就是小看了精灵几百年的钻研和研究... ...」
「是啊... ... 哈... ...」
那些被嘲笑的人被推到在浴缸里。
「要是能施展精灵魔法就容易多了，就算眼见为实也不行。」
「我不认为精灵会服从人类。」
」精灵们不会奴役他们的灵魂，不是吗? 他们不是也喜欢听人类的话吗?」
「那是... ... 诶? 唔唔... ... 唔唔?」
一副难以启齿的特蕾西亚。
如果他们有怨恨的话，那就另当别论了，因为他们在大战期间对此一无所知，所以我不认为他们对人类有任何厌恶的感觉。
」嘿，善良的灵魂，你能帮我一下吗?」
我会对森林深处说这种话。
「如果那样的话，我们就没有立场了... ...」
当然不认为能用。 ... ... 大概吧。

虽然也想为茴芹准备浴缸，但如果烧开水平的水，就需要很长时间。嗯，好不容易有个特蕾西亚，请你帮忙吧。
让特蕾西亚从水源运水的时候做一个茴芹可以坐的浴缸。如果只是浴缸，那就简单多了。用土魔法加固土壤，防止漏水，还会制造阶梯状的入口。
我倒了一杯水，就像特蕾西亚昨晚给我看的那个巨大的史莱姆
「你能做到吗?」
「 ... ... 啊? 不，这是什么意象?」
这次应用的是微波炉的原理。
微波炉通过每秒振动24亿个水分子产生摩擦热来加热物体。不知道精灵魔法能否产生这种分子水平的震动... ... 那么精灵的实力又如何呢。
如果水面呈现蓝光，就能从水面感觉到来自耳朵深处的震动。
「 ... ... 真是个精灵。」
虽然振动速度不到二十亿，但仍足以产生摩擦热，水温开始逐渐升高。
按时间来算，大概花了五分钟调整了片区的拍摄时间，特蕾西亚和茴芹发出了哑然的声音。
「没有火却有水... ...」
「这真的变成热水了吗?」
「精灵太全能了，人类敌不过也难怪。」
虽然三人有些惊讶，但水的确像浇在炉子上一样冒着热气，变成了热水。
浴缸的温度稍微提高了一点，应该没问题。
茴芹像下楼梯一样进入热水中。
「啊... ... 啊啊啊... ...」
他颤抖着慢慢地泡在热水里，一直泡到肩膀
「啊啊啊啊啊... ... 主公啊。这个啊... ... 受不了了。」
「对吧? 啊，在你们发疯之前出去吧!」
」是的，先生」
「 ... ... 你是认真的吗?」
只有一个人没有洗澡，特蕾西亚像看见了难以置信的东西一样东拉西扯。
「特蕾西亚也放进去就好了。」
「我可不想被人用热水煮。」
「凡事都是经验... ... 农夫忍者!」
突然间，一阵寒意袭来。我半途而废，你把热水冷却了吗。
「农夫忍者! ... ... 啊，洗完澡后的裸体真的很危险吗?」
「主人，您冷的话，就到这儿来。」
这样说着，茴芹靠过身来开了个缝隙。... ... 在这里客气的话会不会感冒呢。
「那就失陪一下。」
习惯高温，把身体沉到茴芹前。
对了根据茴芹的尺寸调整的深度所以不能坐啊。
」如果你不介意的话，坐在我腿上」
茴芹向坐立不安的我提议。
「你还好吗?」
「这不是主人骑过就能折断的脚。」
他看起来很自豪，但我会这么做的。
好不容易坐在茴芹的腿上，冷静下来的时候，茴芹的身体一下子靠了过来。
嗯，有些东西本来就很合身，或者更确切地说，是因为我的手臂伸进了乳沟里，而且我的茴芹像抱住我一样伸过来，所以我猜他们是在瞄准我。
」主人，我们只是沉默地泡在里面，你要在里面做什么吗?」
「没什么。一边想着今天也发生了很多事，一边感觉疲劳溶化在热水里就好。如果是景色好的地方，一边眺望风景一边进去就很棒了。」
不幸的是，迷途的森林除了篝火照明的范围外一片漆黑。只要调整一下视野过滤器就可以了，不知道是不是只有火的精灵凑过来，脸都红了，对眼睛不好。
「啊... ... 确实感觉到疲劳消失了。」
「人类就是这样消除疲劳的吗?」
也许是因为我是日本人吧。搞不好这个世界就没有洗澡文化了。
