第二天，农活还是老样子，离开了其他农场，去了几乎没有工作的地方。如果一个月的工作减少到一半以下，那就再好不过了。
没有什么比工作更快的了。收成越早，收成越好，收成也就越少，库存也越少。除非这里有农协。
令人担忧的是其他种族参与了收割工作。自古以来，他们在收获季节都会请人帮忙。似乎是在家族内部供养所谓的季节性劳动者。
他们不想和其他部落有任何瓜葛
「那是很久以前的事了。这边没有人来帮忙。如果你愿意帮忙，那就万分感激了。」
「告诉我最后一件事的那个家伙，说另一个经纪人帮了他的半人马，结果被人嗤之以鼻。」
随着时间的流逝，错误总是从最基层开始的。
　／／／　／／／
那天晚上，不知为何，马雷内夫人把我叫了出来。
回到打捞过一次的餐厅
「那么... ... 哈... ... 马雷内，晚安。」
「晚安，先生。」
说着，加雷利吻了吻马雷内的脸颊，走出了餐厅。
算了，他还是个孩子，睡觉也早吧。
「请吧，我去泡茶。」
「不，我不能强迫怀孕的女人。」
女人不听我的劝阻，站起来开始倒茶。
「没关系。我是第二个，这个时候运动比较好，你坐下吧。」
说着，他向我推荐斜对面的座位。
坐在座位上喝茶，她也坐下来马上开口。
「多亏了大家，帮了大忙。工作得太快了，批发商都吓了一跳。」
「啊，工作做得太多了吗?」
「没关系。在此期间，我们可以准备下一个作物，准备工作结束后，我们就有空了。」
说了一些这样的田地情况之后,
」你和他们相处得不好吗?」
他突然猜对了。
「为什么?」
「没有，工作的时候也没怎么说话。特别是精灵和黑暗精灵。」
「啊，那两个人本来就是敌对的种族。不知道是什么原因，决定一起旅行。」
不能说是个很正经的理由。
「我同意你的看法，如果你有很多奴隶，你会很难受的，不是吗?」
「哈哈... ... 哈?」
突然冒出来的奴隶一词让我无法继续说下去。
在区域和利巴上应该有一条像围巾一样裹着布，让他们遮住奴隶纹，在回到自己房间之前不要拿走。
不知为什么会出现奴隶这个词，女人只是笑眯眯地看着这边。
「精灵称呼你为主人，因为精灵是一个骄傲的种族，如果你称呼别人为主人，他们就会认为你是个奴隶。」
「啊... ...」
哦，该死，我没有转头。
「而且我很羡慕你每天晚上都这么兴旺。」
「 ... ... 什么? 不，不，在别人家里做这种事... ...」
「哎呀，不能误解女人的眼光。特别是这样怀孕的时候，会变得很敏感。」
一边说一边抚摸着大肚子的女人。
这个... ... 不行啊。
我已经处理得很好了
「没关系，女仆们好像没注意到。」
「大本已经注意到了，我不认为应该没问题... ... 我稍微克制一下。」
「没关系，把它当作自己的家，随便你怎么用吧。我有时也和儿子一起用。」
我听说了一件很重要的事。
「 ... ... 儿子?」
我情不自禁地用手表示了身高。最后，女士挥挥手笑了。
「啊哈哈，不会吧! 我以为你早就注意到了，不是吗?」
「 ... ... 是不是有什么特异功能?」
以一种特异的方式。不过你猜对了。
「为什么突然觉得自己被回避了呢?」
「是啊，是不是说闻到了同样的味道?」
那是什么。是可能会犯禁忌的意思吗？
「散发出让别人察觉的气味吗?」
「自己的味道是特别难闻的东西... ... 你介意我们父子的谈话吗?」
」这是你想说的另一个想法吗?」
「好吧... ...」
这位女士用嘴捂住脸颊。这只是一个小小的回报。
「一直瞒着周围的人，肯定有很多矛盾吧。如果有什么想说的，就当做社会学习来听吧。」
「那么，我就说些讨厌的话吧。」
我喝完茶坐直了身子女士开始说话。
不出所料，奥特莱特夫妇是亲生父子。
原来的丈夫是中央管理人员，但因病突然去世。据说，加雷利虽然年纪小，但一直关心着不断悲伤的母亲。
不久，寂寞的目的地转向了小儿子，即使知道禁忌也伸出了手。
伽雷利也接受了他心爱的母亲对小自己的追求，结果怀上了现在的「大儿子」。
当然不能让周围人知道这件事，他们经过深思熟虑，终于搬到了与自己无缘的这个地方，成了有着年轻丈夫的管理员。
「我们互相依赖。我爱加雷利，加雷利也爱我。大儿子和肚子里的孩子都是确凿的证据。... ... 对人类来说，是不是很奇怪?」
「嗯，也不是没有先例。父子、兄妹、亲戚确实也有类似的例子。而且，对他们的目光相当苛刻。」
「人也是这样吗... ...」
「在自己的故乡，爱情和年龄都说了，如果涉及到孩子，就会以强奸罪被抓起来，所以相当严厉。」
「 ... ... 是吗?」
「不过，最近，社会上承认男男女女情侣的活动开始扩散了。」
也就是说，应该和最近成为话题的家人一样认可。
「哦，这些人?」
难道这里没有同性恋吗。
「你们两个都是成年人了。再说，这里的法律是不同的! 和亲生女儿相爱就会被关进监狱，这是没有法律的吧?」
「没有。」女人回答。
「而且，我个人觉得挺好的。把法律、规则和别人的恋爱都考虑进去才奇怪。」
对我来说连常识之类的尺度都没有。「常识」这种既方便又方便的词大多用在眼下... ... 。
「 ... ... 太好了。果然感觉到的气味没有错。」
恢复笑容的女士这样说道。
「我没想到你会提起怀孕的妻子和近亲性交。」
「有人可以分享秘密是很重要的，即使是其他种族的人也一样。因为这样才能说话，才能让他们理解。」
随后，这位女士开始详细地叙述她和儿子之间的爱情。
随着年龄的增长，尾巴变大了、射精的精液增多了、味道怎么样、体位怎么样... ..。
老实说，我没想到女方会想到用近亲性交来讨论这个话题。而且对男人来说。
而且，她这么一下子改变了主意，我觉得我们必须说同样程度的话，告诉她们我们在做什么
「还是加把劲比较好... ...」
行为的内容被修改了。
经过一个多小时的下流话，一位困倦的妇女站起来准备就寝。
「呵呵，这是我第一次把这么尴尬的事公开，而且非常愉快。」
「听起来很浓... ...」
「女人在玩的时候在想些什么?」这些内容真是令人厌烦。
「但是他看起来很开心... ... 哦，是的，是的。你有和那些女人生孩子的计划吗?」
已经和二十个哈比族生孩子的事没有流利地说。
「不能说没有可能性... ... 不是吗? 马雷内先生已经是两个人了，以后也有安排吗?」
「是的，十个还是二十个。只要彼此要求，就一直这样。住几个人是不会有问题的。」
哈哈，这也太夸张了。
我很佩服她，她冲我笑了笑，走出了餐厅。
