WEB 29 百货商店
 
 
一一一一一一一一一
 
 
 
 
 
　我使用『索敌』寻找似乎可以入侵的地方。
 
　正如我所意料，物资的出入口有放哨的学生。
 
　只有一个人吗……虽然出其不意是可以入侵，但也犯不着要我小心翼翼地闯过。
 
　我寻找其他比较容易入侵的地方，这次找到感觉合适的窗户。
 
　虽然窗户上锁了，但也不是问题。
 
。
 
「莫莫」
 
。
 
「汪」
 
。
 
　莫莫从『影子』里出现。
 
　它让影子从窗户的缝隙间钻进去，从内侧将锁打开。
 
　只要有一丁点『缝隙』，『影子』就能从缝隙中钻进去。
 
　真的是方便的技能。
 
。
 
「谢了，莫莫」
 
。
 
　作为回礼我想尽情地摸摸它，不过还是忍住了。
 
　仅仅是把手放在它头上摸了一下，接着让莫莫藏进影子里，立刻潜入窗户里面。
 
　已经利用『索敌』确认过窗户里面没有人。
 
　侵入成功。
 
。
 
　里面虽然昏暗无光，不过我有『暗视』，倒是可以清楚看见。
 
　这里是……营业员的休息室吗？
 
　有摆放着桌子和折叠椅，墙边还有存放物品的储物柜。
 
。
 
「刚才应该有人在这里吧……」
 
。
 
　桌子上还放着饮用水和营养食品。
 
　说不定有人会过来这里，还是尽早离开吧。
 
。
 
　走到走廊上，瓦楞纸箱以及仍没进行销售的商品将通道挤得十分狭窄。
 
　这里这么昏暗，似乎不用烦恼藏身问题。
 
。
 
　装设的荧光灯也没有打开。
 
　果然是没有电力吗……。
 
　像这种百货商店不都应该有发动机或者太阳能发电之类的，那些也行不通吗？
 
　话说究竟是怎么了?我的公寓也是一样，为什么只有电力不能使用？
 
　只有一道渺小的光芒照亮着暗淡的走廊。
 
　固定住的手电筒以一定的间隔摆放着。
 
。
 
「果然有相当多的人在啊……」
 
。
 
　光是现在『索敌』的范围内就有十人以上的反应。
 
。
 
「……嗯？」
 
。
 
　有什么人正往这边过来。
 
　有两个人吗。
 
　我立即启用『潜伏』和『气息消除』，隐藏在商品的阴影里。
 
　是学生过来这边。
 
　其中一个是称之为西野、疑似这些人领头的家伙。另一个则是戴着眼镜，似乎很老实的一位学生。
 
　他们并没有注意到我，走进了休息室里面。
 
。
 
　当看不到那俩人的身影后，我行动起来。
 
　从开门不关的门口走到销售场内。
 
　场内面积十分宽敞，里面也是暗淡无光。
 
　从商品的分区来确认谁在哪里吧。
 
　『索敌』、『气息消除』、『无声移动』、『暗视』在这种情况下果然才能真正发挥出效果。
 
　莫莫也在『影子』里，根本不会被看到呢。
 
。
 
　我边缓慢地走动，边逐渐确认人数以及销售的区域。
 
　结果我知道这家百货商店里总共有二十二个人。
 
　学生是十个人，避难民是十二个人。
 
　过来避难的人们全都集中在场内的一个地方。
 
　而有三位学生则是围绕着他们举起武器监视着。
 
　避难民全都露出疲惫至极的神色瘫坐在地面上，其手上都有饮用水以及食物。应该是学生他们给的。
 
。
 
　嗯？有一位避难民站起来，往学生那边走过去。
 
　记得那个人不就是在门口土下座的大叔吗。
 
。
 
「呐，能不能再给我一瓶水?水已经喝完了……」
 
。
 
「不行，一开始不就说过一个人只有一瓶水了吗」
 
。
 
「可、可是从早上到现在我什么都没有吃……！拜托了，再给我最后一瓶水吧！都这种时候了，再帮帮大叔吧，呐？」
 
。
 
　喂喂，这人到底是有多么厚脸无耻啊……。
 
　其他的学生也是一副「在说什么屁话，这家伙」的表情，瞪视着大叔。
 
。
 
「啧，我去问一下西野。你给我呆在这里别动」
 
。
 
　他吐了口痰，离开自己的岗位。
 
　我还以为他肯定要发怒了，但似乎是忍住了。
 
　呜哇——似乎很不妙啊ー……。
 
　其他两人似乎没有想要离开原地的念头。
 
。
 
　那么，总之场内的情况以及人数已经确认清楚了。
 
　物资待会再来收纳也行，先收集情报吧。
 
　虽然用『聆听』技能偷听了大叔他们的谈话，但似乎并没有什么有用的信息。总是在嘀咕着「坚持到自卫队过来救援就好」或者「我这是在做梦，绝对是在做梦吧」之类的。
 
。
 
　既然要调查只能调查学生那边吗。
 
　我躲在墙边往休息室那边走过去。
 
　毕竟有许多瓦楞纸箱和商品，用来躲藏是正合适。
 
　我把耳朵贴在墙壁上，发动『聆听』技能。
 
　我早已经通过『索敌』得知里面有三个人。
 
　那么，他们究竟会聊些什么事情呢。
 
。
 
「―――收留他们真的好吗？」
 
。
 
「对、对啊，连我们自己人都才勉强坚持得下去，要是还让其他人加入的话……」
 
。
 
　这些声音似乎是刚才的不良少年，以及戴着眼镜的老实人。
 
　好，就称呼为不良君和眼镜君吧。
 
　他们貌似是不同意收留避难民。
 
　嘛，这也是当然吧。老实说，他们无非就是拖后腿罢了。
 
。
 
「……是啊，不过现状上来看我们的确是缺乏人手。柴田，他们已经相信等级和技能了吗？」
 
。
 
「姑且是展现给他们看了，可他们还是半信半疑呢。说心里话吧……我自己都无法相信这样做就能提升等级。而且……」
 
。
 
「而且？」
 
。
 
「那个……似乎有些人对于杀死魔物的行为本身怀有抵触感啊。那些家伙什么事情也没做，只有要求倒是一大堆……该死！」
 
 
 
 
 
果然刚才的事情令他相当火大呢。
 
　不良君用手揍了墙壁。
 
。
 
「说的对啊，那、那些食物明明也是我们拼上性命从其他超市拿过来的……！」
 
。
 
　像是要同意他的说法，眼镜君也嘀咕道。
 
。
 
「……你们两人的心情我能体会，不过现在就忍一忍吧。只要他们的等级提升后，无论是搜索食物还是探索其他地区都会比现在更加轻松」
 
。
 
「可是那些家伙会老实地听从我们的吩咐吗？」
 
。
 
「没有问题，你知道我的『技能』吧？而且……最糟糕的情况下他们也是『弃子』。当出现我们无法应付的魔物时，他们就是用来为我们争取逃跑时间的诱饵。我是这么说过的吧？」
 
。
 
「嘛……虽然是这样」
 
。
 
　呜哇—，你满不在乎地想着这种冷酷无情的事情啊，西野君。
 
　嘛，如果没有这种想法应该就活下不去吧。
 
　是说他还真是相当自信，究竟是拥有什么职业和技能呢？
 
　真令人在意。
 
。
 
「既然已经收留他们了，我们当前更重要的事情是必须赶快增加食粮的库存。等出去外面的两个人回来以后，就重新决定轮流出去找食物的顺序吧」
 
。
 
「了解」
 
「好的……」
 
。
 
「总之，现在要最优先确保食物。幸好只要到附近的超市，还能得到一些容易保存的食材―――」
 
。
 
「不、不好了！」
 
。
 
　一位学生气势汹汹地打开门，走进房间里面。
 
　嘛，虽然我早用「索敌」注意到了。
 
。
 
「怎么了？发生什么事了？」
 
。
 
　西野君并没有为他突然闯进来而感到惊讶，而是询问着他。
 
。
 
「哈啊、哈啊……刚、刚才出去外面的人已经都回来了……哈啊、哈啊……！不、不敢相信。怎么可能、有这种事……！」
 
。
 
「冷静一点，到底发生什么事了？」
 
。
 
　怎么了？
 
　该不会是出现了跟高级猪头人一样强大的魔物吧？
 
　然而他所说的回答却是在我意料之外。
 
。
 
「他、他们说药品店里面空空如也……而且不光是这样，就连这一带的超市和便利店都没有留下半点食物和物品……！」
 
。
 
「……你、你说什么!?」
 
。
 
　听到他的报告，他们三人都发出惊讶声。
 
　……啊咧？该不会犯人，就是我吧？
 
一一一一一一一一一
