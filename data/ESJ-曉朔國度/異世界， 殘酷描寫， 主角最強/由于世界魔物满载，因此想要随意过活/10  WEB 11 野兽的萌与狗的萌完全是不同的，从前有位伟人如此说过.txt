WEB 11 野兽的萌与狗的萌完全是不同的，从前有位伟人如此说过
 
一一一一一一一一一
 
。
 
　诶——，这要怎么办才好啊？
 
　总之先在脑海里默念YSE看看。
 
。
 
≪您的申请已经受理了，莫莫已经加入您的队伍≫
 
。
 
　看样子这样莫莫就成为伙伴了。
 
　嗯，怎么说呢。有太多想要吐槽的，脑袋都整理不过来。
 
 
 
 
 
。
 
　看一下人物状态窗口，最下面又增加一项新的项目。
 
　名为『队伍成员』的项目。
 
　里面有记录莫莫的名字。
 
。
 
　莫莫
 
　柴犬　ＬＶ２
 
。
 
　原来如此，组成队伍就会显示出名字和等级。
 
　柴犬是……。
 
　不，该不会是『种族』吧？
 
　虽然也有可能是狗的品种，不过这是游戏系统，还是『种族』比较适合。
 
。
 
　假如说这是人类，究竟会怎么显示呢？
 
　职业吗？嗯~没有遇见其他人，根本不知道这种事啊。
 
。
 
　话说，给我等一下。
 
　ＬＶ２……？
 
。
 
「……呐，莫莫。你该不会是打倒魔物了吧？」
 
。
 
「汪！」
 
。
 
　莫莫精神奕奕地回应。
 
　也就是说YES？
 
　诶、不会吧、真的假的？
 
。
 
　莫莫摇着尾巴，歪着头示意「怎么了—?」。
 
　好可爱。
 
　这么可爱的汪酱怎么可能打倒了魔物。
 
。
 
　没错，一定是某种误会吧。
 
　啊，对了。不是有确认的方法吗。
 
。
 
「呐，莫莫。你有没有看过这种石头？」
 
。
 
　我从道具包里面取出蓝色的小石头。
 
　这是打倒哥布林的证据，既掉落道具。
 
。
 
　话说这要用在哪里？
 
。
 
　顺便一提，收纳进这玩意时名字就知道了。
 
　道具包的目录里显示的是「哥布林的魔石（极小）」。
 
　魔石……异世界的王道道具。
 
　一般都可以到冒险者公会换钱，或者可以作为强力武器的制作素材。可是现实中根本不存在公会，有的只是无视劳动基准法的黑心企业（社畜制造机）。
 
。
 
　嗯？
 
　当我拿出哥布林的魔石（极小）的瞬间，莫莫猛然晃动尾巴。
 
。
 
「汪汪！」
 
。
 
　给我！那东西给我！似乎是这么说的样子。
 
　诶？你想要吗？
 
。
 
「汪!」
 
。
 
　貌似是想要的样子。
 
　是说它能理解我的意思吗？
 
。
 
　我将拿着魔石的手往右移，莫莫的目光也跟着往右。
 
　往左移，莫莫的目光也往左移。
 
　绕圈圈之后，莫莫的头也绕着圈。
 
。
 
「わふん……」
 
。
 
　啊，眼睛打滚了。……好可爱。
 
　抱歉抱歉。
 
。
 
　总之我将魔石放在莫莫眼前。
 
　它闻了闻魔石的气味，交互看着我和魔石。
 
　我点点头。
 
。
 
　不过它要这东西是要干嘛啊？
 
　当我这么想的瞬间，莫莫叼起魔石，接着直接吃下去了。
 
　诶诶!?吃掉了!?
 
。
 
「等下，莫莫！不行啦，那种东西不能吃的！快吐出来，吐出来」
 
。
 
　然而已经为时已晚。
 
　咔哧咔哧、咕噜。
 
　莫莫满足似的吃掉了魔石。
 
。
 
　诶、诶诶ー……。
 
。
 
　它用着闪闪发光的眼神注视着我。
 
　没有了吗？我还想要更多啦—。
 
　感觉像是在这样说着。
 
。
 
「……わふん？」
 
。
 
　不能给我吗？不行吗？
 
。
 
　呜咕……咕喔喔喔。
 
　给我住手，别用这种眼神看我。
 
　回想起某ＣＭ的吉娃娃。我要怎么办。
 
　结果我败给了莫莫的眼神，从道具包里取出哥布林的魔石。
 
　莫莫超开心地吃着。
 
。
 
「真的假的……这能吃吗？」
 
。
 
　我往嘴里也丢一快试试。
 
　好硬，只是普通的石头。而且好苦，难吃得要死。
 
　这根本不是能咬得动的东西。
 
　为什么莫莫能吃得下去呢？
 
。
 
　哎，算了。反正它开心就行。
 
　結局、俺は倒したゴブリンの魔石を全てモモにあげた。
 
。
 
「啊，话说回来」
 
。
 
　我看着道具包的目录。
 
　里面收纳了许多东西，所以要找到相当『那个』还是有点辛苦的。
 
。
 
「有了……」
 
。
 
　最初被我撞死的那条狗。
 
　那条狗掉落的紫色小石头。
 
　那石头名字也是清楚地记载着。
 
。
 
『Shadow・Wolf的魔石（小）×１个』
 
。
 
　看来那条大狗的名字似乎是Shadow・Wolf。
 
　直译就是影狼吗，这名字太帅气了，可恶。
 
。
 
　是说收纳目录真是方便。
 
　收纳的物品『名字』都能正确地记载进去。
 
　这或许在各个方面上都可以应用。
 
。
 
　我取出影狼的魔石，放在莫莫的眼前。
 
　于是，莫莫以前所未有的气势晃动着尾巴，表情也十分开心。
 
　似乎是非常高兴的样子。
 
　可以吗？可以让我吃这个吗？它的目光如此诉说着，好可爱。
 
　当我说可以吃之后，它就气势汹汹地吃起来了。
 
。
 
　话说回来，当将食物喂给狗的瞬间，为什么心头会小鹿乱撞呢?感到怦然心动。
 
　只有我会这样？不对，只要是爱狗人士都应该能体会。
 
　我悠闲地看着莫莫吃东西的样子。
 
。
 
「嗯？」
 
。
 
　当它吃完影狼的魔石之后，身体抖了一下。
 
　接着像是在确认自己身体一样，当场转了几圈。
 
。
 
「怎么了，莫莫？」
 
。
 
　果然是吃下了不能吃的东西吧？
 
　我担心地看着它，而莫莫则是拍打着我的影子后，精神奕奕地蹭着我。
 
。
 
「汪！」
 
。
 
　没事，不用担心，谢谢了。像是在这样说。
 
　是我的错觉吗，感觉声音比起刚才更有活力了。
 
 
 
 
 
好可爱，我摸了它一会儿。颓废的心灵被治愈了。
 
 
 
 
 
。
 
「那么，莫莫。我接下来要到附近的便利店。你要跟着来吗？」
 
。
 
「汪！」
 
。
 
　莫莫干劲满满地回应我。
 
　嗯~考虑到我的技能和职业，以及魔物的危险性，其实我一个人行动比较好……顾及到莫莫的安全也是。
 
　不过……不知为何我总觉得带着莫莫一起去比较好。
 
　曾经有谁说过人类是以理性思考事情，以感情来行动的生物。
 
　这种时候还是遵循直觉比较好。
 
。
 
 
 
 
 
　于是几分钟过后―――我确信我的直觉是正确的。
 
。
 
　莫莫远比我想象中还要厉害得很。
