翻译：飒君CONAN
 
 
 
WEB 27 某种意义上的强敌出现了
 
 
一一一一一一一一一
 
（颯:建材市场改为百货商店，我发誓我不会再用字典查日本现代名词）
 
一一一
 
 
 
 
 
　我和莫莫准备一起前往百货商店。
 
　从这里走过去需要二十分钟左右吗。
 
。
 
　离开公寓大楼，绕开大马路走在小路上。
 
　尽可能避免太显眼，偷偷地走。
 
　顺便一提，莫莫目前就在我的『影子』里。
 
　如今正待命着，一旦发生什么事都能够立刻现身。
 
。
 
　在街道上走着，来到离百货商店还有点距离的地方，我们遇到了新的魔物。
 
　它似乎是躲藏在道路旁边的茂密草丛里 。
 
。
 
　体型差不多是比起篮球大一圈的程度。
 
　样子是蓝色的半透明球体，而内部则是浮游着小圆球，以及些气泡往上浮起。
 
　幻想世界中的超王道。
 
　与哥布林、猪头人并列的常规魔物。
 
　没错，就是史莱姆。
 
。
 
「噢噢……」
 
。
 
　现实中看到实物之后，它是比我想象中还要不可思议的生物。
 
　就跟某游戏一样，它并没有眼睛。
 
　外形就像是巨大的变形虫一样，又像是 水母的头部一样……难以形容。
 
。
 
　是说很强吗，史莱姆？
 
　『危机感知』并没有多大的反应，应该是不怎么强……。
 
。
 
「莫莫」
 
。
 
「……汪」
 
。
 
　莫莫从『影子』里出来，观察情形。
 
　史莱姆像蛞蝓一样缓慢地蠕动，被它所覆盖的部分草丛都溶解了。
 
　那就是在吃饭吗？
 
　似乎没有注意到这边的样子。
 
。
 
「……打倒吧」
 
。
 
「汪」
 
。
 
　莫莫也点着头，立马将『影子』朝着史莱姆释放。
 
　先是束缚住，再用冰箱砸死。
 
　必胜连招
 
。
 
　延伸而出的影子瞬间就束缚住史莱姆。
 
　然而在下一个瞬间，发生了一件对于我们来说是意料之外的事情。
 
。
 
　ちゅるん、と。（颯:拟声词）
 
　史莱姆从缠绕在它身上的影子缝隙中溜出来了。
 
　扑通一声，掉落在地面上。
 
　接着又若无其事地开始吃草。
 
。
 
　怎、怎么可能……!?
 
　莫莫的影子没有用!?
 
。
 
「呜ー……！」
 
。
 
　莫、莫莫正不甘心着。
 
　有点可爱。
 
。
 
「冷、冷静一下，莫莫」
 
。
 
　我摸着莫莫劝它冷静下来，接着取出冰箱。
 
　没问题的，即便莫莫的影子没有束缚住它，它的动作也很迟钝。
 
　这样就解决！
 
。
 
　释放出来的冰箱，将史莱姆压了下来。
 
　然而事情又是偏离我们的预料。
 
　史莱姆以压平的状态从冰箱的低下蠕动出来了。
 
　就连内部浮游的疑似核心部分也变成平形。
 
　于是史莱姆又像是若无其事一样直接开始吃着草。
 
。
 
「……」
 
。
 
　好火大。
 
　虽然也许是我误会了，但它那副仿佛什么事都没发生一样，开始吃着草的样子真令人火大！
 
。
 
「咕……！」
 
。
 
　莫莫发出威吓声。
 
　冷、冷静一下，莫莫。
 
　我还是第一次看到莫莫这样子，好可爱。
 
。
 
　是说该认真了，史莱姆是物理攻击没有效果吗？
 
　那砍它的话怎么样？
 
　我拿出菜刀，接近史莱姆。
 
　即便是这么接近它，史莱姆也是没有任何反应。
 
　虽说『影子』和冰箱的攻击中我们早就注意到了……。
 
　该不会这家伙没有攻击的手段？
 
　不对，从『敌意感知』没有反应来看，应该是没有这个『意思』吧？
 
。
 
　我对准史莱姆的核心挥下菜刀，没打到!?
 
　史莱姆身体变形了一下躲开攻击，无伤。
 
　再次开始吃着草。
 
。
 
　我砍！我砍！我砍！
 
　ぐにゅーん、ぷるるん、ぽよんぽよん。
 
　史莱姆依然无伤。
 
。
 
　呜噢噢噢……！
 
　好难打。
 
　这什么鬼？
 
　史莱姆是这么麻烦的对手吗!?
 
　既不能砸死，也不能砍死，感觉物理攻击都没办法打败它了。
 
。
 
　物理不行……那其他的攻击手段呢？
 
　比如火。
 
　我取出汽油和打火机。
 
　如果有喷发剂之类的，我就能当场搞出喷火器了，不巧的是我手头上没有。
 
　考虑到自己也有可能会烧伤，于是我让莫莫把『影子』变化成手的形状，让影子拿着汽油和打火机。
 
　于是向史莱姆点火。
 
。
 
「噢」
 
。
 
　效果立竿见影。
 
　刚才还游刃有余的史莱姆，一接触到火就立马啪嗒地暴走起来。
 
　它着急地想要扑灭火，但来不及了。
 
　不久后史莱姆的身体消解，直接消灭了。
 
。
 
≪获得了经验值≫
 
。
 
　脑海里回响着天之声。
 
　史莱姆曾经所在的地方有颗绿色的魔石。
 
　收纳之后，显示的是『史莱姆的魔石（极小）』。
 
。
 
「呼ー……」
 
。
 
　某种意义上，它是以往最棘手的魔物也说不定。
 
　还是第一次遇到物理攻击无效的魔物。
 
　而且耗费那么多时间与道具，经验值并没有那么多。
 
　明明只是打倒一只史莱姆，就用上了大量的汽油。
 
。
 
　嗯~考虑到时间与收获，还是不要狩猎史莱姆比较好。
 
　何况对方不会进行任何的攻击。
 
　又是草食性，感觉也是无害。以后看见史莱姆就直接无视好了。
 
。
 
　是说史莱姆的弱点是火吗。
 
　如果那头高级猪头人也有像是弱点的地方就好了呢。
 
　不会有的吧，那家伙……。
 
。
 
　我将史莱姆的魔石给了莫莫，继续前进。
 
　途中狩猎了三只哥布林。
 
　嗯，果然还是哥布林这边比较容易对付。
 
　不过等级没有提升，遗憾。
 
 
 
 
 
。
 
 
 
 
 
。
 
　之后过了一会儿，我和莫莫抵达百货商店了。
 
　这附近也是跟购物中心并列的商业地区。
 
　百货商店、超市、药品店、书店都有，停车场也挺宽敞，许多人都会来这里逛街。
 
　看样子应该是没有魔物。
 
　只不过有『先来的』。
 
。
 
「所—以—说，是我们先来到这里!***到别的地方，呆子们！」
 
。
 
「怎、怎么这样！这里又不是你们的东西吧！我们也是拼了命才逃到这里的！」
 
「只要让我们呆到自卫队来救援就行了！」
 
「对啊！一般来说这种时候应该都要互相帮助吧！」
 
「快点让我们进去里面啦!要是那些怪物袭击过来要怎么办呀！」
 
。
 
「鬼才知道！自己去死吧！总之这里已经是我们的东西了！」
 
。
 
　百货商店的入口附近。
 
　那边有几位拿着武器的男性不良学生堵在门口，以及有将近十个人的男女逼近他们，向他们抗议。
 
。
 
　呜哇——……。
 
一一一一一一一一一一
