WEB 56 前往购物中心
 
 
一一一一一一一一一
 
「接下来……」
 
。
 
在走进购物中心之前，先把战斗中使用的废车和岩石回收。
 
与水不同，这些都是可以多次使用的武器。
 
得好好地回收回来。
 
。
 
当然，我也没有忘记高级猪头人的魔石和武器。
 
看来一之濑小姐在离开的时候似乎没有捡走。
 
。
 
「……好大啊」
 
。
 
我捡起高级猪头人掉落的魔石。
 
与以往的魔物相比，明显大很多。
 
有拳头那么大。
 
收纳进道具包之后，显示的是『路斐尔的魔石』。
 
。
 
「『路斐尔』……？」
 
。
 
我还以为肯定显示的是『猪头人的魔石』，原来不是吗？
 
路斐尔……这是那头高级猪头人的名字吗？
 
说起来，感觉在晕倒之前好像有听到这个词语。
 
持名魔物。
 
在游戏中有名字的魔物都会很强。
 
考虑到这一点，那头高级猪头人的实力也是可以理解。
 
老实说作为被它逼到走投无路的人而言，只会感到世道不公就是了。
 
。
 
「……先离开吧」
 
。
 
待会再想。
 
根本不知道魔物什么时候还会再袭击过来。
 
我也很在意影狼群体的动向，还是趁着走进里面吧。
 
。
 
「哎呀……」
 
。
 
也许是因为疲劳吧，走得踉跄的双脚都没办法正常走路。
 
。
 
「汪」
 
。
 
于是莫莫跟过来利用『影子』支撑着我。
 
这是多少温柔的孩子啊，被治愈了。
 
作为道谢摸摸它以后，莫莫似乎很舒服似的眯着眼睛。
 
。
 
「……（抖抖）」
 
。
 
『拟态』为衣服的小红似乎也担心着我，抖动着身体。
 
。
 
「我没事的啦，好，走吧」
 
。
 
从莫莫和小红那里得到鼓舞后，我们走进了购物中心。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
「……这还真过分啊」
 
。
 
该说是正如预料吗，购物中心里面变得相当悲惨。
 
墙壁和地面遭到破坏，地面上的商品被踩得到处都是。
 
由于没有灯光，现场暗淡无光，十分令人毛骨悚然。
 
。
 
「似乎没有……魔物啊」
 
。
 
『索敌』既没有反应，也没有感觉到『厌恶感』。
 
那头高级猪头人所带领的猪头人都全部移动到百货商店那边了吧。老实说帮大忙了。
 
现在我这种疲劳的状态想避免发生战斗。
 
。
 
啊，不过我得小心宝箱怪。
 
『索敌』和『危机感知』对那种魔物完全没有效果。
 
。
 
我十分警惕地在店里走着。
 
有座尸体堆积而成的山。
 
应该是跑慢一步的人吧。
 
他们都被堆积在一起，毫无造作地迭出一座山。
 
其中貌似也有疑似自卫队的尸体。
 
我以难以言喻的心情从旁通过。
 
。
 
这三天里我已经完全习惯尸体了。
 
应该也有『抗压耐性』提升的影响吧。
 
不如说如果没有这个技能，这样的光景肯定会造成心理阴影。
 
。
 
「真是讨厌的习惯啊……」
 
。
 
光是看见尸体就会这么情绪紧张。
 
假如某天杀死人，究竟会感觉到多么惊慌呢。
 
我不敢想象。
 
。
 
通过尸体之山，我们继续在店里走着。
 
。
 
「总之这附近就可以吧」
 
。
 
「汪」
 
。
 
我坐在面向不同于刚才所来的其他出口的长椅上。
 
如果是这样，即便发生什么事都能立刻逃跑吧。
 
虽然想补充物资，在店里探索，不过在这之前还有事情要做。
 
就是确认人物状态。
 
由于我晕倒来，所以没能完全将天之声听到最后，所以我想看看有没有改变的地方。
 
。
 
记得等级应该是提升了三级才对。
 
打倒那头高级猪头人，等级提升了三级。
 
我是该觉得光是打倒一只魔物就提升三级了，还是该觉得明明度过那般残酷的遭遇，却只提升三级呢……。
 
。
 
总而言之，打开人物状态。
 
。

正如我所想，ＳＰ增加了６０点数，而ＪＰ则是３0。
 
莫莫和小红的等级也提升了。
 
能力值整体都上升很多。
 
『力量』和『防御』都要立刻达到三位数。
 
还有就是ＨＰ很糟糕。
 
果然我好像还是没有完全痊愈。
 
得好好休息了。
 
。
 
不过更重要的是有个项目令人在意。
 
固有技能的栏目
 
早熟下面的部分。
 
。

。
 
这究竟是什么呢？
 
在跟高级猪头人战斗之前应该没有才对。
 
既然是在固有技能的栏目里，那就应该是某种『技能』……。
 
试着点击一下，什么也发生。
 
纵使我在脑海里想要消耗SP提升等级也是毫无反应。
 
。
 
「怎么回事？」
 
。
 
说起来在晕倒之前感觉天之声似乎有说过『讨伐奖励』，该不会就是当时获得的技能吧？
 
可是这样一来为什么看不到啊？
 
跟以往获得的技能明显不同。
 
真在意啊……。
 
不过既然看不到也就没办法了。
 
。
 
以往的技能和职业都是从字面意思上隐约推断出其中的效果，然而像这样子的技能就完全束手无策了。
 
既然不知道调查的方法，总之先姑且保留吧。
 
。
 
首先是跟平时一样先花掉ＪＰ和ＳＰ吧。
 
特别是职业这边『暗杀者』可以提升到ＬＶ１０。
 
如果跟『密探』的时候一样，应该会解锁上级职业才对。
 
。
 
我立马消耗10点ＪＰ提升『暗杀者』的级别。
 
那么会怎么样。
 
我兴奋不已的等待天之声。
