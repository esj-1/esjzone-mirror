翻译：飒君CONAN
 
 
 
WEB 2 看样子世界是变得奇怪了
 
 
 
。
 
　我醒了过来。
 
　啊啊，睡得真舒服。
 
　不想工作，懒死了。
 
　嘛，今天是休息日，再怎么懒散也可以吧ー。
 
。
 
「是说会在固定的时间段起床，这社畜的本性真的是根深蒂固啊……」
 
。
 
　想上厕所……撒尿。
 
　拖着身体离开被褥。
 
　到厕所撒完尿，神清气爽。
 
。
 
　话说外面感觉好吵。
 
　好像听到了警报声以及汽笛的声音……？
 
　大清早的怎么了？
 
　嘛，算了。
 
。
 
「肚子好饿……」
 
。
 
　结果昨晚回家之后就立刻睡着了，所以什么也没吃。
 
　嗯，吃完早餐再睡回头觉吧。
 
。
 
「嗯～啊、没有下米。真没办法ー，做面包吃吧……」
 
。
 
　毕竟有鸡蛋和腊肉，以及吐司。应该可以做腊肉鸡蛋吧。
 
　好，就做吉卜力饭吧。
 
（颯:原文为ジブ◯飯，其实就是宫崎骏的电影里出现料理的统称，而吉卜力就是动画公司名字）
 
　也就是巴鲁吃的食物。
 
（颯:原文为ぱずー，他是吉卜力制作的动画电影天空之城的主人公）
 
。
 
「……嗯？啊咧？」
 
。
 
　好奇怪。
 
　冰箱里面没电了。
 
。
 
「呜哇，牛奶不是冰的……」
 
。
 
　从冰箱里取出的牛奶是常温的。
 
　明明我想喝冰牛奶的说。
 
　我想拿出一些冰块而拉出盒子，然而冰块已经都溶解成一摊水了。
 
。
 
　怎么回事？
 
　该不会，我拉了下房间里面的荧光灯。
 
　没反应。
 
。
 
「……完了，该不会是跳闸了吧？」
 
。
 
　昨天有打雷吗？
 
　没有吧，印象中天气很好来着。
 
。
 
　我没办法，只能走到玄关那里拉起闸门。
 
　没跳闸。
 
　怎么了？
 
。
 
「喂喂，不会是停电吧？」
 
。
 
　这附近是停电了吗？
 
　刚才响起的汽笛也是因为这样吧。
 
　真是没办法～努力吧東◯電力。
 
（颯:不知道是东北电力还是东京电力，于是不翻了）
 
　煤气……嗯，没问题。
 
。
 
「嘛，暂时的话应该是没问题吧。比起这件事，吃饭吃饭」
 
。
 
　放上长柄平锅，打开火。
 
　倒入食用油，放进鸡蛋和腊肉。
 
。
 
「烤炉也用不了，那面包也用平底锅烤吧」
 
。
 
　如果腊肉鸡蛋做好之后，添加上黄油再用面包片包起来。
 
　有片面包片上面烤得有点焦了。
 
　也就是所谓的三明治，这意外很好吃呢。
 
　再用刚才的腊肉鸡蛋包进去，好，完成。
 
　还加上我喜欢的盐和胡椒粉。
 
　简便的料理。
 
　嗯～不错的气味。
 
　好像很好吃的样子。
 
。
 
「我开动了」
 
。
 
　面包片清脆可口，腊肉柔嫩顺口，半熟的鸡蛋烧香甜美味。
 
　真好吃，如果还有冰牛奶就完美无缺了ー。
 
　我喝着牛奶打开电视，没反应。
 
　啊、对了，停电了。
 
　不然就用手机看一下新闻―――
 
。
 
「……啊—，忘了充电了」
 
。
 
　手机也没电了。
 
　真的假的，今天是什么鬼日子！
 
。
 
「嗯？给我等一下，没有电不就代表连上网都不能上了吗！」
 
。
 
　明明我收藏的小说都在等待着我……。
 
　哈啊……没办法。睡觉吧。
 
　迅速地吃完早餐，躲进被褥里。
 
　总之要洗的衣服就先加水进去。
 
　过会再好好洗干净。
 
　……那家伙的伙食之后再给吧。
 
。
 
「那么，午安」
 
。
 
　于是我开始睡回笼觉。
 
　是说外面从刚才开始就真的好吵……。
 
　到底是怎样啊……。
 
。
 
　嘛，虽然这样我也能睡着就是了。
 
　咕ー……。
 
 
 
 
 
　……。
 
 
 
 
 
　……。
 
 
 
 
 
。
 
 
 
 
 
　醒过来了。
 
　睡得真熟……。
 
　我离开被褥，看了下时间。
 
。
 
「呜哇，已经中午了」
 
。
 
　回笼觉睡了六小时。
 
　从窗帘缝隙间照射进来的阳光完全就是中午的日光。
 
。
 
「好刺眼……」
 
。
 
　对于社畜来说阳光就是毒。
 
　体内的负面能量（主要是上司导致的）被净化了。
 
　不行，不能被净化。
 
　这些能量直到发泄到上司之前都要保存下来。
 
　我唰啦一声打开窗帘。
 
。
 
「……嗯？」
 
。
 
　我歪着头。
 
　我的房间位于破旧公寓的四楼。
 
　房间的位置刚好可以俯瞰到街景……。
 
。
 
「这是、什么鬼光景？」
 
。
 
　最先映入眼帘的是迷之巨木。
 
　大树的高度应该有几十公尺吧。
 
　感觉就像是幻想世界中常见的世界树。
 
　这棵大树从混凝土中迸发而出，穿破各个房屋长得到处都是。
 
。
 
　并且地面上铺装的部分以及裸露的地面犹如龟裂般重重结合。
 
　仿佛就像是将颜色不同的拼图硬是镶嵌而入。
 
。
 
　外头到处都有烟雾袅袅升起，警笛声也一直是不绝于耳。
 
　连车子的汽笛声也同样是接连不断。
 
　在得以听闻的悲鸣中，还混杂着不像是人类的叫声。
 
。
 
　「啊嘎嘎啊啊啊！」之类的、「ＧＩＧＡＡＡＡＡＡ」这种感觉。
 
　没错，简直就像是魔物一样……。
 
　不不不，不会吧。
 
　怎么可能有这种事―――。
 
。
 
「啊，猪头人」
 
。
 
　肥胖的身躯加上猪形人头。
 
　在幻想世界中是王道的魔物。
 
　猪头人走在道路上。
 
　哎呀，大清早的，啊，已经中午了吗。这么有精神真是好呢。
 
。
 
「不不不！」
 
。
 
　好奇怪！
 
　这也太奇怪了！
 
　猪头人!?为什么有猪头人!?
 
。
 
　我再一次望着疑似猪头人的生物。
 
　好真实……感觉不像是假冒的啊……。
 
　背上也没有像是拉链的东西。
 
 
 
 
 
以人偶服装来说，作工实在是精湛无比。
 
　手持的巨大菜刀感觉能轻而易举地砍下人类的首级……啊，仔细看看上面还沾着血。
 
。
 
「…………」
 
。
 
　那位猪头人先生的另一只手上，好像还拿着什么……。
 
　滴答滴答地流着血、圆圆的……嗯，就像是保龄球一样呢。
 
　一定就是这样吧。
 
　虽然有洞的地方看起来就像是嘴巴和眼睛的部位，不过一定是我搞错了。是我看错了呢。
 
　真是Nice boat。
 
（飒:neta来自于某school days要在日本播放最后一集时，毫无预告地终止并换成一张景色图，此图就是一艘船驶行的景色。后来泛指猎奇的场景）
 
。
 
「呜诶……喔喔」
 
。
 
　我吐了。
 
　嗯，没办法。
 
　那是什么鬼？
 
　好血腥。
 
　超血腥的。
 
　这血腥程度完全可以列入限制级。
 
。
 
　其他地方也有猪头人和僵尸一样的怪物在徘徊。
 
　应该是什么提前到来的万圣节前夜吧。
 
　**吗，现在才五月份吧。
 
。
 
「真的假的……」
 
。
 
　面对过于非现实的事态，我的理解能力跟不上。
 
　不过应该是这样。
 
　世界已经改变了。
 
　改变成魔物出现，并袭击人类的世界。
 
　虽然不知道原因，不过应该就是这样吧。
 
。
 
「这不是在做梦吧……？」
 
。
 
　捏一下脸颊，好痛。
 
　简直就像是轻小说和御宅族的妄想一样的展开，我开玩笑地想着。
 
　不，虽然我自己也经常在网络上阅读这一类的小说。
 
　不过没想到现实会发生这种事。
 
　脑袋根本跟不上状况啊。
 
。
 
「呼……」
 
。
 
　桌子上有一盒我放任不管的牛奶。
 
　现在什么水都行，我很想喝点什么。
 
。
 
「呜咕、呜咕……噗啊」
 
。
 
　全部喝光了，头脑可以稍微运转起来了。
 
。
 
「……也就是说昨天的事情难道不是眼花了吗？」
 
。
 
　有点冷静下来的我回想起昨天的事情。
 
　回家途中直接撞死的那头脱离常识的大型犬。
 
　该不会那也是魔物？
 
。
 
　既然如此，至少在昨天的时点上世界就已经发生变化了。
 
。
 
「那么，之后我听到的「那道声音」也是？」
 
。
 
　那只狗消失以后在脑海里回荡的声音。
 
　回想一下。
 
　当时那道声音究竟说了些什么？
 
。
 
「记得是好像是说过……确认到最初的讨伐、获得技能……之类的」
 
。
 
　骤然间，我想到一个可能性。
 
　世界出现魔物。
 
　以及谜团的通知。
 
　这样一来，也会有「那玩意」不是吗？
 
　这种情况下的王道展开。
 
 
 
 
 
。
 
 
 
 
 
「―――Status・Open」
 
。
 
　我开玩笑的。
