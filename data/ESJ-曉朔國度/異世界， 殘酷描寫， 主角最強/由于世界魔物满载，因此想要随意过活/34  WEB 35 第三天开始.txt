翻译：飒君CONAN
 
 
 
WEB 35 第三天开始
 
 
一一一一一一一一一
 
 
 
 
 
　朝阳照射而入，我醒了过来。
 
。
 
「呼啊啊啊……已经早上了吗……」
 
。
 
　睡得真舒服。
 
　果然是太累了吧……。
 
　昨天就觉得了，真想要『疲劳耐性』啊，痛切地。
 
　如果累得半死就可以获得了?虽然我不想这样做就是了。
 
。
 
　看了下时间，已经是早上六点。
 
　应该是原本的习惯问题，我总是会在这时间点醒过来。
 
　不过倒是比起以前还睡得更舒服，究竟是好事呢还是坏事呢……。
 
。
 
「汪！」
 
。
 
　莫莫早就起床了。
 
　它还是老样子很早就起来了。
 
。
 
「早上好，莫莫」
 
。
 
　起床摸摸它，享受莫莫的毛感。
 
　莫莫也用身体磨蹭着我，啊哈哈，这小家伙。
 
　嗯，完全清醒了。
 
。
 
「啊咧……？莫莫，你好像变大了点吧？」
 
。
 
「汪呜？」
 
。
 
　莫莫歪着头，不会错的。
 
　它体型比起昨天还大了一点。
 
　我试着双手抱起它，有点重。
 
。
 
「怎么突然变成这样……？该不会是魔石的影响？」
 
。
 
　又或者是等级提高的影响吗。
 
　可是昨天好像没有变化吧？
 
　比如说等级和摄取的魔石份量还没有消耗？
 
　如果是这样倒是能说明……。
 
。
 
「莫莫，有没有觉得身体很奇怪？或者感到不舒服之类的？」
 
。
 
「咕ー？」
 
。
 
　莫莫像是在说「什么感觉都没有哦」。
 
　反而想展现出自己精神饱满一样，在我的周围反复转圈。
 
　没有……异常吗？
 
　既然没有那就好……。
 
。
 
「我知道了，不过要是有什么地方不舒服，你要立刻告诉我喔?明白了吗？」
 
。
 
「汪！」
 
。
 
　莫莫精神奕奕地回应。
 
。
 
「好，那就吃早饭吧」
 
。
 
　我从道具包取出食材，做了份简单的早餐。
 
　多亏追加的『保存功能』，现在食粮都不会腐烂。
 
　也许是心理作用，比起昨天更能享受吃饭了。
 
。
 
　之后我洗了把脸，简单地擦擦身体。
 
　其实我是想洗澡的，毕竟汗流那么多。
 
　不过没有热水，没办法。
 
　这家里似乎全是电器。
 
　这样不行吧，至少留点利用煤气的家电。
 
　实在逼不得已，我用移动燃气炉烧开水，再加冷水变温水后再拿来用。
 
　顺便也帮莫莫简单洗一遍，由于它不讨厌洗澡，所以洗起来很轻松。
 
。
 
「话说回来，为什么只有电力停掉了……？」
 
。
 
　明明还有自来水和煤气，可只有电力打从一开始就完全停了。
 
　不对，像利用电池运作的小型家电还是可以使用，可除此以外就全部不行了。
 
　虽然没有试过紧急用的发动机，不过那玩意太吵了，感觉会吸引魔物过来。
 
。
 
　发电所是第一天运气不好就被破坏掉了吗。
 
　还是有其他的『原因』呢……。
 
　怎么想也不会得到答案。
 
　即便要特意到那边确认，发电所又实在是太远。
 
。
 
「隔壁镇那边还有电力吗？」
 
。
 
　我连这一点都必须要亲自去隔壁镇确认看。
 
　如果可以看新闻或者上网就好了……。
 
。
 
　啊，对了，收音机。
 
　不是还有收音机吗
 
　记得我手头上是有的。
 
　为什么以前就没注意到呢，笨蛋吗。
 
　我从道具包取出收音机，打开电源。
 
。
 
「不行吗……」
 
。
 
　无论跳转到哪个音频都还是杂音不断。
 
　难道说连『电波』都瘫痪了？
 
　这是怎么回事？
 
。
 
「不行，情报实在是太少了……」
 
。
 
　果然我应该要跟其他集团汇合吗？
 
　不，这样感觉各方面都会不妙啊……。
 
。
 
「只能像昨天一样偷听别人了么……」
 
。
 
　总觉得这样才适合我。
 
　既然如此，果然还是到百货商店吧。
 
。
 
　……不对，在这之前还有一个地方要去。
 
　就先到那边吧。
 
。
 
「那么，莫莫，我们出发吧」
 
。
 
「汪！」
 
。
 
　做好出门的准备，我们走到外面。
 
　如果我没有记错地方，那里应该离这里不远才对。
 
 
 
 
 
。
 
　于是几分钟过后―――。
 
。
 
「噢，有了有了」
 
。
 
　我找到我想要找的地方。
 
　一枚写着『施工中』的看板。
 
　无人利用的建筑机械，钢筋构建到一半的建设中的建筑物。
 
。
 
　没错，我来到的地方就是施工场地。
 
　这里是预定搭建新公寓的建设地。
 
　与百货商店不同，这里对于我来说有着堆积如山的武器。
 
　堆积成山的沙子、铁管、砖块、混凝土以及建筑机械。
 
　无论哪一种配合道具包都是相性极佳吧。
 
。
 
「附近也没有魔物的气息，赶紧回收掉吧」
 
。
 
「汪」
 
。
 
　之后过了几分钟―――。
 
　提升至ＬＶ１０的道具包轻松地将施工现场的资材都收纳进去了。
 
　随便也尝试一下道具包的新用法，这也已经熟练了。
 
　我对于目录上的爆表名单感到满足的同时，和莫莫一起离开了施工场地。
 
。
 
 
 
 
 
「那么，接下来就前往百货商店吧」
 
。
 
　再次开始走动。
 
　跟昨天相比，魔物好少啊……。
 
　『索敌』都没反应。
 
　能感觉到的气息也就只有呆在家里的人们。
 
。
 
　难道说昨天那阵远嗥将这附近的魔物都引诱过去了？
 
　虽然也有可能是离开这里到其他地方了。
 
　强大的魔物姑且不论，连弱小的魔物都不出现了，这下子麻烦了。
 
　经验值都得不到。
 
　如果有练手的哥布林就好了……。
 
 
 
 
 
。
 
　在我们来到距离百货商店还有几分钟路程的地方。
 
 
 
 
 
我发现一个奇妙的物品。
 
。
 
「诶？」
 
。
 
　忍不住就发出这样的声音。
 
　我不禁凝视着『那个』。
 
。
 
「真的假的……」
 
。
 
　明明在昨天根本没有那种东西才对。
 
　在眼前的十字路口。
 
　我注视着伫立于其中心的『那个』。
 
。
 
　那是木制的箱子。
 
　箱子上装饰着豪华物件，中心部分还嵌有钥匙孔。
 
　已经可以确定了吧。
 
　游戏的王道。
 
　无论是谁都会在游戏中看过一次的东西。
 
。
 
「这是宝箱吧……？」
 
。
 
　没错，怎么看放在那边的都是『宝箱』。
 
　真的假的，连这种东西都有吗，这个世界……。
 
一一一一一一一一一一
