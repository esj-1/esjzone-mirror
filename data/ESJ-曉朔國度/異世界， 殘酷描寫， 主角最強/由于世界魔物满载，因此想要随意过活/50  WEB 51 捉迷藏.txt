WEB 51 捉迷藏
 
 
一一一一一一一一一一
 
 
 
 
 
　这已经是第三次听到这道『咆哮』了。
 
。
 
　完了。
 
　完了、完了。
 
　完了、完了、完了、完了。
 
。
 
「莫莫！」
 
。
 
「汪！」
 
。
 
　我大喊之后，莫莫立刻躲到『影子』里避难。
 
　这瞬间，我全力狂奔。
 
。
 
　逃跑吧。
 
　逃跑吧。
 
　逃跑吧！
 
。
 
　尽可能逃远一点。
 
　即便是一步也要拉开距离。
 
。
 
「哈啊哈啊哈啊哈啊……！」
 
。
 
　我全力奔跑着。
 
　恐怕这是我生平跑得最拼命的一次。
 
。
 
≪达到一定的熟练度≫
 
≪逃跑从ＬＶ1等级到ＬＶ2≫
 
。
 
≪达到一定的熟练度≫
 
≪逃跑从ＬＶ2等级到ＬＶ3≫
 
。
 
≪达到一定的熟练度≫
 
≪逃跑从ＬＶ3等级到ＬＶ4≫
 
。
 
≪达到一定的熟练度≫
 
≪恐怖耐性从ＬＶ5等级到ＬＶ6≫
 
。
 
　脑海里响起天之声。
 
　原本我应该会为技能的等级提升而感到高兴，但现在我完全不开心。
 
。
 
　不过多亏了『逃跑』的技能等级提升了，感觉速度比起刚才还要快。
 
　果然这个技能是具有逃跑时提升速度的效果。
 
。
 
　速度越来越快。
 
　距离越来越远。
 
。
 
　然而依然没有消失。
 
。
 
　『厌恶感』并没有消失。
 
　『危机感知』一直都在敲响警铃。
 
　『敌意感知』时常在做出反应。
 
。
 
「哈啊哈啊……」
 
。
 
　我逃跑的同时斜眼看着那家伙的身影。
 
　高级猪头人依然在公寓屋顶上纹丝不动。
 
　你能不能就这样呆在那里呢？
 
　我如此期望着。
 
。
 
　然而，我这份渺小的期待破灭了。
 
　高级猪头人骤然间弯着膝盖。
 
　什么鬼，那种姿势？
 
　简直就像是跳跃前的蓄力一样―――。
 
。
 
　猛然有股寒气袭来。
 
。
 
『オオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオッッ!!』
 
。
 
　咆哮，巨响，冲击。
 
　高级猪头人起跳的同时，高层公寓的屋顶也崩塌了。
 
　如同子弹一般释放而出庞大身躯。
 
　速度极为可怕。
 
　高级猪头人比起瓦砾掉落到地面上还要更快着落。
 
。
 
　咚咚!!!!
 
。
 
　产生一股犹如陨石坠落的冲击。
 
　仔细一看，在我刚才所在的地方附近都形成小型的环形山。
 
。
 
　……不会吧？
 
　那种距离只要一跳？
 
　它的肌肉到底是什么构造啊!?
 
　别给我开玩笑了！
 
。
 
「再荒唐也要有个限度吧……！」
 
。
 
　敌不过，我立即领悟到这一点。
 
　即便如此我还是奔跑着。
 
　尽可能拉开距离。
 
。
 
「咯……」
 
。
 
　高级猪头人扭了扭脖子，看向这边。
 
　给我住手，别看着这边。
 
　别过来这边。
 
　上次不是放过我一马吗。
 
　这次也放过我吧。
 
　求你了。
 
。
 
　然而我的祈祷并没有显灵。
 
　高级猪头人举起手里的菜刀，往这边过来了。
 
　被它追上就是死路一条。
 
　地狱捉迷藏就此开始了。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
　如果贯彻逃跑方针，比起魔物我更有地利。
 
　不知什么时候，我是这样想的。
 
　这世界上出现魔物已经有三天了。
 
　相较于突然出现在这世界上的魔物，土生土长的我更加熟悉地形，并且方向感很好。
 
。
 
　所以我只要彻底逃跑，应该能行得通。
 
。
 
　不过我太天真了。
 
　正因为这种理论行不通，魔物才会是魔物。
 
。
 
「呜噢噢噢噢噢噢噢噢噢噢！」
 
。
 
　我奔跑在狭隘的道路上，正害怕着背后迫近而来的威胁。
 
　总之无论我怎么绕进胡同，跑进迷宫般的小路里也完全拉不开距离。
 
　为什么？
 
　很简单。
 
　那家伙根据我的气味，单纯只是横冲直撞而已。
 
　撞坏墙壁，跳过房屋，踩烂汽车，不顾一切障碍，仅仅是直线冲刺。
 
　障碍物完全没发挥出应有的效果。
 
　真想说开什么玩笑。
 
。
 
　现在彼此之间的距离不到十米。
 
　无论我怎么做都没办法甩开。
 
　该死、该死、该死！
 
。
 
「会……被追上……！」
 
。
 
　我走出错综复杂的巷子，来到开阔的地方。
 
。
 
「莫莫！」
 
。
 
　莫莫回应我的叫声，维持躲在影子里的状态将『影子』延伸到高级猪头人身上。
 
。
 
「咯……？」
 
。
 
　高级猪头人兴味索然地注视着捆绑着自己的『影子』。
 
　『影子』缠绕至它全身，意图束缚它―――噗呷！断开了。
 
　轻而易举地，毫不费力地。
 
。
 
　……连它的行动都没办法停住半秒吗。
 
。
 
　不过我只能赌一把了！
 
。
 
「啊啊啊啊啊啊啊啊啊啊啊啊!!」
 
。
 
　随着大声喊叫，我往高级猪头人头上释放建筑机械。
 
　高级猪头人为头上出现的物体感到惊讶。
 
　建筑机械就这样砸到了高级猪头人的身躯―――并没有。
 
。
 
「假的……吧……？」
 
。
 
　我不禁如此嘀咕着。
 
　高级猪头人并没有被建筑机械所压倒，而是『单手』支撑住。
 
　物体坠落的冲击力让那家伙脚边的地面稍微凹陷进去。
 
　然而那家伙并没有被压到。
 
　光凭单手的力量就支撑住『建筑机械』。
 
　那家伙嗤笑着。
 
。
 
「―――‼」
 
。
 
　『危机感知』、『敌意感知』都达到极限反应。
 
。
 
「―――ォォォオオオオオオオオオオオオオオオオオアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアッッ!!!!」
 
。
 
 
 
 
 
随着一阵咆哮，高级猪头人将手上的『建筑机械』往我这边『丢过来』。
 
　样子轻松到像是在丢棒球。
 
　压倒性的质量以极为可怕的速度迫近。
 
　将我的视野都全数覆盖。
 
　―――避不开。
 
。
 
「～～～～!!（抖抖抖）」
 
。
 
　这瞬间。
 
　『拟态』后的小红身体膨胀到最大限度。
 
　将我身体包容在里面形成球体的形状。
 
　轰然一声，建筑机械撞击到包进了我身体的小红身躯的瞬间，小红的身体将建筑机械的冲击力抵挡了。
 
接着就像是圆球一般，我们往后方飞去。
 
　与墙壁与地面激烈碰撞的同时，小红的身体立体性地在到处弹跳。
 
　我脑袋也摇摇晃晃的。
 
。
 
　究竟弹跳了多久呢？
 
　终于停了下来了。
 
　小红也『拟态』为原本的衣服。
 
。
 
「……呜咕……」
 
。
 
　我按住嘴吧，拼命地忍耐着呕吐的冲动以及眩晕。
 
。
 
「谢、谢了、小红……」
 
。
 
　干得漂亮。
 
无论是『回避』或者『道具包』都来不及。
 
　如果不是小红让身体发生变化，这时候我早就被建筑机械砸死了吧。
 
。
 
「哈啊……哈啊……」
 
。
 
　我以摇晃的视野望向前方。
 
　是高级猪头人。
 
　它正慢慢地走进这边。
 
。
 
　确认下地点。
 
　这里是停车场。
 
　究竟是在哪里？
 
。
 
　稍微移开目光，看到高级猪头人后面有架直升机。
 
　是吗……这里是购物中心啊。
 
　究竟是什么因缘际会呢。
 
　拼命地逃跑，来到的地方却是与这家伙初次相遇的地点。
 
。
 
「哈啊……哈啊……」
 
。
 
　没有其他的猪头人。
 
　停车场的地面到处都是龟裂，建筑物也都纷纷崩塌了。
 
。
 
　高级猪头人则是悠然地举起菜刀，挺直伫立着。
 
。
 
　仿佛在诉说着「不会再让你逃跑了」
 
一一一一一一一一一
