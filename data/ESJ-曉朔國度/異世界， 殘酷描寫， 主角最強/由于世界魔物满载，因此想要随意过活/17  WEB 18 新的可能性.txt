WEB 18 新的可能性
 
 
一一一一一一一一一一
 
　我和莫莫再一次走在街上。
 
　也许是因为昨天下雨，到处都留有水渍。
 
　隐约想避开这些水渍的同时，寻找着能够狩猎的魔物。
 
　理想的情况是单独行动的哥布林或者僵尸。
 
　两只……不，三只的话还可以配合莫莫的『影子』进行连击，迅速解决掉它们。
 
。
 
　问题是出现除此以外的魔物。
 
　特别是猪头人。
 
　从昨天的战斗来看，不光是那头高级猪头人，就连普通的猪头人都具备高超的战斗能力。
 
。
 
　如果有什么弱点就好了啊……。
 
　如果是幻想世界里，经常会出现僵尸怕火，吸血鬼害怕圣水之类的场面。可猪头人倒是没有这种特征性质的弱点。
 
　真要说的话就是性欲比较强，只想到女骑士最适合的对付它的。
 
　嘛，如果遇见它要么就是逃跑，要么就是等它自己离开吧。
 
　至少『现在』只能这样。
 
 
 
 
 
。
 
 
 
 
 
「果然尸体好多啊……」
 
。
 
　尸体都是放任不管，任由它们在风吹雨打。
 
　毕竟也没有人来收拾这种残局。
 
。
 
「话说回来，政府和自卫队究竟在干什么啊……？」
 
。
 
　世界变成这样子已经整整过了两天了。
 
　虽然我觉得各地应该是已经进行救援活动了，可是现在电视和网络都用不了，根本没办法确认这一点。
 
。
 
「是说这种状况中政府还能正常运作吗……？」
 
。
 
　以前看过的真·哥斯拉的电影里还会建立特别对策室，自卫队以及相关部门都会行动。
 
　可实际上究竟能像那样顺利地进行吗？
 
　不，也许只是我对于政治家以及官僚带有偏见而已。
 
　我怎么可能会载到这种事情上!我要逃跑!——类似于这样的感觉最先跑到核隐蔽所之类的，这就是我对于他们的印象。
 
　并且只有在这时候，他们才会无谓地利用权力。
 
　在电影里有看到过吧？
 
。
 
　归根究底，只有日本出现了魔物吗？
 
　还是说世界都是呢？
 
　隐约觉得整个世界上魔物都出现了。
 
。
 
「意外的是发展中国家的存活人数也许会比较多呢」
 
。
 
　『安全』和『生存』完全是两码事。
 
　如果是处于纷争地带的人们，纵使魔物出现也不会完全动弹不得吧？
 
。
 
「核武器又不能使用……」
 
。
 
　不光是魔物，连国民都会一并歼灭……。
 
　之后还会在新闻上播放出「他们是英勇牺牲者，我们必须要将他们的生命铭记于心」类似于这种报道。
 
　呜哇ー，想想都觉得害怕。
 
　而且还真有国家可能会这么做，更加可怕了。
 
。
 
　果然还是边升级边收集情报吧。
 
　可是网络和电视又用不了。
 
　说到底停电实在是太痛苦了。
 
　虽然自来水和煤气还可以使用，但与电力相关的东西都完全停止。
 
　只有这附近的区域是这样吗，还是其他地方也是这样呢。
 
　如果其他地区还有通电，那就应该到那边去。
 
。
 
「真是重新感觉到文明的益处啊……」
 
。
 
　电力真的是很伟大。
 
　正因为伟大，我们才会依赖。
 
　如果这附近有家具全都是电器的家庭，那就应该是地狱吧。只能离家出走了。
 
　如果是手电筒或者闹钟这种利用电池运作的这种类型倒还能使用，然而能使用的家具应该是不会很多吧。
 
。
 
「汪」
 
。
 
　哎呀。
 
　在我想着事情的时候，莫莫叫了一声。
 
　仔细一看，视野的某处有哥布林。
 
　就一只。
 
　是我们的目标了。
 
。
 
「好，莫莫。开始狩猎」
 
。
 
「汪！」
 
。
 
　尽可能不让它注意到我们，直接接近它一刀下去搅动一下。
 
　一点危险都没有就打倒它了，然而等级没有提升。
 
　到升级为止，我究竟要杀死几只才好啊？
 
 
 
 
 
。
 
　后来我们继续在街道上走着，狩猎了三只僵尸，两只哥布林。
 
　总觉得遭遇魔物的几率比起昨天还要低……。
 
　而且很少遇到还活着的人。
 
　这也是因为―――
 
。
 
「意外有很多人选择守家呢」
 
。
 
　走在街道上，我看到有很多人都在公寓以及凉台都用上写着SOS的旗帜，而家里的则是用毛巾挂在窗户外面。应该是准备守在家里吧。
 
　都过了这么久了，感觉也差不多该明显地分类出『逃跑的人』、『战斗的人』、『不能动的人』。
 
。
 
「可是啊……」
 
。
 
　如果是僵尸可能还能忍耐到有人来救援也说不定。
 
　可是现在城市里可都是一些积极狩猎人类的魔物。
 
　面对这些家伙，守家也没多大意义吧？
 
　实际上当我昨天出门的时候，哥布林它们就已经自己在探索公寓的房间了。
 
。
 
「嘛，如果有这种『职业』和『技能』就另当别论了……」
 
。
 
　比如说我职业候补里面的『家里蹲』。
 
　像这种职业比起在外面，大概呆在家里还比较能发挥实力。
 
　不，归根究底就不会是『家里蹲』。
 
　反正我是绝对不会选的。
 
　即便打倒魔物可以提升等级，哪个笨蛋会选择这种『NETA职业』啊？
 
　怎么可能会有嘛。
 
。
 
「嘛，基本上都是会无视的吧……」
 
。
 
　跟这种只会像这样等待别人救援的家伙扯上关系也没有用。
 
　搞不好我还要一直保护到底。
 
　在这种状况中实在是不可能。
 
　应该说很麻烦。
 
。
 
　如果是拥有等级的倒还能考虑一下共同战斗。
 
　于是我决定无视他们的救援信号，继续前进。
 
　嘛，就是那句话。
 
 
 
 
 
你们就祈祷在魔物袭击你们之前，有什么『好人』会出现吧。
 
。
 
　之后我和莫莫继续狩猎着可以打倒的魔物。
 
　于是当我打倒两只乱晃的哥布林时，发生了一件事。
 
。
 
≪经验值达到一定程度了≫
 
≪工藤和人从ＬＶ５升级到ＬＶ６了≫
 
。
 
　太棒了，脑海里回响着升级通知。
 
　确认了一下，跟以往一样得到了20点SP、10点JP。
 
　ＪＰ这样就有１５点了。
 
　立马提升『密探』的等级。
 
。
 
≪需要消耗10点ＪＰ提高密探的ＬＶ吗？≫
 
。
 
　当然是选择YES。
 
　好……会怎么样呢？
 
。
 
≪密探的ＬＶ达到上限≫
 
。
 
≪可以选择上级职业以及派生职业≫
 
≪第二职业已经解锁了≫
 
。
 
≪上级职业『暗杀者』已经解锁了≫
 
≪上级职业『间谍』已经解锁了≫
 
≪上级职业『盗贼』已经解锁了≫
 
。
 
≪职业上已经满足一定条件≫
 
。
 
≪派生职业『告密者』已经解锁了≫
 
≪派生职业『猎人』已经解锁了≫
 
≪派生职业『趁火打劫』已经解锁了≫
 
≪派生职业『驯兽师』已经解锁了≫
 
。
 
　新的可能性。
 
　新的阶级。
 
　已经显现了。
 
一一一一一一一一一
