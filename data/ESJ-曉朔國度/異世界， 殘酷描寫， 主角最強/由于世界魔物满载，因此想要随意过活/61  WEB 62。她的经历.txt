WEB 62.她的经历
 
 
一一一一一一一一一一一一
 
「非常抱歉」
 
。
 
换好衣服后，一之濑小姐向我土下座并道歉着。
 
。
 
「啊、不用、好了啦，反正事情都已经过去了……」
 
。
 
像这种被美少女呕吐一身，对于某些业界来说也许是奖赏，不巧的是我可没有这种特殊的性癖。
 
不过嘛，接下来都是队伍成员了，这点旁枝末节的小事就别介意了吧。
 
……虽然我大概会永生难忘就是了。
 
。
 
顺便一提，小红将她的呕吐物都全部吃掉了。
 
对于小红来说，美少女的呕吐物似乎是奖赏的样子，主要是在进食的意义上。
 
还好被呕吐到地方大部分都是小红所拟态的部分。
 
虽然姑且还是换了件衣服。
 
……内心有点不想穿上小红所拟态的衣服，这件事可是秘密。
 
一之濑小姐移开目光，似乎感到十分抱歉似的低着头。
 
。
 
「对不起……我不知道这种时候要怎么面对你」
 
。
 
「也是呢」
 
。
 
连我自己都不知道要怎么面对被我呕吐到的对象。
 
。
 
「……这时候真希望你能说那句台词」
 
。
 
「非常抱歉，我无法回应您的期待呢」
 
。
 
如果是暧昧的笑容，我倒是会展露出来就是了。
 
话说这孩子性格意外不错呢。
 
完全想不到是家里蹲。
 
。
 
「嘛，这件事就到此为止吧。然后是关于接下来的事情―――」
 
。
 
「啊，说得也是呢。呃……总之我们先去狩猎吧？」
 
。
 
别说得像是モン◯ン啊……。
 
总之先狩猎什么的，这孩子好可怕。
 
（颯:懒得查，应该说超难查出NETA，已放弃）
 
。
 
「汪！」
 
。
 
并且连莫莫也赞同了！
 
莫莫，不行！不行喔！
 
。
 
「……咕呜？」
 
。
 
就、就算你用这种泪眼汪汪的眼神看我也不行！
 
安全第一，这是绝对。
 
我摸着无精打采的莫莫。
 
好可爱，受到治愈。
 
。
 
「啊，不是，在这之前我想确认一下彼此的技能和职业来着？」
 
。
 
今后我们就是同个队伍的成员。
 
还是提前了解彼此的技能比较好吧。
 
。
 
我撇了一眼一之濑小姐所扛着的枪支。
 
那把普通人完全无法得到的长身来福枪。
 
她究竟是在哪里得到这把枪的？
 
。
 
「啊啊，说的也是呢」
 
。
 
她一副理解的样子，抚摸着自己的武器。
 
。
 
「……果然你还是会在意这个吧？」
 
。
 
「如果说不在意那是在说谎」
 
。
 
她说过自己的职业是『家里蹲』。
 
这把武器跟这职业完全不相配。
 
当我等着她的回答时，一之濑小姐的脸色越来越铁青。
 
喂、喂，给我等一下啦……。
 
。
 
「……呜咕。对不起，请稍微等一下」
 
。
 
她按住嘴巴，向莫莫招招手。
 
。
 
「咕呜？」
 
。
 
怎么了怎么了？莫莫以这种感觉走到她身边。
 
莫莫这家伙，意外很亲近她啊，有点嫉妒。
 
。
 
「呼……这样做以后，心情就冷静下来了」
 
。
 
一之濑小姐让莫莫爬在她膝盖上，享受着摸着它的感觉。
 
十分幸福的表情。
 
我懂，我十分能体会你的心情喔。
 
莫莫的摸摸治愈效果可是远远凌驾于最新医疗产品。
 
也许是她手法很舒服吧，莫莫也高兴似的眯起眼睛。
 
不久后冷静下来的她开口说道。
 
。
 
「其实我的职业有两个。其中一个是刚才说过的『家里蹲』。另一个则是名为『狙击手』的职业」
 
。
 
「嘿……」
 
。
 
她这番话令我十分惊讶。
 
职业有两个。
 
换句话说她已经将『家里蹲』提升到上限了吗？
 
不对，她的等级是２０。
 
依据西野君的同伴所说，普通人每提升一级所得到的ＪＰ是1。
 
等级２０根本还不能达到上限才对。
 
……不，也有可能像技能一样提升熟练度就可以升级？
 
还是说除了让最初的职业升满以外，还有其他手段得到第二职业……？
 
。
 
「起初我所选择的是『家里蹲』」
 
。
 
「那个……这么问可能有点失礼，为什么你要选择这种NETA职业？」
 
。
 
我问完以后，她尴尬地移开目光―――原本早就移开就是了。
 
。
 
「……因为我觉得这不是现实」
 
。
 
啊啊，原来如此。
 
的确在最初会认为这种世界会不会是自己在做梦。
 
。
 
「那是在四天前发生的事情―――」
 
。
 
她一边抚摸着莫莫，一边向我说明自己升级的经历。
 
她最初升级是在四天前。
 
也是我撞死影狼那一天。
 
。
 
当时她似乎在公寓自己家里玩着网络游戏。
 
连续熬夜三天，终于疲惫不堪的她姑且休息一下，洗了个澡。
 
于是当她洗完澡回到房间时，窗户正开着。
 
似乎是她忘记锁紧了，粗心大意。
 
。
 
随后似乎是听到毛骨悚然的振翅声。
 
她想着是怎么回事，往那边一看。结果那边有只拳头大小的巨型苍蝇。
 
原来如此，虫型魔物吗。
 
还有这种类型啊。
 
。
 
看到那只巨型苍蝇，她既震惊又害怕。
 
然而她并没有逃跑，而是拿起附近的杂志卷成棍子型尽情地砸下去。……好厉害，这孩子。
 
于是当苍蝇被砸死的瞬间，她脑海里响起升级的声音。
 
。
 
『……等级提升了?』
 
。
 
然而当时她似乎以为是幻听了。
 
苍蝇的尸体也消失不见。
 
一定是因为熬夜三天导致看到不可能存在的幻觉吧，她如此定下结论。
 
然而当她继续玩网络游戏，想要确认画面中自己角色的人物状态时，那东西瞬间出现了。
 
在自己眼前出现了自己的人物状态。
 
。
 
『……这是什么？选择职业……？』
 
。
 
于是她忍不住选择了『家里蹲』。
 
这瞬间，她的能力值大幅度下降。
 
不过取而代之的是得到有趣的技能。
 
那便是『认知妨碍』和『抽奖』。
 
『抽奖』是通过消耗SP可以得到各种方便物品的技能。的确像是家里蹲的技能。
 
。
 
「第一次抽奖，我运气好得到『狙击手套装』」
 
（颯:或许这就是欧皇）
 
。
 
抽到这份奖品时，她的职业里似乎追加了『狙击手』。
 
狙击手拥有附属的技能，并且还附赠一把枪。
 
嘿……通过技能得到的职业可以直接追加上去的吗。
 
虽然是运气性较高的技能，但只要够幸运应该可以得到释放方便的能力。
 
。
 
她觉得反正这是在做梦，于是用这把枪狩猎魔物。
 
狩猎、狩猎、反复再狩猎，不久后她注意到这不是在做梦，而是现实。
 
呕吐、呕吐、反复再呕吐，不久后她思考着该怎么办才能存活下来。
 
。
 
于是她似乎得出跟我一样的结论。
 
提升等级，变得更强。
 
如果可以，还想寻找能够与她合作的人。
 
然后是遇到我们，直到今天。
 
。
 
「―――以上」
 
。
 
她的说明断断续续，但其中拼命的感情已经传达过来。
 
她那与呕吐冲动战斗的同时，也拼命向我说明关于自己经历的样子，我不禁感动起来。……随便拉开一定的距离，以防被呕吐到。
 
。
 
「是吗……好像是很辛苦呢」
 
。
 
「是的……」
 
。
 
她点了下头。
 
。
 
「可是那个，这样好吗？虽然是我自己要问的，但你这么轻易地就将自己的职业和技能告诉我……」
 
。
 
在如今这样的世界上，她应该十分清楚自己的等级和技能、职业是多么重要的情报。
 
明明如此还这么简单就告诉我吗。
 
。
 
「没有……问题」
 
。
 
此时，她第一次略微面向我一点。
 
。
 
「……因、因为我相信那个……和…和人先生和莫莫」
 
。
 
她直面地说出来了。
 
。
 
「……啊，是的。那个……非常感谢」
 
。
 
感觉很害羞，连我都情不自禁地就移开目光。
 
这孩子真的为什么会成为家里蹲啊？
 
拜此所赐，身体感觉有点热热的。
 
并且就连一之濑小姐那边也在说完以后注意到自己的言语，恍然转过头并低着头。
 
。
 
「……」
 
。
 
「……」

好、好尴尬。
 
无话可说的微妙气氛。
 
像为什么这么相信我们，接下来的目标之类的，明明有这么多想要询问的事情，但嘴里就是说不出话。
 
。
 
不如说现在气氛比起被吐一身还要微妙喔。
 
然而这种尴尬在接下来的瞬间就消失无踪。
 
『索敌』有反应了。
 
是魔物的气息。

「……一之濑小姐，抱歉。姑且就先到此为止吧」

「诶？」

我无视茫然若失的一之濑小姐，看着莫莫那边。
 
。
 
「咕噜噜ー……！」
 
。
 
莫莫已经从她膝盖上下来，进入战斗态势。
 
这股气息……这数量，不会错的。
 
我转换心情，露出严肃的表情。

「影狼群体正在接近这边。姑且先离开这里吧」

该死的魔物，给我注意场合啊。
