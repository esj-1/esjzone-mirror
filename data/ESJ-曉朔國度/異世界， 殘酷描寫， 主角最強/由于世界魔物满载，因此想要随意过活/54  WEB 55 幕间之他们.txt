WEB 55 幕间之他们
 
 
 
「……嗯？」
 
 
在暗淡无光的室内，六花醒了过来。
 
身体有股刺痛感。
 
。
 
「啊咧……？这里是、哪里？」
 
。
 
毛毯盖在自己身上。
 
被子？为什么？
 
她按着自己的头，缓慢地撑起身子。
 
。
 
「诶……？」
 
。
 
为什么自己处于这样的状况呢？
 
六花回想着之前发生的事情。
 
对了，记得我们的确是在百货商店遭受猪头人集团的袭击。
 
由于胜算很低，六花他们在百货商店放火，将避难民他们作为诱饵后四处逃跑。
 
虽然她也觉得这是无耻的做法，但也狠下心认为这是为了活下来。
 
后来她和西野一同犹如拖着负伤的身体般踉跄地在雨中奔跑着，跑到附近的一座公园，然后就―――。
 
。
 
「……发生了什么来着？」
 
。
 
之后的事情想不起来。
 
原本『技能』就会导致战斗中的记忆变得暧昧不清，但这次更加严重。
 
所有的记忆都有遗漏之处，身体也是遍体鳞伤。
 
连自己自豪的侧马尾都变得乱糟糟。
 
。
 
「―――你醒过来了吗？」
 
。
 
有道声音。
 
将目光移向那边，西野正坐在椅子上看着自己。
 
。
 
「……小西？」
 
。
 
「我看起来像其他人啊？」
 
。
 
「嗯~譬如说鬼怪之类的？」
 
。
 
「别擅自把我当做死人了，我还活着啦。你看，双脚也有吧」
 
。
 
「啊哈哈，是呢」
 
。
 
面对六花独特的氛围，小西——西野苦笑着。
 
。
 
「然后呢，这里是哪里呀？我记得我是和小西一起逃到附近的公园来着……」
 
。
 
「我也是差不多，醒过来就已经在这里了。……似乎有人将我们搬到这里来了」
 
。
 
「是谁啊？」
 
。
 
「鬼知道，我也不清楚」
 
。
 
西野最后的记忆是淋着雨抵达了公园。
 
后来感觉似乎是遇见了谁，但想不起来。
 
。
 
（说到底好像不是人来着……？狗？不对，究竟是怎样呢……）
 
。
 
总之虽然不知道是谁，但得救了。
 
并且还将包带、伤药以及营养饮料都放在了这里。
 
这么无微不至的照顾，反而会怀疑是否有什么内幕。
 
他将营养饮料递给六花。
 
六花则是一口气喝光了。
 
。
 
「总之现在应该要感谢自己得救了吧。幸好魔物也似乎没有追上来」
 
。
 
「是呢，话说这包带也是那个人给我包的吗？」
 
。
 
六花掀开校服和裙子，注视着包得粗枝大叶的包带。
 
面对她的举动，西野不禁别开目光。
 
。
 
「……稍微给我注意一下自己的举止啊，笨蛋」
 
。
 
「嗯？小西，你说什么了吗ー？」
 
。
 
「……什么也没有」
 
。
 
他有点脸红。
 
随后像是蒙混过关一般说着「比起这件事」。
 
。
 
「得想想接下来要怎么办了啊……」
 
。
 
「怎么办……不就是寻找大家吗？」
 
。
 
「怎么找？在现在不能使用手机的状况下，要找到大家几乎是不可能的喔？」
 
。
 
根本没有取得联系的手段。
 
只能踏实地寻找他们，并且还是在这种魔物横行霸道的极为危险的世界中。
 
如果有取得联系的『技能』倒是另当别论，但不巧的是西野和六花都没有这么方便的技能。
 
。
 
「那要怎么办呢？」
 
。
 
「按照计划行事吧。目标是学校，顺利的话在那边就能跟大家汇合才对」
 
。
 
「呜诶……」
 
。
 
『学校』。
 
听到这词，六花露骨地皱着脸面。
 
。
 
「呐，小西。其实到其他的事务所也可以吧？」
 
。
 
「学校那边比较近，而且到那边效率才更好吧？」
 
。
 
「……」
 
。
 
「我懂你想说什么，不过现在……」
 
。
 
「我知道了啦」
 
。
 
六花鼓起腮帮子。
 
她讨厌学校。
 
学校将六花最重要的朋友夺走了。
 
是仅有讨厌回忆的地方。
 
不过现在非去不可。
 
六花硬是让自己接受。
 
。
 
「柴田和大野如果都没事就好了……」
 
。
 
「一定还活着吧，无论是大野、柴田还是其他人。所以我们也相信他们前进吧」
 
。
 
「……你说得对呢」
 
。
 
回想起亲友的样子，六花寂寞地笑了。
 
。
 
「那么我们走吧，虽然还想稍微休息一阵子，但时间宝贵」
 
。
 
「了~解」
 
。
 
他们的目的地是市中心的母校。
 
两人相信同伴依然活着，决定继续走往前方。
 
 
 
 
 
。
 
。
 
。
 
于是―――。
 
。
 
≪获得了经验值≫
 
≪大野启太从ＬＶ４升级到ＬＶ５≫
 
。
 
≪满足一定的条件≫
 
≪获得了技能『同族杀害』≫
 
。
 
脑海里响起声音。
 
戴着眼镜的学生——大野颤抖不已。
 
他目光焦点都没有对准，嘴唇也是干巴巴。
 
。
 
「哈啊哈啊哈啊哈啊……不、不是的……我、我不是故意的。我不是……有意要这样做……」
 
。
 
他像是在胡说一般嘀咕道，注视着倒在眼前的『他们』。
 
『他们』跟他穿着同样的校服。
 
数量是三人。
 
没有动弹，眼睛也没有光芒。
 
地面留有一摊血。
 
血滩正逐渐地蔓延开来。
 
。
 
「对、对啊，是这些家伙的错……。不听我的劝告擅自行动……我、我我在当时明明都说过逃跑吧……！」
 
。
 
从百货商店逃跑出来以后，他漫无目的地在街道上徘徊。
 
淋着雨拼命地寻找西野和柴田的途中，大野遇见了在商业合作商铺舍弃的学生们。
 
看样子他们似乎也平安无事。
 
―――太好了，他们没事啊。
 
大野为彼此的再会而感到高兴，但对于他们而言似乎不是如此。
 
当看到自己的瞬间，他们犹如魔鬼一般逼近而来。
 
。
 
「怀、怀恨在心也有个限度吧……。什、什么叫『因为你的错差点死掉了』……别开玩笑了别开玩笑了别开玩笑了」
 
。
 
他手上握着一把兰博刀。
 
刀上沾满鲜血。
 
原本是想要自我保护。
 
如果不反击就会被杀。
 
他这么想的，所以捅了。
 
。
 
等级是他比较高。
 
也拥有攻击『技能』。
 
所以杀掉了。
 
轻而易举地、杀死了。
 
杀死―――蓄意杀人什么的―――不对、不对不对不对！
 
。
 
「呜噗……诶诶」
 
。
 
吐了。
 
感觉很糟糕。
 
颤抖压抑不住。
 
从腹部里涌起的难以言喻的不快感和罪恶感折磨着他。
 
。
 
「我没有错、我没有错、我没有错、我没有错、我没有错、我没有错我没有错我没有错我没有错……」
 
。
 
他嘟嘟嚷嚷地主张自己的正当性。
 
如果不这样做，他的精神就会崩溃。
 
。
 
「对了，我还得寻找。西野君、柴田君、六花……我还得寻找大家，大家都在哪里啊。我得快点、快点找到他们……哈哈、……」
 
。
 
他并没有擦拭自己身体上沾到的血迹，也没有注意到这件事。
 
对、对啊。跟大家、跟大家相会。
 
只要见到大家，只要跟大家在一起，自己一定就没事了。
 
应该是这样，绝对是这样。
 
。
 
「所以、所以没事的……没事的……」
 
。
 
他的声音并没有传到谁的耳中，踉跄地走了起来。
 
 
 
 
 
。
 
 
 
 
 
技能：『同族杀害』
 
杀害自身同族者会得到的稀有技能。
 
当杀害同族时，获得的经验值增加同族。
 
以及在与同族战斗时，能力值会有加成。
 
得到此技能者无法再获得『恐怖耐性』、『抗压耐性』。
