翻译：飒君CONAN
 
 
 
WEB 30 误会
 
 
一一一一一一一一一一
 
 
 
 
 
　在偷听学生他们对话的同时，我感觉自己的脸颊僵硬了。
 
　完了，太过于得意忘形了。
 
　测试道具包的性能实在太有趣了。
 
　毕竟可以收纳眼前看到的东西，这样当然会起了劲头啦
 
。
 
　不过，果然把东西都全部收纳得一干二净还是做过头了。这样当然会很显眼吧。
 
　……嘛，虽然我没打算还回去就算了。
 
。
 
「你说……那里没有食物？这是真的吗？」
 
。
 
「是啊……我自己也确认了几遍了。可是他们看起来也不像是在说谎。怎、怎么办啊、西野？这样下去，我们就……」
 
。
 
　过来报告的那位学生的声音十分动摇。
 
　即便隔着墙壁，眼镜君和不良君的动摇神情也传达过来。
 
　当中只有一个人依然是冷静的，就是他们的领头西野君。
 
。
 
「……大野，食粮的储蓄还剩下多少？」
 
。
 
「诶？那个、这个、记得我们还留下差不多三天份量的食粮……不过已经少了刚才给那些人的部分……所以」
 
。
 
「喂！别说得这么不干脆，直接给我说结论，呆子！」
 
。
 
「！非、非常抱歉！」
 
。
 
　不良君向回答得十分墨迹的眼镜君怒吼道。
 
。
 
「住手吧，柴田。你总会像这样立马生气，这可你的坏习惯喔？」
 
。
 
「……对、对不起」
 
。
 
「大野，你慢慢说就好。食粮还剩下多少？」
 
。
 
「……！对、对不起。那个、食粮已经……不多了……大概、只有一天的份量吧……」
 
。
 
　面对眼睛君的回答，大家都沉默了。
 
。
 
「是吗……我知道了。这件事没有让那些过来避难的人知道吧？」
 
。
 
「啊、啊啊、当然」
 
。
 
　虽然我听得十分清楚就是了呢。
 
。
 
「那就行，切记这件事绝对不要让他们知道。一旦他们知道食粮库存不多，也许会发生暴动。你们也跟其他人转告一下」
 
。
 
「我、我明白了」
 
。
 
　如此说完，那位进行报告的学生离开房间。
 
。
 
「哈啊……事情麻烦了……」
 
。
 
「是啊……」
 
。
 
「整理一下思路吧。从状況来看，物资消失不见是在第三次搬运工程结束到第四次搬运开始之前的时间段。时间大概是两小时吧，一般来说，这点时间根本不可能将这附近店铺的商品都抢得半个不留」
 
。
 
「就是这样，不过这样一来―――」
 
。
 
「没错，这些都是拥有这一类『技能』的家伙干的好事。并且从规模来看，技能级别应该是相当高吧」
 
。
 
「啊，应该就是类似于道具包、转移魔法的技能吧？在网络小说里经常有出现这种技能」
 
。
 
　嗯，正确。
 
　真不愧是眼鏡君。既然是网络小说的爱好者，应该就会联想到『鉴定』和『道具包』这两个技能。
 
。
 
「真是给我们添麻烦」
 
。
 
「畜—生，到底是哪个家伙啊！居然偷走别人的东西！」
 
。
 
　不，不良君。
 
　这句话我原地不封地还给你。
 
　毕竟那又不是你们的东西。
 
。
 
「……」
 
。
 
「怎、怎么了，西野君？在想什么……？」
 
。
 
「没什么……只是觉得再怎么样这时机也太巧了。我们在收留避难民的时候，刚刚好就发生这样的事情……」
 
。
 
「西、西野君是觉得那个人盯上我们了？」
 
。
 
「大野，现在可还不能断定是一个人的所作所为。有可能是像我们一样是团队行动」
 
。
 
「啊、是吗……」
 
。
 
　不，就我而已。
 
。
 
「而且也有可能只是偶然而已，也许还呆在这附近」
 
。
 
「要找找看吗？」
 
。
 
「如果可以找的话。不过……假如说那家伙是『有目的性地所有东西都夺走』，那就棘手了」
 
。
 
「这是……什么意思？」
 
。
 
「你还没明白吗？做出这种事情的理由只有一个。就跟我们会收留避难民的原因是一样的。你觉得在如今这种世界上，食粮和日用品拥有多少价值？随着时间的经过，它的价值也会渐渐上涨吧。如果能够巧妙地用来当做交涉的条件，想要多少听话的家伙就有多少」
 
。
 
「你、你说得对啊……」
 
。
 
　不不不，我可没有这种念头。
 
　完了，好深的误会。
 
　我半点都没考虑过这种事情。
 
　只是测试道具包的性能很有趣，于是得意忘形起来了而已，虽然很难说出口……。
 
。
 
「无论如何，再这样逐渐夺走食粮就大事不妙了。我们必须得趁早确保可靠的食粮来源」
 
。
 
「要提高搜索范围吗？」
 
。
 
「虽然很危险，但也只能这样做了。我是想立即开始行动，可不巧的是现在已经是晚上了。在如今魔物到处徘徊的状况下，夜晚出去探索也太危险了。今天就早点休息吧，明天一早立马开始。
 
　还有如果犯人的目标是独占资源，很有可能会盯上这里。你们一定要提高警惕」
 
。
 
「好」
 
「我明白了，西野君」
 
。
 
　于是到这里他们似乎讨论结束了。
 
　三人离开休息室，往销售场内走过去。
 
　看准时机，我也姑且离开了百货商店。
 
。
 
「呼ー……」
 
。
 
　我不禁深深地叹了口气。
 
　怎么办啊，真的、这种情况。
 
　完全是误会我了……。
 
 
 
 
 
。
 
 
 
 
 
　后来我来到距离百货商店有点距离的建筑里，确保休息的空间。
 
　从道具包取出食物，开始吃着为时已晚的晚餐。
 
　莫莫也从影子里出来，我把狗粮给它。
 
。
 
　今天也该睡觉了……发生了许多事情，好累。
 
　明天再来考虑百货商店的物资吧。
 
　如此心想，我躲进被褥里打算睡觉。
 
 
 
 
 
这瞬间，『索敌』有反应了。
 
。
 
「……嗯？」
 
。
 
　这是魔物的气息。
 
　数量……总共是四只。
 
　它们正往这边过来。
 
　不对，正确地来讲并不是我所在的建筑物，而是百货商店那边。
 
。
 
「莫莫，我们要出门了」
 
。
 
「汪」
 
。
 
　我们离开建筑物，往百货商店那边赶去。
 
　压低身子窥视着情况，在那边的魔物是犹如狼一样。
 
　那魔物……跟我当初撞死的魔物很相似啊。
 
　记得名字是影狼来着？
 
　不过体型比起它还要小一圈。
 
　也就是影狼的弱化版吗。
 
　那些家伙在百货商店的周围徘徊，感觉像是在确认内部的情形。
 
。
 
「呜ー……！」
 
。
 
　莫莫也发出低沉的威吓声。
 
　因为它们是狗型的魔物？
 
　似乎是相当警戒的样子。
 
。
 
「学生那边……还没有注意到吗」
 
。
 
　店门虽然有两个人在放哨，但他们没有注意到魔物。
 
　毕竟四周昏暗无光，应该是没办法看到魔物的样子。
 
　是说，他们没有人拥有『感知系』的技能吗。
 
　啊，如果有的话早就注意到我了吧。
 
。
 
　该怎么办呢ー。
 
　感觉要是随意插手，事态会更加恶化……。
 
一一一一一一一一一
