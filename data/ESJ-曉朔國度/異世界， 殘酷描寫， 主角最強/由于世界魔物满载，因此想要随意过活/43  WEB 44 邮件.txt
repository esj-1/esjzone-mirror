翻译：飒君CONAN
 
 
 
WEB 44 邮件
 
 
一一一一一一一一一一
 
 
 
 
 
　刚、刚才的天之声是怎么回事……？
 
　我感觉好像听到与平时截然不同的内容……？
 
。
 
≪满足一定条件≫
 
≪可以获得技能『邮件』≫
 
≪目前有一条未读邮件≫
 
。
 
　脑海里再次回响起天之声。
 
　……邮件。
 
　刚才果然说的就是邮件吧？
 
　原来不是我听错了啊。
 
。
 
「汪！」
 
。
 
「诶、……啊！」
 
。
 
　莫莫的叫声让我回过神。
 
　对了，总之我得先藏起来。
 
　我和莫莫立即走到附近的墙边。
 
。
 
　在那座公寓看来，这里就是死角。
 
　没办法狙击到我们。
 
　嘛，既然那人都放过那些学生，只打死魔物，也就是说至少不是盯上了谁。即便如此，还是小心为妙。
 
。
 
　如果那人有意狙击，早就立马开枪了吧。
 
　……想到这里就感到一股寒意。
 
　虽然不知道对方是谁，可如果有这种念头，我早就已经死了。
 
　认识到这种事实，冷汗都流出来。
 
　笨蛋吗，我……实在太大意了。
 
。
 
「咕呜」
 
「……（抖抖）」
 
。
 
　正当我沮丧的时候，莫莫和小红都在担心我。
 
　啊啊，抱歉让你们担心了。
 
　没错，现在不是沮丧的时候，而是吸取教训不要再发生这种失误。
 
。
 
　好，重新振作起来。
 
　冷静了一些。
 
　那么重新思考―――邮件到底是什么鬼啊!?
 
。
 
　从刚才的天之声可以得知有这么一个叫『邮件』的技能。
 
。
 
　我打开人物状态面板。
 
　在追加获得技能的项目上。
 
　的确是显示着『邮件』。
 
　所需点数是1点，与其他追加技能不同，所需点数比较少啊。
 
　虽然这技能挺令人在意，不过我现在的技能点数是0。
 
　直到下次升级之前都不会有点数。
 
。
 
　可是好在意……。
 
。
 
　至少能让我看看刚才发送给我的邮件内容吧？
 
　我这么想着后，人物状态画面发生了变化。
 
　怎么了？
 
。
 
　在变化的人物状态画面上显示着『邮件菜单』。
 
　看了下项目，里面有着写邮件、收件箱、发信箱、未读信息等等。
 
　底部还有为了输入文字而提供的长形条。
 
　简直就像是立体的电脑画面一样。
 
　还有这种功能吗……。
 
。
 
　项目当中的『未读』栏目正闪烁着。
 
　我尝试着点击了一下。
 
。
 
未读邮件一条。
 
发件人『一之濑 奈津』
 
主题『初次见面』。
 
。
 
　……真的是邮件啊。
 
　这世界到底是变成什么样啊，什么都有吗。
 
　不对，现实中的手机已经用不上了，取而代替的就是以这种形式建立起联系方式？
 
　总、总之还是先看看内容比较好吧？
 
。
 
　话说这家伙是谁啊？
 
　一之瀬 奈津……？
 
　在我认识的人当中没有人的名字是这个，公司里也没有一个姓氏是一之濑的家伙才对。
 
　归根究底，我的交友关系并不是那么广泛。
 
　朋友的数量也是屈指可数的，如果有人叫做这名字我肯定会记得。
 
　到底在哪里见面过？
 
　我看向邮件的文字。
 
 
 
 
 
。
 
实在万分抱歉，突然向您发送邮件。我的名字叫做一之濑奈津。
 
请问您应该是观望到方才的战斗了吧？
 
当时的狙击就是我。
 
。
 
「……哈?」
 
。
 
　我不禁发出声。
 
　什、什么……？
 
　刚才是这家伙干的？
 
　尽管脑海里满是疑问，可我还是决定先看下去。
 
 
 
 
 
。
 
我看到您躲在那边观望着战斗。
 
会发现到您也是偶然。
 
当我在公寓屋顶上观察外边的情况时，便发现了在商业合作商铺里寻找食材的学生们，以及您还有那只汪酱的身影。
 
。
 
原本我是想在当时向您发送邮件的，不过在发送之前魔物便出现了。
 
由于我觉得即便是互不相识的陌生人，要是死了也会令我感到愧疚不安，因此才做出了那般举动。
 
。
 
实在是万分抱歉，导致到让您惊吓一番。
 
不过就结论而言也因此拯救了他们的性命，我觉得真是太好了。
 
。
 
那么请让我言归正传，会如此像您发送邮件，是希望请您务必让我加入您与那只汪酱的队伍。
 
。
 
也许你应该会觉得我为什么突然有这种念头，其实早在之前我就已经得知关于您的事情了。
 
请问您还记得吗？
 
昨天，您有在离此处几公里的商业合作商铺离与几只哥布林战斗对吧？
 
当时我就在远方观望着您的战斗。
 
经过缜密计算过的走位英勇无比，俨然让我衷心感到佩服。
 
。
 
特别是您所携带的那只柴犬，着实美妙至极。
 
它的强大、可爱、毛感，它的一切都将我的心灵紧紧抓而不放。
 
。
 
目睹它的身姿，我就奢望自己也能成为你们的力量。
 
突如其来的联系应该会造成您的困惑，但还请务必给予谅解，若需要考虑还请务必告知一下。
 
我就此等待您的好消息。
 
 
 
 
 
。
 
「……」
 
。
 
　啊——，这什么？
 
　事务邮件？
 
。
 
　虽然我觉得这邮件主要就是在说要不要组队，不过真是生硬。
 
　就像是运营公司发送过来的通知短信一样。
 
。
 
　怎么办？
 
。
 
「……总而言之，似乎不是什么坏人的样子……」
 
。
 
　我不知道这人是谁。
 
　文笔也很生硬。
 
　不过在这个部分上
 
。
 
『特别是您所携带的那只柴犬，着实美妙至极。
 
它的强大、可爱、毛感，它的一切都将我的心灵紧紧抓而不放』
 
 
 
 
 
。
 
　光凭这部分我也可以认为这家伙不是坏人。
 
　你说为什么？
 
　会知道莫莫可爱之处的家伙不可能是坏蛋。
 
。
 
 
 
 
 
嘛，这件事暂且不提。
 
　那么该怎么办……？
 
。
 
　现在还不能确认这不是陷阱，也可以选择无视这条邮件。
 
　不对，这样不行。
 
　毕竟这家伙认识我。
 
　既然都看到我昨天跟哥布林的战斗，那也意味着看到我取出收纳道具的瞬间。
 
　最好还是让那人以为我没有『道具包』，只是拥有类似的技能，这样比较好。
 
。
 
　……难不成就是当时感到的『视线』吗？
 
　当时总觉得似乎被别人看着，如果是这家伙就说得通了。
 
　既然我的技能有可能暴露给那人了，那无视邮件就绝对不妙。
 
。
 
　而且这家伙也有可能是真的想和我们组队。
 
　我想要的是可以远距离攻击的伙伴。
 
　并且那家伙还是具备远距离一击毙命魔物的威力。
 
　如果那家伙成为伙伴，绝对是强大的战力。
 
。
 
　虽然还不是很相信那家伙，但至少还是回复一下比较好吧。
 
　随便写点『总之请让我先考虑一下』，拖延时间。
 
　我点击『回信』项目。
 
。
 
≪未得到技能『邮件』，无法回信邮件≫
 
。
 
　脑海里响起声音。
 
　喂喂，原来没得到『邮件』技能就不能回信吗。
 
　该不会其他项目也是。我想到这件事点了点其他项目，结果是全部行不通。
 
　看样子在没有得到技能的状态下，可以做到的就只是确认已接受的邮件。
 
。
 
　先升级吧。
 
　只能离开这里打倒魔物升级，再立马学习『邮件』。
 
。
 
「待会再回复吧」
 
。
 
　总之先离开这里。
 
　既然对方可以发现我，也就代表那人已经很有可能看穿潜伏和气息消除。
 
　在没办法相信那人并且攻击手段是狙击的情况下，漫不经心地接近实在太危险了。
 
　我们慎重地走到处于那座公寓死角的地方，离开了此处。
 
 
 
 
 
。
 
 
 
 
 
　另一方面，此时在高层公寓的屋顶上―――。
 
。
 
「……写成这样应该没问题吧？」
 
。
 
　她一之濑奈津正在检查邮件的内容。
 
　如果想要不与他人会面，从而取得联系的话，究竟要怎么办才好？
 
　答案就是『邮件』。
 
　这是怕生的她也能够与他人取得联系的手段。
 
　当抽到这个技能时，她都高兴到小跳一下。
 
　如果是这技能，自己也能行。
 
。
 
　只要是自己遇见的对象，技能『邮件』都能够与对象取得联系，是一个方便的技能。
 
　虽然只限定是人类，但也没有问题。
 
。
 
「这样就能见到那只汪酱了……！」
 
。
 
　她就只抱着这份念头发送出邮件。
 
　原本她就考虑过跟别人组队，所以那封邮件所写的内容几乎都是她的真心话。
 
　虽然是说过他是人渣，不过她也明确地对那个男人做出评价。
 
　特化远距离与特化近距离，彼此都能弥补双方的不足之处。
 
　双赢的关系。
 
　嘛，虽然这些事情都只是那只柴犬的附带品就是了。
 
。
 
　刚才偶然发现到他们时，正所谓是感觉到了命运。
 
　剩下的就只是等待OK的回复。
 
　看样子她似乎是半点也没考虑过会被拒绝。
 
　完全就是乐观的想法。
 
。
 
「……躲在墙后面看不到啊，他有没有看到邮件呢？」
 
。
 
　从这里来看，他所躲藏的地方是死角，因此看不到。
 
　姑且再发送一封邮件吧。
 
　也许是因为误以为是可疑邮件，所以就无视了。
 
。
 
　像这种邮件开头都是比较重要的。
 
　文笔也有点生硬，还是稍微改正一下再发送吧。
 
。
 
「……这样还会生硬吗？」
 
。
 
　发送。
 
 
 
 
 
「啊……有错别字。我、我得再发送一次」
 
。
 
　发送。
 
。
 
「……还是再稍微称赞一下汪酱比较好吧？」
 
。
 
　发送。
 
。
 
「嗯，就是这样。这样就能行」
 
。
 
　发送。
 
。
 
「……还没有回复啊……」
 
。
 
　发送。
 
。
 
「以防万一，再发送一封吧」
 
。
 
　发送。
 
。
 
「啊，文章也稍微改一下吧」
 
。
 
　发送。
 
。
 
「然后在―――……」
 
。
 
　发送、发送、发送。
 
　发送、发送、发送、发送。
 
　发送、发送、发送、发送、发送。
 
。
 
　于是她一直在发送邮件，一小时后―――。
 
。
 
「啊，终于有回信了」
 
。
 
　太好了。
 
　虽然发了四十五封邮件左右，不过他终于是注意到了。
 
　好像是因为SP不够，所以没有学习『邮件』技能。
 
　于是为了升级，回信才会这么晚。内容上写的就是这些。
 
。
 
「呼……太好了」
 
。
 
　「请让我稍微考虑一下，所以能不能给我点时间。还有邮件我已经仔细读过了，所以您不需要多次发送也可以的。拜托您了，拜托您了」，回信是往好的方面发展。
 
　嗯，结果不错。
 
　先回信说「我明白了，还请多多指教」。
 
。
 
　话说，他的回信内容为什么这么谦虚，并且像是顾及到这边的心情呢？
 
　她歪着头。
 
一一一一一一一一一一
