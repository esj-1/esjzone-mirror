WEB 45 我没有错！我没有错！
 
 
一一一一一一一一一
 
　……我现在的心情就像是目睹到现代社会的黑暗。
 
　邮件好可怕。
 
　不如说是这女人好可怕。
 
　不，虽然我不知道是男是女，不过这家伙大概是女的。
 
。
 
　完蛋了、完蛋了……。
 
　这家伙是神经病吧。
 
。
 
　什么鬼啊，一小时发45封邮件.！
 
　有点恐怖啊。
 
　都感觉到了狂气。
 
。
 
　差不多每分钟都会在脑海里回响起天之声的≪收到了信息≫。
 
　我都觉得脑袋要变奇怪了。
 
　这或许是以往最痛苦的精神攻击。
 
。
 
　你说有多么痛苦？
 
　都痛苦到『抗压耐性』的等级都提升了。
 
　从ＬＶ５升级到ＬＶ６。
 
　拜此所赐，我感觉轻松了点。真可恶！
 
。
 
「啊——……好糟糕的心情……」
 
。
 
　我蹲下来抱着头。
 
　被个不得了的家伙盯上了啊……。
 
　真的要怎么办呢……。
 
　虽然无论我怎么打滚，感觉都只有不妙的预感就是了。
 
。
 
　啊啊，好想逃跑，好想无视，好想当做没发生过。
 
　说不定可以增加一位远距离攻击的伙伴，我真想把当初会想得这么简单的自己给揍一顿。
 
。
 
　这次狩猎魔物说不定是这三天里最拼命的一次。
 
　无论是寻找魔物的时候，还是战斗的时候都会经常响起天之声的≪收到了信息≫。
 
　升级的瞬间，我二话不说地就学习了『邮件』。
 
　然后我一来就寻找『垃圾邮件』。
 
　我想着能不能拒收呢。
 
　原来没有啊，该死！
 
。
 
　在各种意义上很不妙。
 
　如果不能摸摸莫莫我就要既死了。
 
。
 
「咕呜？」
 
。
 
　莫莫歪着头。
 
　啊啊，好可爱。
 
　光是看到它的举动，感觉被治愈了。
 
　今天我也能努力了啊……。
 
。
 
「……（抖抖）」
 
。
 
　啊啊，抱歉了，小红。
 
　我并不是忘记你了。
 
　嗯，不错的舒适感。
 
　习惯以后意外感觉挺不错的，史莱姆。
 
。
 
「哈啊——……虽然她不是个性很坏的人……」
 
。
 
　只是脑袋进水了而已。
 
　啊不对，这才是致命的吗。
 
。
 
≪受到了信息≫
 
。
 
　还来……。
 
　我不是才刚刚说过让我考虑一下的吗。
 
　到底又怎么了？
 
　我挠挠头打开邮件。
 
。
 
『说起来我有件事情忘了告诉您了。邮件技能提升到等级２以后，聊天室功能就会解锁哦。这样通讯延迟会更加少，并且还能清楚地了解彼此，我建议这样做』
 
。
 
　啧……。
 
　这种情报我可不想知道。
 
。
 
　就维持现在这样。
 
　当聊天室功能解锁之际，我已经能预见到自己会多么痛苦。
 
。
 
　……好，邮件就维持现在这样好了。
 
。
 
『是这样啊，十分感谢您提供的贵重情报。关于组队这件事还请稍微等一阵子。还有不需要频繁地发送邮件喔，我都会仔细看完，并且您的想法也已经完完全全传达给我了』
 
。
 
　好了，发送。
 
　明明都回信说让我稍微考虑一下，还像这样老是给我发邮件啊……。
 
。
 
　这家伙，没有常识也要有个限度吧。
 
　再给我稍微站在对方的角度思考事情。
 
。
 
　并且本人还毫无自觉，性质更加恶劣了。
 
。
 
　偶尔会有这种类型的人呢。
 
　自己明明没有这种意图，可还是无意识地给别人添麻烦。
 
　我真的是不擅长应付这种类型的人。
 
　再稍微站在对方的角度想一想啊。
 
。
 
「汪呜……」
 
「……（抖抖）」
 
。
 
　莫莫和小红以意味深长的眼神看着我。
 
　干嘛，我可是很普通的吧？
 
。
 
　话说回来，这个一之濑奈津到底是什么样的家伙啊？
 
。
 
　等级、职业、技能以及枪。
 
　实在有太多在意的事情了。
 
　特别是枪，到底是在哪里得到的？
 
　如果是气枪也就算了，在这个法治国家的日本中，根本不可能有任何渠道可以得到真正的枪，而且大概还是远距离狙击用的来福枪。即便有也是通过非法的地下交易吧。比如某种自由职业。
 
　从一发就可以解决小狼，并且还是在那种距离下，可以看出威力十分强大。
 
　说到对物来福枪，也就是巴雷特或者赫卡特来着？
 
（颯:玩吃鸡的都知道巴雷特狙击步枪，赫卡特有兴趣的同学自己百度）
 
　虽然我不怎么了解枪械方面的知识，所以几乎是一窍不通就是了。
 
。
 
　连这种枪都能灵活运用的技能。
 
　等级应该相当高。
 
。
 
　并且还有武器，该不会是拥有『制造』的技能吧？
 
　有可能啊……。
 
　既然连『邮件』这种技能都有。
 
　那有『网络』和『邮购』之类的技能也不奇怪。
 
。
 
「再怎么想也是没用的吗……」
 
。
 
　虽然向本人打听是最好的方法，但人家又不可能老实告诉我。
 
。
 
「总之已经拖延时间了，慢慢考虑吧……」
 
。
 
　远距离支援的确是有魅力，可要是遭到背叛就是背后来一发子弹。
 
　不能轻易地就得出结论。
 
　必须要看清楚这家伙是否真的值得相信。
 
。
 
　这段时间还是利用『邮件』进行交流吧。
 
　虽然很讨厌，虽然不情愿。
 
　在过程中慢慢地确认对方的情报以及为人就好。
 
　既然不是直接会面而是使用『邮件』，那就代表对方也是这种想法吧。
 
。
 
「某种意义上也算是邮件网友吗？」
 
。
 
　我边想着这种无所谓的事情，边再次在街上探索。
 
。
 
≪受到信息了≫
 
。
 
　……所以说好快！邮件来得太快了！
 
　到底是有多么没有耐心啊！
 
 
 
 
 
如此这般，在我们的探索中增加一个可疑的邮件网友。
 
 
 
 
 
。
 
 
 
 
 
　另一方面，此时在百货商店―――。
 
。
 
　戴着眼镜的学生——大野平安无事地回归到据点。
 
　同伴们都满脸笑容地迎接他。
 
。
 
「大野，你回来了吗！你没事真的太好了！……其他人呢？」
 
。
 
「嗯、嗯……那个、途中遭到魔物的袭击，大家走散了……」
 
。
 
「……是吗，我知道了。总之光是你能够平安回来就已经很高兴了。真的太好了……」
 
。
 
　西野完全没有展现出怀疑大野所说的样子。
 
　面对他担心着自己，大野感到心很痛。
 
。
 
「是……啊。那个、这些虽然很少……不过还是找到食材了」
 
。
 
　仿佛为了掩饰一般，大野将装满粮食的背包递给西野。
 
　见状，不光是西野，就连他身后的学生们都露出笑容。
 
。
 
「谢谢，你就慢慢休息吧」
 
。
 
「嗯。……啊，柴田君他们呢？还没有回来吗？」
 
。
 
「是啊，比起预定的时间要晚得多，也许是发生什么事了」
 
。
 
「是……这样啊……」
 
。
 
「对。如果再过一小时也没有回来，我是打算出动搜索队的。毕竟现在人手不多，所以跟你一起出去的那些人就先不管了……这点你就谅解一下吧」
 
。
 
「……」
 
。
 
　面对他担心同伴的样子，大野始终感觉到胸口很痛。
 
　走散只是谎言。
 
　自己是抛弃他们逃跑了。
 
。
 
（……没有错，我没有错……！是那些家伙擅自行动，而且还无视我所说的话……对啊，我没有错，什么错也没有啊……！）
 
。
 
　他无数次无数次在心中嘀咕道。
 
。
 
 
 
 
 
　过了几十分钟后，柴田回来了。
 
　他所带来的情报给西野他们产生巨大的动摇。
 
一一一一一一一一一一
