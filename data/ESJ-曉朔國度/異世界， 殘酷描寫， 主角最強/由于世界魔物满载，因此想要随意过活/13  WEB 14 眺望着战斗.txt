WEB 14 眺望着战斗
 
 
 
 
一一一一一一一一一
 
　结果在烦恼过后，我和莫莫都前往购物中心了。
 
。
 
　从这里到那边走路需要十五分钟左右吧。
 
　可是考虑到公寓到便利店所花的时间，还是假设要花上成倍的时间比较好吧。
 
。
 
　路上很安静。
 
　比起前往便利店的时候遇到的魔物还要更少。
 
。
 
　我只遇见一只僵尸和一只哥布林而已。
 
　用洗衣机打倒哥布林时，等级就提高了。
 
　太好了。
 
　我躲到巷子里，直接分配点数。
 
 
 
 
 
。
 
　加上之前的总共有14点JP。
 
　当然『密探』的等级也提升到9级。
 
　接着『轻步』、『观察』、『聆听』、『隐秘行动』的等级都提升一级。
 
　果然职业等级每提升三级，附属的技能等级也会提升一级吧。
 
。
 
　接下来再提高等级的话，『密探』就终于是ＬＶ１０。
 
　一旦提升到ＬＶ１０，应该会发生什么事。真令人兴奋不已。
 
。
 
　接着是ＳＰ。
 
　这边加上上次的总共有23点。
 
　肉体強化、道具包升级到ＬＶ７。
 
　而剑术则是ＬＶ３，潜伏是ＬＶ２。
 
　剩下两点就保留起来。
 
 
 
 
 
。

。
 
 
 
 
 
　确认完人物状态，再次开始移动。
 
　顺便一提，中途我运气好找到不错的武器。
 
　自从看到某本轻小说以后就一直想使用看看的武器。
 
　那把武器顺利地收纳进道具包了。
 
　于是花了二十分钟，我和莫莫来到购物中心的附近。
 
。
 
「……听得见战斗声啊」
 
。
 
「汪……」
 
。
 
　过来的时候就已经听见了，来到这么近的距离就听得更加清晰。
 
。
 
　叫声和开枪声。
 
　以及魔物的喊叫声。
 
。
 
　―――有其他人正在跟魔物战斗。
 
 
 
 
 
。
 
「在那里吗……」
 
。
 
　我们来到距离购物中心几十米的地方。
 
　我和莫莫躲藏在草丛里使用『潜伏』技能。
 
　通过许多验证才知道的，这『潜伏』技能拥有完美的附带效果。
 
　如果我抱着莫莫藏起来，技能效果似乎也会对莫莫产生作用。
 
　真是方便而又美妙的技能。
 
　要说为什么美妙，是因为我有借口可以尽情地抱着莫莫。
 
　软绵绵的，好温暖，好舒服。
 
　并且我摸着莫莫的腹部附近的时候，它还会发出一声「わふぅ……」似乎感觉很舒服的声音。这是什么神ＢＧＭ。
 
　真是美妙的技能。
 
 
 
 
 
。
 
「那么，究竟是怎么样的状况呢……」
 
。
 
　这点距离不需要望远镜。
 
　在购物中心的入口附近，人类正与魔物展开激烈的战斗。
 
。
 
　入口前面杂乱无章地堆积着用来作为障碍物的汽车、卡车、桌子和椅子。
 
　守护着障碍物被破坏掉的大门口的是约十位男性。
 
　手里拿着铁管、杆钩以及劈刀战斗着。
 
　其中也混杂着两位警官。
 
　他们用警棒战斗着，不用枪械吗？……不，早就没子弹了吧。
 
。
 
　战斗的对手是猪头人。
 
　从屋顶的ＳＯＳ旗帜来看，应该是打算坚持到有人来救助，然而现在正是魔物已经突破障碍物的时候吗。
 
。
 
「这样一看好大啊，猪头人……」
 
。
 
　它们有着哥布林所无法匹敌的巨大体格。
 
　与那些战斗中的男人进行比较，也应该有两米以上吧。
 
　手里都拿着巨大的菜刀。
 
　数量总共有五头。
 
。
 
　明明数量上有着成倍的差距，然而猪头人却无视这一点与他们抗衡。
 
　好强……。
 
　毫无疑问比起哥布林还要强更多。
 
。
 
　也许是多亏了厚重的皮下脂肪，警棒以及铁棒之类的打击性武器都起不到作用。
 
　话是这么说，劈刀和兰博刀的长度跟猪头人的菜刀相比，完全不在一个层次。
 
　重点是猪头人那边显然很习惯战斗。
 
　能跟它们匹敌的只有两位警官吧？
 
　那些人也升级了吗？
 
。
 
「怎么看都是人类那边不利啊ー」
 
。
 
　而且我最在意的是呆在那些猪头人后面的一头猪头人。
 
　这家伙的体格比起其他家伙还要大一圈。
 
　相较于其他猪头人的肌肤是黄色，这猪头人的肌肤是红黄。
 
　明显跟其他猪头人有所不同。
 
。
 
「猪头人的上级种？」
 
。
 
　高级・猪头人。
 
　脑海里浮现出这个单词。
 
　『危机感知』也叮铃铃地敲响着警钟。
 
 
 
 
 
。
 
「怎么办呢……」
 
。
 
　应该要帮帮他们吗？
 
。
 
「嗯~……可是啊……」
 
。
 
　稍微想了想。
 
　假设我现在为了帮助那些人而闯进去好了。
 
　只要利用汽车和家具以及莫莫的技能等一系列连击进行突袭，即便是面对在场的魔物说不定也能占上风。
 
　然而终究只是有『可能』。
 
　不是一定。不，死亡的可能性还高过这些。
 
　如果是以往战斗国的哥布林和僵尸那倒另当别论，但猪头人的实力是未知数。
 
　并且还是一群猪头人以及一头显然是上级个体的猪头人。
 
。
 
 
 
 
 
在这种状况中，究竟有什么利益可以让我做到这种程度？
 
。
 
　应该会得到经验值吧，大概等级也会提升吧。
 
　那些被我救助的人们会感谢我吧。
 
　或许还能从他们身上得到各种情报。
 
。
 
　可是―――之后呢？
 
。
 
　他们一定会来请求我。
 
　为了得到能够保护自己安全的『战力』。
 
。
 
　而我的战术是利用道具包。
 
　他们一定会针对道具包刨根问底吧。
 
　毕竟我性格就是这样。
 
　如果有擅长询问或者沟通能力强的家伙在，我没自信可以隐瞒下去。
 
。
 
　一定会各种要求我吧。
 
　在食物上也会说什么分一些给我。
 
　利用一些『现在是这种状况』、『人有困难的时候要互相帮助吧？』之类的自我方便的借口。
 
。
 
　话是这么说，我又不是那种『我会帮助你们，不过你们所有人都要听从我的命令，啊哈哈』之类的性格。
 
。
 
　这样一来，等待我的只是贴身保护他人的保镖结局。
 
　原社畜的我一定会顺应着形势干下去吧。
 
　直到完全逃到安全的地方为止都要继续做下去。
 
　为了互不相识的他人。
 
 
 
 
 
。
 
 
 
 
 
　……老实说，好麻烦。
 
一一一一一一一一一一
