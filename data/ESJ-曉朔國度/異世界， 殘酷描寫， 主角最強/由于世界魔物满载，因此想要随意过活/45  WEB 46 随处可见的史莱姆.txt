WEB 46 随处可见的史莱姆
 
 
一一一一一一一一一一一
 
 
 
 
 
　我有一个非常好的消息。
 
　邮件发送给一之濑之后，我在『邮件』技能上角角落落地都调查个遍。
 
　结果虽然没有找到『垃圾邮件』的功能，不过倒是找到了『静音模式』。
 
　即便受到新的信息，天之声似乎也不会在脑海里响起来。
 
　并且这还是在ＬＶ１的状态下都可以设定的功能，太棒了。
 
　我迅速将邮件设定为静音模式。
 
。
 
　这样脑内就会变成十分安静，还能处于一种舒适的状态。
 
　啊啊，太好了。
 
　真的很重要呢，静音模式。
 
　在举办会议时或者前去会面客户时，得将手机设定为静音模式才是社会人的常识。
 
　在和交易对象商量的途中，如果手机响起来也没有挂断，而是接电话的话就是毫无礼仪教养。
 
　嘛，不管怎么样，这样就能静下心展开行动了。
 
　太棒了。
 
。
 
≪熟练度达到一定水平≫
 
≪技能『邮件』从ＬＶ１升级到ＬＶ２了≫
 
。
 
　…………。
 
　嗯，我就当做是没听到吧。
 
　『邮件』还是维持着ＬＶ１。
 
　就当做是这么回事吧。
 
。
 
≪条件已满足≫
 
≪『邮件』追加了聊天室功能≫
 
≪目前聊天室可以登陆的人数为一名≫
 
≪请问需要进入聊天室吗？≫
 
。
 
「……」
 
。
 
　我无言地选择NO。
 
　接下来还确认了邮件的功能。
 
　……嗯，增加了聊天室功能。
 
　可恶啊！
 
　为什么要在这种时机上提升技能等级啊！
 
　这也太奇怪了吧！
 
　既然要提升技能等级，给我提升其他像是『剑术』或者『要害突击』之类的技能啊！
 
　我会很努力地使用这些技能的啦！
 
。
 
　总而言之，确认看看聊天室功能。
 
　啊，这玩意可以设定ON和OFF，真幸运。
 
　我毫不犹豫地设为OFF。
 
。
 
　增加一封『未读』了。
 
　……以后再看看吧。
 
　这样就好。
 
　不如说我根本不想看。
 
。
 
　好，继续探索吧。
 
　我决定把刚才发生都一切都当做没看见，和莫莫它们一起走了起来。
 
。
 
。
 
。
 
「农田增加了呢……」
 
。
 
　走了一会儿之后，能看见农田了。
 
　毕竟是在郊外中的边境，有这种景色也不足为奇。
 
。
 
「已经是五月份了，也是种植稻草的时期吧ー」
 
。
 
　农田里有着水道，正种植着绿色稻草。
 
　说到底，这些稻草不会再继续种植下去了吧。
 
　毕竟种植的人已经不在了。
 
　话说景色真不错啊ー。
 
　将我那颗因『邮件』而遍体鳞伤的心灵都给治愈了。
 
　而我害怕着看到未读邮件，因为感觉似乎会堆积很多……。
 
。
 
「……嗯？」
 
。
 
　我想着这些事情看着农田，偶然间看见一样东西。
 
。
 
　―――是史莱姆。
 
。
 
　史莱姆正在农田里啪嗒啪嗒地浮在水面上。
 
　而且数量还很多。
 
　连这种地方都有史莱姆吗。
 
　嘛，它们还是老样子什么也没做，单纯只是浮在水面上而已。
 
　身体任由水流所流动，偶尔会缠到稻草上吃着稻草。
 
　真的是很悠闲啊ー。
 
。
 
「咕呜？」
 
。
 
　这些家伙要怎么办？莫莫这么问我。
 
　就算你说要怎么办……。
 
　打倒它们也很浪费时间，就这样放任不管不就好了？
 
。
 
「……（抖抖）」
 
。
 
　当我这么想的时候，小红似乎在抖动着。
 
　它解除拟态，走向莫莫。
 
　怎么了？
 
。
 
「……（抖抖、抖抖）」
 
。
 
　小红抖动着身体。
 
　然后莫莫点着头。
 
　诶，该不会这是在对话？
 
。
 
「汪！」
 
 
 
 
 
　莫莫叫了一声之后，将『影子』伸向了农田。
 
　就这样抓住了一只史莱姆。
 
　史莱姆毫无抵抗地就被抓来到莫莫的旁边。
 
　就像是在钓水母……。
 
。
 
「莫莫、小红，你们要对这只史莱姆做什么啊？」
 
。
 
　我在意起来问了一下，小红慢慢走进史莱姆。
 
　接着它做出惊人的举动。
 
　居然将莫莫抓来的史莱姆『吸收』进去了。
 
。
 
「诶!?」
 
。
 
　太过于震惊，我都喊出声。
 
　小红的身体覆盖住史莱姆，慢慢地吸取它。
 
　被吸取的史莱姆毫无任何抵抗，任由吸收。它的身体逐渐地慢慢变小。
 
　过了一分钟左右吧。
 
　小红将抓来的史莱姆吸收掉，感到满足似的抖了抖身体。
 
　感觉它的体型比起吸收之前还要大了一点点。
 
。
 
　真是十分具有冲击性的场景。
 
　我还不知道……。原来史莱姆还会吃自己的同族吗。
 
　不对，从种族来说与其说是吃掉，不如说更像是融合吧……？
 
。
 
　紧接着小红往我这边过来，再次抖动着身体。
 
　这次又怎么了？
 
　我仔细地端详着，然后小红的身体分裂成两个。
 
　什、什么!?
 
。
 
「「……（抖抖、抖抖）」」
 
。
 
　变为两只的小红抖着身体。
 
　是我的错觉吗，它身体比起还是一只的时候还要小。
 
　体型差不多是之前的三分之二。
 
　它们就这样一直在注视着我。
 
。
 
「嗯？怎么了啊？」
 
。
 
「「……（抖抖）」」
 
。
 
　是我的心理作用吗，感觉有股想要让我称赞它的氛围。
 
　挺起胸膛地说怎么样？我连这种事情都可以做到喔……我个人感觉是这样。
 
　难道说它是为了想让我称赞它，所以才吸取其他的史莱姆吗。
 
。
 
「噢……噢噢，好厉害啊。小红连这种事都可以做得到！嗯，厉害！」
 
。
 
　我说完之后，两只小红都高兴地蹦蹦跳跳。
 
 
 
 
 
虽然完全跳不起来就是了。
 
　看起来似乎想让我摸它，于是摸了摸。
 
　触感凉嗖嗖的，比想象中还要舒服。
 
　小红也是很开心似的抖了抖身体。
 
。
 
「呜ー……」
 
。
 
　接着莫莫也用身体磨蹭我。
 
　看来是希望也能摸摸自己。
 
　哈哈哈，可爱的小家伙。
 
。
 
「啊啊，莫莫也了不起喔。谢谢你帮了小红」
 
。
 
　如此说着并抚摸后，莫莫似乎很高兴地咪起眼睛。
 
　我就继续这样享受了一阵子的抚摸的感觉。
 
　真是幸福的时刻。
 
。
 
「―――你想要吸收更多史莱姆？」
 
。
 
　抚摸完以后，小红如此提议。
 
　看样子小红可以通过吸取史莱姆来强化自己。
 
　对我来说当然是求之不得，不过这样真的可以吗？
 
　姑且是同族，就没有什么抵触感吗？
 
　我问了它以后，它回答没事。
 
　史莱姆似乎并没有所谓的同伴意识。
 
　既然如此，我也没什么可说的。
 
　我和莫莫将史莱姆都抓过来，都给了小红。
 
。
 
　不光是农田里面，排水沟也有许多史莱姆。
 
　应该是任由水流冲走，于是聚集在那边的吧。
 
　多亏这样，才能给小红那么多的史莱姆。
 
。
 
「好，那么我们差不多该离开了吧」
 
。
 
　总停留在同一个地方并不这么好。
 
　正当我想要离开时，距离农田较远的地方『索敌』有反应了。
 
　不是人……是魔物的气息。
 
　并且还是比起以往还要强的气息。
 
。
 
　我立即让莫莫躲在影子里，让小红拟态。
 
　当初低下身，使用『潜伏』窥视着情形。
 
　利用『远望』查看着有反应的地方。
 
。
 
「那是…」
 
。
 
　看到那个魔物，我不禁睁大眼睛。
 
　拥有肥胖的身躯，以及猪头型的魔物。
 
　是猪头人。
 
。
 
　它用鼻子闻着什么，边环视着周围边接近这边。
 
　……它在找什么？
 
。
 
　怎么办？
 
　数量就一头。
 
　是该逃跑呢，还是要战斗呢……。
 
一一一一一一一一一
