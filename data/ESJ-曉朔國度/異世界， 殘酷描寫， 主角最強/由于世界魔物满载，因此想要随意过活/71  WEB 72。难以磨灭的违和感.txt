WEB 72.难以磨灭的违和感
 
 
我窥视着学生们和黑狼的情况。
 
学生的人数是十个人左右。
 
恐怕全都是技能持有者吧。
 
身手跟普通学生截然不同。
 
话说回来―――。
 
。
 
「……为什么那只魔物会在这里呢？」
 
。
 
一之濑小姐嘀咕道。
 
她似乎也是跟我同个想法。
 
。
 
「鬼知道，这种事只有本人才会知情了……」
 
。
 
目前的形势感觉是黑狼那边有利一点。
 
数量上则是学生那边压倒性有利，但黑狼的力量更胜一筹。
 
即便如此学生他们依然能够勇敢奋斗，估计是因为跟我们战斗时受到的伤害让黑狼无法全力发挥吧。
 
黑狼的动作明显很迟钝。
 
相当弱小的样子。
 
。
 
正因为如此我才搞不懂。
 
为什么这家伙会出现在这里？
 
。
 
如果是为了想要更高效率地狩猎人类得到经验值，那袭击避难所的确是合乎道理。
 
不过既然这样，那治好伤势再来也应该无所谓才是。
 
这样袭击的成功率反而会高很多。
 
。
 
那家伙跟我们战斗时，一领悟到形势不利就立马撤退，它就具备着这种智能。
 
那就不可能不懂得这样的道理。
 
然而这家伙却出现在此处正与学生们展开战斗。
 
面对这种情景，我始终都感觉到违和感。
 
 
 
 
 
。
 
「咕嗷嗷嗷嗷嗷嗷嗷!!」
 
。
 
黑狼吠叫着。
 
在脚边展开的『暗』往周围延伸，将物体全都吞噬殆尽。
 
。
 
「各位！别触碰到那些黑色的东西！会被吞进去的！」
 
。
 
在战斗中下达指示的是一位戴着眼镜，似乎很神经质的学生。
 
从他的声音来看，应该是学生会的副会长吧？
 
他在距离前线一步之遥处指挥着学生们。
 
。
 
而抵起眼镜的动作实在是完美无缺。
 
是知识分子系的吧，知识眼镜君。
 
。
 
「呜、呜啊啊啊啊啊啊！」
 
「救救我啊啊啊啊啊啊啊！」
 
。
 
看来有两位学生慢了一步。
 
脚下都被『暗』所覆盖。
 
。
 
「到底在做什么，这些蠢货！还不快点从那走开！」
 
。
 
「可、可是副会长！就算你这么说，脚完全拔不出来啊！」
 
「我也是……总觉得脚被抓住了……」
 
。
 
尽管知识眼镜君焦虑地大喊，然而他们完全无法逃脱。
 
不如说越是挣扎，他们越是沉浸于其中。
 
。
 
「啧，这些无能者……够了！无视这些家伙！其他人继续攻击！」
 
。
 
喂喂，这么轻易就舍弃掉吗，那位知识眼镜君。
 
毫不留情啊。
 
不对，这也是正确的判吗。
 
毕竟要是去救助他们，也许会构成第二次灾害……。
 
。
 
「可、可是副会长……」
 
「要、要见死不救吗？」
 
「那些人现在还活着吧？」
 
。
 
然而在他身旁的学生们似乎无法狠下心来。
 
应该是想要救助他们的心意比较强吧。
 
紧接着，他用手上的铁棒敲打下地面。
 
。
 
「你们这些家伙！究竟是在向谁提意见！我可是从五十岚会长那里得到战斗的全盘指挥权喔？违逆我就相当于是在违逆会长，你们应该知道这意味着什么吧!?」
 
 
 
 
 
「「「……！」」」
 
。
 
闻言，学生们都瞬间缄口不言。
 
。
 
「没错，这样就好。你们这些人只要乖乖闭嘴听从我的指示就好了」
 
。
 
他满足地点了点头。
 
而逃得太慢的那两位学生则是展露出绝望的表情。
 
。
 
「……和人先生，怎么办？」
 
。
 
应该是在问我究竟是否要救人吧。
 
。
 
「就现状来说，救助他们对于我们没有任何利益」
 
。
 
我直白地将自己的想法说出口。
 
救助那两个人，就相当于让我们暴露在他们面前。
 
。
 
「如果我们插手进去，暴露身份，有可能就会受到『魅惑』和『洗脑』」
 
。
 
这是我最惧怕的事情。
 
意味着我们将处于在他们技能的影响之下。
 
那位学生会长拥有利用『声音』介入对象精神的技能。
 
既然如此，其他学生会成员也很有可能拥有相似的技能。
 
至少我想要避免在不知道事情真伪的情况下，漫不经心地出现在他们面前的事态。
 
。
 
「……的确是这样……」
 
。
 
一之濑小姐点点头。
 
虽说可以理解，但她应该不能完全接受吧。
 
仔细回想一下，在给我发送邮件之前，她也救助了不良学生。
 
与我不同，她还没能那么割舍得下吧。
 
。
 
「嘛，一之濑小姐，我想应该是没问题了」
 
。
 
「诶……？」
 
。
 
「看样子援军已经抵达了」
 
。
 
我指着校舍那边。
 
有道人影以可怕的速度疾驰而来。
 
这道人影穿过知识眼镜君的身边，倏然停在正逐渐遭受『暗』所吞没的两人面前。
 
。
 
「呜诶，田中和カト，真是不得了的危机呢。你们两位都没事吧？」
 
。
 
与言语相反，口吻完全没有紧张感。
 
出现的是一位将显眼的头发绑成侧马尾的辣妹系女高中生。
 
。
 
「啊、相坂六花！你这家伙是来救援的吧？喂，为什么要擅作主张!?」
 
。
 
尽管知识眼镜君怒吼着，但她并没有在意。
 
而是抓住两人的手，想要把他们拉出来。
 
然而，为暗所抓住的两人的脚根本无法轻易拔出。
 
。
 
「嗯——果然靠力量是行不通的吗，那么……」
 
。
 
她弯下腰拿起劈刀。
 
见状，两人表情僵硬。
 
他们应该是立马领会到她『想要做什么』。
 
。
 
「诶、等等，相坂同学？」
 
「开、开玩笑……的吧？」
 
。
 
「对不起了，两位」
 
。
 
下一瞬间―――她用手上的劈刀毫不留情地将两人的脚砍断。
 
血沫飞舞，与此同时两人的身体也从『暗』中脱离开来。
 
回响着悲鸣声。
 
她就直接将两人的身体拉出来后，迅速地退到后面。
 
。
 
「好了，成功救出了呢！」
 
。
 
她将两人放在地面上以后，满足地点了点头。
 
面对这毫无犹豫的行动，学生们都茫然若失。
 
。
 
「……哈。你、你们这些人！究竟在发什么呆！别给我停止攻击！」
 
。
 
最快回过神的来的知识眼镜君再次发出指示。
 
其他学生也继续进行攻击。
 
。
 
「我、我的脚……」
 
「好、好痛、好痛啊……」
 
。
 
「对不起了，两位。不过……这样也比起丢掉性命还要好吧？」
 
。
 
面对露出苦闷神色的两人，六花若无其事地说着。
 
嘛，都变成那种样子了，除了这样做以外也的确是没有办法可以救到他们。
 
。
 
「真的是，多么野蛮。所以说不良学生就是……」
 
。
 
知识眼镜君提起眼镜，瞪视着六花。
 
。
 
「可是如果不这样做，这两人都会死掉吧？」
 
。
 
「那也无所谓吧？我又不需要连达成我的指示都做不到的垃圾」
 
。
 
「呜哇，真过分」
 
。
 
「随便你这么说。总之救了这两人后已经满足的话，就给我服从指示」
 
。
 
面对语气桀骜不驯的知识眼镜君，六花感到怒从中来。
 
。
 
「……哈？为什么我要―――」
 
。
 
「六花，服从他指示吧」
 
。
 
在一触即发的氛围中，西野君插手进来。
 
看样子他也是跟着援军过来。
 
。
 
「……小西？」
 
。
 
「现场的指挥官姑且是他，我也会支援的，所以现在服从他的指示吧」
 
。
 
「唔，我知道了啦……」
 
。
 
西野君说服过后，六花才勉为其难地接受。
 
她立刻举起劈刀，朝着黑狼那边冲过去。
 
。
 
当西野君和六花参加战斗，形势开始倾向一边。
 
学生们为了避免遭受『暗』所吞噬而都保持适当的距离，都纷纷采取远距离的攻击方式。
 
恐怕是意图削弱黑狼的体力，拖延为持久战吧。
 
那家伙原本在和我们的战斗中就已经消耗体力了。
 
如果进行持久战，的确有可能会获胜。
 
。
 
「魔物要逃跑了！」
 
。
 
而黑狼也领悟到这一点。
 
强行突破学生的包围圈，逐渐奔向市区。
 
。
 
「喂，别让他逃跑了！快追！」
 
「等下」
 
。
 
知识眼镜君立刻下达追击的命令，然而西野君用手制止住。
 
。
 
「……你想做什么，西野？」
 
。
 
「再追下去很危险，现在还是应该优先治疗负伤者」
 
。
 
「你在说什么蠢话……你也看到那只魔物的实力了吧？那种魔物要是放任不管就危险了。能解决的时候当然是要彻底解决比较好吧，你是想违逆身为指挥官的我吗，五十岚会长将指挥权全权托付的我！」
 
。
 
「那是在单单只有那家伙的前提之下吧？」
 
。
 
「你说什么……？」
 
。
 
「我和相坂在百货商店有跟这家伙相似的魔物战斗过。这些家伙是群体行动的魔物，并且还拥有呼唤伙伴的远吠技能。如果那家伙所逃亡的地方有它的同伴该怎么办？形势瞬间就会逆转，我们会全灭喔？」
 
。
 
西野君所说的，引得周围一片喧哗。
 
应该是知道他说的事情绝不是谎话，也并非是虚张声势吧。
 
知识眼镜君忌讳似的咂嘴。
 
。
 
「……可以，这次我就听从你的『忠告』了。不过你可别忘了，能得到现场指挥权的，终究只有我」
 
。
 
知识眼镜君撂下狠话后，就往校舍那边走过去。
 
见状，西野君深深地叹了口气。
 
其他学生也是放松下来了吧，几乎都当场瘫坐了下来。
 
。
 
「……看样子是结束了呢」
 
。
 
「……」
 
。
 
「……啊咧？一之濑小姐？」
 
。
 
没有回应。
 
怎么了？
 
我往一之濑小姐那边看过去，她正惊讶得张着嘴巴，注视着某一点。
 
。
 
「为……什么……？」
 
。
 
她视线的前方，有一位辣妹系的女高中生。
 
。
 
「小六……」
 
。
 
诶?小六？
 
一一一一一一一一一一一一
