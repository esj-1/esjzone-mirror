WEB 63.莫莫的新技能
 
 
一一一一一一一一一
 
『危机感知』、『敌意感知』都有明显的反应。
 
影狼它们毫无疑问是盯上我们了吧。
 
『无味』ＬＶ４丝毫还不能蒙混过那些家伙的鼻子。
 
那些家伙的嗅觉究竟是有多优秀啊。
 
。
 
「一之濑小姐」
 
。
 
「噫、噫！」
 
。
 
一之濑小姐被我搭话而吓一跳。
 
如今我正背着一之濑小姐，在市区上疾驰。
 
。
 
「总之对不起，突然背着你！从一之濑小姐所说的事情来看，大概我来背着你还跑得比较快」
 
。
 
这不是在性骚扰，绝不是在性骚扰。
 
。
 
「啊、没事……嗯、虽然是这样！但是、那个、紧贴着……！」
 
。
 
对不起，真的对不起。
 
也许你会呕吐出来，但还请再稍微坚持一下。
 
顺便一提，为了以防她摔下来，已经用『影子』将她固定住了。
 
以她的能力值，即便紧紧抓住也会立刻被甩落吧。
 
枪我也用『影子』牢牢固定住。
 
既然我没办法将枪收纳进『道具包』，果然是因为在我的认知中是『他人的所有物』吗？
 
如果可以让她本身收纳进道具包，或者像莫莫一样躲进『影子』就好了，不过不能的事情再说也没用。
 
。
 
「我要提升速度了喔！」
 
。
 
「诶、啊―――呀啊啊啊啊啊啊！」
 
。
 
嘛，以我现在的能力值背着一个女人奔跑根本没什么大不了。
 
如果不是被魔物追着，明明我就可以享受背上顶到的柔软触感……不对，说到底没有被追着也不可能背着她吧。
 
是先有鸡呢，还是先有蛋呢。有点不一样吧。
 
话又说回来―――。
 
。
 
「完全逃不掉啊……」
 
。
 
即便这边提高速度，那些家伙也以一定的速度保持着距离。
 
没有感觉到焦急。
 
正因为可以通过『嗅觉』进行追踪，它们才会采取这种保持一定距离给这边带来压力的行动。
 
集体盯上同个猎物，行动有如进行追踪的猎人。
 
……棘手了。
 
这些家伙真是总追着我们。
 
会憎恨我们的理由……嗯，嘛，毕竟狩猎那么多次呢，该死。
 
。
 
话说该怎么办。
 
越是持久战，这边越不利。
 
既然如此，还是在某处迎敌吧？
 
能够发挥出一之濑的射击，以及我和莫莫技能效果的地方……。
 
。
 
「果然只有那里了吧……」
 
。
 
购物中心的停车场。
 
只能在那边迎击影狼它们。
 
总觉得跟那边真有缘。
 
……虽然只有讨厌的回忆就是了。
 
 
 
 
 
。
 
。
 
 
 
 
 
抵达购物中心。
 
确认一下周围，走向正门附近。
 
。
 
「……呜咕、这里是……购物中心？」
 
。
 
「啊啊，就在这里迎击那些家伙」
 
。
 
从背上下来的一之濑小姐按住自己的嘴巴面向着这边。
 
。
 
「……要战斗吗？」
 
。
 
「是的」
 
。
 
「……我明白了」
 
。
 
我说完以后，她那铁青的表情便为止一变，一切的神色从脸上消失不见。
 
气氛骤然发生变化。
 
眼神锐利，全身的氛围感觉似乎变得冰冷。
 
一之濑小姐拿起枪，确认着周围。
 
。
 
「……和人先生，您能不能用刚才的黑色东西将我带到楼上呢？」
 
。
 
「诶……啊、啊啊」
 
。
 
一瞬间，我为她状态切换之快而感到惊讶。
 
。
 
「我的技能还是在楼上进行狙击比较好，就在那边掩护你们」
 
。
 
「……我知道了」
 
。
 
我如她所说立刻使『影子』发生变化。
 
购物中心二楼的一部分是停车场的高处。
 
对于狙击来说是正好合适的地方吧。
 
。
 
「啊，一之濑小姐。顺便把这玩意带走吧」
 
。
 
「这是……？」
 
。
 
一之濑小姐不可思议似的注视着收下的『东西』。
 
我给她的是针织帽。当然并不是普通的针织帽。
 
。
 
「那是分裂的小红所『拟态』的针织帽，它会起到作用的，所以请戴上去吧」
 
。
 
由于等级提升，小红的『拟态』能力也加强了。
 
虽然要根据拟态物的大小，但现在小红可以拟态为两到三个的物品。
 
。
 
「……非常感谢」
 
。
 
将戴上针织帽的她送到楼上，莫莫也从影子里现身。
 
面向前方，那边有着一群影狼。
 
一、二……总共有十二只吗。
 
好多啊。
 
不过凭现在的我们―――。
 
。
 
「莫莫，小红，一之濑小姐……狩猎了喔」
 
。
 
「汪！」
 
「……（抖抖）」
 
「嗯」
 
。
 
战斗开始。
 
我和莫莫的新技能。
 
就让你们见识一下。
 
。
 
「上吧，莫莫！」
 
。
 
「汪！」
 
。
 
首先我和莫莫分别从不同方向疾驰而出。
 
立刻展开『影子』袭击影狼它们。
 
当然，对方也会伸展『影子』进行应对，这也在预料范围内。
 
。
 
与此同时，我在那些家伙的头上将所有的建筑机械和汽车释放。
 
『影子』和『重量』的同时攻击。
 
。
 
「!?」
 
。
 
影狼它们浮现出惊讶的表情。
 
如果这些家伙跟前天战斗过的个体是相同程度，那应该是承受不住建筑机械的重量。
 
防御住我们的『影子』攻击，如果能逃跑就躲开建筑机械。
 
然而突如其来的建筑机械让那些家伙的动作有转眼即逝的延迟。
 
咚‼随着这样的声音发出，逃慢的几只影狼被砸死了。
 
。
 
≪获得了经验值≫
 
。
 
好，很顺利。
 
我在内心摆出胜利的姿势。
 
然而还不能大意。
 
躲过建筑机械的影狼们飞奔而来。
 
。
 
「莫莫！就是现在！」
 
。
 
我向莫莫发出指示，同时拍了下衣服，向小红传达信号。
 
莫莫点着头，吸着气。
 
紧接着―――
 
。
 
「ワォォォォオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオンッッ!!」
 
。
 
モモ『咆哮』了。
 
它的叫声化为物理性的破坏波及到周遭。
 
是连大气都会为之震撼的超弩级咆哮。
 
。
 
「「「～～～～!?」」」
 
。
 
听到这股『叫声』，影狼它们都后仰着身体。
 
这就是莫莫新学会的『技能』。
 
曾经将我们逼入绝望深渊的怪物高级猪头人的嚎叫。
 
由于吃下它的魔石，莫莫也得到那份力量。
 
。
 
影狼它们的动作完全停止。
 
我们并没有放过这次机会。
 
往它们头上释放建筑机械。
 
。
 
≪获得了经验值≫
 
≪达到一定的经验值≫
 
≪工藤和人从ＬＶ１６升级到ＬＶ１７≫
 
。
 
等级提升了。
 
不过还有剩下的敌人。
 
随后不知有几只影狼被爆头了。
 
。
 
「漂亮，一之濑小姐」
 
。
 
看样子她也没有放过莫莫创造出来的机会。
 
似乎没有受到『叫声』的影响，正如预料。
 
莫莫的『叫声』会不分敌我影响到周围。
 
这也是我为什么将拟态化的针织帽交给她的理由。
 
只要保持一定的距离，并且拟态化的小红守护她的头部，那她应该不会受到任何影响。
 
。
 
……其实我是想事先说明一下的，但没有时间。
 
待会得跟她道歉一下。
 
。
 
然而也还剩下几只影狼。
 
不错的节奏。
 
形势确实是这边有利。
 
再这样应该能行吧？
 
如此想着的瞬间―――有股寒意窜过。
 
。
 
「……！」
 
。
 
『危机感知』、『敌意感知』敲响警铃。
 
我反射性跳离当场。
 
。
 
这判断是正确的。
 
在我刚才所在的地方正扩散着『暗』。
 
。
 
「……这什么啊？」
 
。
 
并不是『影』。
 
看起来就像是将一切吞噬殆尽的无底之『暗』。
 
水滩大小的『暗』逐渐扩大，随着声音作响，从中有什么现身了。
 
。
 
现身的是一条黑狼。
 
体型比起影狼要小上一圈。
 
然而在它身上的压迫感是其他家伙所无法比拟的。
 
。
 
「喂喂……饶了我吧」
 
。
 
不会错的。
 
这家伙是上级个体。
 
影狼的上级种，黑狼吗……。
 
没想到还有这种家伙的。
 
光是这家伙出现，就有股以往的优势都随之一变的错觉。
 
它就是有如此的威胁
 
。
 
「……高级猪头人的级别吗？」
 
。
 
流下冷汗的同时，我定睛注视着眼前的敌人。
 
为什么才过一天又出现这么棘手的家伙啊。
 
。
 
「咕噜噜……」
 
。
 
对方战意满满，向我发出低沉的鸣叫并瞪视着我。
 
并且它在刚才不现身，恐怕是为了观察我们的实力吧。
 
动用同伴，斟酌我们的力量，于是只有在认为获胜之时这家伙才会出现。
 
。
 
多么棘手的对手。
 
明明就是魔物，但颇为聪慧。
 
我还是做好以往的战术都行不通的念头，来跟它战斗比较好吧。
 
不过呢，如果说它以为这样就看穿我们全部的实力就大错特错了。
 
这边还有剩下的『王牌』喔。
 
。
 
没错，就是新技能―――『忍术』
 
一一一一一一一一一一
