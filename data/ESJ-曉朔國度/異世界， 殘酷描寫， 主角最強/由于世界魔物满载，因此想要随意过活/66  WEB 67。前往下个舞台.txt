WEB 67.前往下个舞台
 
 
一一一一一一一一一一
 
 
 
 
 
世界出现魔物的第四天。
 
市中心依然是处于混乱状态之中。
 
突如其来的魔物，遭受隔绝的情报网，崩坏的建筑物。
 
在没有灾害警报以及避难诱导的情况下，人们只能依靠自己的判断行事。
 
是守家呢，还是行动呢。
 
。
 
「―――前往避难所吧」
 
。
 
人们如此寻思而纷纷选择避难。
 
他们在受到外头魔物威胁的同时，也总算是来到了指定的紧急避难所。
 
如今已是第四天，公务所和学校等公共设施都让过来避难的人们挤得水泄不通。
 
。
 
不过到后来并非是有什么进展。
 
大多数人都在干等着避难指示，然而就是完全没有来自于政府和官员的联系。
 
前往避难所就完全了，政府和自卫队应该会为我们做点什么吧。
 
他们就是抱着这种念头而行动，不过他们的希望十分轻易地就粉碎了。
 
。
 
「为什么还没有避难指示？」
 
「政府究竟在干什么？」
 
「我肚子好饿啊」
 
「这里真的是安全的吧？」
 
。
 
聚集在避难所的人们所感到的害怕与不安愈加严重。
 
。
 
为何政府没有行动呢？
 
。
 
其中的理由简单明了。
 
因为没有前例。
 
这又不是Ｂ级电影，究竟会有谁会认真想定过魔物出现并袭击人类之类的场面？
 
实在是颇为非现实的现实。
 
实在是颇为荒唐无稽的事态。
 
即便如此，政府打从最初就想要改变这种事态而行动了。
 
然而进展并不顺利事。
 
即便有意要设立对策本部来解决事态，也会由于法律与责任所阻挠。
 
宪法第九条，魔物是自然灾害吗?这是明文规定过的吗？
 
非法规的措施？那么谁来担负起这份责任？
 
商讨进度十分缓慢，在他们做这种事情的时候，市民也仍遭受魔物的袭击。
 
明明是为了减少魔物所带来的伤亡而互相探讨，然而魔物带来的伤亡仍是在增加。
 
。
 
并且无法使用通信机器也是造成这种原因的其中之一。
 
无论是邮件，亦或是手机与电话都无法使用。
 
即便是想要传达一件事情也全部需要口头说明。
 
上层的指示经过中层、低层、直至远地的其他设施究竟需要花上多少时间呢。在这段时间里伤亡依然在持续增加。
 
否，归根究底，在传达要事的途中就已经频繁发生遭受魔物袭击的案例，导致情报传达本身总是受到诸多限制。
 
并且又要花时间，伤亡又在扩大。
 
正所谓是恶性循环。
 
。
 
「我已经受不了，我要走了」
 
「我、我也是」
 
「我也一样，我还有家人」
 
「没错呀，为什么非得要我们来干啊？」
 
「等、等一下啦！如果我们不在，究竟要让谁来下达避难指示―――」
 
「让那些想要做的家伙去做不就好了」
 
「别开玩笑了！我们可是公务员吧！拿着国民的血汗钱―――」
 
「既然这样，你就留下来，我走」
 
。
 
起初他们也做好了义务上的职责，然而随着时间的经过都逐渐卸下了重任。
 
既有因害怕而放弃职责的人，也有意图跟家人一同逃亡的人，还有拼命阻止这种事情发生的人，以及根据自己的判断擅自下达避难指示的人。
 
事已至此，根本就没有所谓的上下级关系。
 
无论是谁都会自然而然地考虑到自己生存下来吧。
 
混乱招致混乱，这种情形又导致出更加混乱的场面。
 
法律没有履行到身为法律的意义，以往的道德以及价值观都全盘奔溃。
 
在这样的世界中―――。
 
 
 
 
 
。
 
≪获得了经验值≫
 
。
 
脑海里响起通知声。
 
听完以后，不良高中生柴田转过身。
 
。
 
「好，这边已经打倒了！喂大叔，你那边呢？」
 
。
 
「没、没问题！这边也总算是打倒了」
 
。
 
「很好」
 
。
 
柴田从百货商店逃亡出来以后，就一直在往学校的方向前进。
 
成员总共是五人
 
有两位是同班同学，以及避难民的中年男性和大学生。
 
。
 
「柴田君，刚才打倒哥布林后就升级了，点数要怎么用才好？」
 
。
 
「啊啊，大叔记得是选择了『冒险者』吧。那就将『肉体强化』提升到ＬＶ２。毕竟光是提升一级也有很大的不同」
 
。
 
「我、我明白了……」
 
。
 
中年男性听从柴田的指示，将技能点数用掉。
 
看到他的样子，柴田感到钦佩。
 
人要改变还是能改变的。
 
当初那么害怕，只会寻求救助的无能中年男性，如今已作为不错的战力与他们共同行动。
 
虽然不知道中年男性的心中究竟是产生什么样的变化。
 
但至少在当下的情况中是比什么都值得高兴的变化。
 
。
 
「……现在西野君他们究竟会在哪里呢？」
 
。
 
「鬼知道，不过至少应该是不会死掉了吧」
 
。
 
西野应该是跟着六花一起逃跑才对。
 
六花是他们团队里最强的。
 
除非是相当险峻的状况，否则应该还活着。
 
。
 
「总之我们就前往学校吧。西野他们的目的地应该也是那边」
 
。
 
虽然没有确证，但柴田认为这种可能性很高。
 
在猪头人袭击之前，西野已经找到新的活动据点。
 
并且还是能够聚集许多人的地方。
 
因此他们决定先抵达离这里最近的学校。
 
。
 
（请一定要平安无事，西野……）
 
。
 
柴田在心中发誓。
 
自己绝不会再丢下同伴不管。
 
绝对要和他们一同生存下来。
 
在心头下定决意，他和同伴一起前往学校。
 
 
 
 
 
。
 
 
 
 
 
≪获得了经验值≫
 
 
 
 
 
「啊，果然是这样ー。呐ー、姐姐。只要打倒这些家伙就能变得更强了啦ー」
 
。
 
「哦ー、很好很好，做的不错哦，弟弟。那么剩下的就交给姐姐吧！我会全部解决掉的！」
 
。
 
少女如此说着并伸出手掌心，将眼前的哥布林都烧死。
 
面对这样的光景，少年反应十分欢跃。
 
。
 
「噢噢！好厉害！真不愧是姐姐！」
 
。
 
「哈哈哈，对吧、对吧！再继续称赞！」
 
。
 
少年和少女十分开心地逐渐打倒魔物它们。
 
天真无邪的孩子所进行的攻势，持续到他们疲惫而睡午觉为止。
 
 
 
 
 
。
 
≪获得了经验值≫
 
。
 
「队长，周边的魔物全都已经扫荡完毕了」
 
。
 
「辛苦了」
 
。
 
在临时搭建的帐篷中，年轻的自卫队队员向年迈的男性进行报告。
 
。
 
「避难民正在诱导到指定的地方。……可是这样真的好吗？」
 
。
 
「什么事？」
 
。
 
「不是，那个……没有等待上头的指示，擅作主张……」
 
。
 
「我知道你想要说什么。然而上头在这四天里究竟做了什么？」
 
。
 
被称之为队长的年迈男性将手中的咖啡杯放在桌子上。
 
。
 
「要是等待他们的指示，一切就为时已晚了。负责现场的是自卫队（我们）。没有行动的正义毫无意义可言，我们必须尽可能多救到一位国民」
 
。
 
「可是……」
 
。
 
「你听好了，这是『命令』。你想违反身为现场最高负责人的『命令』吗？」
 
。
 
这位年迈的男性是处于目前这个避难所最高地位的人物。
 
而既然已经下达『命令』，他们就非得服从不可。
 
。
 
「……非常抱歉，队长……」
 
。
 
其中的意义，年轻的自卫队队员不可能没注意到。
 
。
 
「你明白就好，派遣出去侦查的直升机怎么样了？」
 
。
 
「三架直升机已经有两架回来了。可是……前往县界的一架直升机到现在还没有回来 」
 
。
 
这份报告让年迈的队长苦涩着脸。
 
派遣到县界的队员全都是有讨伐过魔物的技能持有者。
 
而他们没有回来，这究竟意味着什么呢。
 
他不可能不知道。
 
他立马下达命令让队员集合。
 
 
 
 
 
。
 
≪获得了经验值≫
 
。
 
脑海里响起声音。
 
不过『他』并没有在意，仅仅是注视着从此无法再讲话，变成了尸体的少女。
 
少女就是在刚才被他折断脖子而死的。
 
他满脸笑容地对她如此说道。
 
。
 
「谢了，我挺感谢你的」
 
。
 
 
 
 
 
世界魔物满载的第四天。
 
能够成为主宰者的人们都已经开始崭露头角。
 
。
 
然后―――。
 
。
 
「―――好，那么我们走吧」
 
。
 
「也是呢」
 
「汪」
 
「……（抖抖）」
 
。
 
做好准备，他们也行动了。
 
舞台转移到人类与魔物混杂的市中心。
 
一一一一一一一一
