WEB 65.病娇和有病是不一样的，完全不一样
 
 
一一一一一一一一一
 
总之魔物也离开了，还是跟一之濑小姐汇合吧。
 
我重新打量着周遭。
 
。
 
「呜哇，这还真是凄惨……」
 
。
 
购物中心自不待言，就连周遭的建筑物都完全破烂不堪。
 
刚才黑狼所释放的树干和废车不是撞击到建筑物的墙上，就是掘开地面。
 
简直就像是暴风灾害后一样。……不对，那种灾害应该没有现在这么凄惨。
 
。
 
「好可怕的威力……」
 
。
 
重新让我认识到黑狼是多么可怕的存在。
 
能够将『暗』所吸取的物体都犹如子弹一般全方位发射的力量。
 
虽然跟我『道具包』的攻击方式相似，但本质不同。
 
『道具包』的攻击是由于重量而自然坠落，可那家伙释放的物体则是明显加上了『速度』。
 
换句话说就是我的强化版，现代版王之财宝。
 
那也是我梦寐以求的理想攻击手段，真令人羡慕。
 
。
 
我就不不能使用这种招式吗？
 
既然那家伙是操控『影子』的影狼上级种。
 
那么只要提升『影法师』和『操影』，很有可能就会发现相似的攻击方式。有值得考虑一下的价值。
 
正当我想着这些事情的时候，往正门那边一看就发现了一之濑小姐。
 
她平安无事，似乎没有受到严重的伤害。
 
应该是小红将攻击抵挡下来了。
 
。
 
「……魔物们怎么了？」
 
。
 
「大概是逃跑了，趁着刚才我们因为刚才的攻势而睁不开眼的时候」
 
。
 
「逃跑了……？痛，我还是第一次看到会做出这种不同寻常的攻击的魔物。如果没有这只小史莱姆，搞不好我就已经死了……」
 
。
 
一之濑小姐边摸着小红，边以颤抖的声音说道。
 
的确，如果我没有小红说不定也身受重伤了吧。
 
很了不起喔，小红。了不起，小红。
 
。
 
「…………（抖抖、抖抖）」
 
。
 
不知是否是错觉，拟态过针织帽的小红看起来像是在得意洋洋。
 
嘛，毕竟它为我们做了那么多。
 
应该是身体放松下来了吧，一之濑小姐当场瘫坐下来。
 
。
 
「哈啊……好累」
 
。
 
「辛苦你了」
 
。
 
我从道具包取出茶递给她。
 
随便取出一份给我自己。
 
。
 
「汪汪！」
 
。
 
「好的好的，也有莫莫的那份」
 
。
 
我将碗放在从影子出来的莫莫面前，并倒进水。
 
伸出舌头拼命地喝着水的莫莫好可爱。
 
。
 
「……我以前就很在意了，和人先生的那种能力是『技能』对吧？」
 
。
 
一之濑小姐边喝着茶，边注视着我和莫莫的互动。
 
。
 
「是啊，是名叫『道具包』的技能。可以自由取出或者收纳物品」
 
。
 
再继续隐瞒也没有意义，因此我老实回答。
 
。
 
「呜哇——，好方便呢，真羡慕」
 
。
 
很不错对吧？我不会给你的。
 
她继续喝着茶，中途说了一句「说起来」，随后继续说着。
 
。
 
「我到现在还没问过和人先生的职业和技能来着？」
 
。
 
「啊ー，的确是这样呢……」
 
。
 
其实我也不是有意要隐瞒不说，只是在我说之前魔物就袭击过来了。
 
事到如今就老实说吧。
 
。
 
「嗯~，关于我的职业和技能―――」
 
。
 
我将自己的职业和技能，以及莫莫和小红的能力都告诉她。
 
虽然将底牌都暴露出来了，但感觉如果是她应该可以信任。
 
毕竟她自己地坦率地将技能和职业告诉我，这边也该展现出相应的诚意吧。接下来都是队伍成员了。
 
虽说如此，我还是有仔细注意是否有其他家伙在偷听就是了。
 
说明全部结束以后，一之濑小姐茫然若失的样子。
 
。
 
「诶……忍者？职业是忍者？」
 
。
 
「是的，就是这样」
 
。
 
虽然正确地来讲是『忍者』、『猎人』、『影法师』这三个。
 
。
 
「获得的职业有三个，能使用的『忍术』是五个，并且技能还有三十种以上……？诶、不会吧……诶？」
 
。
 
听完我的职业和技能，一之濑小姐比起预料中还要更加混乱。
 
偶尔还将手抵在嘴角上，嘀咕着「呜哇，我的技能会不会太少了……？」。
 
。
 
嗯，果然我的技能相比较其他人多了许多。
 
虽然至今以来都由于没有比较对象，所以没产生实感就是了。
 
果然『早熟』的效果好厉害。毕竟ＳＰ和ＪＰ获得量是一般人的十倍。
 
。
 
「……和人先生」
 
。
 
「嗯，怎么了？」
 
。
 
「总之我可以叫你做『开挂的混º蛋』吗？」
 
。
 
「驳回」
 
。
 
「为什么呀!」
 
。
 
「这是当然的吧！这种别名我可不想要」
 
。
 
绝对要拒绝。
 
于是一之濑小姐注视着自己脚边的同时、
 
。
 
「呜哇ー哪有这样的ー哪有这种人呀ー明明我都是『家里蹲』这种NETA职业可是对方居然是忍者并且还有（可爱）的小莫莫和史莱姆成为伙伴这完全就是开挂混º蛋吧话说就算是我也想要就职帅气的职业只是不小心就选择这种NETA职业而已而且我也想要这么可爱的宠物可对方居然已经全部得到了啊啊真羡慕真羡慕真羡慕真羡慕―――」
 
。
 
「……那个，一之濑小姐？」
 
。
 
啊咧，一之濑小姐的样子好奇怪。
 
。
 
「说到底错的完全就是抽奖明明只要出现更好的奖品就好了这样一来我也能更加轻松地提升等级也能早点遇到小莫莫和人先生的事情也是我可以不带偏见地看待他哎呀虽然初次见面的时候的确是将他当做是小莫莫附赠的不过现在我已经认同他了既值得依靠也是被我呕吐一身也不会多加在意的一个好人话说回来对于家里蹲来说无论什么事都难度太高了啦啊啊不行真的不行了因为胃好痛所以就呕吐出来什么的真的是好差劲―――」
 
。
 
她在说什么？
 
呐，这孩子在说些什么？
 
眼睛里没有光芒。
 
。
 
「一、一之濑小姐！请你振作一点！一之濑小姐！」
 
。
 
「……啊，是的」
 
。
 
我用力地摇动她肩膀后，一之濑小姐终于回到现实当中。
 
她与我眼神交汇就立马转过头，好打击我。
 
。
 
「……那个，对不起。让你看到我尴尬的一面了」
 
。
 
「啊，没事」
 
。
 
「其实我从以前开始就有一不注意就会埋头思考的习惯……诶嘿嘿」
 
。
 
不，刚才怎么看都不是这种层次吧。
 
别给我笑着蒙混过关啊。
 
怎么说呢。
 
邮件也好，刚才也好，这孩子应该是有病吧？
 
是我太早让她成为伙伴了？
 
。
 
「那个，总而言之，我们先离开这个地方吧。差不多要到中午了，就找个地方吃点饭吧。然后在边吃饭边商量一下今后的事情。这样可以吧，一之濑小姐？」
 
。
 
「好、好的、说的对呢！就这样做吧」
 
。
 
对于说个不停的我，一之濑小姐点点头。
 
明明她只要不说话就是美少女，怎么说呢这股遗憾的感觉。
 
不行，我得更加振作一点。
 
。
 
「好，莫莫和小红也觉得这样可以吧？」
 
。
 
「汪」
 
。
 
莫莫精神奕奕地回应着。
 
然而小红并没有回应。
 
。
 
「……小红？」
 
。
 
当我触摸拟态为衣服的小红后，小红突然解除拟态恢复成原本的史莱姆姿态。
 
。
 
「呀，怎么了？」
 
。
 
拟态为一之濑小姐的针织帽的小红分身也解除拟态，回归本体。
 
接着小红激烈地抖动着。
 
。
 
「小红，你怎么了，怎么这么抖动？」
 
。
 
「……」
 
。
 
小红没有回答，一直都在抖动着。
 
究竟是怎么了啊？
 
不对，等一下啊？
 
之前好像也有过这种情形―――。
 
。
 
「啊，该不会是―――」
 
。
 
我恍然大悟，确认下人物状态面板。
 
……果然是这样。
 
。
 
小红
 
红色史莱姆ＬＶ１０
 
。
 
小红的等级提升到10级了。
 
是在刚才与影狼的战斗中提升的等级。
 
换句话说―――
 
。
 
「～～～！！（抖抖抖）」
 
。
 
小红更加剧烈地抖动着。
 
随后耀眼的光芒包围住它。
 
。
 
「这该不会是……进化吧？」
 
。
 
同莫莫一样。
 
等级达到上限所产生的成长现象。
 
。
 
几秒过后，光芒渐渐消失。
 
从中则是已进化后的小红姿态。
 
 
 
一一一一一一一一一一
