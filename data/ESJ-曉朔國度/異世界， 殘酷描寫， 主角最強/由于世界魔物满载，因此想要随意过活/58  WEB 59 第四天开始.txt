翻译：飒君CONAN
 
 
 
WEB 59 第四天开始
 
 
一一一一一一一一一一一一
 
 
 
 
 
我醒了过来。
 
睡得真香。
 
姑且确认一下周围有没有魔物的气息。
 
『索敌』也没有反应，应该没问题吧。
 
。
 
「小红，将『拟态』解除一下」
 
。
 
「……（抖抖）」
 
。
 
让小红解除拟态后，我深深地伸着懒腰。
 
。
 
「莫莫、小红、早上好」
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
莫莫当然是早就起床了。
 
而小红……说起来史莱姆要睡觉吗？
 
问了一下，小红似乎像是完全不懂的样子抖了抖身体。
 
嘛，算了。反正它很有精神。
 
。
 
那么，今天终于是一之濑小姐成为伙伴的日子。
 
嗯~……果然会紧张啊。
 
无论如何，这也是第一位『人类』的队伍成员。
 
并且还是女性，坎坷不安。
 
我头发没问题吧?没有乱糟糟的吧？
 
。
 
「啊，得跟她商量好会面的地点和时间」
 
。
 
向一之濑小姐发送邮件。
 
回信正如预料很快就发过来了。
 
地点是附近的公寓，时间是一小时过后吗……。
 
。
 
「还有多余的时间呢」
 
。
 
先解决早餐的问题吧。
 
从道具包里面随便取出一些食材，作为早餐。
 
有面包、水果、色拉、牛奶。莫莫则是给它狗粮。
 
吃完后剩下的垃圾全给小红吸收掉。
 
小红也津津有味地吃着，真是十分环保。
 
。
 
吃完早餐后，洗头、刮胡子、还有是检查头发状况。
 
嘛，这是姑且吧。姑且先端正好自己的形象。
 
照照镜子一看，映照出我平时一副气色不好的表情，真可恶。
 
。
 
时间还有剩，再稍微探索一下吧。
 
会面的地点只要从这里走几分钟就到了。
 
再稍微在购物中心里面逛一下也应该没问题吧。
 
难得猪头人它们已经不在了。
 
。
 
「……话说回来」
 
。
 
昨天那些骷髅兵该不会是在猪头人它们离开后才出现的吧？
 
如果不是这样，时机也太凑巧了。
 
。
 
「魔物之间也会互相斗争呢」
 
。
 
毕竟在农田遇到的猪头人就是受到一群影狼所追赶，它们之间的关系应该也是相当杀伐吧。
 
嘛，话是这么说，我要做的事情也依然是狩猎它们。
 
不这样做，这边就会被杀。
 
。
 
于是剩下的时间就继续在购物中心里面探索。
 
就到昨天没能完全走到的地方吧。
 
顺便将似乎有用的东西都回收掉。
 
我走过一楼、二楼、停车场都进行着这道步骤。
 
停车场有大量遭人遗弃的汽车。
 
我当然是全都收纳进道具包。
 
连有破碎部分的支柱也回收，毕竟应该能成为不错的武器。
 
。
 
「话说回来，尸体好少啊……」
 
。
 
二楼和停车场并没有尸体。
 
在这座购物中心所杀死的人们果然就是在那座堆积成山的尸体中吧。
 
不过究竟是有什么意图才会特意将尸体集中在一处？
 
。
 
说起来魔物打倒以后尸体会消失不见，得到经验值，还会留下魔石。
 
可是人类依然还是存留着尸体状态。
 
这种不同之处究竟是怎么回事？
 
。
 
「……仔细想想，真是扑朔迷离啊，这世界的系统」
 
。
 
魔物突如其来地出现，世界也变得可以利用宛如游戏一般的系统。
 
归根究底，魔物为什么会出现啊?魔石又是什么呢？
 
不清楚的事情实在太多了。
 
即便要以『本来就是这样的』这种说法来接受，古怪的地方也太多了。
 
譬如说物理法则之类的，这些各方面……。
 
。
 
「……嗯?」
 
。
 
想着事情的时候，『索敌』有反应了。
 
并不是魔物的气息，而是人类。
 
。
 
该不会是……幸存者？
 
不对，可能性很低吧。
 
以猪头人的嗅觉能力，根本不可能完全躲藏得了。
 
除非像我这样拥有『无味』和『气息消除』，否则应该就会被发现吧。
 
。
 
「也就是说是从外面过来的……？」
 
。
 
就跟骷髅兵一样，看上了猪头人已经不在的时机吗?
 
总而言之，过去看一下吧。
 
我让莫莫躲进影子里，发动『气息消除』、『无声动作』、『无味』前往气息所在的地方。
 
。
 
「……是那边吗」
 
。
 
那是在与我进来的地方完全相反的出入口。
 
有两位女性。
 
好年轻啊。一位是十几岁多一点，另一位则是二十多岁。
 
。
 
「不好意思，有谁在吗——？」
 
。
 
「呐，果然我们还是回去吧？不会有问题的啦，大家一定已经逃跑了」
 
。
 
「……没事。我们跟之前不同，现在已经提升等级了。这一次不会再、输……输给魔物了。我绝对要找到」
 
。
 
「你声音这么颤抖根本就没有说服力……」
 
。
 
两位女性背着背包，举起疑似长尖叉的武器走在场内。
 
。
 
「哦呀……？」
 
。
 
其中一个人我有印象。
 
我注视着架子旁边的那位女性容貌。
 
是一位眼睛下有颗黑痣特征的黑发美女。
 
。
 
不会错的，那位女性就是住在我隔壁的新婚太太。
 
还活着吗，那个人……。
 
也就是说她所寻找的难不成是她老公？
 
我有股微妙的讨厌预感，决定继续观望她们的行动。
 
。
 
。
 
。
 
此时在另一边―――。
 
。
 
「糟了……胃好痛……」
 
。
 
一之瀬奈津正蹲在某座公寓的房间里。
 
结果经过一天，她的精神并没有安定下来。
 
。
 
「呜呜……好紧张」
 
。
 
不对，不如说时间的经过反而让她更加不安，根本没办法正常入眠。
 
要是见面后笑话我该怎么办、要是讨厌我该怎么办、真的没有在添麻烦吗？邮件会不会发太多了?摸摸小狗会不会摸过火了？
 
在她想着这些消极事情的时候，回过神来已经天亮了。
 
。
 
「不过为了跟莫莫见面……加油吧我……！」
 
。
 
一之濑做出加油的姿势，为自己打气。
 
不过果然跟不认识的人说话就会紧张，对精神上很不好。
 
毕竟她已经高中辍学一年。
 
与外界毫无联系的宅居生活令她怕生更加严重。
 
即便世界已经涌现魔物也依然毫无改变。
 
。
 
「在他晕倒的时候明明就简单地见到面……咕」
 
。
 
通过抽奖得到的回复药。
 
当为了给那位男性治疗而靠近时，自己的心情一定是不寻常的吧。
 
冷静地回想一下，都觉得害羞得要死。
 
。
 
「我该以怎么样的表情见他才好呢……」
 
。
 
估计对方并没有注意到吧，各方面都很不妙。
 
。
 
「呜呜ー……做好觉悟吧……做好觉悟啊……！」
 
。
 
不过……即便如此，我也有想要共同跨越这份苦难的人（狗）在。
 
好想摸一摸，好想受到治愈。
 
因此她站了起来。
 
即便是怕生也要克服。
 
全都是为了新的伙伴。
 
为了单单一只汪酱。
 
。
 
「要等我哦，莫莫……。我会努力的……！」
 
。
 
她扛起伙伴来福枪，打开公寓的大门。
 
表情犹如身经百战的战士决意要奔赴战场一般。
 
姑且提醒一下，她只是出门跟别人会面而已。
 
一一一一一一一一一一
