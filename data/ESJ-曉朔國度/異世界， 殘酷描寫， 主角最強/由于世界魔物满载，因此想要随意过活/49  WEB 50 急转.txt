一一一一一一一一一
 
　难不成……注意到我了？
 
　不过又是为什么？
 
　西野君也拥有『感知』系的技能吗？
 
　虽然我觉得昨天应该是没注意到我，该不会是升级之后学到的吧？
 
　还是说其他的技能？
 
。
 
「谁在……那边……」
 
。
 
　西野君以虚无的眼神往这边伸出手。
 
　怎么办……？
 
　应该要出来吗？
 
　周围没有魔物的气息。
 
　既也没有人的气息，也没感觉到被谁看着。
 
　不过……不过―――。
 
。
 
「汪」
 
。
 
　啊，莫莫!?
 
　当我烦恼时，莫莫就先出现在他们的面前。
 
　见状，西野君的神情阴霾。
 
。
 
「……狗？啊咧，你是……？咕，拜托了……谁都行，有谁来……―――」
 
。
 
　他没能把话说到最后。
 
　似乎是失去意识了。
 
　莫莫慢慢地走近他。
 
　接着在闻了闻他们之后就看向我。
 
。
 
「汪」
 
。
 
 
 
 
 
　应该是在说我出来也没问题吧？
 
　真没办法……。
 
。
 
「小红……再给我增加个兜帽」
 
。
 
「……（抖抖）」
 
。
 
　现在所穿的雨衣是小红『拟态』的衣服。
 
　因此可以随意地改变服装的搭配。
 
　我深深地戴上小红多拟态出来的兜帽。
 
　姑且是为了以防万一。
 
　避免被他们看到长相。
 
　然后再出道具包里面取出『背包』并背着。
 
　再往背包里面随意地放点东西，营造出感觉。
 
　这样就行。
 
　看上去完全就是避难民。
 
　我下定决心，往西野君他们走过去。
 
。
 
「……完全是失去意识了啊」
 
。
 
　无论怎么摇晃西野君还是另一位少女都毫无反应。
 
　衣服到处都有烧焦的痕迹，也有些地方渗透出血迹。
 
　西野君身上有许多擦伤和割伤。
 
　少女那边就更严重了，衣服一部分都染成了红色。
 
　果然是跟猪头人发生过战斗了。
 
　不过遇到那头高级猪头人，真亏他们能活着逃出来。
 
。
 
「汪」
 
。
 
　莫莫咬住我雨衣的衣摆拉着我。
 
　似乎在说救救他们。
 
　救助吗……。
 
。
 
「……不过几乎没有我可以做到的事情喔？」
 
。
 
　应急措施我也没有做过。
 
　别说可以做得到，真的就光有知识而已。
 
　我顶多只能将他们背到附近的无人房屋，以免淋到雨。
 
　莫莫注视着烦恼中的我。
 
。
 
「……呐，如果我这时候对他们袖手旁观，莫莫会讨厌起我吗？」
 
。
 
　无意间就说出这种事情。
 
　结果我心里还是没办法完全相信其他人吧。
 
　也许是害怕别人知道道具包的存在以及食粮。
 
　一之濑小姐也是一样。
 
　虽然她想要伙伴，但也仍没有完全迈出一步。
 
。
 
「……汪呜」
 
。
 
　莫莫表现出有点烦恼的样子，然后用身体磨蹭着我。
 
　像是在说「没事的，我一直都会是你的伙伴」。
 
　是吗……太好了。
 
　放心了。
 
　所以、
 
。
 
「……嘿咻」
 
。
 
　我把背包移到前面，背起西野君。
 
　……呜喔，人意外地很重啊。
 
　如果能力值没有提升，根本就背不起来。
 
。
 
「没办法一次性背走两人呢，分开背吧」
 
。
 
「咕ー呜……？」
 
。
 
　我望着表情茫然的莫莫。
 
。
 
「总而言之，他们就先背到那边的房屋里面。这样就可以了吧，莫莫？」
 
。
 
「！……汪！」
 
。
 
　莫莫尽情地摇晃尾巴，还围着我绕起圈。
 
　似乎是十分开心的样子。
 
　嘛，真没办法……这也是为了不让莫莫讨厌我。
 
　我把西野君和少女背到无人房屋里。
 
。
 
。
 
 
 
 
 
　在房屋里我让两人躺着。
 
　总之先将湿透的衣服脱下来―――脱不下来!?
 
　被雨水淋透的衣服比起想象中还要难脱。
 
　……用毛巾擦干吧。
 
。
 
「啊对了，止血……」
 
。
 
　西野君那边虽然还好，但少女这边就流很多血了。
 
　用包带卷起来……不对，先是消毒吧。
 
　总之掀开衣服，看一下伤口。
 
。
 
「……啊咧？」
 
。
 
　看着伤口，血已经止住了。
 
　从衣服上渗透的血量来看，我还以为伤势很严重……。
 
。
 
「……这是怎么回事？」
 
。
 
　再怎么说这伤势也恢复得太快了吧？
 
　该不会这孩子拥有这一类的『技能』？
 
　『ＨＰ回复』？不对，应该是『自我治疗』或者『自我再生』吗？
 
　总之虽然她脸色很差，但似乎不会立刻死掉了。
 
　太好了。
 
。
 
　啊，不过还是姑且消毒一下再用包带卷上去吧。
 
　还有为了御寒帮他们盖上毛毯，桌子上也放着一袋装满预备的包带、外用药、营养饮料等物品的袋子。
 
。
 
　这样就足够了吧。
 
　不如说我已经没有其他相关知识了，这已经是尽力而为。
 
。
 
「这就好了吧，莫莫？」
 
。
 
「汪！」
 
。
 
　莫莫似乎也很满意。
 
　那么，在两人醒过来之前先离开吧。
 
。
 
　走到外面，雨已经停了。
 
　乌云消失不见，到处都有阳光倾注而下。
 
　是小阵雨吗？
 
　毕竟在背着西野君他们的时候，雨势就已经变弱了。
 
。
 
「……得赶紧走了」
 
。
 
　比起想象中还要花时间。
 
　不过既然西野君他们是从百货商店逃过来的，也就是说现在猪头人它们就在那附近。
 
　那么购物中心那边反而没多少魔物才对。
 
　只要突破那边，直接往市中心的方向前进就好了吧。
 
。
 
　应该说现在为了避开影狼群体以及猪头人群体，就只有这条路可以走。
 
。
 
　虽然会路过一之濑小姐的公寓附近，不过应该没事吧。
 
 
 
 
 
反正估计又不会开枪打我。
 
　话说她自己也逃跑不就好了？
 
　在那座高层公寓上的确是最适合俯瞰城市进行狙击，可遗憾的是魔物的威胁正在旁边迫近。
 
。
 
「姑且还是发条邮件告诉她逃跑比较好吧……」
 
。
 
　虽然她在各方面上感觉很不妙，不过要是死了我也不是很舒服。
 
。
 
「哦呀……？」
 
。
 
　打开邮件界面一看，没有『未读』邮件。
 
　啊咧？我还以为肯定又发邮件过来了……。
 
　她正在休息吗？
 
　我感觉到微弱的违和感，发送邮件。
 
。
 
「……嗯？」
 
。
 
　骤然间，我看着公寓那边感觉到违和感。
 
　怎么了？
 
　好奇怪。
 
　从那座公寓上传来『厌恶感』。
 
　并且还是比起影狼群体和百货商店那边所感觉到的『厌恶感』还要大。
 
。
 
「怎么回事？」
 
。
 
　我仔细地端详着，感觉看到屋顶有什么东西。
 
　是一之濑小姐吗？
 
　不，不对。
 
　我在意起来，正打算使用『远望』―――这瞬间。
 
。
 
　随着一阵轰然巨响，公寓屋顶的一部分崩塌了。
 
。
 
「什!?」
 
。
 
　什么？发生了什么？
 
　屋顶的一部分完全凹陷下去，震起的碎片随着重力掉落下来。
 
　灰尘泛起，从中悠然现身的是一头魔物。
 
。
 
「―――啊」
 
。
 
　映入我眼帘的存在。
 
　即便隔着一段距离，『远望』技能依然清楚地目睹到『那家伙』的身姿。
 
。
 
「高级猪头人……」
 
。
 
　高层公寓上伫立着一头红铜色猪头人。
 
　为什么……？
 
　为什么你会在那边？
 
　你不是应该在百货商店吗？
 
。
 
　我看向百货商店ホ。
 
　有股『厌恶感』，不过比起刚才比较不『强烈』……很弱。
 
　就跟影狼群体所感觉的差不多。
 
。
 
「啊……」
 
。
 
　难不成―――难不成……。
 
。
 
　不经意间，脑海里想到最坏的想法。
 
。
 
　可以俯瞰周围的建筑物。
 
　寻找猎物的绝妙场所。
 
。
 
　为什么我会觉得这种地利只有『人类』会活用？
 
。
 
　如果说那家伙也注意到这种优势，单枪匹马地离开百货商店，为了想要更有效率地寻找『接下来的猎物』的话―――。
 
　如果说从而看上了那座高层公寓的话―――。
 
。
 
　一之濑小姐没事吧？
 
　还是说……。
 
。
 
　高级猪头人在屋顶上缓缓地环顾四周―――紧接着、
 
。
 
「―――啊」
 
。
 
　与我双眼交汇了。
 
　全身不寒而栗指的就是这种感觉吧。
 
。
 
　明明距离隔着这么远。
 
。
 
　明明在对面看来，这边是死角才对。
 
。
 
　然而高级猪头人的目光显然是望着我这边。
 
。
 
　那家伙看到我，嗤笑着。
 
　衷心感到喜悦一样。
 
　像是在说「终于遇到了」一样。
 
。
 
『――――喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔!!!!』
 
 
 
 
 
。
 
　―――『咆哮』回荡着。
