WEB 49 遭遇
 
 
一一一一一一一一一一
 
 
 
 
 
　地点是百货商店―――。
 
。
 
　西野和六花赶往门口时，状况已然处于劣势。
 
　面对几头猪头人，学生们和避难民完全是防御的一方。
 
。
 
「你们都没事吧!?」
 
。
 
「西、西野！」「这些家伙好强！凭我们根本是完全应付不过来……！」「究竟要怎么办才好啊？」「可恶！我怎么忍受得了自己死在这地方啊啊啊啊啊啊!」「别过来啊啊啊啊，你们这群怪物！这些怪物！这些怪物啊啊啊啊！」
 
。
 
　所有人都因两人的到来而感到安心，但声音并非从容。
 
。
 
（这种状况……不妙啊……）
 
。
 
　光是这里举目所及之处都有十头以上的猪头人。
 
　并且每一头猪头人都很强。
 
　远比昨天战斗过的哥布林和小狼还要强得多。
 
。
 
（如果是我的同班同学姑且不论，但避难民他们根本就不是对手……可恶，明明我还没有让他们提升到最低限度的级别）
 
。
 
　实际上早有几位避难民丢下武器，跑到销售场内深处颤抖着身体。
 
　从昨天到现在的连续战斗，让他们心灵奔溃了吧。
 
。
 
（该死……一群没用的**。平时就只会无谓地抱怨别人，一点用处都派不上吗）
 
。
 
　事到如今，还是不要将逃亡到店里的避难民当做是战力比较好。
 
　虽说如此，如今这种状况并没有简单到光凭在场的人们就能够应付。
 
　最令人在意的，其实还是呆在它们后方的红铜色猪头人。
 
。
 
（……层次明显跟其他家伙不同，那就是柴田说过的魔物吗……）
 
。
 
　红铜色的猪头人现在依然没有行动的迹象，仅仅是远远眺望着他们。
 
　西野的脸上流下冷汗。
 
　不妙，他的本能在告诉他那家伙一旦行动就『结束了』。
 
。
 
「……六花」
 
。
 
「什么事？」
 
。
 
　西野向旁边举着劈刀的六花询问道。
 
。
 
「你能对付那头红铜色的猪头人吗？」
 
。
 
「嗯~……」
 
。
 
　六花懒散地回应着他，并定睛注视着前方的猪头人。
 
　仅过了几秒钟。
 
　接着她深深地叹了口气。
 
 
 
 
 
「……抱歉，小西。那家伙我是不行了，会死」
 
。
 
　口吻还是跟平时一样毫无紧张感。
 
　然而这反而比起什么都更清楚地诉说着如今的状况，以及敌我之间的战力差距。
 
。
 
「『狂化』到最大极限也不行吗？」
 
。
 
「嗯，行不通。我赢不了」
 
。
 
「是吗……」
 
。
 
　既然六花都断言到这种地步，那这边是没有胜算了。
 
　既然如此，就只有一个方法。
 
　西野做好觉悟，大声喊道。
 
。
 
「各位！实行作战四！想方设地活下来吧！」
 
。
 
「诶!?」「不会吧、第四……？」「真的假的……」
 
「喂、喂！等一下，我可没听说过这种号数的作战喔？」
 
「我、我也是啊！究竟是什么作战呀？」「有什么办法吗？」
 
。
 
　听到西野的声音，反应明显分为两种。
 
　亦既是学生与避难民。
 
　西野的作战粗略地分为１～４。
 
　而第四道作战则是他想定中最糟糕的一个计划。
 
。
 
　那便是―――『将避难民作为诱饵逃离这里』。
 
。
 
　当然，只有与他共同行动的学生们才听过这道作战。
 
　避难民他们都不知情。
 
　而学生们知道这意味着什么，神色都略微有点阴霾。
 
　避难民虽然不知道其中的具体，但都以为是应该有什么办法吧，抱着这样的一份渺小的希望。
 
。
 
「边维持着防线，边慢慢后退到店里！一旦猪头人它们来到店里就立刻实行作战！后门和窗户！听好了吧！」
 
。
 
「了~解」
 
。
 
　六花毫不犹豫地点头说道，其他学生们则是有点犹豫不决。
 
。
 
「再这样下去，你们都会死喔！我们绝对要活下来！是这样对吧！」
 
。
 
「「「……！」」」
 
。
 
　听到西野拼命的叫喊，学生们也做好觉悟。
 
　避难民他们不知道这是什么意思，仅仅是听话照做而已。
 
。
 
（……没错，绝对不能死在这里，绝对……！）
 
。
 
　他们决死的作战就此开始。
 
　为了活下去而牺牲别人的作战。
 
 
 
 
 
。
 
。
 
。
 
　另一边，此时―――。
 
。
 
　『厌恶感』更加强了啊……。
 
　我们为了避免遇到影狼群体和猪头人群体，而慎重地在街道中行动。
 
　如果是哥布林这种程度的魔物姑且不论，但以我如今的等级要对付一群像猪头人和影狼这种层次的魔物根本是不可能的。
 
　何况猪头人那边还有高级猪头人。
 
　当时是运气好才放我一马，可我不认为下次还会这么幸运。
 
　绝对不能再遇见它。
 
。
 
「从位置上来看，刚刚好就像是被两群魔物夹在中间吗……」
 
。
 
　并且双方都是嗅觉优秀的魔物。
 
　相性太差了。
 
　要想顺利离开究竟要怎么办才好……？
 
　我边想着边仰望天空，只见天空乌云密布。
 
　雨水滴落在鼻子上。
 
。
 
「……又要下雨了吗……」
 
。
 
　最近总是在下雨。
 
　在大雨到来之前，我想尽可能地拉开距离。
 
　总而言之，先往没有『厌恶感』的方向前进吧。
 
　战斗也是能尽量避免就尽量避免。
 
。
 
「……嗯？」
 
。
 
　抬头看着天空，突然看到远处冒气袅袅黑烟。
 
　火灾吗？
 
　从烟雾的大小来看，规模相当大。
 
。
 
　那边记得是百货商店所在的方向吧。
 
　也是猪头人群体所前往的方向。
 
　难不成……。
 
。
 
　我有点在意起来，打算给一之濑小姐发一封邮件。
 
　如果在她那边的高层公寓上，应该可以立刻得知火灾的发生地吧。
 
　我打开邮件界面。
 
　有一封『未读』邮件。
 
　看了下发送时间，似乎是在刚才发送过来的。
 
　打开看看，就是以下的内容。
 
。
 
『百货商店似乎发生了火灾呢，猪头人它们也前往那边，在百货商店的避难民们很有可能与它们发生战斗了。猪头人它们貌似很积极地在狩猎人类，我们也必须要小心行事了呢。啊，说起来关于加入队伍这件事已经得出结论了吗？还请多多指教』
 
。
 
　情报好快！
 
　这人把我想打听的事情先发送过来了吗。
 
　还有，居然若无其事地催促我队伍的事情。
 
　真是贯彻落实啊，这个人……。
 
。
 
　不过，原来如此……。
 
　果然那阵烟雾是百货商店那边的吗。
 
　并且还是跟一群猪头人战斗。
 
　我只能说节哀顺变。
 
。
 
「如果他们能顺利逃跑就好了……」
 
。
 
　嘛，虽然我自己也没有余力去担心别人就是了。
 
　我避开有『厌恶感』的方向再次前进。
 
　途中雨下得变大起来，所以让小红拟态为『雨衣』。
 
。
 
「……嗯？」
 
。
 
　雨中走着走着，『索敌』有反应了。
 
　并不是魔物，而是人的气息。
 
　数量是两人。
 
　地点是……附近的公园吗。
 
　居然还有人不呆在家里，而是在外面游走，真是稀奇呢。
 
　我感到有点意思，往『索敌』有反应的方向过去。
 
。
 
　于是接近以后感觉到违和感。
 
　好奇怪……。
 
　这些人半点都没有离开『索敌』所反应的地方。
 
　怎么回事？
 
。
 
　我抵达有反应的地方。
 
　是一座仅有秋千和长椅的小公园。
 
　那些人就躲藏在角落的茂密草丛里。
 
。
 
「那不就是……」
 
。
 
　我躲在建筑物的阴影窥视着样子。
 
　有印象的容貌。
 
　是学生。
 
　一个是西野君。
 
　另一个则是绑着侧马尾的女高中生。
 
　名字是……六……什么来着？
 
　哎，算了。
 
　总之问题是两人的状况。
 
。
 
「……这不是遍体鳞伤吗」
 
 
 
 
 
两人的伤势严重到离他们较远的我都能知道。
 
　特别是那位女高中生，流了很多的血。
 
　样子相当不妙。
 
　该不会是受到猪头人的袭击后逃跑过来的吧？
 
　接着西野君往这边看了过来。
 
。
 
「……是谁在那边？」
 
一一一一一一一一一
