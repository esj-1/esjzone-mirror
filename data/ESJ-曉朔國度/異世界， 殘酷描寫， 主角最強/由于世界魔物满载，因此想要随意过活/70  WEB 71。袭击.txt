WEB 71.袭击
 
 
离开学生会室，再次走在校舍里。
 
那位会长在各种意义上都很不妙。
 
。
 
毕竟那位会长在说话的时候，『危机感知』就有反应了。
 
。
 
并不是魔物，也不是在战斗，单纯只是偷听，『危机感知』就出现反应。
 
换句话说，当时发动着某种『技能』。
 
隔着墙壁连我都受到影响，也就是说原因估计是『声音』。
 
毕竟她的声音有种不可思议的感觉，说不定是给听者带来某种影响的技能。
 
要比喻的话就是『魅惑』和『洗脑』一样。
 
既然都有『交渉术』技能，那存在这类技能也不足为奇。
 
就事实而言，西野君的态度从中途开始显然改变了。
 
应该是受到她的技能影响吧。
 
。
 
而『危机感知』虽然有反应，但我自己没有影响，也就是说这技能的能力是对指定的任意对象发动，或者是根据距离越远而效果会更加弱小……。
 
无论如何，我运气不错。
 
如果这是令所有听到声音的人类平等给予影响的技能，那如今我和一之濑小姐就―――。
 
想到这里，感觉背脊一凉。
 
这次的对象不是魔物，我只是单纯收集情报。说不定是这种想法导致我粗心大意。
 
也是因为他们无法认知到自己，于是我骄傲自满了。
 
。
 
「……可恶，我是笨蛋吗，太大意了……」
 
。
 
既然不知道对方有什么技能和职业，明明小心行事才是上上策，我究竟是多么粗心大意。
 
。
 
「那个，和人先生，请问你怎么了？从刚才开始表情有点可怕来着……？」
 
。
 
一之濑小姐对我这么说，我才回过神。
 
。
 
「啊，抱歉，我只是对于自己的愚蠢而有些惊呆了而已……」
 
。
 
「……哈？」
 
。
 
一之濑小姐歪着头，露出一副感到不可思议的表情。
 
是吗，与我不同，一之濑小姐并没有『聆听』技能，应该是不知道里面究竟在谈论什么事情吧。
 
我将学生会室里面的谈话内容，以及关于学生会长可能拥有的技能相关都告知给一之濑小姐。
 
。
 
「……原来如此，这样的确很棘手呢」
 
。
 
听完我所说以后，一之濑小姐点了点头如此说道。
 
顺便一提，我们现在位于远离学生会室的楼梯角落。
 
在没有人影的地方谈话比较好，即便他们不能用技能认知到我们。
 
。
 
「那位学生会长，我想应该就是五十岚同学吧。当我还是高一的时候，她就已经加入学生会了……毕竟当时好像发生过什么事，是一位名人」
 
。
 
「嘿，是这样啊……诶？」
 
。
 
给我等一下。
 
刚才感觉好像听到了什么不能听漏的信息……。
 
。
 
「……高一的时候？」
 
。
 
「啊……」
 
。
 
一之濑小姐也察觉到自己说了些什么。
 
。
 
「那个、是的……今年我姑且会升上高三……如果有上学的话，不过辍学了，就在去年……」
 
。
 
真的假的。
 
事到如今我才知道这件事。
 
怪不得肌肤挺光滑的。
 
。
 
「嘿ー，原来是这样啊」
 
。
 
我随意地点点头之后，一之濑小姐自嘲地笑了。
 
。
 
「就是这样。那个……其实想笑也不用忍的哦？我自己也知道的，高中辍学的家里蹲……就是典型的那种人……哈、哈哈」
 
。
 
「哈？不，我又不是想嘲笑你来着？」
 
。
 
「诶……？」
 
。
 
一之濑小姐经常会这样自虐，我还是清楚地告诉她比较好。
 
。
 
「我又不是因为学历才让一之濑小姐成为伙伴的吧？无论是否初中毕业的还是家里蹲，这些都是毫无关联的小事。最重要的只是一之濑小姐不顾自己的安危，救助了我和莫莫而已」
 
。
 
当时如果没有一之濑小姐的掩护，我必然会被高级猪头人杀死。
 
不对，在那之前她如果没有给我带来情报，我早就有可能被逼到绝境了。
 
。
 
「所以我是这么想的，这个人可以相信，想要让她成为伙伴。对吧，莫莫」
 
。
 
「汪」
 
。
 
从影子出来的莫莫也像是在说“对哟”一样点了点头。
 
然后直接爬上一之濑小姐的膝盖上，用身体磨蹭着她。
 
呜，感觉莫莫这家伙最近挺粘一之濑小姐的……。
 
羡慕嫉妒恨。
 
。
 
「……」
 
。
 
另一边，一之濑小姐一副茫然若失的样子。
 
啊咧？我有说什么奇怪的事情吗？
 
我倒是觉得说得挺不错的。
 
紧接着一之濑小姐嘀咕一句、
 
。
 
「……刚才好狡猾」
 
。
 
「哈？」
 
。
 
她倏然地别过脸。
 
。
 
「没什么，比起这件事，接下来我们要怎么办？」
 
。
 
「啊、啊啊，你说得对」
 
。
 
的确是这样。
 
不小心偏离主题了。
 
。
 
「总而言之，我觉得姑且离开这里」
 
。
 
如果可以我也想再收集情报，但考虑到那位会长的技能，还是不要久留比较好。
 
要是有『耐性技能』倒是另当别论。
 
。
 
「这样好吗？」
 
。
 
「嗯，而且―――」
 
。
 
我从楼梯的角落窥视着样子。
 
看到忙碌似的来回奔跑的学生们。
 
。
 
「……我不觉得随意乱走是好方法……」
 
。
 
这座学校比起预料中还要安定，毫无疑问是那位会长和学生会的成员所为。
 
倘若是借助『技能』控制意识，又或者是感情的话，这么安稳也可以理解。
 
虽说这些人还真是表里不一，但至少也像这样将避难民和学生组织起来了。
 
应该是为了避免无谓的争执吧。
 
虽说有点担心西野君他们会怎么样，不过对于他们来说具有讨伐魔物技能，拥有技能的人才应该是贵重的战力，应该不会乱来吧。
 
。
 
而且这次光是知道这里上层拥有这种技能就已经是充分的收获了。
 
得到一次不错的教训，在对于人类也不能大意的意义上。
 
我们只要以我们自己的步调筹备物资，踏实地提升等级，见机行事再来收集情报就好。
 
。
 
根本没有必要拘泥这里，还不如到其他避难所收集情报。
 
像这样将对象和地方都看透以后，再来决定是否要参加这种团伙就好。只要利用『邮件』，即便不用直接见面也可以在不暴露真实身份的情况下进行接触。
 
行事谨慎比较好，至少在当下这种世界里。
 
。
 
「总之我们接下来就要按照计划行事，边提高等级边筹备物资吧」
 
。
 
在来到这里之前，一之濑小姐说过的农业合作社的仓库和批发市场。
 
只要到这些地方就好，也是得到大量物资的机会。
 
这样一来，下次升级的时候就提高扩展功能吧。啊，如果可以做到还真想坐摩托车过去，就没有什么办法可以不要那么显眼吗。
 
我以这种样子思考着今后的预定，正想站起身的时候―――这瞬间。
 
。
 
『敌意感知』和『危机感知』发动了。
 
。
 
「……这是？」
 
。
 
有股『厌恶感』。
 
这是魔物的气息。
 
并且还是相当强烈，正逐渐接近这边。
 
地点是校门那边吗。
 
。
 
「唔ー……」
 
。
 
莫莫也察觉到了吧。
 
已经在有所戒备了。
 
。
 
「是魔物！是魔物出现了ー！」
 
。
 
听到了叫喊声。
 
校内引发动摇的气息。
 
以及啪嗒啪嗒的下楼声。
 
他们也似乎察觉到异状了。
 
。
 
「特征是什么！数量多少!?」
 
。
 
「是一只超大的像是狼的魔物！不过太强了！而且还拥有奇妙的能力，快点支援一下！」
 
。
 
「我知道了！」
 
。
 
超大的狼？能力？
 
。
 
「难不成……」
 
。
 
我看向一之濑小姐那边。
 
她也移开目光点了点头。
 
。
 
「莫莫」
 
。
 
「汪」
 
。
 
莫莫也迅速躲在『影子』里。背起一之濑小姐，我们赶到校门那边。
 
 
 
 
 
。
 
「绝对不要让它入侵！」「就在这里将它挡住！」「喂——，救援来了喔！」「敌人就只有一只！」「喂，别大意！」「大家一起解决掉它！」
 
。
 
我们在校舍的角落窥视着情形。
 
校门附近聚集着许多学生，战斗早已经开始了。
 
。
 
「咕噜噜噜噜……」
 
。
 
该说是正如预料吗，与他们战斗的魔物就是我们所认识的。
 
全是覆盖着黑色体毛的狼形魔物。
 
赤红的眼瞳闪耀着光芒，其脚边扩散出吞噬一切的『暗』。
 
。
 
不会错的。
 
那就是与我们战斗过的黑狼。
 
真没想到居然会这么快就见到面了。
 
可是……它为什么会在这种地方啊？
 
一一一一一一一一一一一一
