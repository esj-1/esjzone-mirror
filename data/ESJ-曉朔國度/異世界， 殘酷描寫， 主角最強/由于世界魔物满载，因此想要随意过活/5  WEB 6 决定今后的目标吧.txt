WEB 6 决定今后的目标吧
 
 
 
一一一一一一一一一
 
　那么，开始检验获得的技能吧。
 
　话是这么说，需要尝试的也就只有『肉体強化』和『道具包』。
 
。
 
　我的身体能力究竟提升到什么程度了呢ー。
 
　力量是「１３」，原本的数值是「３」。
 
　单纯有四倍多，比红还更快。
 
（颯:NETA估计是机动战士高达里某个红色有角的高达有着一般机体的三倍速度）
 
。
 
「啊，说起来……」
 
。
 
　我走向厨房。
 
　眼前有一台冰箱。
 
　这玩意超重的。
 
。
 
「嘿咻……」
 
。
 
　我抱住冰箱，缓慢地抱起来。
 
。
 
「喔喔……完全感觉不到重量」
 
。
 
　能够轻松地抱起来。
 
　咕，如果拥有这个技能，当初搬家的时候明明就可以更加轻松地搬东西。
 
　这台冰箱大到门外的走廊都只能勉强通过。
 
　啊，不过有装小脚轮喔，很厉害对吧。
 
　呜！水流下来了！好冰！
 
　我连忙从冰箱身上放开手。
 
　啊~湿透了。
 
。
 
　换衣服。
 
　走着，接下来是试着搬起家里家具中最重的洗衣机。
 
　这也很轻……倒也不是，不过可以搬起来。
 
　好厉害……明明以前需要两个人才可以搬得起来。
 
。
 
　接着是轻轻地跳一下。
 
　我的手轻而易举地就碰到天花板。
 
　真的假的。
 
　我得意忘形了，这次力气稍微大一点地跳起来。
 
　啵咚。
 
　跳起来。好痛。
 
　都撞出个洞了。
 
。
 
　再来是拿出勺子，用手握住。
 
　咕唏。
 
　喔喔……连超能力者都会吓一跳的握力……。
 
　我自己都感到害怕。
 
。
 
「嗯~身体能力感觉是全方面提升了啊，这还真厉害」
 
。
 
　啊哈哈哈，力量涌现出来了啊ー。
 
　之类的。
 
 
 
 
 
。
 
　那么身体能力已经确认完毕，接下来是检验『道具包』吧。
 
　『道具包』正如刚才所尝试的一样，能力是收纳手里持有的物品或者接触到的物品。
 
　正所谓是幻想世界的王道。
 
　与『鉴定』可以并称为双大。
 
　有没有这两项能力相当于在异世界是否是开挂。
 
。
 
　毕竟有些小说光靠这技能就能开后宫。
 
　是王道且必胜的技能，大概。
 
。
 
　于是试试看ＬＶ１究竟能收容多少东西吧。
 
　总之先依次收容吧。
 
。
 
　结果是替换用的衣服、内裤、手套、防寒用品、工口同人本、魔芋（处理完毕）、桌子、洗衣机、狗粮等，我所拥有的家具都收容到道具包里面。
 
　即便是ＬＶ１似乎也能收容相当多的东西。
 
　超方便的，这是神吗。
 
。
 
　完了，这还真有点开心的。
 
　虽然现在是处于不妙的紧急事态中，但这种幻想世界感真是欲罢不能。
 
。
 
「啊、如果有东西已收容的目录就好了……」
 
。
 
　东西实在是放太多了，我都记不清我放进什么东西了。
 
　事到如今，我才意识到这种理所当然的事情。
 
。
 
≪请问需要显示已收容道具的目录吗？≫
 
。
 
　我这么想着，脑海里就响起一道声音。
 
　诶？有目录吗？
 
　我在心中默念好的之后，眼前出现跟人物状态画面相同的窗口。
 
。
 
「喔喔，已收容的东西按照顺序显示出来了……」
 
。
 
　收容的顺序似乎是从上面开始的。
 
　并且还有纵向滚动条。
 
　当物品处于复数状态时，是以×数来表示。
 
。
 
　譬如说以塑料瓶来举例
 
　水５００ｍｌ　×３瓶
 
　茶２ℓ　　 ×１瓶
 
。
 
　显示的就是这样子。
 
　分类也十分清楚。
 
　超有能，神吗这是。
 
。
 
「这样就不用担心我会忘记放进什么了」
 
。
 
　哎呀，真厉害啊，道具包。
 
　我能理解在异世界为什么这玩意会是王道了。
 
。
 
　顺便一提，为了知道保存效果怎么样。我试着将烧水倒进水筒里并收纳进去。
 
　经过一段时间再拿出水筒看看。
 
　温度很普通地变温了。
 
。
 
　嗯~没有保温功能或者时间停止的功能吗？
 
　不对，毕竟才ＬＶ１，可能提升等级以后功能会更强大也说不定。
 
　无论如何，这都是十分方便的技能。
 
。
 
　某种程度的存取之后，在运用上似乎更加得心应手了。如果我想要让东西出现在手中，心中这样默念就会实现这种效。
 
　而在无法出现于手中的情况下，则是会出现在眼前。
 
。
 
　其他还有当进行收容时，如果是自己的所有物，即便隔着某种程度的距离也可以成功实现。
 
　虽然具体哪方面才会认同为『所有物』这点有些微妙，只能多次尝试之后就会知道了吧。
 
　顺便一提，能够收容的范围为一公尺左右。
 
　技能等级提高，应该会扩大收容范围吧。
 
 
 
 
 
。
 
　那么，技能验证基本上就结束了。
 
　接下来要怎么办呢。
 
。
 
　总之先吃完饭睡午觉吧？
 
　不不不，逃避现实是要闹哪样。
 
。
 
　现在外头还能听到魔物的叫声，以及不知是什么人的悲鸣声。
 
　也许是技能『聆听』的影响，耳力比起以前还要好，现在能够清楚地听见那些声音。好害怕啊。
 
。
 
　嗯~，怎么办呢。
 
　是应该要出门吗，还是闭门不出呢。
 
。
 
「可是就算不出门，食物也是问题啊……」
 
。
 
　冰箱里面没剩下多少食物了。
 
　几片面包和鸡蛋，以及蔬菜和几种水果，再来是纳豆、肉类、咸菜。
 
　还有就是几碗方便面和为数不多的下酒菜。
 
　水龙头虽然还有水可以用，可一旦连水都停了就无计可施了。
 
。
 
「啊，对了。总之先将水装进空塑料瓶吧」
 
。
 
 
 
 
 
我想在资源回收日那天将所有东西都一次性扔掉，因此留有将近几十个空塑料瓶。
 
　洗干净的话应该还能用吧。
 
　把水都装进去，收容到道具包里面。
 
　我得提前确保最低限度的饮用水，以防万一。
 
　收容了12个两升的塑料瓶。
 
。
 
「调味料和平底锅也收容进去吧，还有米也是」
 
。
 
　就算没有电饭煲，只要利用瓦锅和饭盒也能行吧。
 
　于是大米也收容进去。
 
。
 
「还有厨具也放进去吧」
 
。
 
　柜橱里所有的厨具也全部收容了。
 
　ＬＶ1能放进这么多东西，而且价格居然只要1个点数。
 
　来来来，大家现在也立刻入手一个道具包吧！
 
 
 
 
 
。
 
 
 
 
 
「嗯~，果然还是应该出门吧……」
 
。
 
　虽然也可以选择继续呆在家里，可如果那些猪头人冲进来就无处可逃了。
 
　更重要的是食物很少。
 
　能维持几天，可如果这段期间里周遭的魔物更加多的话就完蛋了。
 
　而且停电实在是麻烦。
 
　在家里面没有任何收集情报的手段。
 
。
 
　即便多少有些风险，还是出门到外面找人，或者收集情报才是上上策吧。
 
　一旦发生什么事，最糟糕的情况下我只要逃跑就行。
 
　我所得到的技能能力都是货真价实。
 
　而且跟魔物相比之下，地形是我这边有利。
 
　只要贯彻逃跑的念头，应该是不会死翘翘才对。
 
。
 
「再来就是有什么可以作为武器的吗……」
 
。
 
　话是这么说ー。
 
　又没有这种时候该有的棒球棒或者那些难以启齿的铁棒之类的。
 
。
 
「……果然只有这个了吗」
 
。
 
　我手里的是做饭时使用的菜刀。
 
　只有这玩意能当做武器来用了。
 
　哎，没有的东西再强求也是无济于事。
 
　在惊心动魄的场面里可是王道凶器，杀伤力也十分优秀。
 
　葬送了无以计数的花心大萝卜，在某种意义上也是适合杀人的武器。
 
　而且我的武器并不单单只有菜刀。
 
。
 
「嗯~可是单手拿着菜刀到处乱悠悠，在别人看来完全就是可疑人物啊……」
 
。
 
　说了也是没办法吗。
 
　我单手拿着菜刀，打开玄关的大门。
 
。
 
　好，终于要前往外面的世界。
 
一一一一一一一一一
