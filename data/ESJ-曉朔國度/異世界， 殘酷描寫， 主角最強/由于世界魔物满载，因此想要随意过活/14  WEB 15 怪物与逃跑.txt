WEB 15 怪物与逃跑
 
 
一一一一一一一一一
 
 
 
 
 
　―――老实说，好麻烦。
 
。
 
　想到这件事时，我意识到自己意外很没用、
 
　不过这是那么不好的事情吗？
 
　无论是谁都是最珍惜自己的性命吧？
 
　在这样的世界里。
 
　比起别人更优先自己。
 
　思考自己能存活下去的方法。
 
。
 
　这是『错误』的吗？
 
。
 
　什么错也没有。
 
　应该是没错的才对。
 
。
 
　正因为如此，我才不能理解那些人的观念。
 
　障碍物被攻破，拼命战斗的他们。
 
　为什么他们不逃跑，而是这么拼命地战斗到底？
 
。
 
　很简单。
 
　恐怕他们是在争取时间。
 
　争取让其他人――也就是同样呆在购物中心里面的人们逃跑的时间。
 
 
 
 
 
「绝对不能让它们通过这里！我们要尽可能为大家争取时间！」
 
「「「喔喔喔喔喔喔喔‼」」」
 
。
 
 
 
 
 
　像是验证我的猜测，战斗中的男人大喊道，鼓舞着自己人。
 
　没有错。
 
　他们正如字面意思上赌上性命争取时间。
 
　尽可能争取让其他人逃跑的时间。
 
。
 
「……真的假的。这种事还真做得出来……」
 
。
 
　我绝对不会像他们这么做……。
 
　为什么那些人要这么拼命？
 
　是因为自己的家人和恋人在里面吗？
 
　嗯~虽然即便是这样，比起他人我也会优先自己就是了。
 
。
 
「果然还是逃跑吧……」
 
。
 
　不行了。
 
　看样子我不擅长应付那些人。
 
　天空也是乌云遍布，总觉得似乎要下雨了。还是趁早离开吧。
 
。
 
「くぅーん？」
 
。
 
　被我抱在怀里的莫莫发出声音。
 
　像是在问「这样好吗?」。
 
 
 
 
 
「……啊啊，别管那些人。莫莫……你反对吗？」
 
。
 
「…………汪」
 
。
 
　隔了片刻，莫莫回答道。
 
　似乎要我自己决定。
 
　莫莫同意了，我有点放心了。
 
。
 
「我知道了。那么莫莫，你配合我的信号解开『影子』。与此同时我也会解除『潜伏』。然后一口气逃跑吧」
 
。
 
「汪！」
 
。
 
　莫莫赞同我的提议，我正想解除『潜伏』。
 
　骤然间―――战况发生了变化。
 
。
 
「嗯？」
 
。
 
　呆在它们背后的那头高级猪头人开始行动了。
 
　男人们的脸上浮现出紧张神色。
 
。
 
　它想做什么？
 
　高级猪头人往前走了几步，慢慢地吸气。
 
　这瞬间，我全身感到不寒而栗。
 
。
 
　―――！！完了！！有什么要来了！
 
　某种极其不妙的『什么』！
 
。
 
　证据就是其他猪头人停止战斗，与男人们拉开距离。
 
　并且像是在防备什么一般，双手交叉于眼前。
 
　男人愕然地看着猪头人它们的古怪行动。
 
。
 
「莫莫！将『影子』缠绕在我全身!现在立刻！」
 
。
 
　莫莫也在本能上察觉到危险了吗，在我发出指示的同时就已经操控『影子』了。
 
　莫莫『影子』覆盖我的全身。
 
　在这种状态中，我将莫莫藏在我身下并当场趴了下来。
 
。
 
　下个瞬间，高级猪头人『咆哮』了。
 
。
 
『喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔！！！！』
 
。
 
「～～～～～！！」
 
。
 
　那道咆哮声化为物理性的破坏波及到四周。
 
　嘎吱嘎吱，大气都为之颤抖。
 
　地面惨遭拨开、粉碎，裸露的地面迸发出龟裂。
 
　购物中心以及周围建筑物的玻璃全数粉碎得完无体肤，纷纷掉落在地面上。
 
。
 
　仅仅几秒钟。
 
　那令人感到是永远的短暂咆哮给周围带来莫大的灾害。
 
。
 
「―――！哈啊、哈啊、哈啊……莫莫，你没事吧？」
 
。
 
「……くぅーん……」
 
。
 
　虽然声音微弱细小，但还是有回应。
 
　什么鬼，刚才的『叫声』是……？
 
。
 
　有那种无视物理法则的叫声吗？
 
　不对，那估计也是『技能』。
 
　虽然不知道技能名称，但应该是能带来广范围破坏效果的『叫声』技能。
 
　而使用技能的就是那头高级猪头人。
 
。
 
　我稍微探出身子，看着前方。
 
。
 
「什……!?」
 
。
 
　入口附近战斗的人们都倒在地面上，全身都流出血。
 
　死了。
 
　那毫无疑问是死了。
 
。
 
　高级猪头人举起手。
 
　以此为信号，其他猪头人都纷纷闯进购物中心。
 
　应该是要狩猎里面的人类吧。
 
。
 
　看到其他猪头人走进购物中心，高级猪头人也迈开步伐。
 
　然而，不经意间它停下脚步。
 
　我想着怎么了―――它的目光正看着『我这边』。
 
。
 
「！！！」
 
。
 
　目光相对。
 
　明明这里离那家伙少说也有几十米。
 
　明明我已经使用『潜伏』技能了。
 
。
 
　然而那家伙的视线正对着我这边。
 
。
 
　接着那家伙―――嗤笑了。
 
　看到我，它打从心底感到高兴般嗤笑了。
 
。
 
　糟糕、糟糕糟糕糟糕、糟糕糟糕糟糕！
 
。
 
「哈啊哈啊哈啊哈啊哈啊」
 
。
 
　心脏激烈地抖动着，仿佛都要震裂一般。
 
　呼吸也紊乱。
 
。
 
　死。
 
　这一字浮现在脑海里。
 
。

　脑海里响起许多通知声。
 
　然而我现在根本没有余力去听这些。
 
。
 
　我不想死。
 
　我不想死。
 
　只有这件事占据了我的大脑。
 
。
 
　下雨了。
 
　雨水滴在我的脸上。
 
。
 
「噫！」
 
。
 
　一瞬间我以为是那头高级猪头人的攻击，因而发出没出息的叫声。
 
　而高级猪头人看到我这幅样子，用鼻子嗤笑以后就移开目光了。
 
　于是它就消失到购物中心里面。
 
。
 
　太好了。
 
　看样子是放过我一马了。
 
。
 
　我和莫莫立刻解除『潜伏』和『影子』。
 
　接着立马当场逃跑。
 
。
 
　不妙。
 
　那家伙很不妙。
 
。
 
　逃跑了。
 
　全力逃跑了。
 
　在持续不断的雨中，尽可能地跑得远远的。
 
　跑到那头高级猪头人看不到的地方。
 
　那家伙不一样。跟其他魔物相比显然『不同』。
 
。
 
「哈啊…哈啊……！」
 
。
 
　不行。
 
　不能再这样继续下去。
 
。
 
　会死，迟早绝对会死。
 
。
 
　我不想死。
 
　我不想死。
 
。
 
「可恶……可恶……！」
 
。
 
　必须要变得更强。
 
　跟莫莫一起。
 
　变得比起现在还要强。
 
。
 
　为了存活下去。
