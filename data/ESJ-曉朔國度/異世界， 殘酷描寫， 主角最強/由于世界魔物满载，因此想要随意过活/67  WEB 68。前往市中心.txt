翻译：飒君CONAN
 
 
 
WEB 68.前往市中心
 
 
一一一一一一一一一一一一一
 
 
 
 
 
接下来，我们就前往市中心吧。
 
话是这么说，也不是走过去。
 
这样太花时间了。
 
。
 
「我们就坐这玩意过去吧」
 
。
 
我从道具包里面取出『物品』。
 
。
 
「这是……摩托车吗？」
 
。
 
「是啊，摩托车灵活性较强，速度也很快」
 
。
 
虽然还有其他已回收的汽车和摩托车，不过还是选择有钥匙插在上面并且损伤较少，没有爆胎的车辆。
 
虽然这辆摩托车多少是有点损坏，不过用来驶行应该是没问题的吧。
 
诶、盗品？事到如今还要说什么。
 
。
 
「可是这样……真的没问题吗？」
 
。
 
「没有问题」
 
。
 
我知道一之濑小姐的担忧。
 
她应该是在担心摩托车的『声音』是否会吸引魔物过来吧。
 
不过没事，在这一点上没有问题。
 
我启动摩托车的引擎。
 
。
 
「诶？」
 
。
 
正如我所预料，一之濑小姐惊讶起来。
 
毕竟摩托车的『声音』小到惊人。
 
。
 
「虽然我也是在刚刚才注意到就是了呢。技能『无声动作』似乎也对我所触碰的物体产生效果」
 
。
 
我是在刚才利用小红所拟态的菜刀砍着家具的时候察觉到的。
 
起初我只是觉得小红的性能真不错，可在我持续不断的尝试过程中就感觉到了违和感，注意到这是其他技能效果。
 
转职为『忍者』时已经将技能都全部检验完毕了，可这点倒是完全遗漏了。虽然也是因为道具包、『忍术』、『投掷』主要是用来攻击，所以很难注意到就是了。
 
顺便一提，在触碰的物体上产生的效果比起我自己还要弱上一些。
 
就像现在这样，声音低到十分安静的程度。
 
。
 
不过这种优点很不错。
 
如果与技能同时运用，既能让魔物难以察觉到我们，又可以节省体力。
 
并且即便魔物察觉到了，除非是移动速度较快的魔物，否则还是可以直接逃离。
 
至少以影狼的速度，我们是可以甩开的。
 
汽油还有很多，在燃料方面也没有任何问题。
 
。
 
「不是，我想说的不是这件事……」
 
。
 
「啊啊，难不成你想说的是驾驶技术吗？这点也没问题喔」
 
。
 
毕竟我已经利用刚才升级得到的ＳＰ学习新的技能『骑乘』。
 
『骑乘』的效果似乎也适用于驾驶技术，即便是摩托车也能充分发挥出技能的能力。虽然不会合体就是了。
 
。
 
顺便一提，刚才升级得到的点数都已经用光了。
 
ＪＰ那边是将『忍者』和『影法师』提升到ＬＶ５。
 
ＳＰ那边则是学习新技能『骑乘』、『交涉术』。并且将『骑乘』提升到ＬＶ２，、接着是『忍术』到ＬＶ６，『无味』和『无声动作』提升到ＬＶ５。
 
这样就能应付速度较慢的敌人，以及嗅觉优秀的敌人……我是这样觉得。
 
还有『交涉术』的话，你想嘛……可能性不是很重要吗？
 
。
 
我骑在摩托车上确认着机能。
 
嗯……没问题，这样应该可以开。
 
。
 
「小红，你分裂一下『拟态』出安全头盔给我。莫莫就躲到『影子』里」
 
。
 
「汪」
 
「……（抖抖）」
 
。
 
这样容貌就不会暴露了。
 
莫莫也躲在影子里，不用担心它会落下。
 
我将小红所拟态的另一个头盔丢给一之濑小姐。
 
。
 
「那么一之濑小姐，我们出发吧」
 
。
 
「…………」
 
。
 
然而一之濑小姐拿着头盔僵硬不动。
 
究竟是怎么了？
 
。
 
「啊，该不会对于坐在男性后面果然还是抱有抵触感吧？」
 
。
 
虽然这种事就只能让她忍耐一下了……。
 
我会尽可能不会意识到背后的触感。
 
。
 
「不对，不是这样的……」
 
。
 
一之濑小姐很尴尬地移开目光。
 
。
 
「……我一坐车就会立马呕吐出来」
 
。
 
「……」
 
。
 
是这样啊……。
 
这种事倒是无能为力了啊ー……。
 
。
 
「真的很对不起……」
 
。
 
一之濑小姐似乎感到十分抱歉似的低下头。
 
 
 
 
 
「没事，既然是这样也是没办法的事情啦」
 
。
 
硬是让她上车，结果边走边吐也太不妥的。
 
还有魔物会通过气味追踪上来的危险性。
 
。
 
「……（抖抖）」
 
。
 
随即一之濑小姐手中拟态为头盔的小红抖动着。
 
只要戴上自己就没问题，似乎是在这样说。
 
小红……你该不会是想要将一之濑小姐的呕吐物都全部解决掉吧？
 
。
 
「……（抖抖）」
 
。
 
YES，似乎是在这样说。
 
嘛，对于小红来说，呕吐物仅仅是食物罢了。
 
。
 
『汪』
 
。
 
并且就连躲在影子里的莫莫也利用『影子』变出疑似安全带的物体。
 
利用这些固定住一之濑小姐就没有问题了，似乎在这样说。
 
这些小家伙，该不会是把一之濑小姐当做是搬家时的物件了吧？
 
不，我希望这是它们纯属的善意行动。
 
。
 
「……一之濑小姐，莫莫和小红都做到这份上了，你能不能稍微努力一下呢？」
 
。
 
我说完以后，一之濑小姐犹豫着犹豫到最终才点了点头。
 
。
 
「………………好的」
 
。
 
那是非常微弱，并且毫无自信的声音。
 
不过我得到她的同意了。
 
。
 
很好，我们出发吧。
 
将一之濑小姐固定在后座之后，我们出发前往市中心。
 
。
 
后来应该是晕车实在太痛苦了，在途中一之濑小姐似乎得到了技能『晕车耐性』。应该可以说是结果不错吧。
 
顺便一提，呕吐物全部让小红处理掉了。
 
真是十分值得依靠的史莱姆。
 
。
 
。
 
。
 
 
 
 
 
随后过了几十分钟。
 
我们终于来到了市中心。
 
将摩托车停在杳无人迹、尽可能不是很显眼的地方。
 
。
 
「姑且就在这里下车吧」
 
。
 
「……好滴……」
 
。
 
即便戴着头盔也能立刻知道一之濑小姐的脸色很苍白。
 
她摇摇晃晃地从摩托车上下来。
 
即便如此也依然牢牢扛着枪支，真不愧是她。
 
确认到一之濑小姐已经下车，我将摩托车放进道具包里。
 
。
 
「……然后呢，我们首先要到哪里？」
 
。
 
「说的也是……我们就到这附近的超市和便利店，将物资都全部回收吧。还有这一带应该有着高中学校，就在那边收集情报吧」
 
。
 
虽然是偷偷打听。
 
依靠我和一之濑小姐的技能与职业，应该可以在建筑里顺利地行动。
 
不如说一之濑小姐反而在建筑物里面才能发挥出本领。
 
。
 
「高中学校……」
 
。
 
「是啊，就是我的母校。一之濑小姐是―――啊」
 
。
 
说完以后我才注意到自己说错话了。
 
她是家里蹲，并且说过是高中辍学。
 
应该不怎么愿意回想起来吧。
 
。
 
「啊，不用顾虑到我。我也是在那边的高中就读的。嘛，虽然只有讨厌的回忆就是了……」
 
。
 
一之濑小姐的神色上出现阴霾。
 
。
 
「学校吗……一定是遭受魔物的袭击而变得破烂不堪了吧，感觉这样也挺不错的……不如说我反倒是希望能毁掉呢。毁掉就好了，那种地方。毁掉吧，毁掉吧、给我毁掉吧。我讨厌学校，最讨厌学校了，也讨厌那些从来没有过来帮助我的同班同学。大家要是跟学校一起被毁掉就好了呢，呵呵呵呵呵呵，啊哈哈哈哈哈哈」
 
。
 
「你、你冷静一下，一之濑小姐!」
 
。
 
我摇了摇她肩膀，她才终于恢复正常。
 
。
 
「啊咧……？对不起，我好像又埋头想着事情了……」
 
。
 
「啊，不，没事的……」
 
。
 
不对，虽然完全不是没事就是了。
 
这孩子果然有点神经病。
 
好深的负面情绪……。
 
。
 
「那个，如果不喜欢还是到其他的地方吧？」
 
。
 
这种会刺激到心理阴影的地方，根本没必要特意过去也行吧。
 
我只是因为比较近才想过去，并非是其他理由。
 
。
 
「没有问题」
 
。
 
然而一之濑小姐干脆地拒绝我的提议。
 
。
 
「不，可是―――」
 
。
 
「没有问题」
 
。
 
「啊，好的」
 
。
 
固执到底。
 
。
 
「而且……有一个人」
 
。
 
「诶？」
 
。
 
「……我有一位关系不错的朋友，如果还没死就好了……」
 
。
 
「是这样啊」
 
。
 
关系不错的朋友吗。
 
我就没有这种朋友啊。
 
在这方面还真有点羡慕。
 
（颯:社畜表示学生时代的同学早忘得一干二净，联系方式?不存在的）
 
。
 
「要是那人也还活着就好了」
 
。
 
「……是的」
 
。
 
我真心地说出口后，一之濑小姐虽然依然没有正面相对，但也浮现出些微笑。
 
如此这般，我们解决掉物资收集问题后就决定前往学校。
 
一一一一一一一一一一一一
