WEB 42 进化
 
 
一一一一一一一一一一
 
 
 
 
 
「汪！」
 
。
 
　影子中的莫莫精神奕奕地飞奔而来。
 
　噢噢，莫莫！你究竟是变成―――啊咧？
 
　……没怎么变化？
 
。
 
「……莫莫？」
 
。
 
「汪呜？」
 
。
 
　我仔细端详着莫名的样子。
 
　它的样子就跟影子包裹住之前别无二致。
 
　啊咧？这是怎么回事？
 
　我还以为等级达到上限就会『进化』了呢……？
 
。
 
　不对，仔细一看似乎多了几分『可怕度』？
 
　……是我的错觉吧。
 
　还是平时可爱的莫莫。
 
　摸摸。
 
　嗯，不错的毛茸感。
 
　莫莫也似乎很舒服似的眯起眼睛。
 
。
 
「莫莫，你有没有哪里觉得奇怪?比如说感到痛苦或者奇怪的感觉之类的？」
 
。
 
「汪！」
 
。
 
　莫莫像是在说「没事」，精神奕奕地摇着尾巴。
 
　看样子似乎没有问题……。
 
。
 
「嗯~……」
 
。
 
　啊，对了。
 
　只要打开人物状态面板，确认一下不就好了吗。
 
　迅速打开，看看项目上的队伍成员。
 
。
 
莫莫
 
暗杀犬ＬＶ１
 
。
 
　如此显示着。
 
　…………暗杀犬？
 
　喂，这名字总觉得好吓人啊。
 
　既然都有警犬和导盲犬，那有暗杀犬也不奇怪……吧？
 
　不对，这也太奇怪了吧。
 
。
 
　嘛，如果考虑到莫莫和我的战斗风格，那倒是可以接受。
 
　利用『影子』夺取对象的自由和视觉，趁机接近并捅死。
 
　嗯，这的确是暗杀啊。
 
。
 
　令人信服的进化。
 
　不对……不如说这算是进化吗？
 
　真要说的话，感觉类似于我转职为上级职业。
 
　我还以为肯定会『进化』成类似于那只影狼一样的感觉。
 
　我想过如果那样子的话，莫莫不就变成魔物的样子吗。
 
　不过从结果来说还是毫无变化，太好了。
 
　如果依然是原本的姿态，还是可以如同往常一样随意行动。
 
　更重要的是这样比较可爱，这很重要，是最重要的事情。
 
。
 
「话说回来……」
 
。
 
　看着莫莫，我突然想到一件事。
 
　狗主人和狗的职业都是『暗杀者』『暗杀犬』，这样会怎么样……？
 
　不会太吓人了？
 
　如果被其他人看到，我有百分百的自信被人误解就是了。
 
。
 
　嗯，绝对隐瞒不过的。
 
　要是暴露了，在各种意义上都很不妙。
 
　我可没有自信可以跟别人好好解释。
 
。
 
　结论，就跟以前一样继续行动吧。
 
。
 
　嗯，这样就好。
 
　……总觉得我似乎是顺利走上单人（孤独）的道路了？
 
　这应该是我的错觉吧？
 
。
 
「咕？」
 
「……（抖抖）」
 
。
 
　哎呀，在考虑事情，莫莫它们都担心地看着我了。
 
　抱歉抱歉，我没事的啦。
 
　嗯，没错。
 
　我还有莫莫和小红。
 
　绝不是单人……才对。
 
。
 
　而且目前我没打算要跟别人打交道。
 
　虽然想要远距离类型的伙伴，但终究只是觉得有的话就好。
 
　并不是说我有这么渴望孤独。
 
。
 
　即便要和他人打交道，我也希望会是事务性的关系。
 
　双方都有利益，不会过多干涉彼此。
 
　这样的关系是最好的。
 
　嘛，虽然现实中是不可能有这么好的事情吧……。
 
。
 
「总之先离开吧……」
 
。
 
　我还想确认一下莫莫进化过后的能力以及技能相关。
 
　在这里说不定还会被人看到。
 
　为了以防万一，我们离开了此处。
 
 
 
 
 
。
 
 
 
 
 
　后来随便来到一家没人的房屋里，确认一下莫莫的新技能相关。
 
　我可不会做出一来就在实战中进行验证这种蠢事。
 
　一定要事先确认完毕。
 
。
 
　结果就是莫莫得到跟我相同的技能。
 
。
 
　『气息消除』、『無无声移动』等技能，以及看透对象要害的技能。
 
　而且操控『影子』的技能能力也更上一层楼。
 
　效果范围更加广泛，影子的束缚力和攻击力也增强了。
 
　还有身体能力也提高许多。
 
　大概就跟以前战斗过的影狼相同甚至更高。
 
　不对，在反应和动作的灵敏度方面完全是超越影狼了。
 
　总的来说，现在就是以前莫莫的加强版。
 
。
 
　验证也结束了，接下来就提升等级并顺便测试一下。
 
。
 
「哦，刚刚好啊……」
 
。
 
　『索敌』有反应了。
 
　几只哥布林。
 
　而且也有大型哥布林。
 
　作为测试莫莫的实力来说，或许是刚刚好的对手。
 
。
 
「上吧，莫莫，小红」
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
　往哥布林所在的地方过去。
 
　它们似乎没有察觉到我们。
 
　在接近到极限的时候，我向莫莫发出信号。
 
。
 
　莫名点着头，释放出『影子』将哥布林它们束缚住。
 
　释放『影子』的速度以及缠绕住对象的所需时间。
 
　无论哪一项都是以往的莫莫所无法相提。
 
。
 
　轻而易举地就将那些哥布林给无力化了。
 
　莫莫进化过后，『影子』拘束力强大到连那只大型哥布林都甚至无法解开。
 
　虽然它拼命抵抗，但解不开影子。
 
。
 
「汪！」
 
。
 
　而且光是这样还没有结束。
 
　莫莫更加强着影子的束缚力，哥布林它们直接就被绞杀了。
 
　哥布林它们的头部和身体都折向奇怪的方向，气绝身亡，掉落魔石。
 
　好强。哥布林这种程度的魔物，即便是团体也似乎已经不是莫莫的对手了。
 
。
 
　唯一存活着的大型哥布林也被我一击致命。
 
　哥布林和大型哥布林的要害位置是一样的。
 
　弱点也是一样。
 
　捅进去转动一下刀柄。
 
 
 
 
 
大型哥布林就轻易死亡了。
 
。
 
≪获得了经验值た≫
 
≪经验值达到一定的水平≫
 
≪工藤和人从ＬＶ１１升级到ＬＶ１２了≫
 
。
 
　天之声告知我升级了。
 
　跟昨天在超市战斗时截然不同啊、
 
。
 
　有种顺利变强的感觉。
 
　不过不能骄傲。
 
　不如说正因为自己的等级提升了，所以才更加清楚我跟『那家伙』之间的实力差距。
 
。
 
　―――还不够强。
 
。
 
　会让我这么觉得。
 
　而且我莫名感到不安。
 
　再稍微升点级之后，还是赶快离开这座城市说不定比较好。
 
 
 
 
 
。
 
 
 
 
 
　另一方，这时候在购物中心―――。
 
。
 
　那头魔物正感到无聊。
 
　没有乐趣，也觉得索然无味。
 
。
 
　他踩着躺在地面上的尸体，边思考着。
 
　究竟要到什么时候才会出现能够排除这份无聊的猎物？
 
。
 
　无论怎么等待，总是出现一些毫无半点感觉的杂鱼。
 
　都是一群见到自己的身姿就会感到胆怯、瘫软、迷茫着是否要逃跑的软脚虾。
 
。
 
　他所期望的并非是蹂躏。
 
　而是战斗。
 
　战斗才能满足自己的本能，它是神圣的仪式，能够给自己带来活着的充实感。
 
　与此对应的对手。
 
　究竟要到什么时候才出现？
 
。
 
　虽然对于从空中出现的人抱有些期待，但期待落空了。
 
　像刚才出现的人们，即便是**也有不错的地方。
 
。
 
　十分烦躁的他，将拳头揍向大地。
 
　同胞们都在惧怕。
 
　他知道的。
 
　他已经对弱小的同胞感到失望。
 
。
 
　啊啊，已经不行了。
 
　不会再等了。
 
　已经忍耐到极限了。
 
。
 
　既然不过来这边，那就亲自去找吧。
 
　为了找到能够满足这份渴望的相应猎物、相应的对手而出发吧。
 
。
 
　他大叫着。
 
　其叫声震撼大地，撕裂大气。
 
。
 
　他行动了。
 
。
 
　来吧―――如同沉睡的时间已经到此结束了。
