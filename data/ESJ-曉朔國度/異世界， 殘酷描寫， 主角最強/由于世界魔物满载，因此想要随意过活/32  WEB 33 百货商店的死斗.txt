WEB 33 百货商店的死斗
 
 
一一一一一一一一一一
 
 
 
 
 
「唔噢噢噢噢噢噢噢噢噢噢!!!」
 
（颯:老早想吐槽，不应该是嗷呜呜之类的吗）
 
。
 
　影狼吠叫着。
 
　大气嘎吱作响，其他的魔物也都在惧怕。
 
　好可怕的魄力。
 
　不过我还是能承受得住。跟『那家伙』的咆哮比起来，这还算可爱的。
 
。
 
「莫莫！它要来了！」
 
。
 
「汪！」
 
。
 
　影狼从脚边释放出『影子』。
 
　『影子』分切成无数条，犹如鞭子一般袭向我们。
 
　果然可以使用跟莫莫一样的技能吗。……不对，对方才是真货吧。
 
 
 
 
 
「啧！」
 
。
 
　我从道具包取出物品。
 
　即时制作的墙壁。
 
　『影子』缠绕在衣柜身上，紧紧绷住将其破坏掉。
 
　衣柜的碎渣飞舞于空中。
 
　这瞬间，影狼已然飞奔而来。
 
。
 
「！―――好快啊，喂！」
 
。
 
　我和莫莫往左右躲闪。
 
　虽然在『索敌』就已经知道影狼的速度，但远比我想象中还要更快。
 
　勉强躲开攻击。
 
。
 
「嘎呜！」
 
。
 
　然而影狼直接扭过身子，往我这边冲过来。
 
。
 
「什!?」
 
。
 
　不光是速度！
 
　这家伙的动作『灵敏度』根本不是哥布林和小狼可以匹敌的！
 
　犬类生物特有的轻巧行动，犹如弹簧一般的动作。
 
　有如子弹一样的速度，影狼的獠牙正在迫近！
 
　『危机感知』最大限度地敲响警铃。
 
　障碍物―――不对，不行了，来不及！
 
。
 
「呶噢噢噢噢噢噢噢！」
 
。
 
　我硬是利用达到三位数的敏捷数值，扭过身体避开攻击。
 
　影狼的獠牙穿过我刚才所在的地方。
 
　好危险。
 
　只要晚了一步避开，手腕就会被咬断了。
 
。
 
「汪！」
 
。
 
　在我躲开的瞬间，莫莫操控我的『影子』想要束缚住影狼。
 
。
 
「嗷呜!!」
 
。
 
　然而影狼仿佛在诉说『别小看我！』一样，利用自己的影子抵消莫莫的『影子』。真的假的，这种距离、这种速度还能抵挡下来？
 
。
 
　不过它的动作停顿了一瞬间，这是机会。
 
　我朝着影狼头上释放出自动贩卖机。
 
　这样就解决！
 
。
 
「―――噢噢噢噢噢噢!!」
 
。
 
　影狼吠叫着。
 
　于是在它的脚边更迸发出无数的影子。
 
　影子变化成蜘蛛网，将自动贩卖机接住了，使其停止在空中。
 
。
 
「什……!?」
 
。
 
　喂，这是在开玩笑吧？
 
　那架自动贩卖机究竟有几百公斤啊？
 
　这是我和莫莫葬送多数魔物的必胜战术。
 
　然而完全被它破解了。
 
　掉落在蜘蛛网上的自动贩卖机发出咔嚓声响，逐渐被压瘪。
 
　……要是被那『影子』抓到就完蛋了。
 
　根本就逃不掉，而且还会直接被勒死吧。
 
。
 
　不妙……。这家伙远比预料中的还要强得多。
 
。
 
　从『危机感知』的反应我还以为比起大型哥布林还要强一些，但不是这样。比想象中还要强。
 
。
 
　不过还是有胜算。
 
。
 
　从当初撞死的魔物以及小狼可以看出这些家伙的『防御』很低。
 
　一旦攻击命中，应该就可以瞬间给与伤害。
 
。
 
「问题是攻击要怎么打到它……」
 
。
 
　棘手的是它的速度以及『影子』。
 
　如果不对这两点想点办法，根本很难打到它。
 
　正当我思考着该怎么办时，『敌意感知』又显示出新的反应。
 
。
 
「咕噜噜噜……！」
 
「呜呜呜呜呜」
 
「呜噢！」
 
。
 
　又出现新的小狼。
 
　数量是五只。
 
　所以说为什么你们总要过来这边啊！
 
　给我去那边啊！那边去！百货商店那边！
 
。
 
　我往百货商店那边撇了一眼。
 
　学生们还在与魔物战斗中。
 
　依然没有注意到这边吗？
 
　明明应该听得到刚才影狼的远嗥才对，即便一片漆黑看不到什么，但应该会意识到除了自己人以外―――还有『谁』在这里战斗中。
 
　如果可以，我也想尽量避免显眼的行动……。
 
。
 
「可怎么说都没办法吗……」
 
。
 
　如果不认真起来跟它们战斗，这边就死定了。
 
　它们就是这么强大的对手。
 
　达到ＬＶ１０的道具包。
 
　也该是让我见识一下能力了吧。
 
。
 
「呜喔喔喔喔喔喔!!」
 
。
 
　影狼边吠叫，边猛然冲过来。
 
　小狼们也紧跟其后。
 
　我再次朝着它们头上释放洗衣机。
 
。
 
「嘎呜!!」
 
。
 
　然而躲开了。
 
　我连续不断地释放沉甸甸的家电。
 
　虽然是打中小狼了，但并没有命中影狼。
 
　我明明是在头上极限的距离下释放家电的，但它利用『影子』停止家电的坠落，趁机逃了出来。
 
　……面积不够吗，既然如此―――。
 
。
 
≪获得了经验值≫
 
。
 
　哦，打倒小狼的经验值吗。
 
　頭の中に声が響く。
 
。
 
「这边！」
 
。
 
　我边朝着那家伙释放家电阻碍它的行动，并且边跑向停车场里面。
 
　我将眼睛看到的『东西』都全部收纳进道具包。
 
　虽然影狼速度很快，但只要利用道具包和莫莫的『影子』妨碍它，还是可以勉强拉开距离。
 
　好，数量已经足够了。
 
　剩下的就是选好地点。
 
　比起这里，还是对面那边比较好。
 
。
 
「莫莫！到这边！」
 
。
 
「汪！」
 
。
 
　我与莫莫汇合，离开停车场来到大马路上。
 
　影狼也追着我们离开停车场。
 
　十分宽敞的公路。
 
　这里没有障碍物，因此双方的身姿都能清楚看见。
 
　而我和莫莫则是站在公路中间。
 
。
 
「咕噜噜噜……」
 
 
 
 
 
。
 
「咕噜噜噜……」
 
。
 
　影狼应该是在戒备陷阱吧，与我们保持距离并瞪视着。
 
　不过抱歉了。
 
　当你被诱引到这里时，你就已经输定了喔。
 
。
 
「接招吧―――！」
 
。
 
　我朝着那家伙释放出我的王牌。
 
　那就是我一直想使用而取走的爱车――不对，应该是『废車』在那家伙的头上释放。
 
。
 
「!?―――咕喔喔!!」
 
。
 
　就连影狼也对废车的大小于重量感到惊讶吧，它迸发出比起刚才还要多的『影子』，将废车抵挡住。
 
　于是趁着废车停止坠落的瞬间，它意图逃脱。
 
。
 
「还没完！」
 
 
 
 
 
　然而我朝着它想要逃跑的方向『前方』，再次释放汽车。
 
。
 
「―――!?」
 
。
 
　影狼的神情再次因惊愕而扭曲。
 
　骤然出现的汽车之『壁』。
 
　并且我还围绕着它释放出第两辆、第三辆汽车。
 
。
 
　这些都是刚才丢弃在停车场的汽车。
 
　我已经验证过道具包可以收纳『丢弃的物品』，因此在刚才的战斗中就先收纳起来了。
 
　问题是道具包能否将这些汽车都全部收纳，不过多亏提升到ＬＶ１０，收纳能力似乎也提高了。
 
。
 
『咕……咕噜噜……』
 
。
 
　它的周围全部被汽车堵住，已经无处可逃。
 
。
 
「怎么了？你只要跳过车子不就可以逃跑了？」
 
。
 
　即便我这么说，那家伙也是苦涩着脸而已。
 
　啊啊，就是这样，我知道喔。
 
　既然莫莫的技能是你的翻版，那『弱点』也绝对是一样。
 
。
 
「如果脚没踩到地面上，『影子』就不会出现」
 
。
 
　没错，这就是『影子』技能的弱点。
 
　如果没有处于脚踩在地面上的状态，就无法操控『影子』。
 
　以前有问过莫莫真是太好了。
 
　嘛，虽说前肢后肢只要有一条腿踩到就行，所以一般情况下在战斗中应该是没有问题的。
 
。
 
　不过像这样四面八方都被堵住的情况下就不行了。
 
　它自满的速度都没办法发挥出来。
 
　一旦想要跳过去，那瞬间『影子』就会消失。
 
。
 
　唯一的悬念就是它是否拥有跟莫莫一样的『潜入影中』的技能。然而从它身处于这样的局面也不使用这招来看，这家伙应该是没有这个技能。
 
　不知是莫莫的固有技能，还是小狼之间的个体差异呢。
 
。
 
　我再那家伙头上追加一辆汽车。
 
　无处可逃。
 
　那么就只能利用『影子』防御到底。
 
　然而释放出来的『影子』也是有极限。
 
。
 
「将军了……莫莫」
 
。
 
「汪！」
 
。
 
　莫莫的影子捕捉到影狼。
 
　虽然它刚才还会用影子抵消到，但现在没有这份从容。
 
　莫莫的影子缠绕在影狼身上，它无法随心所欲地动弹。
 
　应该是领悟到自己败北了吧，影狼呻吟地发出悔恨声。
 
。
 
「再见了，影狼。是你输了」
 
。
 
　我又在那家伙支撑着汽车的影子上追加一辆汽车。
 
　这瞬间，影子应该是迎来了极限，轻而易举地就消散了。
 
　剩下庞大的重物将影狼压倒了。
 
。
 
≪获得了经验值≫
 
≪经验值达到一定的水平≫
 
≪工藤和人从ＬＶ８升级到ＬＶ９了≫
 
。
 
 
 
 
 
　呼——，总算是打倒了。
 
　那么，学生他们看起来还在战斗中，还是赶紧逃跑吧。
 
一一一一一一一一一
