WEB 81 我依然没有做好觉悟
 
 
一一一一一一一一一一一一一
 
 
 
 
 
　我在近距离将刀刃（小红拟态的）架在魔物师的脖子上。
 
　虽说是临场想出的作战，不过似乎很顺利。
 
。
 
「―――那么，魔物师。如果你不想死就乖乖听我的话照做吧？」
 
。
 
　如此说道的同时，我内心正颤抖着。
 
　应该是多亏了『交涉术』吧，说话相当流利。
 
　不过实际上内心是紧张不已。
 
　毕竟我现在刀刃所对付的并非是魔物，而是人。
 
　我现在正把刀刃对着人。
 
　并且还是年纪尚轻的少女。
 
。
 
　明明面对着魔物就很普通，可为何换成人之后就会这么紧张呢？
 
。
 
　我知道答案。
 
　单纯只是我还没有割舍得开而已。
 
。
 
　―――不能伤害他人。
 
。
 
　在日常中这是理所当然的想法。
 
　而在如今的世界中，这种想法只会碍事。
 
　这样的想法就存留在我心里。
 
　理性是知道的。
 
　赶紧杀死她比较好。
 
　即便不杀死，也可以将她的手脚弄伤，再绑起来也行。
 
　明明知道这一点却没有付诸行动，也不愿行动……。
 
。
 
　『抗压耐性』的等级更高的话，这种感觉也会越来越淡薄吗？
 
　还是说拥有其他的耐性技能―――『忌讳感耐性』或者『罪悪感耐性』就会不一样了呢。
 
　……不行，现在别考虑多余的事情。
 
　先注重眼前。
 
。
 
「别给我抵抗，现在立刻让那些魔物停下来，身为操控者的你应该办得到吧？」
 
。
 
　闻言，魔物师刚才紧绷的表情稍微柔和了一些。
 
　一副像是惊讶，又像是安心的表情。
 
。
 
「……还真是温柔呢」
 
。
 
「我是想尽可能安稳地解决事态」
 
。
 
「安稳……呢……」
 
。
 
　魔物师虽然想说什么，但还是老实地举起双手。
 
　以此为信号，暴乱的魔物们突然老实下来。
 
。
 
「怎、怎么了？」
 
「这些家伙突然安分下来了」
 
「怎、怎么回事……？」
 
。
 
　食堂里还存活的学生们都有些动摇。
 
　即便如此，声色中还是带有些安心。
 
。
 
「―――『回归』」
 
。
 
　接着她说出口以后，包围住食堂的『暗』也都逐渐消失不见。
 
　而安分下来的魔物们则是由脚边的暗所吸收，消失了踪影。
 
　刚才的骚乱仿佛是虚假一般，现场充满着寂静。
 
。
 
「得、得奖了？」
 
「没事了吗……？」
 
「呐，在葛木同学旁边的人是谁呀？」
 
「戴着兜帽，完全看不到面孔啊……」
 
「是那个人救了我们吗……？」
 
。
 
　……真引人注目，虽然也是没办法的事情。
 
　西野君和六花ち也同样是定睛注视着这边
 
　一之濑小姐……依然没有放下枪支。
 
　像是在诉说着随时都可以开枪一般。
 
　没事了，至少在当下的状况中。
 
。
 
「……真意外，没想到你会老实地听从我说的话」
 
。
 
「那当然是因为我不想死啦」
 
。
 
　她嫉恨地瞪视着我。
 
。
 
「……真是误算，根本没想到这里还有你这种家伙，可恶」
 
。
 
「是吗」
 
。
 
「啊~啊~真遗憾啊。我还以为可以得到大量的经验值呢……」
 
。
 
　游戏结束了吗，她宛如在玩游戏一般嘀咕着什么。
 
　那是与我刚才所想的事情是完全相反。
 
　完全就是没把他人当做是人看待的措辞。
 
。
 
「你疯了吧……」
 
。
 
　我忍不住说出这句话。
 
　于是魔物师像是感到意外似的睁大眼睛。
 
。
 
「是吗？无论是谁都会珍惜自己吧？在这种满街是魔物的世界里，无论等级提升多少不也是不够吗。我可是觉得这样比起照顾他人，受到他人扯自己后腿还要好得多喔？你就没想过要随心所欲地活下去吗？」
 
。
 
　随心所欲地活着……吗。
 
　她的主张套用在如今的世界上，说不定的确是正论。
 
　拥有力量之者才是绝对，如今的世界中已有的价值观和权力都毫无意义可言力。
 
　既然如此―――胜利的就是随心所欲的生存者。
 
。
 
「我不会……否定。毕竟我的生存方式也不是值得称赞」
 
。
 
「嘿~……」
 
。
 
　她直直地注视着我，以只有我能听到的声音嘀咕道、
 
。
 
「……呐，现在也不晚啦，跟我合作怎么样？」
 
。
 
　她提出这样的提议。
 
。
 
「哈？」
 
。
 
「看样子你应该是拥有隐秘、暗杀系的职业和技能吧？我可以通过技能将这些能力进行『強化』喔？还有我的魔物军团。作为同伴而言也不会有损失。怎么样，这是不错的提议吧？」
 
。
 
　强化技能和职业的技能……？
 
　该不会像我的『道具包』和『早熟』一样是稀有技能吧？
 
　这就是她可以驯服这么多魔物的原因……。
 
。
 
「……你以为我会同意吗?我可无法相信你」
 
。
 
「相信我吧，我自己也不想死。而且从你身上隐约感觉到和我相同的『感觉』。我觉得一定会成为不错的搭档喔？」
 
。
 
　才刚刚认识她究竟在说些什么鬼。
 
　归根究底这家伙可是疯子，想要杀光学校里的人。
 
　别听她说了。这家伙只是在说些漂亮话，寻找我破绽而已。
 
。
 
「抱歉―――」
 
。
 
「「啊ーーーー！找到了ーーー！」」
 
。
 
　食堂里响起的声音将我原本想要拒绝的声音覆盖过去。
 
　往那边望过去，门口那边有对双胞胎。
 
　那不就是……刚才来到这里的时候擦肩而过的双胞胎吗。
 
。
 
　他们身体到处都有与魔物战斗的痕迹。
 
　果然是技能持有者吗。
 
。
 
「看样子援军也已经抵达了」
 
 
 
 
 
。
 
「……」
 
。
 
　我说完以后，她像是终于放弃一般低下了头。
 
　从反应来看，那对双胞胎似乎并非是她的同伙。
 
。
 
「喂~，给我投降吧ー！」
 
「啊哈哈哈，坏蛋让我找到了！」
 
。
 
　如此说着，少女手上出现火球，而少年手上出现螺旋状的石头。
 
　嘿，魔法系的技能吗。
 
　果然还是有这一类的技能呢。
 
　在我像这样悠闲思考着的时候―――『危机感知』和『敌意感知』有反应了。
 
。
 
「「坏蛋，给我从那个人身边离开ーーー！」」
 
。
 
「―――哈？」
 
。
 
　这完全是意料之外的一击。
 
　不对，正确地来说是两击吧。这种事怎么样都好。
 
　惊人的是他们的攻击是向『我』释放的。
 
。
 
「什……!?」
 
。
 
　我几乎是反射性地避开两人的攻击。
 
　而这些攻击就直接打到食堂的墙壁上，一部分是燃烧起来，一部分是击碎了。
 
　好、好危险……。要是打中了，毫无疑问就死翘翘了吧……。
 
。
 
「你、你们干什―――噗哈!?」
 
。
 
　她不可能会放过这次机会。
 
　全力扭转身体，抬起膝盖全力踢中我的侧腹。
 
　就这样直接后跳着与我拉开距离。
 
。
 
「哈哈，真幸运~」
 
。
 
「你、你这家伙……！」
 
。
 
　开玩笑的吧？再怎么是擅长隐秘和奇袭，我的『防御』可是超过三位数吧？
 
　而且虽说是出其不意，但刚才的速度就连我也反应不过来喔？
 
　不对，比起这件事。
 
　我斜眼看着门口附近的双胞胎。
 
　这些家伙为什么攻击我―――？
 
。
 
「啊咧?呐，姐姐，十香姐姐说过的坏蛋不是女人吗—？」
 
「弟弟啊，你在说些什么啊！年轻的女性可是被人用刀架在脖子上！而且那男人的打扮也是十分可疑，绝对就是坏蛋吧！」
 
「对喔！说的也是ー！真不愧是姐姐！」
 
。
 
　……看样子他们似乎是将我误以为事件的罪魁祸首。
 
　这还是我有生以来首次真心想要揍死小鬼。
 
　该死，居然做出多余的事情来……！
 
。
 
「啊哈哈……我还以为游戏结束了，不过幸运女神似乎还是站在我这边呢」
 
。
 
　她嗤笑着。
 
　这瞬间听到连续的微弱开枪声。
 
　是一之濑小姐的狙击。
 
　在魔物师离开我的瞬间，她就已经立刻行动了。
 
　对准的是太阳穴和大腿。
 
　目的应该是即死，又或者是造成她行动不便吧。
 
　为她的毫不犹豫感到钦佩。
 
　然而―――。
 
。
 
「没用的」
 
。
 
　子弹在距离她几公分前『停止下来』。
 
　又是『暗』从子弹中守护着她。
 
。
 
「轮到你出场了！小狗！」
 
。
 
　刹那间，巨大的野兽从她脚边出现。
 
　第三次邂逅。
 
。
 
「―――嗷嗷嗷嗷嗷嗷嗷!!」
 
。
 
　黑狼发出吼叫声，从中现身。
 
。
 
「那条狗是怎么回事……」
 
「总觉得好像很强啊……」
 
。
 
　双胞胎都被突然出现的黑狼夺走目光。
 
　其他学生也是同样，其实还有人抱着胳膊瑟瑟发抖。
 
　这也意味着黑狼的强大给他们带来的印象是多么深刻。
 
 
 
 
 
。
 
　不过……这又怎么了。
 
。
 
　现在这家伙根本不是万全状态。
 
　经历过两次战斗，身体应该相当虚弱才对。
 
　实际上我和学生们给它造成的伤势都还没有痊愈。
 
　现在赢得了它。
 
。
 
「露出一副赢得了的表情啊喂？」
 
。
 
　紧接着，仿佛像是要颠覆我的想法一般，魔物师露出令人毛骨悚然的笑容。
 
。
 
「太天真了，让你见识一下吧，这家伙就是我的王牌」
 
。
 
　她将手放在黑狼身上。
 
。
 
「―――『使魔強化』」
 
。
 
　下个瞬间，黑狼的身体剧烈地抖动一下。
 
　它全身浮现出黑色的烟气，伤口也逐渐愈合。
 
　体型更加庞大，缠绕在身躯上的暗也比起以往更加强烈。
 
　它现在显然比起以前还要强。
 
。
 
　强化魔物的……技能……？
 
　我呆然不动地注视着眼前的光景。
 
　不光是我，在场所有人都纹丝不动。
 
　全都受到眼前的光景所影响。
 
　这也意味着黑狼的威压感是多么压倒性。
 
。
 
「那么，战斗继续」
 
一一一一一一一一一
