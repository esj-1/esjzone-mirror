翻译：飒君CONAN
 
 
 
WEB 60 汇合
 
 
 
 
一一一一一一一一一一一一一
 
 
 
 
 
我躲在架子的阴影里，眺望着走进来的两位女性。
 
其中一位是住在我以前公寓隔壁的新婚妻子。
 
……说起来名字不知道啊，是叫什么来着？
 
。
 
另外一位是身穿学校制定运动衫的短发女孩子。
 
大概是高中生吧，正所谓是运动少女的少女。
 
两人举着长尖叉缓慢地在场内走着。
 
。
 
「应、应该没问题吧……？猪、猪头人它们是不在吧？」
 
。
 
女高中生那边相当惧怕的样子。
 
连拿着武器的手都在颤抖。
 
。
 
「没事的，如果在这里早就袭击我们了……大概」
 
。
 
「别什么大概呀ー！」
 
。
 
新婚妻子那边表情僵硬，但步伐稳扎。
 
是意外有胆量呢，还是说在年纪比自己小的人面前逞强着呢。
 
。
 
嗯~，应该没问题吧？
 
虽然两人看起来等级似乎不高。
 
话说这两人来干嘛的？
 
补充物资……感觉也不像。
 
从刚才的交谈来看，是来寻找幸存者的？
 
。
 
「没事的啦，石泽小姐。您老公一定已经逃跑了。所以说呢？我们快点回去吧ー……」
 
。
 
啊，果然是来寻找幸存者的。
 
还有新婚妻子的名字似乎是叫石泽。
 
运动少女拼命地在说服石泽小姐，然而她依然逐渐往深处走去。
 
。
 
啊，记得再继续走下去……就是那个了吧。
 
有种讨厌的预感啊，我边想着边看着两人。正如预料，她们看到『那个』后发出悲鸣。
 
那就是猪头人堆积起来的尸体山。
 
。
 
「这、这是什么呀？……呜咕、呜……」
 
。
 
运动少女像是无法直视般尽情地吐出来。
 
啊——，果然一般都会这样啊。
 
如果没有『抗压耐性』，估计这会成为我一生的心理阴影。
 
。
 
「怎、怎么会……不会吧……」
 
。
 
另一边，石泽小姐茫然若失地看着尸体山。
 
正确地来讲是尸体山的某一处。
 
看着一位男性。
 
。
 
「为什么……这不是真的。不要、亲爱的……不要……不要啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊！」
 
。
 
石泽小姐如此大叫的同时奔向男性的尸体。
 
记得那人是第一天在这里战斗的其中一个。
 
是吗，原来是她老公啊……。
 
。
 
「啊，亲爱的……对不起，对不起……」
 
。
 
石泽小姐将尸体抱在怀里，哭泣着一阵子。
 
运动少女也像是无法直视地别开目光。
 
。
 
……老实说看到这种情形心情也不怎么好。
 
差不多该到时间了，趁现在还没注意到早点离开吧。
 
虽然有兴趣知道她们是从哪里过来的，不过跟一之濑小姐见面的时间已经要到了。
 
果然还是不能迟到。
 
。
 
「……咕呜？」
 
。
 
这样好吗？莫莫如此询问我，答案早就心里有数。
 
。
 
「……好啊，我们根本没必要硬是扯上关系」
 
。
 
虽然是见过面的邻居，但也就仅此而已。
 
。
 
「……请努力活下去吧」
 
。
 
仅对她们说出这样一句声援，我们就离开了购物中心。
 
单单三天就彻底改变的世界。
 
世界已然是魔物满载，生命低贱，处处充满危险。
 
现在到处都产生这样的悲剧了吧。
 
。
 
……正因为如此，我们才非得避免死亡努力存活下去。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
接着过了几分钟。
 
又有人造访购物中心。
 
。
 
「哦，果然就跟小西猜测得一样呢。猪头人它们似乎不在」
 
。
 
「是啊，太好了。这里可是前往学校的最近路线」
 
。
 
两位学生——西野和六花在停车场眺望着购物中心。
 
。
 
「怎么办？要到里面看一下吗？」
 
。
 
「……还是不要了吧。里面的东西的确有魅力，但现在我想尽快抵达目的地」
 
。
 
「嗯——，嘛，说的也是。……啊咧？好像有谁出来了哦？」
 
。
 
「你说什么……？」
 
。
 
两人将目光移向那边，购物中心从中出来了两位女性。
 
其中一位是他们认识的人物。
 
身穿运动衫的短发少女。
 
当看到她，两人的表情都发生变化。
 
。
 
「该不会是……葛木同学？」
 
。
 
突然叫到自己的名字，少女从而吓了一跳。
 
于是当她看到停车场的西野他们后便睁大眼睛。
 
。
 
「……诶、不会吧？西野君？还有相坂同学？你、你们还活着？真的？」
 
。
 
短发少女以惊讶与混杂点安心的声音说道。
 
名为葛木的少女是西野和六花的同班同学。
 
三人都奔跑到一块，为彼此的再会感到高兴。
 
。
 
「葛木，好久不见ー。你平安无事呢ー」
 
。
 
「相坂同学也是……我好担心你们喔。你们两人在那一天都没有来学校……」
 
。
 
「……嗯，是呢。呐，葛木之前都在做什么呀？还有在那边的人是？」
 
。
 
「啊啊，嗯，说的也是。呃，要从哪里开始说起好呢―――」
 
。
 
运动少女——葛木将之前的经历都告诉给两人。
 
三天前，也就是世界改变当天她还在学校进行社团活动。
 
当时遭到魔物的袭击，与其他学生相互合作存活下来。
 
于是现在学校附近的魔物变少，目前是作为避难所各处收留难民，她这么简单地说明一下。
 
。
 
「……是吗，似乎是辛苦啊」
 
。
 
「西野君你们也是啦……。如果大野君和柴田君也没事就好了呢」
 
。
 
「是啊，他们一定还活着」
 
。
 
「呐？你们两人打算今后要怎么办？如果不介意，就跟我们―――」
 
。
 
「当然，让我们跟你们一起行动吧。反正我们也是前往学校的途中」
 
。
 
西野用力点头，对此葛木露出安心的表情。
 
她的脸颊也微微泛红。
 
。
 
「太好了。那我们一起走吧？」
 
。
 
「啊啊，知道了」
 
。
 
西野和六花就这样与同班同学再会，决定一起前往学校。
 
。
 
「啊，说起来你们两人都打倒魔物了吧？职业和技能都选择了什么？」
 
。
 
无意的询问。
 
一瞬间，两人的表情都阴暗起来。
 
。
 
「……那个，我是―――」
 
。
 
「我是选择了『交涉者』，六花则是『冒险者』。所以战斗方面就交给六花了。……虽然作为男人实在是没面子。选择完以后似乎没办法变更了」
 
。
 
「嗯~，这样啊」
 
。
 
没有半点怀疑的葛木点点头。
 
见状，西野安心地摸着胸口。
 
。
 
「……这样好吗？」
 
。
 
六花偷偷地以西野听得到的音量询问道。
 
。
 
「……啊啊，毕竟不知道会发生什么事。姑且还是隐瞒起来比较好吧，我们『真正的职业』」
 
。
 
「……也是呢」
 
。
 
「你们两人都在干什么呀?快点走吧ー」
 
。
 
「好，现在就去」
 
。
 
两人结束谈话。
 
没错，绝对不能让别人知道。
 
自己真正的等级和职业，特别是关于那个『技能』。
 
。
