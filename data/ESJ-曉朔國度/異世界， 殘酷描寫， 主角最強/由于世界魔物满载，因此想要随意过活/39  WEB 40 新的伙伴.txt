WEB 40 新的伙伴
 
 
一一一一一一一一一一
 
 
 
 
 
　我注视着在脚边抖动着身体的红色史莱姆。
 
　史莱姆以似乎感到期待的样子看着我。
 
　不，虽然没有眼睛就是了。
 
。
 
　看来这家伙是想成为我的伙伴。
 
　虽然我自己说也有些不妥，不过我到底有什么吸引它的因素啊。
 
　这只史莱姆是眼瞎吧，啊，它没有眼睛来着。
 
　我挠挠头。
 
。
 
「……上次我也说过了，我可没打算让你成为我们的伙伴―――嗯？」
 
。
 
　说到途中，我注意到一件事。
 
　红色史莱姆的体内，除了一颗像是梅干一样的小核心之外还有『另一个』漂浮的东西。
 
。
 
「那是……魔石吗？」
 
。
 
　红色史莱姆的体内漂浮着棕色的魔石。
 
　这颗魔石的形状感觉在哪里看见过，不如说……。
 
。
 
「那该不会是我的魔石吧？」
 
。
 
　我说完以后，史莱姆抖动着身体。
 
　它大概是想说正确吧。
 
　这家伙……。
 
。
 
「还给我，那可我想要给莫莫―――」
 
。
 
「汪！」
 
。
 
　正当我想把手伸向史莱姆时，莫莫在这瞬间穿插进来了。
 
。
 
「……莫莫？」
 
。
 
「汪！汪呜」
 
。
 
　莫莫向我吠叫着，仿佛是在庇护着史莱姆。
 
　诶、莫莫、怎么了……？
 
　我做了什么让你觉得讨厌的事情吗？
 
　糟糕，有点想哭了……。
 
。
 
「汪、汪！」
 
。
 
　看到狼狈的我，莫莫慌张地想要解释「不是的」。
 
　接着它将前肢放到史莱姆的上面，并看着我。
 
　而史莱姆的身体则是因为它的举动而抖了抖。这种样子有点可爱。
 
。
 
「莫莫……你该不会是想让这只史莱姆成为伙伴吧？」
 
。
 
「汪」
 
。
 
　莫莫点点头。
 
　看来莫莫是想让这只史莱姆成为伙伴。
 
　话说，难不成……。
 
。
 
「呐，莫莫。你刚才把魔石丢到外面，难不成是想给这家伙吗？」
 
。
 
「汪」
 
。
 
　莫莫再次点头。
 
　果然是这样吗，刚才吃饭时莫莫做出的迷之行动。
 
　我还觉得喜欢魔石的莫莫会这么做还真是奇怪，原来是这么回事。
 
。
 
　莫莫察觉到这只史莱姆就在外面。
 
　于是应该是有什么想法，所以将魔石给了这只史莱姆。
 
。
 
　给它魔石的理由……能想到的就是强化，或者是『技能』。
 
　就跟莫莫吃掉影狼的魔石得到『影子』技能一样，这只史莱姆该不会也是……。
 
　我看着史莱姆，刚刚好它也完全消耗掉宝箱怪的魔石了。
 
。
 
「汪」
 
 
 
 
 
「……（抖抖）」
 
。
 
　莫莫向着史莱姆吠叫着。
 
　于是史莱姆抖了身体，让身体发生『変化』。
 
。
 
「噢、噢……」
 
。
 
　我不禁喊出声。
 
　史莱姆的身体逐渐进行变化，变成空空的塑料盒。
 
　啊，这不就是我给这家伙吃的垃圾吗。
 
。
 
「变身……不对，是拟态技能吗……」
 
。
 
　这好厉害。
 
　光凭外表根本看不出有什么区别喔。
 
　试着摸一下，触感也是相同。
 
　只不过重量不一样。大概是史莱姆原本的体重。
 
　而且跟宝箱怪一样，『索敌』没有反应。
 
　好厉害的技能。
 
。
 
「汪」
 
。
 
　莫莫再次吠叫以后，史莱姆恢复成原本的姿态。
 
　于是恢复原样的史莱姆和莫莫一起用着期待的眼神注视着我。
 
。
 
　……难不成刚才的演示就是这些小家伙想打动我的方法？
 
　比如「我可以做到这种事!」什么的。
 
。
 
「……莫莫，你该不会是察觉到了吧?如果把宝箱怪的魔石给了史莱姆，它就可以得到拟态技能」
 
。
 
「……汪呜」
 
。
 
　猜的，隐约有这种感觉。
 
　莫莫像是如此回答道。
 
　也就是所谓的野生的直觉吗？
 
　如果是这样就太厉害了吧，莫莫的直觉。
 
。
 
≪红色史莱姆注视着您，似乎是想要成为伙伴。请问需要加入为伙伴吗？≫
 
。
 
　天之声再次响起。
 
　我看着莫莫和史莱姆。
 
　接着深深地叹了口气。
 
。
 
「哈啊……我知道了啦，莫莫。是我输了」
 
。
 
　拥有拟态技能的史莱姆。
 
　这样的确是不会妨碍到我的技能。
 
　还不只是这样，它具备物理攻击无效，作为战力而言无可挑剔。
 
。
 
　重点是……这是莫莫第一次主张自己的意见。
 
　那么我只能选择相信了吧。
 
　无论是莫莫，还是这只史莱姆。
 
。
 
　这一次，我在脑海里选择YES。
 
。
 
≪红色史莱姆成为伙伴了≫
 
≪满足一定条件≫
 
≪可以获得职业『魔物师』≫
 
≪职业『驯兽师』统合成『魔物师』≫
 
≪当选择『魔物师』时，初始等级为ＬＶ２≫
 
。
 
　哦，增加的新的职业了吗。
 
　『魔物师』吗……。
 
　从天之声可以看出它应该是『驯兽师』的上级职业。
 
　虽然很在意这职业能得到什么技能，不过还是等到第三职业解锁的时候吧。
 
。
 
「……（抖抖）」
 
。
 
　应该是对于我接受请求而感到开心吧，史莱姆抖了抖身体。
 
　莫莫也像是在说「太好了呢」，摸着史莱姆。
 
　汪汪、抖抖、摸摸。
 
　……这什么鬼，好可爱。
 
。
 
　看了下人物状态。
 
　在最下面的项目队伍成员上追加一项史莱姆。
 
　显示的是红色史莱姆ＬＶ５。
 
　果然魔物也是有等级的吗……。
 
　与莫莫不同，名字并没有显示。
 
　话说没有名字，要叫它有些不方便吧……。
 
。
 
「呐，红色史莱姆。如果你不介意，我来给你取名字吧？」
 
。
 
　我说完以后，史莱姆展现出抖动的反应。
 
 
 
 
 
看样子是想让我取名字。
 
。
 
「嗯~……那就『小红』怎么样？」
 
。
 
　毕竟是红色史莱姆，那就小红了。
 
　连我自己都觉得我这命名真简朴。
 
　不过红色史莱姆似乎很中意的样子。
 
　高兴似的抖了抖身体。
 
　噢，身体像饼干一样伸得直直的。
 
。
 
　看着人物状态，增加了红色史莱姆的名字。
 
。
 
莫莫
 
柴犬ＬＶ９
 
。
 
小红
 
红色史莱姆ＬＶ５
 
。
 
　莫莫的等级又是若无其事地提高。
 
　差一点就要提升到ＬＶ１０。
 
。
 
「那么我们走吧」
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
　于是红色史莱姆——小红成为了同伴。
 
　我们和新伙伴一同离开了公寓。
 
 
 
 
 
。
 
 
 
 
 
　另一方，此时―――。
 
。
 
「呜啊啊啊啊啊啊！柴、柴田！救、救救我……啊啊啊啊啊啊啊啊啊啊啊!!」
 
。
 
　鮮血飞舞，又有一位同伴倒下了。
 
　柴田茫然若失地看着那副样子。
 
。
 
　意义不明。
 
　这是什么状况？
 
　为什么会变成这样？
 
 
 
 
 
。
 
　为了搜索食粮，他们扩大探索范围来到了这里。
 
　中途十分顺利。
 
　出现的魔物也只有一两只哥布林和僵尸。
 
　提高等级后，它们根本不是我们团队的对手。
 
。
 
『再稍微往前面一点吧』
 
。
 
　不知是谁先说出这句话。
 
　没有人持反对意见。
 
　打倒了魔物，提高了等级，得到新技能，这些事情令柴田他们得意忘形了。
 
　能行，如果是我们就可以对付更强的魔物。
 
　得到大量的食粮，让大家吓一跳吧。
 
　就是抱着这种念头增加搜索范围而来到了这里。
 
。
 
　然而，他们的步调唐突结束了。
 
　他们来到了大型购物中心。
 
　在那边有着远远超出想象的『怪物』。
 
。
 
「……什、什么啊，那家伙……！」
 
。
 
　躲藏在附近草丛离的不良学生——柴田身体颤抖着。
 
。
 
「呀啊啊啊啊啊啊啊啊啊！」
 
。
 
　又听到一位同伴的悲鸣声。
 
　救人？不，不行。根本不是对手。
 
　那家伙的层次不同寻常。
 
。
 
「哈……哈啊哈啊哈啊……！可、可恶！」
 
。
 
　柴田急促地反复呼吸着，他做出了决断。
 
。
 
「喂！柴田!?你要到哪里去啊!?还、还不快点救助那些人―――呜、呜啊啊啊啊啊啊啊啊啊!!」
 
。
 
　柴田背对着同伴的悲鸣声逃跑了。
 
　他选择了『逃跑』。
 
　趁着那家伙的注意力都在同伴的时候，为了尽可能拉开距离，柴田奔跑着。
 
。
 
（抱歉……大家，抱歉了……！）
 
。
 
　他在心中对于舍弃同伴这件事忏悔着。
 
　即便如此也没有停下步伐。
 
。
 
「……我、我得快点告诉西野。快点……快点……！」
 
。
 
　必须要有谁……有谁活下来传达给其他人不可。
 
　将那头魔物的存在告诉给其他还活着的伙伴。
 
　他颤抖地奔跑着，耳中那头魔物的叫声一直是不绝于耳。
 
一一一一一一一一一
