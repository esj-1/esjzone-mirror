WEB 64.忍术
 
 
一一一一一一一一一一一一一
 
「咕噜噜……」
 
。
 
黑狼发出低吟声。
 
仿佛与此呼应一般，那家伙的『暗』逐渐扩展开来。
 
看起来就像是漆黑沼泽或者泥巴一样。
 
「厌恶感」的反应十分强烈。
 
还是避免触碰到那玩意比较好。
 
。
 
「咕嗷嗷嗷！」
 
。
 
随着黑狼大叫，那家伙周遭扩展的『暗』以像是浊流一般逼近而来。
 
好快，不过这速度还是可以避开。
 
我边往后退，边『观察』那家伙的攻击。
 
。
 
「啥!?」
 
。
 
那是什么鬼？那家伙的『暗』所触碰的废车都逐渐吞没进去。
 
不单单是这样，就连周遭的物体都全部渐渐沉浸于『暗』中。
 
就像是无底沼泽的光景。
 
。
 
……哎呀，现在可不是悠闲地观望的时候。
 
抱歉了，那些东西就还给我吧。
 
我打算将要被吞没的废车回收回来。
 
然而……。
 
。
 
「……没办法回收!?」
 
。
 
我始终无法将其回收进道具包。
 
无论怎么在心中默念，依然是无法进行回收。
 
被那玩意所缠住就意味着完蛋了吗。
 
究竟是怎样的原理啊？
 
。
 
「啧」
 
。
 
总之也就是说不能无谓地进行攻击吗。
 
可是远距离攻击在打到那家伙之前就会被『暗』所吸取，从而防御下来。
 
话是这么说，要进行肉搏战也得对那家伙周围展开的『暗』想点办法。
 
。
 
「莫莫！」
 
。
 
我向莫莫发出信号。
 
仅过零点几秒。
 
莫莫『咆哮』。
 
。
 
「ワォォォォオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオンッッ!!」
 
。
 
破坏的咆哮。
 
然而黑狼应该早就意料到了吧。
 
那家伙立马利用『暗』将自己围住。
 
那『暗』连冲击力也能吸收吧。
 
表面上在振动而已。
 
莫莫的『咆哮』也对那家伙不起作用吗。
 
。
 
不过这样就行。
 
我的目的并非是莫莫的咆哮能给它造成伤害。
 
而是妨碍它的『视野』，争取时间。
 
我趁机发动了『技能』。
 
。
 
莫莫的咆哮结束了。
 
黑狼也解开『暗』。
 
这瞬间，那家伙的表情发生变化。
 
这也是当然的吧。
 
毕竟『我』就在那家伙的眼前。
 
。
 
「嘎呜!?」
 
。
 
究竟是怎么过来的？
 
它应该是这么想的吧。
 
很简单。
 
我只是打造出『踏脚处』来到这里而已。
 
毕竟你的『暗』要将废车完全吞没需要几秒的时间。
 
所以我只是朝着暗之沼泽释放废车和岩石，在吞没之前利用这些踏脚处接近它而已。
 
。
 
『我』挥下手上的猪头人的菜刀。
 
在当下脚边的踏脚处吞没之前，只要解决这家伙―――。
 
。
 
「嘎呜呜！」
 
。
 
然而事态不可能这么顺利。
 
在刀刃命中之前，那家伙扭转身体躲开攻击。
 
好可怕的身手。
 
。
 
「嘎嗷！」
 
。
 
黑狼立刻进行反击。
 
于周遭所展开的『暗』变化为巨大的手将『我』抓住。
 
黑手用力紧握，身体到处都发出骨折声。
 
接着当身体承受能力迎来极限时―――砰!『我』化为白烟消失不见了。
 
。
 
「!?」
 
。
 
黑狼的表情再次惊愕起来。
 
应该是在想「发生什么事了?」。
 
。
 
「怎么了？为什么要这么惊讶啊？」
 
。
 
在离它有点距离的地方，我向它搭话说道。
 
。
 
「嘎……嘎？」
 
。
 
黑狼像是觉得「怎么回事」般歪着头。
 
嘛，初次看到这种情况根本不可能明白的吧。
 
。
 
如果要揭开其中的内幕，其实刚才的是我的新技能『忍术』。
 
而这招则是其中之一，既『分身之术』。
 
。
 
这招忍术正如字面意思所述，能力是创造出自己的分身。
 
最大能够创造的分身是五个。
 
身体能力几乎都与我相同。
 
每一个分身都会让我十秒消耗3MP。
 
与本体不同，分身无法使用技能。可即便如此它也是强大的可以弥补这缺点的技能。
 
毕竟无论如何，我都可以毫无风险地向敌人进行突击。
 
。
 
「这样好吗？你看看后面」
 
。
 
「!?」
 
。
 
应该是注意力全在我身上吧。
 
黑狼并没有注意到身后还有一个『我』正迫近它。
 
分身还有一个喔。
 
。
 
这次它无法避开攻击了。
 
分身所持有的菜刀略微伤到那家伙的身体。
 
。
 
「嘎嗷呜！」
 
。
 
然而在下个瞬间，分身被那家伙的『暗』所抓，消失不见。
 
我可不会手下留情。
 
我使用『投掷』将菜刀和小刀丢向那家伙。
 
那家伙意图利用『暗』进行防御。
 
然而，在这瞬间、
 
。
 
「ワォォォォオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオンッッ!!」
 
。
 
莫莫的咆哮再次发动。
 
『投掷』并非是为了直接攻击到它。
 
而是像这样为了让那家伙将注意力都放在我身上。
 
尽管它想再次利用『暗』来抵御攻击，但来不及。
 
。
 
「！」
 
。
 
无法承受得住的黑狼被振飞了。
 
并且还有颗子弹打中滚落在地面上的它身体。
 
是一之濑小姐的掩护射击。
 
真是完美的毫不留情，一之濑小姐。
 
。
 
「嘎……」
 
。
 
黑狼踉踉跄跄地站起来。
 
刚才那套攻击似乎给它造成巨大的伤害。
 
果然这家伙是影狼的上级种，弱点也是相同的。
 
物理防御力都一样很低。
 
虽然『暗』能力的确是威胁，但只要攻击命中就能确实给它造成伤害。
 
它很强，不过并非是像高级猪头人一样荒唐到不讲任何道理的对手。
 
。
 
然而也不能大意。
 
这点我已经吃过许多教训了。
 
毕竟处于优势并想要给与最后一击时，才是最危险的时刻。
 
。
 
「嘎呜嘎呜！」
 
「嘎嗷嗷！」
 
。
 
它那些影狼同伴都聚集在它身边。
 
黑狼与我们保持一定距离的同时，定睛注视着我们。
 
下一击直接解决掉。
 
就在我如此思忖的瞬间。
 
。
 
「―――ゥゥウウウウオオオオオオオオオオオンッ!!」
 
。
 
黑狼骤然间吠叫起来。
 
与此相呼应，『暗』从它身上喷发而出，逐渐往四面八方扩散。
 
。
 
自暴自弃的攻击吗？
 
不对，错了。
 
我从扩散开来的『暗』里察觉到『厌恶感』。
 
这是―――!?
 
。
 
「莫莫！快闪开！」
 
。
 
在我大叫的同时，各式各样的物体从那家伙的『暗』中气势汹汹地飞奔而出。
 
既有刚才吞没的废车、岩石，也有建筑物和树木以及尸体等。
 
各种重物犹如子弹般全方位释放出来。
 
那『暗』可以自由地取出吞没的物体吗！
 
。
 
「啧！小红！」
 
。
 
「……（抖抖）！」
 
。
 
莫莫已经躲进『影子』，所以没有问题。
 
小红立刻膨胀身体，从犹如子弹而来的重物中保护我。
 
一之濑小姐那边也有小红的分身跟着，应该是没问题吧。
 
。
 
在小红的守护下，我们在风暴中无限弹跳。
 
身体也翻转不停。
 
有种变成洗衣机里衣物的感觉。
 
虽说是第二次，果然还是很痛苦。
 
。
 
在这种风暴中，那家伙打算攻击过来吗？
 
不过并没有它并没有追击过来。
 
攻势停止了。
 
视野恢复。
 
那里早没有黑狼的身姿。
 
。
 
「诶」
 
。
 
藏起来了吗？
 
就跟起初的时候一样躲进『暗』里面了？
 
可是也没看见其他的影狼。
 
。
 
究竟在哪里？
 
虽然觉得没用，但我还是发动『索敌』。
 
于是我感觉到黑狼它们的气息越来越远。
 
『厌恶感』也渐渐减弱。
 
。
 
「难不成……『逃跑』了？」
 
。
 
突然到访的寂静。
 
我警惕着这是什么陷阱，什么也不做。
 
。
 
「汪！」
 
。
 
莫莫从影子里出来走近这边。
 
实在是过于突然就闭幕了。
 
选择『逃跑』的魔物。
 
对于我来说这种事也是首次经历。
 
。
 
「……」
 
。
 
果然那只黑狼跟其他魔物不同。
 
它会观察这边的手法，一旦形势不对就会逃跑。
 
居然还有这种魔物……。
 
。
 
「麻烦了，让它逃跑了……」
 
。
 
既然它会逃跑，反过来说现在就很有可能打倒它。
 
即便我想进行追击，它只要保持距离躲进暗里，我就束手无策。
 
。
 
「好棘手……」
 
。
 
那头黑狼一定还会出现在我们面前吧。
 
它会变得更强，选择利于自身的状况下出现。
 
虽然战斗是获胜了，但现在无法为这件事尽情地感到高兴。
 
一一一一一一一一一
