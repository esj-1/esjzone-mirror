WEB 7 初次遭遇
 
 
 
一一一一一一一一一一一一
 
 
 
 
 
　我伫立在玄关的大门。
 
　接下来就不是我以往度过的日常了。
 
　而是魔物横行霸道的未知世界。
 
。
 
「……」
 
。
 
　平凡的大门感觉十分沉重。
 
　呼，看来我是紧张了―――啊，只是生锈了而已。
 
　咔嚓，门打开了。
 
。
 
　走到外面。
 
　跨出第一步。
 
　没问题。
 
　『气息察知』和『敌意感知』，以及『隐秘行动』都在发挥效果。
 
　现在周围没有魔物……大概。
 
　是说这三个技能。
 
　呆在家里根本不知道其中的效果吧。
 
　来到外面之后，才初次发挥效果的技能。
 
　这方面有点感到不安。
 
 
 
 
 
。
 
　首先确认一下左右两边。
 
　是破旧公寓里早已看惯的走廊。
 
　墙上到处都是龟裂，天花板上安装的荧光灯有着蜘蛛网。
 
　没问题，没有魔物。
 
　至少在「这一楼」。
 
。
 
「呼……」
 
。
 
　手里紧握的菜刀真是不牢靠。
 
　虽说只有这玩意可以用来作为武器，但外表上看完全就是那种可疑人物。
 
　我屏住呼吸，压低身子躲藏在栏杆下的墙壁里进行走动。
 
　这样一来，外面就不会看到我。
 
。
 
　首先是确认一下邻居。
 
　我隔壁的房间里住着一对新婚夫妇。
 
　话是这么说，我也不是很清楚详细情形。
 
　我，基本上是不会跟邻居有所交流的。
 
。
 
　只是因为墙壁太薄了，夜晚隔壁经常会传来性行为的声音。
 
　翌日，看到肌肤顺滑的太太，我有种难以言喻的心情。
 
　这么贤淑并且还是巨乳的美女，在夜晚居然能叫得那么激烈。
 
　那一天，魔芋用了好多……不不，别想了。
 
　会感到悲伤的，各种意义上。
 
（颯:知道魔芋是某种发电用的东西就行，别问我发电是什么）
 
。
 
　笃笃。
 
　轻轻地敲门。
 
　……没有反应。
 
。
 
　转动把手。
 
　……打不开，还在家里？
 
　还是说已经逃跑了？
 
。
 
　该不会是在我睡回笼觉的时候？
 
　有可能啊。
 
　仔细想想，世界变成这样子已经经过了半天。
 
　其他人会有所行动也不是什么奇怪的事情。
 
　虽然我在睡觉就是了！
 
。
 
　可能在我睡觉的时候有过来确认过我是否平安无事。
 
　于是没有反应就直接逃跑了吗。
 
　嗯，非常有可能呢。
 
　不如说世界都变成这样子了，还能睡回笼觉睡了六小时的我太奇怪了吧。
 
　这也是没办法的啦，我实在是太累了。
 
　错的不是俺，错的全都是社会。
 
（颯:关西腔都出现，然而我无所畏惧，毕竟关西腔很带感，有自学过一些）
 
 
 
 
 
。
 
　总之看看其他的房间吧。
 
　我按照顺序敲敲门，转动把手。
 
　全部是锁着的。
 
　其中也有没锁的房间，不过里面东西都是弄得乱七八糟。
 
　恐怕是慌慌张张地逃跑所导致的吧。
 
　四楼都是这样了，那楼下也是一样吧。
 
　姑且调查一下。
 
。
 
　走下楼梯。
 
　在走廊上走着的时候就觉得了，声音小到没有，真毛骨悚然。
 
　恐怕这就是技能『轻步』的效果吧。
 
　即便移动迅速也不会发出声音。
 
　真方便呢。
 
　如果这样再加上鬼步舞不就完美了？什么的？
 
。
 
　说起来，现在我在其他人看来是什么样子的呢？
 
　比如说很不起眼的状态之类的？
 
　嗯~自己没办法确认真是心急如火。
 
 
 
 
 
。
 
　来到了三楼。
 
　延伸而去的走廊上谁也不在。
 
　从离楼梯最近的房间开始敲门。
 
　没有反应。
 
。
 
　下一个房间，以及再下一个房间都没有反应。
 
　最后则是最边缘的房间。
 
。
 
　敲门。
 
　没有反应。
 
　转动把手。
 
　门打开了。
 
。
 
「喔」
 
。
 
　随着嘎吱作响声，门打开了。
 
。
 
「……有谁在吗—？」
 
。
 
　我尽可能压低音量试探里面的反应。
 
　……没有回应。
 
　一个人都没有吗？
 
　我慢慢地窥视里面。
 
。
 
「…………嗯？」
 
。
 
　有谁……在这里。
 
　感觉到气息。
 
。
 
　不过……这什么啊？脑海里回荡的『讨厌的感觉』？
 
。
 
　这是……『敌意』？
 
。
 
　这时我恍然醒悟。
 
。
 
　下一瞬间。
 
。
 
「ギギィィィイイイイイイイイ！！」
 
。
 
　房间深处有什么人发出声音冲了出来！
 
。
 
「！！」
 
。
 
　我反射性地将菜刀向前挥。
 
　锵！发出这样的声音。
 
　是刀与刀碰撞发出的声音。
 
。
 
　向我袭击而来的某种什么映入我眼帘。
 
　那并不是人
 
。
 
　小学生程度的绿色肌肤身躯。
 
　仅仅在腰上裹着肮脏的小布块，手持一把短剑。
 
　在幻想世界的魔物中的王道。
 
。
 
「哥布林……吗？」
 
。
 
　……不，也许有正式的名称也说不定。
 
　不过外貌上完全就是哥布林。
 
　因此就以哥布林来称呼它。
 
。
 
　哥布林慢慢地凝视着我。
 
　它体格比起我还要小。
 
　手腕也是肌肉发达，整体感觉是弱不禁风。
 
　然而感觉到的威压却好可怕。
 
　这并非是像游戏和动漫之类的隔着屏幕登场的人造物。
 
　而是真正的本尊才有的感觉。
 
。
 
　原来如此，这还真令人害怕……。
 
　也许是多亏了『恐怖耐性』，我已经相当冷静了。如果没有这个技能，或许就会颤抖不停也说不定。甚至双脚都发抖到没办法逃跑。
 
。
 
「ギギ……」
 
。
 
　哥布林与我保持距离并瞪视着我。
 
　对于首次的攻击被我挡下而感到警惕了吗？
 
。
 
 
 
 
 
「哼，很好……一下子不就有幻想世界的感觉了吗」
 
。
 
　我也举起菜刀。
 
　接着露出无畏的笑容。
 
。
 
「ギィ……！」
 
。
 
　哥布林举起短剑，表情上像是在说随时放马过来!。
 
　哈哈哈，我燃起来了。
 
。
 
　既然对方干劲满满――――――我就逃跑了喔！
 
。
 
　我转过身，当场逃跑了。
 
　再见了，哥布林！
 
。
 
「ギィッ!?」
 
。
 
　听到从后面传来的一道感觉像是「诶!?等等」的声音。
 
。
 
　于是哥布林连忙走出玄关，追在我身后。
 
　好。
 
。
 
　这瞬间，我转过身与哥布林正面相对。
 
　面对我又一次迷之行动，哥布林一瞬间呆站着。
 
　趁这个机会、
 
。
 
「―――道具包・Open」
 
。
 
　我依然举着菜刀，就这样使用了『道具包』。
 
　取出的是『洗衣机』。
 
　我就是取出了那台超重超大的洗衣机。
 
　从道具包里取出的东西是尺寸过大的情况下，它会在我眼前出现。
 
　而哥布林就在我眼前。
 
　结果会怎么样呢？
 
。
 
「グギャッ!?」
 
。
 
　哥布林被突然出现的洗衣机压在地上。
 
　毕竟那台超大超重的洗衣机几乎跟玄关和走廊是同个宽度。
 
　而长度也很大，所以即便它往后退也来不及了。
 
。
 
「作战成功」
 
。
 
　嗯，感觉良好地击溃它了。
 
　虽然它还没有死。
 
。
 
「ギィ……ギィィィ……！」
 
。
 
　它刚刚好被洗衣机从上面正面砸中，现在是被压在洗衣机下面。
 
　似乎是没办法怎么动弹。
 
。
 
「收纳」
 
。
 
　我将洗衣机收纳到道具包里。
 
。
 
「ギ？ギィ……？」
 
。
 
　压住自己的东西突然消失不见，哥布林神情惊讶。
 
　我露出笑容。
 
。
 
「再来一遍」
 
。
 
　在哥布林站起来之前，我又取出洗衣机。
 
　尽可能对准头部的位置―――咚。
 
。
 
「グギャ」
 
。
 
　当然，哥布林又被砸了。
 
。
 
「嗯~还活着吗……」
 
。
 
　哥布林痉挛地动着。
 
　虽然我是比起刚才还要高的位置上取出来的—，还是有点不够吗。
 
　我收纳了洗衣机，再取出来。
 
　咚。
 
。
 
「グ……ギャ……」
 
。
 
　重复这道流程。
 
　收纳、取出、咚。
 
　收纳、取出、咚。收纳、取出、咚。
 
　收纳、取出、咚。收纳、取出、咚。收纳、取出、咚。
 
。
 
　究竟重复了几遍呢。
 
　响起一阵恶心的声音。
 
　哥布林完全是一动不动。
 
　看样子似乎是死了。
 
　这一瞬间，哥布林的尸体消失不见，留下一颗蓝色小石头在走廊上。
 
。
 
≪获得了经验值≫
 
≪经验值达到一定水平≫
 
≪工藤和人的ＬＶ从1提升到２了≫
 
。
 
　脑海里响起一阵声音。
 
　看来刚才应该是升级了。
 
。
 
「呼……」
 
。
 
　我深深地吐了口气，当场瘫坐下来。
 
　啊啊，好可怕。
 
　哎呀，勉强打赢它了，可喜可贺。
 
。
 
　后背的汗水狂流不停，好可怕。
 
　不过，怎么说呢。
 
　首次意识到自己杀死魔物了，果然要做还是可以做到的。
 
。
 
　道具包真的是太厉害了。
 
　啊，结果菜刀完全就没用上。
 
一一一一一一一一一
