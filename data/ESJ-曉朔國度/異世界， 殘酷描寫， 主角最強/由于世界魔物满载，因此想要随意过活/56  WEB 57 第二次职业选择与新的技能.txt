WEB 57 第二次职业选择与新的技能
 
 
一一一一一一一一一一
 
 
 
 
 
我眺望着新显示出来的职业。
 
『凶手』、『处刑者』、『忍者』吗。
 
。
 
「……原来如此」
 
。
 
『凶手』吗。
 
嗯嗯，原来如此。
 
…………这要怎么读啊？
 
似乎没办法普通地读出来？
 
应该是きょうしゅ吧……？
 
我从道具包取出国语字典查了一下。
 
。
 
啊，「きょうしゅ」是对的。
 
意思是指目的是杀人的人，又或者是其中的手段呢。
 
原来如此，的确是『暗杀者』的上级职业。
 
。
 
「…………咕」
 
。
 
对于无法念出来的自己感到羞耻……。
 
我姑且也是社会人士。
 
词汇量也太低了吧……。
 
。
 
「咕呜？」
 
。
 
莫莫那股「怎么了」的眼神好痛苦。
 
什么事也没有，什么都没有啊，莫莫。
 
不，嘛，就是那个。这种词语在日常中根本不会出现，所以不算数。
 
就当做是这么一回事吧。
 
反正又没有别人看到。
 
我重振精神，选择职业。
 
。
 
「要选的话就是『凶手』或者『忍者』吧……」
 
。
 
『暗杀者』的强化版感觉就是『凶手』。
 
『处刑者』是提高隐秘性，提高战斗能力吧?
 
『忍者』总觉得有股NETA职业的印象，不过感觉是相当强大的职业。
 
毕竟在游戏和漫画里看到的忍者，假如再现在现实当中应该是十分强大才对。
 
。
 
「……现实中要是可以使用那种『忍术』就方便了啊……」
 
。
 
像分身之术、变化之术之类的，以及土遁弱点是雷遁之类的。
 
这种招式要是可以实际施展出来就十分方便了吧。
 
虽然忍者基本上几乎都是不会躲藏，但作为增加行动方式的其中一种方法也是物有所值。
 
。
 
「以往的行动也对某些敌人没有作用啊……」
 
。
 
像猪头人和影狼之类的隐秘行动不通用的魔物。
 
特别是高级猪头人这种完全没办法逃跑得了的怪物。
 
根本无法保证今后不会出现这样的事情。
 
在避免不了的战斗中，想要跨越难关果然就需要『力量』。
 
既然如此，偏向战斗系并且还能强化自身行动方式的『忍者』就十分具有魅力。
 
。
 
「决定了」
 
。
 
转职为上级职业需要4点数。
 
我做好觉悟，选择了『忍者』。
 
好，来吧。
 
。
 
≪暗杀者变更为上级职业『忍者』≫
 
。
 
≪职业转职为忍者。获得了技能『忍术』。获得了技能『投掷』。获得了技能『五感强化』。获得了技能『无味』。获得了技能『无声动作』。获得了技能『隐蔽』≫
 
。
 
≪技能无声移动与无声动作进行统合≫
 
≪技能无声动作从ＬＶ１升级到ＬＶ３≫
 
。
 
≪技能潜伏与隐蔽进行统合≫
 
≪技能隐蔽从ＬＶ１升级到ＬＶ２≫
 
 
 
 
 
。
 
脑海里响起天之声。
 
我在心中白痴胜利的姿势。
 
太好了，得到我想要的技能了。
 
。
 
『忍术』吗。获得技能的同时，能够使用的能力也浮现在脑海里。
 
噢噢……这还真是方便的力量。
 
尝试的时候可以期待了。
 
。
 
并且其他获得的技能似乎很不错。
 
特别是『无声动作』、『无味』。
 
似乎可以用来应付猪头人和影狼。
 
。
 
那么，接下来是第三职业吧。
 
我要在这次新出现的派生职业加上以前获得的职业中进行选择。
 
。
 
市民、冒险者、事务员、交渉者、家里蹲、无业游民、修行僧、厨师、骑手。
 
告密者、趁火打劫、魔物师（ＬＶ２）。
 
暗器师、逃亡者、偷运者、影法师。
 
。
 
真迷茫啊ー。
 
原本是想选择『魔物师』的，可这次派生职业也难以舍弃。
 
特别是『影法师』。
 
这大概是因为与莫莫共同行动而增加的职业。
 
我想应该是可以操控『影子』的职业。
 
我与莫莫的共同作战会更加强大吧。
 
。
 
可如果是基于与莫莫和小红共同作战的意义上，又想要『魔物师』。
 
『偷运者』感觉像是可以强化道具包的职业。
 
怎么办呢……。
 
。
 
「汪！」
 
。
 
「嗯……？」
 
。
 
当我烦恼着的时候，莫莫爬到我的膝盖上。
 
怎么了？
 
。
 
「汪呜」
 
。
 
莫莫看着我，操控着自己的影子。
 
该不会是在告诉我吧？
 
。
 
「莫莫推荐的是『影法师』吗？」
 
。
 
「汪！」
 
。
 
是哦，莫莫点点头。
 
嗯~……。
 
。
 
「……（抖抖）」
 
。
 
连小红也抖着身体。
 
似乎是同意莫莫的想法。
 
小红也推荐『影法师』吗……。
 
。
 
「……如果是『魔物师』，我觉得跟你们的合作会更加强大喔？」
 
。
 
「汪汪！」
 
「……（抖抖）！」
 
。
 
试着问了一下，它们都摇摇头。不，虽然小红没有头就是了。
 
它们像是挺起胸膛地在说「即便没有『魔物师』，我们自己也能做好」。
 
。
 
「好，那就选择『影法师』吧」
 
。
 
第三职业的学习点数也是需要4点。
 
我选择了『影法师』。
 
。
 
≪第三职业变更为『影法师』。获得了技能『操影』≫
 
。
 
获得的技能只有一个吗。
 
不过考虑到莫莫的案例，随着技能级别的提高，也许使用幅度和能够使用的技能也会增加。
 
反正光是操控『影子』就已经相当强大了
 
。
 
ＪＰ还有１８。
 
虽然想要使用，不过还是先提升技能那边比较好吧。
 
职业等级每当达到３、６、９，就会令附赠技能的级别提升一级。
 
既然如此，先使用ＳＰ提升技能等级后再提升职业等级的话，这样就可以节约整体消耗的SP才对。
 
。
 
于是先提升技能那边吧。
 
点数还有６３。
 
。
 
首先是将忍术、投掷、五感强化、无味、隐蔽、操影都提升到ＬＶ３。
 
这样就剩下35点数。
 
。
 
「接下来是……提升点数消耗量比较高的『观察』吧」
 
。
 
观察目前是ＬＶ９，下一级就是满级。
 
说不定还能获得上级技能『鉴定』。
 
试试看吧。
 
我兴奋不已地提升观察的级别。
 
。
 
≪观察的等级已经达到上限≫
 
≪条件已满足≫
 
。
 
好，来吧！
 
。
 
 
 
 
 
≪―――获得了技能预测≫
 
。
 
「……」
 
。
 
可恶啊！原来不是鉴定啊！
 
不不，虽然这貌似也是十分方便的技能啦！
 
是吗ー……观察的上级技能不是鉴定吗……遗憾。
 
鉴定要怎么样才能得到啊……。
 
。
 
哎，算了。
 
过去的事情再怎么后悔也无济于事。
 
并且是上级技能，应该足够派上用场吧。
 
我重新振作，提升其他的技能。
 
。
 
说起来『肉体强化』也差不多要满级了。
 
点数还有很多，也提升这技能吧。
 
能力值上升并非是坏事。
 
我提高肉体强化的级别。
 
。
 
≪肉体强化的等级达到上限≫
 
≪条件已满足≫
 
≪获得了技能肉体活性≫
 
。
 
噢噢，这技能感觉似乎很强！
 
肉体活性吗……究竟是什么效果？
 
赶紧确认下人物状态，许多能力值都提高的。
 
肉体强化是只针对『力量』、『防御』、『敏捷』、『技巧』而增加，而肉体活性则是包含ＨＰ和ＭＰ在内的所有数值进行+５的加成。
 
连一直以来都是0的魔力和魔防都有增加。
 
好厉害，真是得到相当不错的技能。
 
。
 
剩下的点数就将肉体活性提升到ＬＶ３，忍术和操影则是ＬＶ４，预测是ＬＶ２。
 
之后使用ＪＰ将忍者和影法师提升到ＬＶ４。
 
正如预料，附赠技能的级别都增加一级。

这样就好了。
 
总觉得身体充满力量。
 
力量和防御也突破三位数，现在应该变得相当强了吧？
 
不，不能大意。
 
我得脚踏实地，慎重行事。
 
绝对不能骄傲自满。
 
。
 
「接下来就是这个」
 
。
 
我从道具包里面取出『路斐尔的魔石』。
 
将它送给了莫莫。
 
虽然小红也是大活跃，但多亏莫莫我才振作起来。
 
 
 
 
 
「汪汪！」
 
。
 
莫莫很高兴地吃着魔石。
 
虽然魔石很大，但莫莫还是咬碎了，好暴力啊。
 
。
 
之后我们验证新得到的技能和武器。
 
武器是那头高级猪头人持有的菜刀。
 
道具栏目里显示的是『猪头人的菜刀（上级）』。
 
。
 
尝试了下这把菜刀的感觉以及技能。
 
忍术和操影以及莫莫新获得的技能，无论哪个都是强大的实战技能。
 
虽然与高级猪头人的战斗很艰苦，但说不定也确实是有相应的回报。
 
特别是忍术好厉害。
 
没想到在现实中还能做到那种事情。
 
简直都到了感觉白活到现在的程度。
 
。
 
「那么我们走吧」
 
。

技能充分验证完毕以后，我们继续在购物中心里探索。

如果有东西还完好无损就好了。
