翻译：飒君CONAN
 
 
 
WEB 38 我想成为纵使被盯着看也能好好说出NO的人
 
 
 
一一一一一一一一一一
 
 
 
 
 
　离开百货商店，我走在街道上。
 
。
 
「嗯~……」
 
。
 
　边走的同时，回想着刚才学生们的谈话。
 
　『早熟』所带来的增加ＳＰ和ＪＰ的效果。
 
　虽然跟其他人比较之后才明白，不过这技能比我想象中还要更作弊。
 
。
 
「虽说我能得到这技能只是单纯运气好，不过这样一来我就更想要技能的说明啊……」
 
。
 
　即便获得技能和职业，人物状态面板上也不会出现说明。
 
　因此只能实际获得并尝试一下，从名字来推测个大概。
 
。
 
「果然技能说明是需要『鉴定』的吧」
 
。
 
　『鉴定』。
 
　解析技能的技能。
 
　我重新认知到它的必要性。
 
。
 
「虽然不知道获得条件……不过感觉像是『观察』的上级技能」
 
。
 
　『观察』是我得到『密探』职业的时候获得的技能。
 
　现在的级别是４。
 
　如果提升到ＬＶ１０，很有可能会得到『鉴定』不是吗？
 
　虽然终究只是我的推测，但值得尝试一下。
 
。
 
「感知系技能，再加上『观察』的级别也都要提升啊……」
 
。
 
　以往只是满不在乎地就放任不管了，不过说到底『早熟』是『固有技能』。
 
　从这点来看就跟其他技能不同。
 
　有必要了解清楚这个技能的事情。
 
　看来需要做的事情增加了。
 
。
 
「くぅーん？」
 
。
 
　莫莫向我询问道「没事吧?」。
 
　哎呀，完全热衷于思考事情当中了。
 
。
 
「没事喔，莫莫，你不用担心啦」
 
。
 
　我蹲下来摸着莫莫的头。
 
。
 
「……汪呜」
 
。
 
　类似于「那就好」的回应。
 
　我不禁苦笑起来。
 
　莫莫真是爱操心，真可爱。
 
。
 
「嗯?那是……？」
 
。
 
　突然间，我望过去看到那里有史莱姆。
 
　就是建设在墙边的垃圾场。
 
　有几只史莱姆就混在垃圾堆中。
 
。
 
　连这种地方都有这些家伙吗。
 
　由于『危机感知』和『敌意感知』没有反应，我都直接遗漏它们了。
 
　它们并没有做什么事情，仅仅是在原地动弹着而已。
 
　啊，不对，仔细一看垃圾正慢慢地被溶解。
 
　这些家伙什么都吃吗。
 
　在幻想世界的小说里，史莱姆一般都是杂食，看来现实中的史莱姆似乎也不是例外。
 
。
 
　嗯~怎么办。
 
　要狩猎这些家伙应该很费时间吧……。
 
　反过来回报很少。
 
。
 
「嗯？」
 
。
 
　仔细一看，史莱姆里面有一只颜色不同的史莱姆。
 
　其他的史莱姆都是蓝色的，但那家伙是红色的。
 
　该不会是史莱姆的上级种？
 
　我就这样继续眺望着，而红色的史莱姆也没有在做什么，仅仅是在原地蠕动着而已。
 
。
 
「真是不可思议的生物啊……」
 
。
 
　某种意义上是谜团最深的魔物了。
 
　嘛，总之就放任不管好了。
 
　反正经验值也少，不管它也没有危害。
 
。
 
「啊，对了」
 
。
 
　稍微有点好奇心，我从道具包里面取出『垃圾』。
 
　是吃完饭以后的便当塑料盒。
 
　我把塑料盒丢给史莱姆。
 
 
 
 
 
『……』
 
。
 
　史莱姆并没有反应―――不，有反应了。
 
　它们将塑料盒覆盖住并将其溶解。
 
　……虽然溶解非常慢就是了。
 
。
 
「嘿ー……」
 
。
 
　我仔细地端详它们的样子。
 
　差不多经过十分钟，史莱姆吃完塑料盒了。
 
　它们抖动着身体，又继续到垃圾场吃起垃圾。
 
。
 
「这些家伙真能吃啊」
 
。
 
　有种在看着大型微生物的感觉。
 
　嗯？怎么了？
 
　颜色不同的红色史莱姆缓慢地往这边过来了。
 
　『敌意感知』和『危机感知』都没有反应。
 
　姑且提高警惕离开距离。
 
　莫莫也准备好随时进入战斗态势。
 
。
 
≪红色史莱姆注视着您，似乎是想要成为伙伴。请问需要加入为伙伴吗？≫
 
。
 
　脑海里响起天之声。
 
。
 
「诶？」
 
。
 
　一瞬间我以为是听错了。
 
。
 
≪红色史莱姆注视着您，似乎是想要成为伙伴。请问需要加入为伙伴吗？≫
 
。
 
 
 
 
 
　天之声再次在脑海中响起。
 
　并不是我听错了。
 
。
 
　真的假的，魔物也有这种事情发生吗。
 
　话说回来，果然史莱姆也具备『意识』啊。
 
　史莱姆抖动着身体注视着这边……在我看起来是这样。
 
　该不会认为是我给它诱饵的吧？
 
。
 
「嗯~……」
 
。
 
　可是啊……。
 
　老实说很微妙。
 
　虽然网络小说里史莱姆成为同伴是王道剧情，可是在现实中就有点……。
 
　看着伫立在垃圾场的红色史莱姆，我思考着。
 
　虽然是无害，但也没有什么特别的好处啊……。
 
　不如说考虑到我技能的特性，有它在反而会碍事吧？
 
　『潜伏』和『气息消除』的效果只针对我。而躲在『影子』离的莫莫虽然是例外，但也适用于这点。
 
　而且它走得很慢，还非得背着它走动不可，战斗中似乎也是派不上用场。
 
　有它也是正如字面意思上的包裹。
 
。
 
　我在脑海里选择NO。
 
。
 
≪申請已经拒绝了≫
 
。
 
『……』
 
。
 
　是我的心理作用吗，感觉红色史莱姆似乎有点悲伤似的抖了抖。
 
　大概是我的错觉，我很想这么认为。
 
　我的错觉，绝对
 
。
 
「……我、我们走吧，莫莫。去寻找其他的魔物吧」
 
。
 
「……汪」
 
。
 
　嗯，没有做错。
 
　我应该没有做错。
 
　再次迈开步伐后，肚子响了起来。
 
。
 
 
 
 
 
「……说起来，现在差不多到中午了吗」
 
。
 
　看着手表上的时间，已经过了十二点了。
 
。
 
「总而言之，先休息一下吧，莫莫」
 
。
 
「汪」
 
。
 
　寻找似乎可以休息的建筑物。
 
　……那边就好吧。
 
　非常近的一座高层公寓。
 
　背对着抖动中的史莱姆，我和莫莫往那边走去。
 
 
 
 
 
。
 
 
 
 
 
　另一方面，这时候在某座公寓屋顶上―――
 
。
 
「……果然在可以看见的范围内，魔物变得越来越少了呢」
 
。
 
　通过照准器看着城市风景的同时，『她』嘀咕道。
 
　魔物出现的第三天。
 
　她从来没有从这座公寓踏出一步，一直都在这里反复进行狙击。
 
　她的努力有了回报，等级提升了相当多，点数也得到了许多。
 
。
 
　然而差不多也该到了极限。
 
　射击范围内出现的魔物数量变少了。
 
。
 
「差不多该离开了吧……」
 
。
 
　考虑到自己的职业和技能，还是不要离开这里比较好。
 
　然而即便呆在这里，目前也已经要到达极限了。
 
。
 
「使用点数强化技能和数值，然后找到目标建筑物，接着继续蹲在那边就好了吗……」
 
。
 
　只要找到符合职业条件的公寓或者高层大厦，然后将那里作为新的据点就好了。
 
。
 
「可以的话，还真想要帮手啊。……话说回来，那个男人和汪酱还活着吗」
 
。
 
　她想起昨天来到这里的男性和柴犬。
 
　拥有着有趣的技能。
 
　虽然到最后她都没有上去搭话，他们直接就站起来离开了。
 
。
 
「……嘛，毕竟是人渣，我觉得大概还活着就是了」
 
。
 
　不对，与其说是人渣，不如该说是阴险狡诈吧。
 
　从超市的战斗方式以及在这里看到他的反应，她是这么觉得。
 
　为什么会这么觉得。
 
　那是因为自己也是这样。
 
　不想死，想要活着死。
 
　所以才会打倒魔物，提升等级，谋划出掌握到安全的方法。
 
　话又说回来，她又如此想着。
 
。
 
「那只汪酱好可爱啊……」
 
。
 
　紧紧陪伴在那男人的身边，晃着尾巴的样子。
 
　那是多么的可爱。
 
　她很想摸摸它，很想受到治愈。
 
　她并非猫党，而是狗党。
 
　特别是柴犬就是她的最喜。
 
　如果不是这座公寓禁止饲养宠物，她自己也想养一只这么可爱的汪酱。
 
　她恍惚着，脸颊都泛起红晕。
 
　仿佛就像是恋爱中的少女。
 
。
 
「……真想再见它一面啊，那只汪酱」
 
。
 
　在她的脑海里，男人的容貌早已经暧昧不清，而莫莫的样子则是美化了两百倍左右。
 
　怕生的人，基本上都不会记住别人的长相。
 
。
 
「―――『地图』」
 
。
 
　如此嘀咕后，她的人物状态面板发生变化。
 
　变成显示成立体的城市地图。
 
。
 
「……嗯，这里就好」
 
。
 
　离百货商店有点距离的某座高层公寓。
 
　如果是这里就可以符合自己职业的条件。
 
　虽然有必要再凑齐家具和食物，嘛，这些就先不管吧。
 
。
 
「那么就出发吧」
 
。
 
　她把手法已经完全熟练的长枪背负在肩膀上。
 
　如此这般，她一之濑奈津为寻求新的狩猎场而行动。
 
　其心里的某处在祈祷着与那只柴犬的再次相遇。
 
一一一一一一一一一
