WEB 37 与他人比较
 
 
一一一一一一一一一一
 
　到达百货商店了。
 
　我藏在附近，窥视着情形。
 
。
 
「放哨的人数比起昨天还少啊……」
 
。
 
　……门口有两人。
 
　而且其中一个人并不是学生，总觉得有点老，衣服也不一样。
 
　大概是避难民。
 
　应该是昨天跟学生一起与魔物战斗的人吧。
 
　等级提高，得到技能，于是被他们认同是同伴吗。
 
。
 
「再稍微走近一点看看吧……」
 
。
 
　『索敌』的等级也提升了，现在只要接近到墙边应该就可以知道里面有多少人吧。
 
　多亏『气息消除』和『无声移动』，那些放哨的人并没有注意到我。
 
　跟昨天一样，我轻松地来到窗户的旁边。
 
。
 
「……人的气息还真是少呢」
 
。
 
　『索敌』范围内可以感觉到的气息顶多也就５～６人。
 
　与昨天相比，明显少了很多。
 
　因为这附近的物资已经不多了，所以提高搜索范围了吗？
 
　毕竟昨天西野君也说过类似的事情。
 
。
 
　是说没有西野君的气息啊……。
 
　他也出去了吗？
 
　虽然他看起来不像是会不顾别人劝解出门探索的类型就是了……。
 
。
 
「要怎么办呢……」
 
。
 
　对我来说既是奢求的状况，也是不愿发生的状况。
 
　虽然我现在只能选择赶紧拿走物资，立马逃跑就是了……。
 
。
 
「不过毕竟才过了一天，应该还在戒备我吧……」
 
。
 
　昨天的商讨中，西野君明显是在警惕着『拿走物资的人物』―――也就是我。
 
　并且他还告知很有可能会来到百货商店 。
 
　他就是在大家的面前如此公告的，正确地来讲他是向学生们这么说。
 
　在这种情况下，真的可以漫不经心地进去吗？
 
　说不定在昨天的魔物战中，已经有人升级并得到探知系的技能了。
 
。
 
　嗯~，可是我都难得来到这里了，什么收获也没有也太那个……。
 
　在施工现场已经得到一些不错的东西了，所以老实说还真绰绰有余。
 
　而且还得到『保存功能』，根本不用烦恼食物的问题。
 
　也就是没必要冒着风险这么贪婪资源……吗。
 
。
 
　还是说在不会暴露的程度下偷偷拿走一点？
 
　外面放着一些园艺用的木桩和方材。
 
　这附近还有很多，就算稍微拿走一点也应该不会发现吧。
 
。
 
「―――……技能―――……」
 
。
 
「……嗯？」
 
。
 
　我听见不远处传来的声音。
 
　而且还听到『技能』、『职业』等词语。
 
。
 
「在销售场内吗？」
 
。
 
　我慎重地贴着墙边移动，发现到他们是集中在场内的某处说着什么。
 
　姑且确认一下周围有没有人。……应该没问题。
 
　我把耳朵贴在墙上，发动『聆听』技能。
 
。
 
「―――那么我再说一遍。首先所有人都要打开人物状态面板。在心中默念『人物状态』应该就会出现才对」
 
。
 
「呜喔！真的、真的出现了喔！」
 
「哈?你在说什么……真的出现了!?」
 
「不可思议的面板呢……这究竟是怎样的结构？」
 
。
 
「啊—，我们姑且是称之为人物状态面板。千万要注意别让别人看见自己的面板」
 
。
 
　……这。
 
　该不会是在教授这些已升级的人关于技能的知识吧？
 
　声音的主人应该是昨天那些不良学生的其中一个吧。
 
　感觉在门口放哨的学生当中似乎有这家伙的声音。
 
。
 
「我待会再来说明各个项目，所以先看看写着ＳＰ、ＪＰ的地方。你们的等级都是显示１吧？」
 
。
 
　学生向避难民询问道。
 
　虽然没有回应，但我想他们大概是点头了。
 
。
 
「昨天西野应该是说过了，无论是谁只要打倒一只魔物就绝对会升到一级。这样就可以看到这个人物状态。你们应该都听到脑海离响起的声音吧？『获得了经验值』」
 
。
 
「啊、是啊……。不过这又怎么了?」
 
。
 
「大叔，接下来才是重点。项目下面有ＳＰ、ＪＰ吧，可以通过这些来选择职业和技能。简单来说，就是可以得到与魔物战斗的力量」
 
。
 
「战、战斗的力量……？」
 
。
 
「没错，ＳＰ应该是2点数，ＪＰ则是有1点数才对。你们所有人都是这样没有错吧？」
 
。
 
「是、是啊，有的」
 
「我也……」
 
「我也是」
 
。
 
「这样就可以选择一种职业和两种技能。你们点击一下技能和职业的栏目。在那边显示出来的就是你们可以获得的能力与职业」
 
。
 
　学生们再次确认面板。
 
　……嗯?话说，给我等一下。
 
　ＳＰ２点数、ＪＰ１点数？
 
　记得我初期点数是20点ＳＰ、10点ＪＰ才对。
 
。
 
　这差距是怎么回事？
 
　从刚才的口吻来看，没有个体差异，所有人的初期点数应该是一样的吧。
 
　那为什么我的点数比较多？
 
　能想到的就是……。
 
。
 
「―――『早熟』的效果吗……？」
 
。
 
　经验值加成（推测）和增加获得的点数。
 
　如果这就是『早熟』带来的效果―――。
 
。
 
「这、这要怎么选才好啊？」
 
「那个，我这里出现了『公司员工』、『市民』、『冒险者』、『高尔夫球手』等多种来着……？」
 
「那、那个……我的技能项目也有好多―――」
 
。
 
「啊，那么不好意思，能不能写在纸上。千万不要故意写漏喔。再过一会西野就回来了，还是商量一下再做决定比较好。如果是那个人，应该会为你们选择出最适合生存的。总之不要擅自选择技能和职业」
 
。
 
 
 
 
 
避难民都在跟学生商量关于自己的技能和职业。
 
　从听到的来看，似乎没有人拥有稀有的职业和技能。
 
　也没有人具备道具包。
 
　难不成这是相当稀有的技能吗？
 
。
 
　话说以往是没有比较对象所以才没办法判断，该不会我是属于相当幸运的？
 
　『早熟』和『道具包』。
 
　毕竟从最初开始我就已经得到这两个技能了。
 
。
 
「……嗯？」
 
。
 
　『索敌』有反应了。
 
　这股气息是……西野君他们吗。
 
　看样子是回来了。
 
。
 
　反正也得到了相当有意义的情报，该撤退了吧。
 
　走的的时候稍微在不会被发现的程度下借走了一些木桩和方材。
 
。
 
　在学生们发现我之前，我已经离开这个地方。
 
一一一一一一一一
