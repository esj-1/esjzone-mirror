WEB 73.各自的关系性
 
 
 
「为什么……为什么？」
 
。
 
犹如在说梦话一样，一之濑小姐重复着同样的言语。
 
该不会是熟人……不对，难不成是刚才说过的关系不错的朋友？
 
……那孩子吗？
 
虽然在我看来完全就是跟一之濑小姐相反的类型……。
 
。
 
不，这件事以后再说吧。
 
现在重点是黑狼那边。
 
怎么办？该追上去吗？
 
现在还能追得上。
 
西野君他们并不知道那家伙的同伴几乎是半毁的状态。
 
黑狼自己也是相当虚弱。
 
这是解决它的大好机会。
 
。
 
「汪」
 
。
 
莫莫已经从影子里出来，咬着我的衣摆催促着我。
 
莫莫应该也是赞成追上去吧。
 
无论何时都挺积极的。
 
。
 
好，我决定了。
 
利用莫莫的嗅觉追上那家伙。
 
之前是被它们追得半死。
 
也轮到我们反击了吧。
 
这次要解决掉它。
 
。
 
「一之濑小姐，我们要追上黑狼，请上来我背上吧」
 
。
 
毕竟还是背着她比较快。
 
而且一之濑小姐不怎么有胸部的感触，所以奔跑的时候也不会分散注意力。
 
好，COME ON。
 
。
 
「……啊咧？一之濑小姐？」
 
。
 
没有回应。
 
究竟是怎么了？
 
往背后一看，一之濑小姐依然还是注视着六花。
 
。
 
就那么在意吗……喂！你在干嘛啊！
 
不敢相信，一之濑小姐挥开我的手。
 
随即以踉跄地步伐走到前面，开着口、
 
。
 
「六―――」
 
。
 
笨―――尽管我想要制止，但来不及了。
 
 
 
 
 
「～～～～～！（抖抖抖）」
 
 
 
 
 
然而在刹那间，在一之濑小姐身上的小红分身将她的口堵住。
 
并且莫莫也随即利用『影子』将一之濑小姐束缚住。
 
。
 
「唔ー唔ー！」
 
。
 
「抱歉了，你就暂时维持这样吧！」
 
。
 
我抱着一之濑小姐全力奔跑。
 
立刻离开了这里。
 
好危险，差一点我们就要暴露了。
 
。
 
「你应该是遇到旧识了吧，不过现在请忍耐一下」
 
。
 
「……」
 
。
 
「在场还有那些学生会的成员，以及其他许多的学生。现阶段暴露我们的身份并非是上上策」
 
。
 
「……」
 
。
 
「反正你也看到熟人了，应该也在『邮件』的送信对象之内了吧。待会利用这种方式进行联络就行。如果实在是很想见面，想要指定秘密地点的方法要多少就有多少。所以现在还请忍耐一下」
 
。
 
我边奔跑着，边拼死地说服一之濑小姐。
 
……啊，完了。边说话边奔跑好痛苦。
 
侧腹那边超痛的。
 
明明数值提升以后都可以抱着一位女性奔跑，为什么这方面还是在常识范畴之内啊，可恶。
 
。
 
嗯？话说回来，等一下啊？
 
其他人要是看到我们现在的样子会怎么想？
 
。
 
戴着摩托车头盔的男人（我）拿着枪支（一之濑小姐的），抱着一位遭受黑色绳子（莫莫的影子）所捆绑的美少女（一之濑小姐），并且嘴里还塞着东西（小红的拟态），气喘吁吁地全力奔跑着。
 
。
 
……完了，这种情形怎么看都是出局了。
 
完全就是绑架事件。
 
嘛，算了。
 
反正因为技能的缘故，谁也不会注意到。
 
。
 
我们抵达距离学校有些距离的地方。
 
来到这里应该就没问题了吧，周围也没有人类的气息。
 
解绑一之濑小姐。
 
。
 
「……咳，哈啊哈啊哈啊……」
 
。
 
「很抱歉对你做了这么粗暴的事情，不过―――」
 
。
 
「对、对不起！」
 
。
 
将一之濑小姐抱下来的瞬间，她土下座了。
 
啊咧？
 
。
 
「对不起，因为我的擅自行动给和人先生添麻烦了……。明明都组成队伍了……真的很对不起」
 
。
 
她声音颤抖，如今似乎都要哭泣一样。
 
别这样，女性要是哭出来，总觉得会无条件地变成是我这边做错了一样，所以真的别这样。
 
话说回来，她怎么了？
 
跟刚才的态度完全判若两人。
 
啊，难道说是冷静下来，于是回到平时的状态了吗。
 
。
 
「没、没事……总之都勉强赶上了，把头抬起来吧」
 
。
 
「不行，真的对不起……」
 
。
 
所以说别这样。
 
毕竟罪恶感好强，虽然不知道是为什么。
 
。
 
「总、总而言之，姑且休息一下吧。你看，刚好在那边有没人住的房屋」
 
。
 
现在这种情形根本没办法追踪黑狼。
 
为了让她情绪安顿下来，先休息一阵子吧。
 
我们走到无人的房屋里面。
 
。
 
「你应该口渴了吧，来，请喝吧」
 
。
 
「……」
 
。
 
我从道具包取出罐装咖啡递给一之濑小姐。
 
デミタス◯ランデ（Demitasse）微糖。又甜又美味。
 
似乎是只限定于东北的商品，是真的吗。
 
。
 
「汪」
 
。
 
「好的好的，莫莫也有份喔」
 
。
 
虽然不是罐装咖啡，而是水就是了。
 
把水倒进碗里，再拿给莫莫。
 
随便找个地方坐下来，我也吐了口气。
 
嗯，好甜。糖分和咖啡因在身体中渗透着。
 
。
 
「稍微休息一阵子就出发吧。现在应该还可以借助莫莫的嗅觉追踪得到才对」
 
。
 
「汪」
 
。
 
莫莫点点头。
 
嗯，那就拜托你了。
 
我摸着莫莫的头后，它就身体瘫软了，真可爱。
 
。
 
「……你不问吗？」
 
。
 
一之濑小姐翻弄着手中的罐装咖啡，如此嘀咕着。
 
。
 
「问什么？」
 
。
 
「那个，刚才的事情……」
 
。
 
我挠挠自己的脸颊。
 
。
 
「是呢，要是说我不在意那就是在说谎……」
 
。
 
老实说超在意的。
 
不过―――
 
。
 
「我不想因为硬是打听内幕，导致损害到现在跟一之濑小姐的关系」
 
。
 
「……」
 
。
 
从今以后我还想和她作为队伍的成员处在和谐的关系。
 
而且这也是她自己的问题，并不是我可以漫不经心地就可以插手的。
 
无论是谁都有着不想告知的心事。
 
我也是一样。
 
。
 
「所以就交给一之濑小姐了。想说的时候就请说出来吧，我会等到那时候」
 
。
 
想说就说出来。
 
不想说也不用说出来。
 
这不是由我来决定的。
 
所以我选择等待。
 
。
 
「……好的，非常感谢」
 
。
 
我说完以后，一之濑小姐微微露出笑容。
 
应该已经没事了吧。
 
。
 
「那么我们出发吧，还要追上黑狼，可以吧？」
 
。
 
「好的」

追上黑狼吧。
