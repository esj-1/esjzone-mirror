WEB 25 自卫队VS高级猪头人
 
 
一一一一一一一一一一
 
　我使用『远望』技能观察着直升机的状况。
 
　直升机已经飞到购物中心的附近。
 
　幸好没有那么多障碍物。
 
　我才可以清楚看到购物中心的周遭。
 
。
 
「……那架直升机不怎么要降落呢」
 
。
 
　直升机正围绕着购物中心飞行。
 
　嘛，毕竟到处都是高级猪头人的叫声造成的损伤，所以应该在确认周围是否有魔物吧。
 
。
 
　至少在空中可以看见的范围内是没有那么危险的家伙。
 
　毕竟高级猪头人它们就在购物中心里面。
 
。
 
　购物中心里面大概已经是成为地狱了吧……。
 
　晚一步逃跑的人绝对已经被杀了。
 
　假如就算还活着，女性也应该成为猪头人的RBQ了。
 
　根本谈不上是……活着吧。
 
。
 
「哦，要降落了吗」
 
。
 
　一直在盘旋飞行的直升机终于要降落在停车场附近了。
 
　从中逐渐出现疑似自卫队员的人们。
 
　一……二……总共是五人吗。
 
　奇妙的是人数就跟当初在那座购物中心与猪头人战斗的人们是一样的。
 
（颯:虽然我记得是十个人）
 
　他们举着枪支，丝毫不敢大意地环顾四周。
 
　果然好厉害，没有一丝无谓的动作。
 
　与此同时，我也对于他们即便世界变成这样也依然在履行自身职责这件事感到纯粹的尊敬。
 
　我认识的朋友里面也有原本是自卫队的家伙，从那家伙身上听说到训练内容根本不是闹着玩的……我绝对是做不来这种事情。顺便一提，那家伙受伤了，结果医生嘱咐别再继续干下去，于是退役了。
 
。
 
「他们有等级和技能吗？」
 
。
 
　世界已经成这样子。
 
　在日本最有机会与魔物正面抗衡的就是他们这些自卫队或者警察了。
 
　毕竟他们有着文明的武器——枪械。
 
。
 
「既然车子撞死魔物都能得到经验，那用枪杀死魔物也一定能得到经验吧」
 
。
 
　自卫队的成员们保持队形，行事谨慎地前进。
 
　从他们的行动来看，显然已经跟魔物多次战斗过了。
 
　只不过有一件事情比较在意。
 
。
 
「……啊咧？怎么有的人拿着枪，有的人就没有呢？」
 
。
 
　不对，正确来说没有拿枪的人是举着小刀的。
 
　……为什么装备这么不统一？
 
　这世界最值得依赖的武器不就是枪械。
 
　为什么要拿起小刀。
 
　装备不够吗？不对，怎么可能有这种蠢事。
 
　难道是有什么『理由』吗？装备不能统一的理由。
 
。
 
「汪！」
 
。
 
　莫莫叫道。
 
　我心想怎么了，往它的视线看去。正好有一头猪头人从购物中心出来。
 
　完全是放松警惕了吧。
 
　它正打着哈欠。
 
。
 
「猪头人没有察觉到吗？」
 
。
 
　啊啊，对了。从它那边的位置是看不到自卫队的吧。
 
　明明就有直升机的声音，没有提高警惕吗？
 
。
 
　走到外面，猪头人才将视线望向自卫队。
 
　看样子它终于注意到他们了。
 
　然而在那一瞬间，自卫队的成员早已经行动了。
 
。
 
　他们举起枪对准猪头人，一同开枪。
 
　在弹雨的攻击下，猪头人当场倒下了。
 
。
 
「好厉害啊……这不是单方面碾压吗……」
 
。
 
　猪头人的战斗能力再怎么估算也有哥布林的几倍。
 
　然而这么简单就死了……果然枪械真是开挂。
 
　气绝身亡的猪头人消失不见，当场留下一颗蓝色魔石。
 
　猪头人的魔石也是蓝色的吗。
 
　是说这种距离都能清楚看见，连我自己都惊叹『远望』的能力。
 
　等级提高以后，究竟能看到多么远处的东西呢。
 
。
 
　也许是听到开枪声了，购物中心的大门又出现两头猪头人。
 
　然而在弹雨的攻击下，两天猪头人也是立马死亡。
 
　蓝色的魔石掉落在地面上。
 
。
 
「嗯？刚才他们明显是在猪头人从大门出来之前就已经做好开枪准备了吧……？」
 
。
 
　这已经不是反应力可以说得通的行动了。
 
　而是从最初开始就已经知道，所以配合猪头人出现的瞬间一同开枪，应该是这种感觉的行动。
 
　已经確定，那些自卫队的成员是技能持有者。
 
　虽然不知道是『索敌』还是『敌意感知』，总之其中有队员是拥有感知系的技能吧。
 
　与我不同，他们还能分享彼此之间的情报。所有人肯定是有着某种职业和技能吧。
 
。
 
「……这不是行得通吗，按照这样子的话」
 
。
 
　我无意识地握紧拳头，露出笑容。
 
　压倒性的火力。
 
　远距离的攻击手段。
 
　团结的行动再加上有等级与技能可以辅助他们。
 
　我不认为只能近距离战斗的猪头人还有胜算。
 
。
 
「这样就―――」
 
。
 
　安全了。
 
　当我这么想着。
 
　下一瞬间。
 
 
 
 
 
。
 
 
 
 
 
　―――听到了『咆哮』。
 
 
 
 
 
。
 
『――――噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢!!!!』
 
 
 
 
 
。
 
　咆哮的回荡声连离那里较远的这里都传到了。
 
　大气颤动，大地摇晃。
 
。
 
　我反射性抱着莫莫，准备抵抗冲击波。
 
　而莫莫也立刻展开『影子』。
 
。
 
「没、没事的，莫莫。距离这么远应该是没效果的……」
 
。
 
「咕——……」
 
。
 
　莫莫担心似的看着我。
 
　不会错的，是『那家伙』。
 
　是那头高级猪头人拥有的『叫声』技能。
 
。
 
「―――!?自、自卫队呢……？」
 
。
 
 
 
 
 
他们没事吧？
 
　我撑起身子，看着购物中心那边。
 
　虽然他们多少有些身体颤抖，也有跪下来的成员，但所有人都还活着。
 
　而在队伍最后面的疑似队长的人物，他指着飞机那边发出什么指示。
 
　到底在说什么？
 
　果然连我也听不到他们在说什么了。
 
。
 
「……高级猪头人呢？」
 
。
 
　在哪里？
 
　我不认为那道叫声是在建筑物里面释放的。
 
　高级猪头人应该就在外头的某处才对……。
 
。
 
「什……!?」
 
。
 
　我的猜测是对的。
 
　爆炸声和冲击力。
 
　自卫队乘坐的直升机发生爆炸了。
 
　被从天而降的某种『什么』给破坏掉了。
 
　在猛烈燃烧的火焰与黑烟之中，一头红铜色肌肤的高级猪头人悠然地现身。
 
　其手上握着一把巨大的菜刀。
 
。
 
「不会是跳起来将直升机踹下去的吧……？」
 
。
 
　到底是什么脚力啊，那头怪物……！
 
　它另一只手上握着疑似直升机驾驶座上的男性尸体。
 
　高级猪头人毫无做作地讲尸体投掷出去。
 
　明明看起来就像是轻轻用手一丢，然而尸体却以不同寻常的速度飞向他们。
 
。
 
　伙伴的尸体被丢过来。
 
　一瞬间产生了动摇。
 
　其中一位队友没有避开，随着尸体一同被撞飞到购物中心的墙壁上。
 
　从墙上滑落下来以后，队员就纹丝不动。
 
　是晕过去了吗，还是死了呢。虽然在这里看不出来，但他大概是不行了……。
 
。
 
　拿枪的两名队员边大喊着，边开枪。
 
　然而高级猪头人将双手的手腕交叉于脸前，直接在弹雨中突破出来。
 
。
 
「开玩笑的吧……!?」
 
。
 
　高级猪头人悠然自得地在猛烈的弹雨中前进。
 
　没、没有效果吗？
 
　不对，仔细一看。子弹打中了，也伤到它了。
 
　然而也就仅此而已，根本还不是致命伤。
 
。
 
　……嗯？怎么回事？
 
　子弹打中的瞬间，我看到高级猪头人的身体上浮现出像是血管的东西。
 
　某种技能吗……？
 
。
 
　总之可以轻松夺走猪头人性命的枪械，在高级猪头人面前根本是毫无作用。
 
　它就这样接近自卫队，用手上那把巨大的菜刀将持枪的两人的脑袋给砍了。
 
。
 
　疑似队长的人物举起小刀，接近高级猪头人。
 
　那把小刀将高级猪头人的腹部切开―――并没有。
 
　反而是被它抓住了手腕。
 
　尽管那人拼命抵抗，但高级猪头人并没有放手。
 
　他那被抓住的手腕弯向奇怪的方向，露出痛苦的表情。
 
　接着就这样无力地被砍掉脑袋，气绝身亡。
 
。
 
　剩下一个人。
 
　他是刚才因为咆哮声而屈膝跪下的队员。
 
　他茫然地眺望着眼前的光景。
 
。
 
　高级猪头人缓慢地走进最后一个人。
 
　也许是完全被吓呆了，他甚至都没有任何逃跑的举动，当场瘫坐着。
 
　高级猪头人挥下了菜刀。
 
。
 
　仅仅一分钟左右，自卫队的成员全灭了。
 
一一一一一一一一一
