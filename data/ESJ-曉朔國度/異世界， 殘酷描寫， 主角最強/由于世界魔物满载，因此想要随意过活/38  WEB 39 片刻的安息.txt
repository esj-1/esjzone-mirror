WEB 39 片刻的安息
 
 
一一一一一一一一一
 
　走进公寓。
 
　虽然有居民的气息，但数量很少。
 
　似乎有人蹲在家里。
 
　话是这么说，也是在楼上。
 
　应该不用在意吧
 
。
 
　随便找个没人的房间走了进去。
 
　明明房门都上锁了，里面还是乱七八糟。
 
　地板和家具都有砍伤的痕迹，玻璃窗也打碎了。
 
　大概是哥布林干的好事吧
 
。
 
　总之我将地板上散乱的家具和玻璃碎片都收纳进道具包。
 
　确保空间。
 
。
 
「好，我们吃午饭吧」
 
。
 
「汪！」
 
。
 
　莫莫在要吃饭的时候回应会比起平时还要精神。
 
　一旦拿出它喜欢的狗粮，尾巴还会摇晃得很激烈。
 
。
 
「好，那今天就吃这个吧」
 
。
 
　我取出了比起往常还要便宜的狗粮。
 
。
 
「汪…………咕呜呜……」
 
。
 
　莫莫顿时就无精打采。
 
　尾巴也是瘫软下来。
 
　真是奢侈的家伙，已经完全会挑剔食物了。
 
　对于这种贪吃好货的家伙……。
 
。
 
「―――看好了，这边」
 
。
 
　取出比起平时还要昂贵的狗粮。
 
。
 
「！―――汪！汪汪！」
 
 
 
 
 
当看到食物的瞬间，莫莫立马露出笑容。
 
　它激烈地摇晃尾巴，仿佛在说「给我，把那个给我!」似的缠着我的脚。
 
　喂喂，你抱着这么紧我都不好活动了啦。
 
　真是拿莫莫没辙。
 
　这东西库存可不怎么多，所以只有这一次喔。真的是。
 
。
 
「那么，我也做点什么吃的……」
 
。
 
　不然从道具包里面随便拿出点方便面……。
 
　啊—，可是这种现成的食物我已经吃腻了啊 。
 
　还是做点什么吧。
 
。
 
　走到厨房。
 
　燃气炉……还能用吗。
 
　自来水似乎也没问题。
 
　好想吃拉面啊，就做炒面吧。
 
。
 
　适当地切切从超市那里拿来的蔬菜和猪肉，并炒了一下。
 
　然后放进拉面，还有天かす（油炸小吃）。
 
　添加调味料和中华だしの素（调味品）再炒一下。
 
　最后再加点白芝麻和鲣鱼干就完成了。
 
　虽然很简单，但很好吃啊。
 
　好，做好了。
 
。
 
「那么我开动了」
 
。
 
「汪！」
 
。
 
　我大口地吮吸着拉面。
 
　嗯，好好吃。
 
　这种便宜货的味道真不错。
 
。
 
　豆芽与卷心菜的清脆口感与天かす以及调味料相互结合。
 
　即便只加了点胡椒粉也十分美味。
 
　饮料是乌龙茶。
 
　适合味道浓厚的饭菜呢。
 
。
 
　莫莫也是肚子饿扁了吧，它狼吞虎咽地吃着。
 
　喂喂喂，吃得这么快小心咽着了。
 
　我边这么想着，边继续吃饭。
 
 
 
 
 
。
 
　哈——……好美味。
 
　吃完饭，我躺在地板上。
 
　自破碎的玻璃窗照射而入的阳光十分舒适。
 
　想睡觉了。
 
。
 
「汪呜ー……」
 
。
 
　喂，莫莫。
 
　我肚子上可不是你坐的地方。
 
　嘛，算了。
 
　我就这样享受着抚摸它的感觉。
 
。
 
「……」
 
。
 
　……该怎么说呢，这股安心感。
 
　在魔物满载的非日常中感觉到的日常氛围。
 
　让我重新认识到平时的生活是多么美好。
 
　嘛，虽然社**活就算了。
 
。
 
「一般情况下，连这种食物都没办法感到满足吧……」
 
。
 
　在食粮生产已经停止的当下，如果我道具包没有提供保存功能，那就只能从难以保存的食材开始利用。
 
　现在是五月中旬。
 
　这附近的食材都差不多回收完毕了，不过其他地区超市里的新鲜食品也差不多该开始发臭了。
 
　在这样的状况中，即便一直在节约食粮，最终也只会剩下没有味道的干面包和水而已。
 
。
 
　话是这么说，这种情形并不会发生在我身上。
 
　我已经确保足够份量的食粮，并且道具包目前还增加了『保存功能』。
 
　我手头上的食粮并不会腐烂，也不会有所损伤。
 
。
 
　『保存功能』似乎是维持着最初放进道具包里面的状态。
 
　比如说一碗倒进热水的杯面收纳进道具包，过了一阵子之后再取出来。
 
　于是拉面并不会冷掉，也没有泡开，维持着最初收纳进去的状态。
 
　与其说是『保存功能』，或许更应该说是『时间停止』功能。
 
　这是什么挂，神吗。
 
。
 
　虽然不知道其中是什么原理，总之这样就彻底解决食粮问题了。
 
　万一这样食粮也还是不够的话，那就只能钓钓鱼，或者从种菜开始做起了。不过那也是以后的事情吧。
 
。
 
　而且我还打算继续收集食粮，虽然不是昨天那种速度就是了。
 
　增加食粮的库存并不是坏事，要是到了魔物的胃里才令人恼火。
 
　就跟西野君说的一样，食粮在今后会成为交易的材料。
 
　……嘛，虽然我不觉得自己能够与他人交涉就是了。
 
。
 
　道具包真的是厉害。
 
　如果连『动态的物品』也能收纳就最好了……。
 
。
 
　即便是如此方便的道具包也并非是万能。
 
　依然存在不能收纳的物品。
 
。
 
　譬如说『生物』。
 
　生物根本无法收纳。
 
　我可不是拿莫莫来做实验的喔？
 
　只是随便找了只停落在附近电线上的乌鸦进行实验，结果没办法收纳。
 
　虽然植物种子、树枝和土倒是没问题。
 
　这方面果然也是通过我的认知，或者天之声判断的吧。
 
。
 
　接下来是『动态物体』。
 
　这也没办法进行收纳。
 
　如果不是静止的状态，似乎就不能收纳的样子。
 
。
 
　如果可以收纳那就是最强了吧。
 
　将掉落物收纳起来，调整方向再释放的话，那将会成为拥有可怕动能的子弹吧。
 
　现实版王之财宝。
 
　回想起某位金闪闪的英灵。
 
（飒:不用我多说）
 
。
 
　真想试一下啊……。
 
　道具包的追加功能中并没有增加能够收纳的种类。
 
　没有的东西再怎么强求也没用，不过那种招式可是装载着男人的浪漫。
 
　迟早真想变成那样子。
 
。
 
「啊，对了莫莫。魔石你也要吃吗？」
 
。
 
「汪！」
 
。
 
　刚才打倒宝箱怪的魔石，以及来这里途中得到的魔石。
 
　我全都给了莫莫。
 
　到中途为之，莫莫还是很高兴地吃着。
 
　不过……
 
。
 
「……咕」
 
。
 
「嗯？怎么了莫莫？」
 
。
 
　莫莫一直在盯着最后留下的一颗―――『宝箱怪的魔石（极小）』。它不吃吗？我这么想着看了看它，它慢慢地叼起魔石。
 
。
 
「……」
 
。
 
　然后直接看着窗外。
 
　接着―――把魔石从破碎窗户中丢了出去。
 
。
 
「啊！你在干什么啊，莫莫」
 
。
 
　就算不想吃也用不着丢掉吧。
 
。
 
「汪」
 
。
 
　然而莫莫像是在说「这样就好」。
 
　到底哪里好了啊……。我是完全搞不懂。
 
　嘛，算了。
 
 
 
 
 
出去外面的时候拿回来就好。
 
。
 
　虽然还想再稍微悠闲休息一阵子，但也差不多该走了。
 
　做好准备，将室内有的东西都全部回收。
 
。
 
「那么我们走吧，莫莫」
 
。
 
「汪」
 
。
 
 
 
 
 
　好，出发吧。
 
　打开玄关的门。
 
。
 
「嗯……？」
 
。
 
　这瞬间，『那家伙』映入眼帘。
 
　在走廊的角落伫立着，抖动着身体的是―――。
 
。
 
「……你该不会是追上来了吧……？」
 
。
 
　我如此询问后，红色史莱姆似乎感到开心似的抖了抖身体。
 
一一一一一一一一一
 
 
没有和谐字，不懂度娘乱吞什么
