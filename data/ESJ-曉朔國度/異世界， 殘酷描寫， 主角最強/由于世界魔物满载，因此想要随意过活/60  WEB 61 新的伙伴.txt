WEB 61 新的伙伴
 
 
 
一一一一一一一一一一一一一
 
 
 
 
 
我们离开购物中心，前往会面地点。
 
走路要几分钟的路程。
 
没有魔物的气息，很快就可以到达了吧。
 
。
 
「就是这里吗……」
 
。
 
毫无问题地抵达目的地公寓。
 
啊，话说回来我好像没有问过详细的汇合地点来着。
 
是在正门吗？还是说某处的无人房间里？
 
总之先给一之濑小姐发封邮件……啊，对方已经发来了。
 
点击未读，看一下内容。
 
。
 
『就在正门处』
 
。
 
简短的文字是如此显示的。
 
。
 
「诶……正门？」
 
。
 
往那边看过去，谁也不在。
 
不过……有某种『违和感』。
 
。
 
「……咕呜？」
 
。
 
莫莫也歪着头注视着正门附近。
 
嗯，总觉得好奇怪啊。
 
怎么说好呢，……这种像是看着立体绘画之类的错觉绘画时所感觉到的奇妙违和感。
 
啊，又发了一封邮件过来了。
 
。
 
『对不起，我现在就解除『技能』，还请稍微等一下』
 
。
 
「？」
 
。
 
我歪着头，随即感觉到了『视线』。
 
就跟以前与哥布林战斗的时候所感觉到的视线一样。
 
感觉是从公寓正门那里。
 
我仔细凝神一看。
 
紧接着―――。
 
。
 
「诶？」
 
。
 
景色慢慢地『扭曲』。
 
扭曲立马消失不见，从中有位黑发少女正将枪抗在肩膀上跪坐着。
 
。
 
「诶、啊……」
 
。
 
不顾混乱中的我，她站起来低下头。
 
。
 
「……像这样直接见面还是第一次呢。初次见面，我叫一之瀬奈津，还请多多指教」
 
。
 
如此说着，她低下了头。
 
。
 
「啊，初次见面。我是工藤和一」
 
。
 
我也低下头进行自我介绍。
 
虽然双方已经都认识了，不过姑且做一下形式上的寒暄。
 
。
 
「这边是队伍成员里的莫莫和小红」
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
莫莫也精神奕奕地打招呼。
 
小红依然是『拟态』为衣服的状态，所以是多少抖动了下。
 
。
 
顺便一提，一之濑小姐已经知道小红的事情。
 
在我晕倒的时候，她似乎见到它们了。
 
。
 
「那个，刚才的『技能』……是什么？」
 
。
 
「嗯，我的职业是『家里蹲』，当时就得到一个名为『认知妨碍』的技能。效果是别人难以认知到自己的存在。嘛，虽然有很多限制就是了」
 
。
 
一之濑小姐流畅地回答道。
 
原来如此，认知妨碍吗。真是方便的技能啊。
 
适合『家里蹲』的风格。
 
可话又说回来……。
 
。
 
「……」
 
。
 
「……请问怎么了？」
 
。
 
我一直在注视着一之濑小姐，于是她歪着头询问道。
 
嗯，我已经忍不住了。
 
直接打听吧。
 
。
 
「那个，我想冒然询问一下……请问您从刚才开始到现在为什么都不肯面向我呢？」
 
。
 
从刚才开始她一直都是看着旁边。
 
毕竟她容貌端正，所以这样也宛如是一幅画。不过这应该不是与他人交流时该采取的态度吧。
 
。
 
「……我想了很多」
 
。
 
一之濑小姐叹了口气。
 
那是有点疲惫的口吻。
 
。
 
「这是件令人害臊的事情，我是个怕生的家里蹲。并且是严重到光是跟初次见面的人对话就会有呕吐的冲动」
 
。
 
「哈啊……」
 
。
 
嗯，虽然在邮件交流里隐约察觉到就是了。
 
。
 
「所以我在想……这样的我能够与异性结成队伍吗」
 
。
 
「啊啊，这样」
 
。
 
「于是想到了，既然通过邮件可以正经地交谈，那只要不正脸相对不就可以普通地聊天了……我的猜测是对的」
 
。
 
这种猜测真希望不是对的。
 
斜视我的同时还说着话，我有股非常微妙的感觉。
 
。
 
「咕呜？」
 
。
 
莫莫一副完全搞不懂的样子歪着头。
 
嗯，莫莫你用不着知道哦，继续保持这么纯洁就好。
 
。
 
「于是今后还请多多指教」
 
。
 
一之濑小姐斜眼看着我，低下头。
 
。
 
≪一之濑奈津正看着您，似乎想成为伙伴，请问需要加入为伙伴吗？≫
 
。
 
不不，根本没在看我。
 
毕竟她是面向着其他方向。
 
。
 
「……」
 
。
 
哎，算了。
 
反正早就决定好了。
 
我选择YES。
 
。
 
≪申请已经受理，一之濑奈津已加入您的队伍≫
 
。
 
她应该也听到天之声了吧。
 
侧脸看起来似乎有些安心的样子。
 
看到队伍成员的栏目，记载着一之濑小姐的名字。
 
我看看……。
 
。
 
一之濑 奈津　ＬＶ２０
 
。
 
「……诶？」
 
。
 
我再一次看着窗口。
 
 
 
 
 
。
 
一之濑 奈津　ＬＶ２０
 
。
 
……诶，真的假的，等级好高!?
 
这人等级比还高四级啊。
 
究竟是狩猎多少魔物了！
 
。
 
「那个，一之濑小姐。冒昧询问一下，您究竟是怎么提升到这么高的等级―――嗯？」
 
。
 
偶然看着一之濑小姐，她身体正颤抖着。
 
我以为怎么了而看着她，结果她的脸色越来越铁青。
 
。
 
「等、一之濑小姐！你怎么了!?」
 
。
 
我不禁跑进她，双手按着她的肩膀。
 
随即看着她的脸―――
 
。
 
「……对、对不起，虽然我以为没问题……不过似乎是达到极限了」
 
。
 
「诶？」
 
。
 
『危机感知』虽然敲响警铃，但为时已晚。
 
下个瞬间、
 
。
 
「呜呜啊啊啊」
 
。
 
逆转。
 
她的呕吐物尽情地吐到我身上。
 
有股酸味，并且现场气氛极度尴尬。
 
。
 
「对……对不起……」
 
。
 
「啊，没事……」
 
。
 
如此这般，我与一之濑小姐的初次见面就是对上眼被吐了一身，实在是过于冲击性。
 
……好想哭。
 
。
 
。
 
。
 
 
 
 
 
此时在另一边―――。
 
猪头人群体目前正离开百货商店，前往购物中心。
 
理由是因为一头脱离群体，擅作主张的猪头人。
 
。
 
「路斐尔还没回来」
 
。
 
由上天赋予名称，拥有红铜色肌肤，实力在群体中也是别具一格的个体。
 
压倒性的实力与独特的价值观使得它在群体内是不同寻常的存在，即便如此也没有人质疑它的实力。
 
这样的它没有回来。
 
对于它们而言，这显然是异常事态。
 
。
 
「寻找它」
 
。
 
没有人提出异议。
 
它应该还没死吧。
 
然而很有可能陷入某种不妙的事态中。
 
。
 
特别是这几天频繁下雨。
 
路斐尔虽然很强，但它弱点反而是比起普通猪头人还要更加显著。
 
也许是因为某种理由淋到雨，陷入无法动弹的状态。
 
必须得快点找到它。
 
。
 
「……呶？」
 
。
 
突然间，一头猪头人用鼻子闻了闻。
 
紧接着其他个体也环视着周围。
 
。
 
―――被包围了。
 
。
 
「滚出来吧」
 
。
 
猪头人大叫道之后，屋檐上出现一只魔物。
 
拥有黑毛的狼―――影狼。
 
其他的个体也陆续不断地出现。
 
它们都同样登上高处，俯瞰着猪头人它们皆。
 
这反而刺激到猪头人它们的自尊心。
 
。
 
「耍小聪明」
 
。
 
弱狗一般的家伙居然俯瞰着我们，这能原谅吗。
 
纵使没有路斐尔，我们的实力也在影狼之上。
 
猪头人它们立刻举起手中的武器。
 
。
 
「杀了」
 
。
 
随着号令下达，猪头人它们进行突击。
 
它们终究只不过是狗畜生。
 
根本不是我们的敌人……才对。
 
。
 
然而随后在猪头人它们的脚边出现巨大的『暗』。
 
。
 
「什么!?」
 
。
 
猪头人它们感到动摇。
 
『暗』没多久就向四周扩散，将猪头人它们纷纷吞没。
 
怎么可能，猪头人它们都这么想着。
 
影狼操控的『影子』根本不可能有这种能力。
 
这是别的―――。
 
 
 
 
 
「……!?」
 
。
 
不经意间察觉到厌恶的气息，猪头人它们往向那边。
 
不知是何时之间呆在那边的呢。
 
在它们的前方的道路中央有一匹黑狼。
 
体型比起其他影狼还要小一圈。
 
然而其周围的气氛明显不同。
 
―――上位个体。
 
并且还是相当强。
 
猪头人它们都咽了口气。
 
。
 
「呜嗷嗷嗷嗷嗷嗷！」
 
。
 
随着黒狼的吼叫，『暗』更加扩大。
 
尽管它们拼命抵抗，但依然无法挣脱。
 
猪头人它们陆续吞没进『暗』里。
 
不光是猪头人，就连周遭的建筑物都全部逐渐淹没在『暗』里面。
 
。
 
「咕喔喔喔喔喔喔喔―――……」
 
。
 
见状，猪头人才知道影狼要呆在高处的理由。
 
它们并非是为了俯瞰自己而这样做。
 
而是为了避免遭受『暗』所吞没，因而进行避难而已。
 
。
 
然而即便它们注意到也已经束手无策。
 
在完全吞没之前，猪头人与引发这种现象的黑狼双眼交汇。
 
它的眼神，犹如在观望虫子一般。
 
到此为止了吗，猪头人领悟到自己的命运。
 
。
 
（……路斐尔，抱歉了……）
 
。
 
它临死前向应该呆在某处的同胞谢罪。
 
于是十头以上的猪头人群体全灭。
 
。
 
≪获得了经验值≫
 
。
 
脑海里响起抑扬顿挫的声音。
 
黑狼无视它，将『暗』消除。
 
它对于猪头人根本没兴趣。
 
仅仅是因为它们似乎会阻碍到自己人所前往的目的地，因此排除掉而已。
 
。
 
「……很近」
 
。
 
黑狼用鼻子闻了闻，注视着某个方向。
 
它闻到细微的气味。
 
那是自从世界变奇怪后，它所寻找的一只雌性的气味。
 
。
 
「……给我等着」
 
。
 
于是影狼群体再次行动。
 
新的威胁即将迫近。
 
一一一一一一一一一一
