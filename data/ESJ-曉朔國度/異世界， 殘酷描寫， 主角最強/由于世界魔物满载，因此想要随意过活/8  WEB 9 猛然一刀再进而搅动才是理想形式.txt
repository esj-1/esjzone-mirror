　如此这般，我单手拿着菜刀再次出门。
 
。
 
「好，努力吧我」
 
。
 
　鼓起干劲，再次出门。
 
　咔嚓咔嚓！
 
　咕，这扇门还是老样子好沉重！该死的破旧公寓！
 
。
 
　下到二楼。
 
　也许是多亏了『隐秘行动』的级别提高了，感觉自己的存在感比起刚才还要更加薄弱。
 
　配合『轻步』的效果，简直就像是忍者。
 
　不对，毕竟我还单手拿着菜刀，真要说应该是小偷？
 
。
 
「嗯……？」
 
。
 
　有几扇房门敞开着。
 
　『敌意感知』、『危机感知』有反应。
 
　哥布林从敞开的玄关门出现了。
 
　好，洗衣机，该你出场了。
 
。
 
　……不不，等一下。
 
。
 
　哥布林并不是一只。
 
　二……三、四……喂喂，总共有四只吗，好多啊。
 
　我躲藏在墙壁的阴影里，偷窥着哥布林的状况。
 
。
 
「ギギ」
 
「ギィ、ギギィ」
 
「ギャッギャ」
 
。
 
　哥布林它们似乎在说些什么。
 
　是说哥布林好厉害啊。
 
　明明世界变成这样子的时候我还是晕头转向的，可现在已经习惯了。
 
　可在哥布林看来也就只会认知到狩猎场发生了变化吧？
 
　虽然不知道它们的智商如何，也不好判断就是了。
 
。
 
　其中一只哥布林指着楼上。
 
　其他的哥布林们也似乎点点头。
 
。
 
　……难道说刚才楼上那一只哥布林是这些家伙的伙伴吗？
 
　比方说到楼上侦查之类的？于是它没有回来而觉得有古怪……？
 
。
 
　还是说它们是听到洗衣机砸死人的声音了？
 
　嗯，毕竟都发出那么大的声响。
 
　那当然会觉得古怪啦。
 
　笨蛋啊我！可恶！
 
　那也是没办法啊！首次战斗我根本就是惊慌失措的！
 
。
 
　―――完了。
 
　其中一只哥布林往这边过来了。
 
。
 
　怎么办？
 
　是要战斗吗，还是逃跑呢。
 
。
 
　不，等等。
 
　这是机会。
 
　一口气得到大量经验的机会。
 
。
 
　更重要的是那些哥布林跟刚才那家伙不同，都没有武器。
 
　我的数值也比起刚才还要高，无论是武器还是地形都是这边有利。
 
　……上吧。
 
。
 
　想要逃跑怎么样都能逃。
 
　可如果这时候逃跑也只会遭遇同样的场面。
 
　有可能一直逃跑下去吗？答案是否定的。
 
　既然如此，为了『习惯这种世界』，为了提高等级，我应该要在这里将那些哥布林打倒吧。
 
。
 
「……」
 
。
 
　一只哥布林正往这边走过来。
 
　没问题，它没有注意到。
 
。
 
　还有一点……再一点点……就是现在！
 
　当哥布林走在走廊拐弯处的瞬间，我塞住哥布林的嘴，用菜刀刺进它的心脏。
 
。
 
「―――ギッ!?」
 
。
 
　哥布林应该是松懈了吧。
 
　轻而易举地就刺中胸口了。
 
　猛然一刺进而搅动。
 
　哥布林当场倒下。
 
　也许是一击毙命吧，当场掉落一块蓝色小石头。
 
。
 
≪获得了经验值≫
 
≪技能熟练度达到一定程度≫
 
≪抗压耐性从ＬＶ１升级到ＬＶ２≫
 
。
 
　脑海里回响着声音。
 
　与此同时，刚才所感觉到的不快感和罪恶感也减轻了。
 
。
 
「ギィ!?」
 
「ギギィ！」
 
「ギィ、ギィィ！」
 
。
 
　其他哥布林注意到我。
 
　它们凶狠地进行威吓，往我这边冲过来。
 
　好，意料之中。
 
。
 
　冷静下来，仔细看着观察。
 
　最后面的房间门―――好，是关着的。
 
　其他房间也是一样。
 
　那就没问题了。
 
　我也尽情地奔赴「前方」。
 
。
 
「道具包Open」
 
。
 
　取出了『冰箱』。
 
　它在我家里的家具当中也是与洗衣机一样重，一样大的。并且还装有小脚轮。
 
　我以得到身体强化的力量尽情地将冰箱往前推！
 
（颯:必杀技《老汉推车》）
 
。
 
　「「「ギィィィ!?」」」
 
。
 
　哥布林们发出惊愕声。
 
　眼前突兀地出现一个以可怕速度迫近而来的庞然大物。
 
　哥布林们的抵抗毫无作用，直接被撞到墙上。
 
　咚！响起一阵清脆声响。
 
。
 
「呼ー…」
 
。
 
　将冰箱回收。
 
　哥布林们全都眩晕过去了。
 
　也许是撞得太狠了，其中也有一些哥布林手脚都弯向奇怪的方向。
 
　太好了，事情顺利。
 
　果然家具就是强大，必杀技就是方便。
 
。
 
「那么，最后一击……」
 
。
 
　既然没有留下石头，也就意味着这些哥布林还没有死亡。
 
　我用手上的菜刀捅进哥布林的胸口。
 
　嗯，别想太多。机器式地搞定吧。
 
　猛然一刺进而搅动。
 
。
 
　哥布林轻而易举地就死了。
 
　捡起蓝色石头。
 
。
 
≪获得了经验值≫
 
≪经验值达到一定的水平≫
 
≪工藤和人从ＬＶ２升级到ＬＶ３≫
 
。
 
≪获得了经验值≫
 
≪经验值达到一定的水平≫
 
≪工藤和人从ＬＶ3升级到ＬＶ4≫
 
。
 
　等级提升了。
 
　而且还是一口气两级，这等级提升速度。
 
　果然那个叫做『早熟』的技能绝对是有经验值加成效果。
 
。
 
「呼……」
 
。
 
　好累。
 
　话说『抗压耐性』的级别都提高了，我是感觉到相当亚历山大了啊。
 
。
 
　不过下一次就能更顺利了吧。
 
　最初的奇袭我就明白了。
 
　只要找好位置，单凭哥布林这种对手就能在不被察觉到的情况下杀了它。
 
。
 
「人物状态的确认……还是趁现在搞定比较好吧」
 
。
 
　迅速完事吧。
 
 
 
 
 
确认完周围没有魔物，我走到刚才哥布林进去的房间里。
 
　Status Open。

　感觉自己的身体好像很灵活的样子。
 
。
 
　不过无论变得有多强，老实地从正面挑战对手之类的也是错误。
 
　果然要奇袭或者暗算对手。以必杀技为重点进行战斗吧。
 
　从我性格来看感觉比较适合这种。
 
　而且我还有洗衣机和冰箱这些可靠的伙伴呢。
 
。
 
 
 
 
 
　接下来是确认道具包的功能具体如何。
 
　级别都提高了，有什么追加的功能……嗯？这种感觉……。
 
　脑海里隐约浮现出使用方法。
 
。
 
　喔喔，可以同时取出两件物品了！
 
　在我的手上同时出现苹果和圆珠笔。PPAP。
 
（颯:PPAP（Pen-Pineapple-Apple-Pen Official），简单来讲，某个搞笑艺人拿着苹果和笔跳着舞）
 
。
 
　看样子技能级别提高，能够同时取出多种物品。
 
　最大可以同时进行四件
 
　真的好方便，这是神吗。
 
　这样一来，战术选择性就更多了。
 
。
 
　之后我做过许多尝试，道具包的追加功能差不多是这样子的。
 
。
 
　１收纳的容量增多。
 
　２不同物品可以同时取出。（最多四件）
 
　３收纳、取出的范围更加广泛。（最大五公尺）
 
 
 
 
 
。
 
　啊~如果还有将收纳物品的时间停止住就完美了啊ー。
 
　嘛，奢侈的事情说再多也是无济于事。
 
　级别提高之后或许就会有也说不定。
 
。
 
　那么，下去一楼吧。
 
　随便先从这房间借走几样似乎派得上用场的东西。
 
　没事的，如果遇到原主我会还给本人的。如果还活着的话。
