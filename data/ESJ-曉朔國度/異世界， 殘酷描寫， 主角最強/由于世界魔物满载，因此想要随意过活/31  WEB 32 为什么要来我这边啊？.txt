WEB 32 为什么要来我这边啊?
 
 
一一一一一一一一一一
 
 
 
 
 
　周围魔物的气息逐渐增多。
 
　而且不光是一个种类。
 
　还有僵尸和哥布林的气息。
 
　那道远嗥的效果不光可以吸引同种的魔物，连周围的魔物也可以吗？不对……其他魔物有可能只是因为小狼它们的行动而想要采取渔翁之利。
 
　无论如何，是我大意了。
 
　也许是我判断错该撤退的时机了。
 
。
 
　好，逃跑吧。
 
　虽然四周到处都有魔物的气息，但凭借我的能力值还是可以冲出包围圈。
 
　不过在这之前，
 
。
 
「……我得对这些家伙想点办法啊」
 
。
 
「咕噜噜噜……」
 
「唔唔唔唔……！」
 
。
 
　从黑暗中出现的小狼们将我和莫莫包围了。
 
　数量是四只。
 
　应该是刚才的远嗥中最先赶过来的吧
 
　明明我已经使用『潜伏』和『气息消除』了，可还是察觉到我，也就是说……。
 
。
 
「果然是靠气味知道我的吗……」
 
。
 
　虽说正如预料，但相性太差了。
 
　我技能全部的优势都毫无作用了。
 
。
 
「话说是对面那些家伙在呼唤你们吧?给我去那边」
 
。
 
「咕噜噜噜……」
 
。
 
　虽然试着跟它沟通一下，可当然的语言根本不能相通。
 
　不如说它们警惕地逐渐接近过来。
 
　干劲十足的样子。
 
。
 
「……莫莫，要上了喔」
 
。
 
「汪!」
 
。
 
　没办法，这些家伙就在这里杀了。
 
　虽然或许是可以逃离这些家伙，可它们的嗅觉很出色。
 
　如果它们记住我和莫莫的『气味』，追踪过了就棘手了。
 
　何况它们还是群体行动。
 
　要是事态发展成无限追踪上来就麻烦了。
 
。
 
　我从道具包取出菜刀并举起着。
 
　小狼们边保持警戒，边慢慢接近过来。
 
　似乎是意图占据前后左右，一齐袭击过来吧。
 
　不过这边可是有一打多的战斗方法。
 
　小狼们再往前踏出一步。
 
。
 
「―――就是在效果范围之内了」
 
。
 
　这瞬间，我朝着小狼它们的头上释放出自动贩卖机与电器家具。
 
　必杀的家电杀法。
 
　面对头上突如其来掉落的家电，小狼他们都惊讶了。
 
。
 
「グルッ!?」
 
「ガウッ!?」
 
「――ガッ!?」
 
。
 
　三只小狼没办法避开，都被家电压在下面。
 
　与哥布林相比，防御力似乎很低的样子。
 
　响起一阵低沉声响。
 
。
 
≪获得了经验值≫
 
≪经验值达到一定的水平≫
 
≪工藤和人从ＬＶ7升级到ＬＶ8了≫
 
。
 
　天之声告知我升级了。
 
　虽然很高兴，但我现在没有从容。
 
　艰难避开刚才一击的一只小狼，往这边过来了。
 
。
 
「莫莫！」
 
。
 
「汪！」
 
。
 
　莫莫的『影子』延伸出去。
 
　影子将小狼的腿缠住，阻碍它的行动。
 
。
 
「ガァッ!?」
 
。
 
　小狼身体无法随心所欲地动弹。
 
　虽然小狼也立刻想要释放出『影子』，然而我不会给它机会。
 
　我再次朝着它的头上立刻释放出自动贩卖机。
 
。
 
≪获得了经验值≫
 
。
 
　地面掉落了四颗紫色魔石。
 
　我迅速捡起来丢给莫莫。
 
　莫莫完美地接住魔石，并直接吃下去。
 
。
 
「越来越多了啊……」
 
。
 
　小狼的远嗥仍在持续。
 
　魔物们逐渐接近过来。
 
　包括哥布林和僵尸在内，光是我能看到的就有几十只。
 
　并且也确认到大型哥布林的身影。
 
　到底是在哪里藏有这么多魔物啊……。
 
　还是尽量寻找魔物比较少的地方。
 
。
 
「好，莫莫！那边应该可以逃得掉！我们走吧」
 
。
 
「汪！」
 
。
 
　抱歉了，学生们。
 
　我走了。
 
　你们就努力活下去吧。
 
　我撇了一眼百货商店那边。
 
　学生们正合小狼和哥布林奋斗中。
 
　也有几位避难民参加战斗的样子，其中也有人拿着木材。
 
。
 
　我心想西野君究竟在哪里而看了看，他就在那些人的后面。
 
　他大声地发出指示的同时，还将手指向伙伴。
 
　那是什么？从手上散发出来的渺小光芒。
 
　被光芒照到的学生们，看起来速度和力量都提高了。
 
　该不会是支援魔法?还有这一类的技能吗？
 
。
 
「不过还真的是奇妙的行动啊……」
 
。
 
　像是在贯彻防守一样。
 
　而且……总觉得与他们战斗的魔物数量比我想象中还要少吧？
 
　明明就有那么多？
 
　其他魔物到哪里去了？
 
。
 
「嗯……？」
 
。
 
　于是我注意到了。
 
　在距离店门口一段距离的停车场中心。
 
　魔物们都聚集在那边。
 
　一位女高中生像是配合他们对魔物进行夹攻。
 
　她有着辣妹系的穿着打扮，头发则是编织成侧马尾。
 
　挥舞着两把劈刀，以魔物为对手进行奋斗。
 
。
 
「不会吧……那是什么鬼？」
 
。
 
　从那位女高中生的身体上浮现出宛如热气一般的雾气。
 
　即便是在黑暗之中，她的眼瞳也看似纯红灿烂。
 
　她的劈刀将魔物它们的身躯切开，被她踢飞的哥布林飞舞到几米高的天空。啊，看到内裤了。
 
　纵使小狼想用『影子』束缚女高中生，她也强行切断影子，继续挥舞着劈刀。
 
　魔物随着她的吼叫声陆续不断地被砍死。
 
　即便是从旁看来也是异样的光景。
 
。
 
「很明显是技能吧，那个……」
 
。
 
　『強化』……？
 
　不对，是『狂化』吧？
 
　大概是提升战斗能力的技能。
 
　该不会她就是不良团体的主力吗？
 
　其他学生像是不想阻碍到她一样，都贯彻防御战。
 
。
 
 
 
 
 
「……意外地能行嘛」
 
。
 
　学生他们的战力似乎比我想象中还要完备。
 
　按照那种情况，他们应该能自己解决吧。
 
。
 
「……哦，现在可不是悠闲地眺望战斗的时候，赶紧逃―――」
 
。
 
　如此思忖的瞬间―――骤然感到一阵寒意。
 
。
 
　『危机感知』敲响警钟。
 
　『敌意感知』产生反应。
 
　什么？
 
　有什么在这里……！
 
。
 
「呜嗷嗷嗷嗷嗷嗷嗷嗷嗷―――………」
 
。
 
　听见了远嗥。
 
　我往声音的方向看过去，在刚才我们休息的建筑物屋顶上。
 
　那边有一只魔物。
 
。
 
　月亮在它背后衬托下，它的样子甚至都带有一种梦幻感。
 
　多亏『暗视』，我清楚地看到它的长相。
 
　它比起小狼还要大上一圈。
 
　不会错的。
 
　就是我在世界变成这样子之后首次撞死的魔物。
 
。
 
「……影狼」
 
。
 
　并且它比起我当初撞死的还要更加庞大。
 
　应该有跟狮子同等程度的体型。
 
　在黑暗中也显得红色辉煌的双眸，显然是在注视着我和莫莫。
 
。
 
「所以说为什么目标是我们啊……给我去那边」
 
。
 
　虽然问了它一下，但不可能有回应。
 
　说到底，这段距离根本就不知道它能不能听到。
 
　然而在我看来，影狼似乎在嗤笑。
 
　它身体缠绕着影子，从建筑物上面下来。
 
。
 
「……看样子它是没打算放过我们呢」
 
。
 
「汪」
 
。
 
　莫名也从『影子』出来，进入战斗态势。
 
　如果不对这家伙想点办法，估计是没办法逃跑了。
 
　我依然看着影狼，迅速地操控人物状态。
 
。
 
≪请问需要消耗10点ＳＰ提高道具包的ＬＶ吗？≫
 
。
 
　选择YES。
 
。
 
≪已经消耗ＳＰ了，道具包提升为ＬＶ１０≫
 
。
 
　道具包终于达到ＬＶ１０。
 
。
 
　这瞬间，天之声再次在脑海中响起。
 
　闻言，我的笑容更加灿烂。
 
。
 
「好，上吧。莫莫！」
 
。
 
「汪！」
 
。
 
　鼓起精神，我和莫莫往前方疾驰而去。
 
一一一一一一一一一一
