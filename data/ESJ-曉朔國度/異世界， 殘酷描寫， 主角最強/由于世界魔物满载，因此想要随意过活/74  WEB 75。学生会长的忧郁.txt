WEB 75.学生会长的忧郁
 
 
 
一一一一一一一一一一
 
 
 
 
 
「―――以上便是本次骚乱的报告」
 
。
 
「……原来如此」
 
。
 
五十岚十香在学生会室里面听取怎学生们的报告。
 
报告内容是关于黑狼的袭击。
 
仔细听取伤亡状况的同时，将其中的内容都理顺清楚。
 
。
 
「田名君和加藤君的情况如何？」
 
。
 
「……两人现在都在保健室静养。只不过……以后的日常生活会受到影响，毕竟那两人都……那个，膝盖以下的部分都已经切掉了……」
 
。
 
前来报告的男学生露出忧伤的神色。
 
刚才说到的两人就是遭受黑狼袭击时，由六花所救助的受『暗』吞噬的学生。
 
虽说作为代价，他们失去了双脚。
 
即便知道不这样做就会丢失性命，但应该还是没办法理顺情绪吧。
 
。
 
「我明白了，谢谢你的报告」
 
。
 
她慰劳过来报告的男学生，让他离开房间。
 
于是只剩下学生会的成员仍留在房间里，五十岚会长深深地叹了口气。
 
。
 
「辛苦您了」
 
。
 
在旁边整理报告的女书记向她搭话说。
 
。
 
「哎，要说不辛苦也是在骗人呢，毕竟我也是人……」
 
。
 
今天已然是魔物出现后的第四天。
 
她从来没有正经地睡过觉。
 
这座学校的全体指示与方针都是由她和学生会所实施。
 
虽说一定程度上的判断都交给现场的人员自主决定，但拥有最终决定权的还是她。
 
食粮、睡觉、资源、以及其他诸如此类。需要做的事情数都数不清。
 
根本就没有时间休息。
 
如果没有『疲劳耐性』、『抗压耐性』，她这几天早就因为这些庞大的事务处理工作中倒下来。
 
。
 
（话又说回来……）
 
。
 
五十岚十香边整理着学生的报告，边思考着。
 
是关于本次的袭击事件。
 
。
 
（可疑之处还真是挺多的呢……）
 
。
 
将报告整理以后，发现几处不自然的地方、。
 
魔物突然出现，又轻易撤退。
 
早已身受重伤。
 
明明具有群体行动的习性，但却单枪匹马地冲来。
 
报告中可以发现这些令人在意的地方。
 
。
 
（……有许多可能性，不过恐怕是―――）
 
。
 
得出结论之后，响起敲门声。
 
她说着「请进」给出入室许可后，一位男学生走了进来。
 
那是位戴着眼镜，似乎很神经质的学生。
 
。
 
「失礼了，我回来了」
 
。
 
「啊啦，宫本副会长。辛苦了」
 
。
 
「我并不辛苦，比起我会长您才是更加辛苦吧。……那些资料是？」
 
。
 
「是关于刚才魔物的报告书，刚刚整理好的」
 
。
 
「……这种事请交给我和她来做吧，虽然并非没有休息，但还请您再稍微休息一下吧」
 
。
 
面对苦劝的宫本，五十岚苦笑着苦言。
 
毕竟刚才的女书记也是对她说过这样的话。
 
自己看起来有那么疲劳吗？
 
。
 
「呵呵，你的心意我心领了，不过没事的。这些做好以后，我会好好休息的」
 
。
 
「还请注意身体，毕竟万一您发生什么事就不妙了。是说这次的骚乱如果可以彻底解决那只魔物，我们也根本用不着把时间花在后续处理上。都怪西野那家伙无视我的命令……真是的，所以说我才讨厌这些不良学生……」
 
。
 
「你还是不要对他说太多的坏话比较好哦？别看他这样，他也是优秀的学生」
 
。
 
「……会长还真是对他抱有相当高的评价呢？」
 
。
 
对于她那番宛如在偏袒西野的措辞，宫本露骨地蹙起眉头。
 
。
 
「是呢，毕竟我和西野君以及相坂同学都是同个初中就读，当时他就很有名气了。定期考试总是名列第一，性格认真、行动积极，老师他们都对他赞不绝口。即便是在班上也没有任何人讨厌过他」
 
。
 
「……这是真的吗？」
 
。
 
宫本露出一副无法相信的表情。
 
五十岚见状，高兴似的笑了。
 
。
 
「呵呵，大家都会这么说，不过这是真的。……说起来相坂同学与那时候相比，外貌真是改变了许多呢。当时她还是黑发，戴着眼镜……。啊啊，真怀念呢」
 
。
 
「……过去是过去，现在是现在吧？他现在就是品行不良的学生，这点并不会有所改变」
 
。
 
「是的，不过我觉得这终究只不过是表面罢了。毕竟人的本质并不会轻易改变，他毫无疑问会成为对我们而言不错的同伴」
 
。
 
「……！会、会长！会长的意思是比起我，西野他更优秀吗!?不可能！不良学生可都是一些自我中心的一群人！根本就配不上我们这种组织着大家，领导大家的存在！证据就是那家伙的周围都是一群没用的家伙！应该立刻将他们赶出―――」
 
。
 
应该是忍耐到极限了，宫本大喊大叫着 。
 
然而、
 
。
 
「―――宫本副会长」
 
。
 
宫本吓了一跳而抖了下身体。
 
仅仅一瞬间，室内的气温陷入宛如绝对零度以下的错觉。
 
她的口吻就是如此冰冷。
 
仔细一看，五十岚不愉快似的蹙起眉头。
 
。
 
「再继续说出贬低同伴的话语，我可不会原谅你哦？」
 
。
 
「……非、非常抱歉」
 
。
 
宫本老实地低下头。
 
那是不容分说的压倒性的威压感。
 
见状，她莞尔一笑。
 
这瞬间，紧张兮兮的氛围犹如是虚假一般消失不见。
 
。
 
「请不要这么担心，并不是因为他优秀，我就再也不会信赖你。从今以后也请你要多多帮助我哦？」
 
。
 
「会、会长……」
 
。
 
这番话轻而易举地宫本恢复情绪。
 
实在是太轻松了，五十岚在内心苦笑着。
 
。
 
「那么这件事就到此为止。接下来我们就回到工作的话题吧，我有点事情想要拜托你」
 
。
 
「是什么呢？」
 
。
 
「我希望你从探索班里选出几名成员，在这座学校的周围进行搜索。无论是什么小事都无所谓，请调查一下有没有不自然的疑点」
 
。
 
「……也就是说？」
 
。
 
也许是无法揣摩五十岚的真意吧，宫本询问她这是什么意思。
 
。
 
「……我认为本次的魔物袭击是有人而为」
 
。
 
「什么！？」
 
。
 
她的发言使得宫本以及其他学生会成员都感到震惊。
 
学生会室一片议论纷纷。
 
。
 
「这、这是什么意思，会长？这次的袭击是人为的？」
 
。
 
宫本反问回去。
 
五十岚点点头，双手在桌子上缓慢地相抱。
 
。
 
「是的，暗地有人在指示……这样来考虑，本次袭击的诸多疑点就解释得通了」
 
。
 
「怎、怎么可能……有这种事……」
 
。
 
「你没办法断定没有吧?比如操控魔物的技能和职业。虽然我不是很清楚这方面，不过听说在游戏里这种职业和技能都是常有的」
 
。
 
「虽、虽然……是这样！可是这样一来，对方的目的是什么!?操控魔物袭击人类究竟有什么意义……！」
 
。
 
「意义有的吧，就是『经验值』」
 
。
 
「诶……？」
 
。
 
「我们只要打倒魔物就会得到经验值。那么就不能反过来认为魔物杀死人就能得到经验值？」
 
。
 
「这样……的确」
 
。
 
「接下来终究只是我的推测，使役魔物的情况下，魔物得到的经验值会以一定的比例分给使役者的话？」
 
。
 
战斗全都可以交给魔物处理，自己只要在安全圈内悠然自得地得到经验值。
 
如果说这种假设是正确的，那这种手段该说是相当不错吧。撇除过于心狠手辣，无法认为是人类所为这点的话。
 
。
 
「这种事……这、这样一来……」
 
。
 
宫本终于注意到事情的严重性。
 
。
 
「对，很有可能会再次进行袭击吧。正因为如此，我才想要尽可能拟定对策」
 
。
 
「我明白了，我会找到拥有探知系技能的人员组成搜索班」
 
。
 
「好的，那就拜托你了。如果这座学校周遭有可疑行动的人，请尽早通知我」
 
。
 
「好的，那么我就此告辞」
 
。
 
宫本勤恳地低下头后，离开了学生会室。
 
她也将工作分配给其他成员，于是学生会室只剩下她一个人。
 
在寂静的学生会室里，她独自思考。
 
。
 
（那么，接下来我就清查一下『内部』吧）
 
 
 
 
 
她终究只是暗示出对方是外部者的可能性，但野想到另一个可能性。也就是敌人是在内部的情况下。
 
这种情况下除了自己以外的所有人都是嫌疑者，即便是学生会的成员也不是例外。
 
正因为如此，她当时才没有将这件事说出口。
 
。
 
（真是的，虽然不知道是谁，但还真是给我添麻烦了呢……）
 
。
 
是外部的人呢，还是内部的人呢。
 
无论是哪一边都极其棘手。
 
不对，是更加难以解决
 
。
 
（啊啊，真令人厌恶、令人厌恶、令人厌恶。该死的畜生，是谁允许过你对我可爱的所有物出手的）
 
。
 
愤怒的情感、怨恨之念逐渐涌现。
 
五十岚十香是支配者。
 
最喜欢令他人服从自己、服侍自己。
 
他人带着敬意面对自己的样子，无论何时都会令她感到愉悦。
 
最近利用自己的技能让西野屈服自己的瞬间，一股难以言喻的快感便充斥全身。
 
。
 
这座学校是为了自己而存在。
 
这些技能、学生会、学生们、避难民都全部是为了自己所支配而存在。
 
如果能作为支配者存在于世界上，她可以不辞劳苦。
 
毕竟她认为这是处于高处之者的理所当然的义务上。
 
。
 
而正因为如此，她无法忍受自己支配下的人们受到伤害。
 
拥有一切决定权的是自己。
 
无论是想要爱惜，亦或是取乐，还是惩罚。
 
然而这位污秽的大罪人无礼地将这些都糟蹋了。
 
她不可能会原谅。
 
。
 
（使役魔物的人物……该说是『魔物师』吧……我绝对不会让你逞心如意，绝对会抓到你的狐狸尾巴）
 
。
 
对素未谋面的敌人燃起敌意的同时，五十岚十香正拟定着今后的对策。
 
一一一一一一一一一一
