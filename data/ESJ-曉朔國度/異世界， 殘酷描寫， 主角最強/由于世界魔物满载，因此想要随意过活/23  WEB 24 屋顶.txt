翻译：飒君CONAN
 
 
 
WEB 24 屋顶
 
 
一一一一一一一一一一
 
 
 
 
 
　怎么办？
 
　应该要追上去吗？
 
　可是那架直升机前往的方向是那座购物中心，里面就有那头高级猪头人。
 
。
 
　虽然只是我的猜测，但那家伙还呆在那里。
 
　证据就是我的『危机感知』正敲响着警钟。
 
　还不能到『那边』去。
 
　绝不能过去，它如此诉说着。
 
。
 
　上次我是好奇心使然，无视这道『警钟』擅自行动。
 
　结果就是遇到高级猪头人，甚至都遭遇生命危险。
 
　我不是才刚反省过自己，决定不再靠近那边吗。
 
。
 
　可这件事真的能构成我不追上自卫队的理由吗？
 
　不……不过……可是啊。
 
　日本的自卫队很优秀。
 
　听说肉搏战方面在世界上也是名列前茅，而且应该还拥有枪械武器吧。
 
　自卫队拥有杀伤力远比自动贩卖机和洗衣机还要强大的武器，即便魔物多少有点不好对付，我也不认为他们会输。
 
。
 
「嗯~……」
 
。
 
　我烦躁的挠挠头。
 
　怎么办？追上去吗？
 
　还是说要找个地方暂且观望情形呢。
 
。
 
　现在我就有『远望』技能，即便不靠近也可以偷窥到状况。
 
　这附近比较高的地方……如果是公寓的屋顶说不定就可以利用 『远望』俯瞰到购物中心的情形。
 
　判断是否安全之后再来行动也不迟吧？
 
　嗯，大概这样才好。
 
　应该要安全第一。
 
。
 
「这样一来……那座公寓大楼附近就刚刚好吧 ？」
 
。
 
　从商业合作社的正面延伸百米左右的地方有一座公寓大楼。
 
　如果是那边的屋顶应该就可以观望到远处的情形吧。
 
　而且我不怎么感觉到那座公寓大楼有多大的『厌恶感』。
 
　到那边应该没问题。
 
。
 
「莫莫，你就继续呆在『影子』里，我要移动了」
 
。
 
「汪」
 
。
 
　在莫莫保持着呆在『影子』里的状态下，我们行动了。
 
　当然，我也完全发挥出『气息消除』、『无声移动』、『索敌』以及『敌意感知』。
 
　毕竟我得时常提防敌人的奇袭。
 
。
 
　移动的途中打倒三只僵尸。
 
　都是秒杀。现在僵尸都可以轻松打倒了。
 
　不过意外的没有多少魔物气息啊。
 
　这附近没有那么多魔物吗？
 
。
 
「啊咧……？」
 
。
 
　突然间我在地面上找到在意的东西。
 
。
 
「……魔石」
 
。
 
　红色的魔石。
 
　收纳进去看了一下，显示的是『僵尸的魔石（极小）』。
 
　就跟便利店的时候一样，打倒魔物之后没有注意到魔石，于是就落在这里了？
 
　嘛，我就心怀感激地收下吧。
 
　再次开始行动。
 
。
 
「啊咧？又有」
 
。
 
　走了几十米，我又捡到了魔石。
 
　运气真好。
 
　然而，自从我接近公寓大楼以后，我开始感觉到违和感。
 
。
 
「……又有啊」
 
。
 
　地面上又是魔石。
 
。
 
「这已经是第几次了……？」
 
。
 
　我捡到的魔石已经不下二十个了。
 
　好奇怪，再怎么说这也太奇怪了。
 
　只是不同的人物偶然将魔物打倒，没有注意到魔石就走掉了？
 
　这种事情有可能发生吗？
 
。
 
　那假如是同一个人物所做的，那为什么要放任魔石不管呢？
 
　为什么不捡起来？
 
　因为没有注意到？
 
　还是说『没办法捡起来』？……到底是怎样啊，这种情况。
 
。
 
「…………嗯？」
 
。
 
　当我在郁闷时，『索敌』突然感觉到人类的气息。
 
　而且还不是一个人。
 
　光是索敌范围内就感觉到了五个人。
 
。
 
「莫莫，我要稍微过去一趟了喔」
 
。
 
「汪」
 
。
 
　反正目的地也不远了。我有点在意起来，于是往气息发出的方向前进。
 
　我躲藏在墙壁的阴影处窥视着样子，于是看到几个成年人连忙从家里或者公寓的窗户跳出来。
 
。
 
「喂！你们看啊！是直升机！自卫队来救助我们了喔！」
 
「是那边！就在购物中心的方向！」
 
「得救了！我们得救了吧！」
 
「快点！赶紧过去！」
 
「啊、亲爱的你等一下……！」
 
「**！钱包和印鉴就别管了！赶紧走吧！」
 
。
 
　那些大叫的人们表情上没有一丝从容。
 
　明显表现出「想快点得救，尽可能脱离这种状况」的样子。
 
　应该是在呆在家里观望情形，不过看到自卫队的直升机，于是连忙开始行动了吧。
 
。
 
　嗯，我非常能体~会他们的心情。
 
　也许是十分慌张吧，里面还有人连鞋子都没穿。
 
　那些人都不管其他人的安危，争先恐后地跑向购物中心。
 
。
 
　喂喂，行动这么引人注目真的好吗，魔物也在喔？
 
　你们这不是连像样都武器都没有吗。
 
　既然都能背着那么重的背包，那再拿一把菜刀不也可以？
 
　虽然我不会特意给他们忠告就是了。
 
。
 
　我眺望着他们一会儿，再次前往公寓大楼。
 
　嗯，当然是无视啦。
 
　因为啦，这对我又没什么好处。
 
。
 
　到达公寓大楼了。
 
　结果来到之前总共捡了将近三十颗魔石。
 
。
 
「真是空荡荡的……」
 
。
 
　『索敌』范围内感觉不到多少人类的气息。
 
　虽然还有人呆在家里，但从整座公寓大楼的规模看只是微不足道的数目。
 
　我爬上没有人影的楼梯，前往屋顶。
 
　虽然是有电梯，但估计是用不了吧。
 
　魔物的气息也没有。
 
　因此我几分钟就爬到顶层了。
 
。
 
「……没有上锁啊」
 
。
 
　一般来说，屋顶都是禁止入内的。
 
　为什么可以打开？
 
　……有谁在屋顶吗？
 
。
 
「……」
 
 
 
 
 
。
 
「……」
 
。
 
　我慢慢地转动门把手，走到屋顶上。
 
　利用『索敌』确认完屋顶上没有其他人在。
 
　舒适的微风吹拂着脸颊。
 
　嗯，没问题……谁也不在。
 
　话说屋顶上还真令人兴奋不已啊。
 
　有一种空荡荡的开放感，如果可以自由地占用这里……无论是谁都会这么妄想吧？
 
　走到最顶点，能够望到城市的景色。
 
　噢噢，比起我住的破旧公寓还看得更清楚。
 
　使用『远望』后，也能像是在近距离看到购物中心的附近情形。
 
。
 
「好，莫莫你也可以出来了喔」
 
。
 
「汪」
 
。
 
　莫莫从影子里出来以后，呆在我的旁边。
 
　我们决定就潜伏在这里，观望直升机的情况。
 
一一一一一一一一一一
