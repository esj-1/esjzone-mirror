WEB 79.魔物军团
 
 
一一一一一一一一一一
 
 
 
 
 
「―――六花，快离开！」
 
。
 
西野君如此大喊着。
 
六花则是反射性地往后一跃。
 
随后她刚才所在的地方都遭受『暗』所覆盖。
 
正所谓是千钧一发。
 
『暗』侵蚀着食堂内部。
 
其他学生也都露出不敢相信的表情注视着暗，同时还往后退着。
 
。
 
「……啊ー啊ー，为什么事情会变成这样子呢……」
 
。
 
其元凶。
 
在暗的中心，刚才倒下的少女―――否，魔物师站起身。
 
她的额头上并没有原本开枪应该打中的『伤口』。
 
在距离额头的几公分处展开着『暗』，不久后就以漩涡状逐渐消失不见。
 
于是只留下子弹掉在她的手中。
 
她兴趣盎然地注视着子弹、
 
。
 
「……究竟是谁开枪的啊？就我所知，学校里面应该没有人拥有『狙击』技能才对……」
 
。
 
她目光尖锐地瞪视着周围。
 
此时的她已然没有刚才那副柔弱、无力的运动少女的影子。
 
她所释放的威压感强烈到犹如在面对魔物一般的错觉。
 
。
 
「哎，算了。反正无论如何，你们都得给我死在这里……啊哈哈哈哈哈哈……」
 
。
 
她的嘴角犹如月牙般张开，按住额头仿佛无法忍耐一般大声狂笑。
 
完全无法想象是同一个人。
 
她的变化甚至会使得别人认为人格是否完全改变的程度。
 
。
 
「你、你在说什么啊，葛木同学……？你究竟是怎么了！」
 
。
 
离她有点距离的地方大喊的是知识眼镜的宫本君。
 
他似乎还没有理解到如今的状况。
 
不如说几乎是所有学生都完全理解不了究竟是怎么了。
 
仅仅是茫然若失地目睹事态的发展。
 
能理解到状况的也就只有我们……还有西野君吧。
 
他丝毫不敢大意地注视着魔物师。
 
虽说身旁的六花也同样是还没理解到事态就是了。
 
。
 
「啊，干嘛啊，宫本副会长？」
 
。
 
魔物师瞪视着宫本君。
 
仅仅如此，宫本君就被吓得瘫坐在地面上。
 
。
 
「噫……」
 
。
 
「呜哇、好逊。什么嘛，稍微瞪你一下就怕成这样啊。平时那副很伟大的样子到哪里去了ー？」
 
。
 
口吻像是瞧不起的人。
 
她在暗上面悠然自得地地走向知识眼镜君。
 
毫无任何架势，毫无防备。
 
这反而显得毛骨悚然。
 
每当她踏出一步，脚边的『暗』也会随之扩散开来。
 
。
 
「别、别过来！」
 
。
 
「老实说我还对你抱有点期待呢ー。毕竟是那位五十岚会长的心腹，我还以为脑袋会更加灵光一点……不过只是一个跟班吗」
 
。
 
她像是感到失望一般，从宫本君身上移开目光。
 
随后看着西野君那边。
 
。
 
「果然还是你这家伙比较麻烦啊，是吧，西野君哟ー」
 
。
 
「……这就是你的本性吗，葛木同学？」
 
 
 
 
 
「本性?我原本就是这种性格啦。只不过……像那样做比较方便自己不是吗。你想想嘛，男生不都是对于柔弱、想要保护的那一类女生没辙吗。譬如说这种勇敢地提升等级，想要为大家而努力的性格之类的。啊哈哈哈哈哈哈」
 
。
 
不知有什么奇怪的，她遮掩嘴角笑着。
 
然而不经意间，笑容从她脸上消失。
 
。
 
「……本来是想隐藏性格活下去的。忍耐着，忍耐着，一直在扼杀着自己，我就是通过这样做活过来的，就是抱着这种念头的……」
 
。
 
不过，她说道。
 
摊开双手，浮现出灿烂的笑容、
 
。
 
「世界发生了变化！看吧西野！这样的世界！魔物出现，到处都是死亡，甚至还有等级和技能这种像是游戏一样的概念！我为什么要如同以往的自己一样？有必要再忍受吗？我想要活得随心所欲！我想要活得像是在活着生！不被任何事物所限制！以往世间的束缚和权力都不复存在！其他人会怎么样我半点兴趣都没有！随心所欲的存活者才是胜者不是吗！就是这样对吧，啊哈哈哈哈哈哈哈」
 
。
 
她像是打从心底感到愉悦般主张自己的看法。
 
回答她的并不是西野君，而是距离远一点的宫本君。
 
。
 
「怎、怎么可能！你在说什么啊，葛木同学！你知道我们现在究竟处于多么危险的状况中吗！没想到你居然有着这种愚蠢的想法……！」
 
。
 
他像是失望一般哀叹着。
 
对此，她一副很无趣的样子注视着。
 
。
 
「啊，是吗。随便你怎么失望吧，小跟班」
 
。
 
啪的一声。
 
她打了下响指。
 
暗开始出现波纹。
 
暗之中有魔物爬出来。
 
。
 
「吱吱……」
 
。
 
那是目光虚无的哥布林。
 
。
 
「有操控的魔物的魔物师―――这点是正确的，不过呢」
 
。
 
她嗤笑着。
 
暗更加扩散，从中有什么在蠢蠢欲动。
 
第二只哥布林从暗中爬出来。
 
并且在它旁边还有猪头人、僵尸逐渐现身。
 
。
 
「你们应该不会想到数量有这么多吧」
 
。
 
学生们神色惊愕。
 
虽说如此，我也同样被眼前的光景夺走目光。
 
我也预料到魔物师说不定驱使着多数魔物。
 
不过……这远远超出我的想象。
 
。
 
光是粗略一看，魔物都有五十只以上。
 
并且种类五花八门。
 
连大型哥布林和影狼都混杂在其中。
 
她狂笑的同时，举起手。
 
 
 
 
 
「―――魔物们，给我杀。无论是谁都全部杀掉吧。你们所有人都给我变成经验值！」
 
。
 
随着号令下达，魔物们开始进攻。
 
它们举起武器，露出獠牙，袭击着学生们。
 
。
 
「呜、呜哇啊啊啊啊啊啊啊啊啊啊！」「快、快逃吧！」
 
「根本打不过，这种数量！」「救救我，谁来救救我啊啊啊啊啊啊啊！」
 
。
 
「你、你们到底在干什么！战斗啊！战斗！」
 
。
 
宫本君拼命地想要制止学生逃跑，但并没有多大效果。
 
学生们都陷入轻微的恐慌症。
 
魔物军团是压倒性的。
 
这幅光景早就将他们的心灵击溃了。
 
而魔物们则是毫不留情地袭向他们。
 
。
 
「六花，打破窗户！」
 
。
 
「了解~！」
 
。
 
六花用手上的劈刀将附近的玻璃窗打破。
 
西野君应该是领悟到无法应付这种军团吧，立刻选择了逃跑。
 
。
 
「喂喂，你们想逃吗。再稍微给我悠着点吧！」
 
。
 
她迅速举起手。
 
这瞬间，脚边的『暗』以极可怕的速度延伸而去，形成网格状将玻璃窗堵住。
 
。
 
「什么!?」
 
。
 
两人立刻扭转身体，逃离了『暗』。
 
然而同样想要从窗户逃跑的一位男学生则是被『暗』抓到了。
 
。
 
「这、这是什么啊，完全动不了……!?」
 
。
 
尽管他想要脱离而拼命地挣扎，但越是动弹，这些犹如蜘蛛丝一般的细小暗线也就越将他身体缠得更紧。
 
这时一头猪头人逐渐走向他。
 
。
 
「不、不要―――啊啊啊啊啊啊啊！」
 
。
 
被暗所束缚而失去自由的男学生已然束手无策。
 
响起一阵声响后，他全身是血的倒下了。
 
而魔物师则是高兴似的看着他这种下场。
 
。
 
「我说过了吧，别想逃跑」
 
。
 
「葛木……！」
 
。
 
西野君和六花举起武器，瞪视着她。
 
即便如此，她的表情也毫无变化。
 
。
 
「这种表情真不错，真令人兴奋不已……！不过你们的对手可不是我哟」
 
。
 
脚边展开的暗中又爬出三头猪头人。
 
还有吗……。
 
。
 
「六花……」
 
「我知道啦！」
 
。
 
在猪头人袭向他们的同时，西野君他们也做出行动。
 
战斗开始了。
 
。
 
「那么―――」
 
。
 
我们这边也差不多开始行动了吧。
 
现在是到处都发生战斗的乱战状态。
 
这样一来，即便我们多少有点动作也应该不会暴露吧。
 
无论如何，如果不对『暗』所展开的宛如蜘蛛网一般的东西做点什么，我们也没办法逃离这里。
 
。
 
「小红，拜托你了」
 
「……（抖抖）」
 
。
 
我拜托以后，拟态为衣服的小红分裂出一部分，变化成猪头人的菜刀。
 
。
 
「一之濑小姐就在这里进行掩护」
 
。
 
一之濑小姐点了点头，举起枪。
 
。
 
「好，莫莫……我们上吧」
 
「汪」
 
。
 
魔物数量这么多。
 
既然对方的如意算盘是这样的，那我这边也赚取大量的经验值吧。
 
我将兜帽戴紧一些，与莫莫一同奔赴魔物军团。
 
一一一一一一一一一一
