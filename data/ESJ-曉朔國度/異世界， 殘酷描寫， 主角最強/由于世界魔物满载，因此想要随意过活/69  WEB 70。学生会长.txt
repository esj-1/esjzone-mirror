翻译：飒君CONAN
 
 
 
WEB 70.学生会长
 
 
一一一一一一一一一一
 
我们在周围晃来晃去，寻找比较容易入侵的地方潜入校内。
 
虽然是有看守的人，但并没有察觉到我们。
 
如果没有高级猪头人和黑狼那种探知能力，果然是没办法察觉到的。
 
太好了，最近总是遇到技能效果没有发挥作用的敌人，我都已经感到不安了。不过技能果然还是有在发挥效果的。
 
并且在入侵校舍内部以后，一之濑小姐的『认知妨碍』也发动了。
 
配合上我的技能效果，这样一来根本不可能察觉到我们吧。
 
事实上即便我们在走廊上行走着，他们也没有注意到。
 
这样就能堂堂正正地暗中探索。
 
。
 
话说回来，好真是有几年没看见母校了。
 
真怀念呢。
 
我并没有穿上室内鞋，而是直接走进去。
 
虽然可能会说现在是注意这种事情的时候吗，但我总是会隐约觉得好像做了什么不该做的事情。
 
邀请别人一起上厕所是好啦，但如果是上大号就会受人嘲笑，在高年级与低年级之间还有着默认的规定；虽然这些事情在踏出社会以后就会觉得这种拘泥到异常的倾向怎么样都好，不过在某种意义上，这里所形成的阶级制度比起社会上还要严重。
 
学校真的是不可思议的地方呢。
 
直截了当地说，根本就没有什么有意义的回忆。
 
一之濑小姐的经历倒是笑不出来。
 
。
 
「……和人先生，你的道具包里面有汽油和打火机吗？」
 
。
 
有是有，但不给你。
 
我无视嘀咕着可怕事情的一之濑小姐，环顾着校内。
 
跟我记忆中的印象相比较之下，没怎么变化。
 
这样也正好方便我。我和一之濑小姐牵着手走在校舍内。
 
如果不这样做，彼此的技能效果都不会产生作用。
 
……得注意自己手心上不要流汗。
 
还有一之濑小姐，你的手真是柔软呢，十分感谢。
 
许多人与我们擦肩而过。
 
。
 
「对不起，请问哪位是田中小姐的家人―――」「受伤的人往这边，想要物资在对面！」「不会有事的，不会有事的，所以请冷静下来」「姐姐，我肚子好饿喔ー」「忍耐一下，弟弟。来，吃颗糖！」「太棒了，不愧是姐姐！」「探索队回来了喔ー」「喂，你听说了吗？西野他们还活着」「真的假的」「据说现在正和会长他们相见」「有男性想要协助探索的请往这边―――」
 
。
 
我利用『聆听』偷听擦肩而过的人们以及附近的人所交谈的内容。
 
……人数比我预料的还要多。
 
校门不用说，就连庭院以及与附近马路衔接的地方到处都是人。
 
虽然几乎是避难民和学生，但在这里的人数估计有将近一百人。
 
。
 
「……比想象中还要镇定呢」
 
。
 
「的确……」
 
。
 
虽然小孩子他们在走廊上跑着，到处乱嚷着，不过并没有听到悲鸣声以及哭泣声。
 
走廊到处都张贴着校内的示意图，玄关那边也设好一个传言板。
 
已经完全组织化，整备完毕。
 
恐怕这间学校也有着『领袖』吧。
 
就像西野君掌握住百货商店一样。
 
。
 
不过在这种非常时期将这种人数都完全统领起来，究竟是什么样的家伙啊？
 
除非是具有相当高超的影响力和统率力，否则不可能控制得住这些人数。
 
……既然要收集情报，那就选择这家伙的地方？
 
。
 
『索敌』所反应出来的强烈气息有三处。
 
办公室、学生会室以及校门附近。
 
在校门附近的一定是侦查回来的那些人吧。
 
。
 
说起来，刚才有听到西野君被会长叫走了。
 
会长……大概是指学生会长吧？
 
那么，到那边去是最好的？
 
。
 
「……一之濑小姐，你身体没事吧？」
 
。
 
我看了一下一之濑小姐那边。
 
这里对于她而言并非是喜欢的地方。
 
就事实而言，她的脸色并不怎么好。
 
。
 
「……没、没有问题」
 
。
 
她在逞强是显而易见强。
 
脸色很差，眼神游移，并且耳朵很红。
 
不过按照这种情况，说不定就可以获得『抗压耐性』吧？
 
一之濑小姐，你就想办法忍住呕吐冲动努力一下吧，你可以做到的。
 
然后得到技能。
 
在心中鼓励着一之濑小姐，我们就前往学生会室了。
 
。
 
。
 
 
 
 
 
此时在学生会室―――。
 
。
 
「好久不见了呢，西野君」
 
。
 
在学生会室，西野与某位人物面对面。
 
。
 
「……真是好久不见了，会长」
 
。
 
西野君毫不掩饰不想见面的态度，适当地做出回应。
 
面对他的态度，被称之为会长的少女将手抵在嘴边，莞尔一笑。
 
。
 
「请别露出这么不情愿的表情啦，我可不记得有做过让西野君讨厌我的事情来着？」
 
。
 
「嘛，你说的对。不过在我和相坂看来，像会长这样的人就像是不同世界的人……」
 
。
 
少女在西野眼前坐在椅子上抱着胳膊。
 
她是这座学校的学生会长——五十岚十香。
 
是一位能以秀丽端庄、聪明伶俐加以形容的典型的美少女。
 
。
 
而她―――也是西野最想保持距离的人物。
 
。
 
「才没有这回事，我也是普通的女高中生。跟西野君你们没什么区别哦？」
 
。
 
闻言，西野不禁要大笑。
 
究竟是那张嘴敢这么说。
 
。
 
「会长也真是会开玩笑呢，普通的女高中生可没办法像这样将别人组织起来。我在来这里的途中就已经听说过了，是会长一直在行动将大家都组织在一起的吧？这是一件很了不起的事情」
 
。
 
 
 
 
 
「我只是很努力而已，魔物出现过后，许多老师比起我们还要先逃亡。因此只能依靠剩下的人想点办法。所以说，如今的状况完全是因为大家齐心协力打造出来的结果」
 
。
 
「是这样吗」
 
。
 
「就是这样」
 
。
 
对于西野随意的回答，会长满脸笑容地回应。
 
很难办。她始终是满脸笑容，完全无法看穿内心深处的心思。
 
不如说呆在她身后的学生会成员还比较容易理解。
 
面对西野随意的应答，在她后方戴着眼镜的一位像是知识分子的男学生―――记得是副会长――以焦虑难熬的表情注视着自己，其他成员也是同个反应。
 
。
 
（……没有将六花带来是正确的选择啊）
 
。
 
她比起大脑更活用于身体，这种方式的抗衡她应该很不擅长吧。
 
即便本人没有这种意图，也很有可能会将自己人的情报说漏嘴。
 
。
 
（不过，现在不能轻举妄动了……）
 
。
 
来到学生会室，看到他们就立刻明白了。
 
包括五十岚会长在内，学生会的成员全都是技能持有者。
 
既然不知道他们拥有什么样的技能，那就不能轻举妄动了。
 
即便表面上是狐假虎威，话题的主导权也是在他们那边。
 
。
 
「然后我想进入主题，西野君和相坂同学已经打倒过魔物了吧？」
 
。
 
―――来了。
 
西野在内心加强警惕。
 
。
 
「……是啊，就是这样」
 
。
 
「果然如此……那你们应该也知道技能和等级吧？」
 
。
 
「是的，我们知道」
 
。
 
他没想要隐瞒隠。
 
反正一起过来的石泽和葛木已经传达给对方了。
 
虽说传达的是『虚假』的情报。
 
。
 
「那么我就直接开门见山，请问你们可以成为我们的伙伴吗？」
 
。
 
「伙伴吗？」
 
。
 
「是的，我们如今正处于极为危险的状况之中。深受魔物威胁的同时，也无法进行任何联络。根本不知道救援何时到来，就连食粮也经常是维持勉强过得去的份量。我想要对如今的状况做点什么，想要改变这种情形」
 
。
 
五十岚会长低着头，哀叹着如今的状况。
 
她的举动、口述全都十分灿烂夺目，具有魅惑观者般的魅力。
 
学生会的成员都「嗯嗯」地点着头。
 
。
 
「正如会长所说」
 
「就是这样，大家一同跨越这种危机吧！」
 
「对，对」
 
。
 
西野以冷酷的眼神看着他们。
 
。
 
（……为了找到走散的柴田他们，这些人的协助是必不可少的。可是跟这些过分乐观的家伙组队真的好吗？）
 
。
 
他会来到这里只是为了得到关于同伴的情报。
 
虽然他们的战力的确具有魅力，但始终还是合不来。
 
毕竟他们是跟为了生存而能够舍弃一切的西野是完全相反的类型。
 
。
 
「谢谢大家」
 
。
 
五十岚会长如此说着，从席位上站起来。
 
她直接走到西野的面前，握住他的手。
 
。
 
「……会、长？」
 
。
 
她的动作颇为自然，西野的反应慢了一步。
 
这也是致命的破绽。
 
。
 
「拜托你了，西野君。能不能帮帮我们呢？」
 
。
 
她的『声音』通透地传达到西野的耳中、
 
。
 
「以前是有人诉说过你们品行不良，不过我相信你和相坂同学都是十分出色，满怀正义感的人」
 
。
 
这些话舒适地侵蚀着他的脑海、
 
。
 
「大家一起共同跨越过危机吧？」
 
。
 
在她炫目闪耀的眼瞳里映照出自己十分迷茫的表情、
 
。
 
「……啊，是啊。如果您不介意，我们愿意帮助你们」
 
。
 
回过神来，西野已经点头同意帮助他们。
 
。
 
「谢谢你，西野君……」
 
。
 
如此微笑的五十岚会长就犹如女神大人一般。
 
西野的心脏跳动不已，感觉体温都提高了。
 
恐怕就连脸上也是红彤彤的吧。
 
。
 
「那就拜托西野君你们补充物资了。因为我想让技能持有者优秀探索市区」
 
。
 
「啊啊，我明白了……」
 
。
 
西野点了点头。
 
啊啊，没错。
 
自己在这之前都在怀疑着什么呢。
 
明明她都这么真诚地求助自己。
 
为了她，自己得尽力而为。
 
这样的想法逐渐在心头涌现。
 
。
 
「那么现在请休息一下吧，如果有指示之后会传达给你」
 
。
 
「啊啊，是这样啊，会长。从今以后一起努力吧」
 
。
 
西野露出灿烂的笑容离开了学生会室。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
于是在他的气息远离以后，男学生副会长开口说道。
 
。
 
「……事情很顺利呢」
 
。
 
「是呢，虽然多少有些戒备，但既然已经变成这样子应该没问题了吧」
 
。
 
那是与刚才截然不同的冷淡声音。
 
五十岚会长坐在自己的椅子上，露出一抹冷淡的微笑。
 
。
 
「『技能』真的是方便呢。居然就连警惕性那么强的他都这么简单的屈服了……啊啊，真是兴奋不已，呵呵，呵呵呵……」
 
。
 
「既然是会长的力量，这也是理所当然」
 
。
 
「呵呵，谢谢称赞。不过我对他说的事情也是真的哟？毕竟我是真心想要对现在这种状况做点上面。我希望可以回到以前那样安稳的学院生活……」
 
。
 
没错，这样的世界她可敬谢不敏。
 
好想回到安稳的学院生活，好想取回安稳的学院生活。
 
那段支配一切，无论是谁都服侍着自己的时光。
 
她那带有忧郁的举动，令学生会的成员都不禁屏住呼吸。
 
见状，她的笑容更加的灿烂。
 
 
 
 
 
。
 
。
 
 
 
 
 
而我则是流下冷汗隔着墙壁听到这些谈话。
 
……这位学生会长比我想象中还要不妙。
 
我和一之濑小姐连忙离开了此处。
 
一一一一一一一一一一
