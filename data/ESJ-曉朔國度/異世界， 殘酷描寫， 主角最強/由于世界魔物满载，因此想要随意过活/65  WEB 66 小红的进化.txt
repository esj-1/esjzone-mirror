WEB 66 小红的进化
 
 
一一一一一一一一一一
 
 
 
 
 
我们都注视着进化结束后的小红。
 
。
 
「没有变化啊……」
 
。
 
「没有变化呢」
 
。
 
「汪」
 
。
 
「……（抖抖）」
 
。
 
外观上跟以前几乎没有变化。
 
依然是红色半球体的史莱姆。
 
莫非跟莫莫同样是外表没有变化的进化吗？
 
确认下人物状态。
 
。
 
小红
 
伪装（Fake）・史莱姆ＬＶ１
 
。
 
种族名改变了。
 
看样子小红是进化为伪装史莱姆这种魔物了。
 
Fake―――是假冒品、仿造品的意思吧。
 
吸取宝箱怪的魔石，驱使『拟态』能力，也许这样也能说是有小红风格的进化。
 
既然『拟态』能力得到强化，对我们而言也是一支强心剂。
 
我双手抱起小红。
 
。
 
「小红，恭喜你进化了。今后也期待着你的表现喔」
 
。
 
「……（抖抖）」
 
。
 
―――会努力的。
 
感觉像是在这么说。
 
小红振奋自己并抖动的样子有点令人欣慰。
 
 
 
 
 
。
 
接下来，虽然很在意小红的能力，但在这之前还是到别的地方吧。
 
在这里太显眼了，我还是想到能够稍微静下心的地方。
 
。
 
我们为了休息而走到附近的建筑物里面。
 
没有到购物中心里面是因为我顾虑到一之濑小姐。
 
一之濑小姐并没有『抗压耐性』。
 
还是不要让她看见那堆尸体山比较好吧。
 
实际上她光是和我交谈都似乎要呕吐出来。
 
一之濑小姐在这方面的承受能力跟一般人是一样的。
 
因此我们适当地走进一座建筑物。
 
。
 
「虽然屋檐和墙壁多少有点毁坏，但想休息应该是没有问题吧」
 
。
 
「嗯」
 
。
 
我从没有天花板的屋檐眺望着蔚蓝的天空，并做着午餐的准备。
 
要做什么好呢？
 
嗯~就咖喱饭吧，料理包的那种。
 
用热水加热咖喱料理包和佐藤的米饭。
 
（颯:原文是サ◯ウのごはん，既佐藤食品工业株式会社名下的产品）
 
。
 
「……好香的气味」
 
。
 
咖喱的香味也让一之濑小姐喜出望外。
 
果然大家都喜欢咖喱饭呢。
 
紧接着我听到一之濑小姐的肚子传来的一道「咕~」的催促声。
 
。
 
「……」
 
。
 
一之濑小姐害羞地别过脸……啊，原本就没面对我呢。
 
只是她的耳朵和脸颊有点红晕，噗。
 
。
 
「好了，请吃吧。啊，调味料你就根据自己的喜爱放进去吧」
 
。
 
我从道具包适当地取出调味包。
 
即便是调味包也有胡椒味和香辛料等口味，估计味道是完全不同吧。
 
顺便一提，我比较喜欢将香辛料磨碎再添加生姜。
 
生姜并不是大块状，因为我比较拘泥于使用切好的碎状。
 
我很喜欢这样的味道。
 
。
 
「啊，那不好意思，我就不顾虑了……」
 
。
 
嚯，一之濑小姐是辣椒油吗。
 
真是厉害的强者，这家伙很能干。
 
。
 
「汪！」
 
。
 
莫莫吃的狗粮似乎也很美味，都吃到脸颊鼓鼓的。
 
小红则是在……适当地吃着附近的废物品。真自由……。
 
。
 
「啊呜……果然咖喱饭好好吃哦」
 
。
 
是啊，用汤匙吃一勺，再一勺。
 
啊，说起来跟别人一起吃饭还真是久违啊。
 
毕竟回到家就我独自一人，工作的时候也没有跟谁一起吃饭。
 
有莫莫陪着我，我也没觉得寂寞就是了―――。
 
 
 
 
 
「……不过这样意外很不错呢」
 
。
 
「诶？」
 
。
 
「没事，什么都没有」
 
。
 
我随便地蒙混一下，继续专心致志地吃饭。
 
嗯，咖喱饭真好吃。
 
。
 
。
 
。
 
 
 
 
 
那么吃完午饭，也该检测一下小红的能力吧。
 
虽然我是知道『拟态』能力一定是强化了……。
 
。
 
我问了下小红究竟能做到什么样的程度。
 
紧接着小红抖动着身体，发生了变化。
 
这是―――。
 
。
 
「……『武器』吗？」
 
。
 
小红所拟态的是我使用的『猪头人的菜刀』。
 
实际用手拿起来看看，没有半点违和感。
 
简直就像是真货，能直接使用吗？
 
。
 
「……（抖抖）」
 
。
 
小红在催促着我使用看看。
 
因此我用它随意地砍向附近的家具。
 
直接砍成两半。
 
。
 
噢噢，好厉害。
 
跟真货没什么区别……不对，这锋利度比起真货还要优秀吧？
 
话说回来―――。
 
。
 
「小红，你在什么时候把『猪头人的菜刀』吃掉了啊？」
 
。
 
小红应该只能『拟态』自己所吞噬的物品才对。
 
进化过后就改变了吗？
 
。
 
「…………（抖抖、抖抖）。………（抖抖抖抖）」
 
。
 
嗯嗯，原来如此。
 
搞不懂。
 
。
 
不，虽然我隐约真的知道它想说的事情就是了。
 
。
 
听小红所说，它似乎在为了从魔物的攻击中保护我而将我全身包裹住时丝毫就把握到我所持有的武器的形状和锋利度。
 
由于进化过后使得『拟态』能力进一步强化，现在不用直接吞噬，在某种程度上可以依靠自己的想象『拟态』出来。
 
并且取决于想象，刀刃还能变化为鞭子，手柄伸长也能变化为长枪。
 
。
 
……什么鬼，这孩子太优秀了吧？
 
有点令人吓一跳。
 
。
 
「……开挂史莱姆」
 
。
 
一之濑小姐也嘀咕一句。
 
嗯，我也这么觉得。
 
。
 
「……！（抖抖）」
 
。
 
给我订正！小红激烈地抗议着。拟音吗！
 
它这副样子有点可爱。
 
。
 
顺便一提，虽然能够拟态的范畴更为广泛，但即便如此也还是无法拟态为『生物』。
 
嘛，要是连这点都做得到就真的是开挂了。
 
不过能够变为武器这点实在太棒了。
 
防御能力也不错，今后应该会更加依靠小红了吧。
 
 
 
 
 
。
 
。
 
 
 
 
 
那么小红的能力也已经确认完毕，接下来就是决定今后的目标吧。
 
。
 
「……不就是狩猎魔物提升等级吗？」
 
。
 
这种无限硬拼到底的作战真有一之濑小姐的风格。
 
明明就是怕生的家里蹲，为什么在这方面就这么积极呢引？
 
。
 
「虽然是这样，可我觉得还要同时进行物资与情报的收集比较好」
 
。
 
食物、饮用水、安全的睡床。
 
在这种魔物满载的世界里，这些都是梦寐以求的贵重且重要的东西。
 
。
 
现在还有远距离火力的一之濑小姐，队伍的平衡也更加完善了。
 
考虑到刚才的战斗，除非是出现上级种或者持名魔物，否则应该是不会输的吧。
 
既然如此，我还是重视战斗以外的部分比较好。
 
。
 
现在已经是世界发生改变的第四天。
 
即便别人已经组成某种程度上的规模的集团也不足为。
 
就像西野君他们一样。
 
。
 
能够集中人的地方大概就是学校，公务所、运动公园等之类的紧急避难所吧。
 
就从这些人当中收集到情报。
 
并且我也很在意政府和自卫队的动向如何。
 
结果我依然不知道被高级猪头人全灭的自卫队究竟是为了什么而来。
 
嘛，虽然我十分怀疑世界变成这样还有能够正经行动的政治家吗。
 
不过事先调查一下也没有损失。
 
。
 
「……可是和人先生，物资搜查姑且不论，可我们能够跟别人进行普通的交涉吗？」
 
。
 
「呜咕……」
 
。
 
这孩子真的是说到痛处了啊。
 
不，我很清楚这一点喔。
 
清楚到令人生厌的程度。
 
我和一之濑小姐的沟通能力都是绝对性的不足这件事。
 
。
 
正因为如此，我们才要偷偷寻找。
 
并且我还有未取得的技能『交涉术』。
 
只要得到这技能，对待人际关系应该就更擅长……才对。
 
。
 
「一之濑小姐，只要努力总能行得通。你不觉得「可能性」这词十分美妙吗？」
 
。
 
「不过可能性是零，再怎么做答案也还是零哦？」
 
。
 
住手，别说这样的话，会伤到人的。
 
啊，就连亲口说出来的她也消沉起来。
 
这就是自爆攻击吧。
 
莫莫和小红的视线好刺人。
 
。
 
「总、总之我们就前往市中心吧。然后在中途边狩猎魔物，边寻找生存者他们应该会集中起来的地方」
 
。
 
然后暗中偷听进行情报收集。
 
毕竟一之濑小姐比较怕生，而我又不擅长心理战。
 
这应该是最好的选择吧。
 
一之濑小姐也赞同了。
 
。
 
……啊咧？结果还不是跟以往做的事情是一样的吗？
 
。
 
不，在意的话就输了。
 
话说回来，市中心吗。
 
现在公司那些同事正在做什么呢？
 
还活着吗?还是说已经死了呢？
 
。
 
「……」
 
。
 
嘛，怎么样都好啦。
 
毕竟我跟他们的交情又不是那么深。
 
反正应该不可能再见了吧。
 
。
 
那么，我们就出发吧。
 
一一一一一一一一一一
