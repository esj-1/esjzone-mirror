WEB 58 探索与新的技能
 
 
 
 
一一一一一一一一一
 
 
 
 
 
继续探索购物中心。
 
购物中心在这附近是占地最大的建筑物。
 
即便是走着也可以走得很远，十分宽敞。
 
外头已经暗下来了，还是快点解决掉事情吧。
 
首先是前往食品销售区。
 
确保食粮。
 
。
 
「……果然有很多东西已经不行了啊」
 
。
 
绕完食品区后出来的感想就是这样。
 
特别是肉类和蔬菜相当凄惨。
 
应该是猪头人它们乱吃一通的吧。
 
吃剩的都丢得到处都是。
 
哥布林也是这样，这些家伙只袭击人类并不会再进食。
 
。
 
「果然魔物袭击人类的理由是因为『经验值』吧……」
 
。
 
既然没有将人类视为『食物』，那我会思考它们『狩猎』本身的意义也是自然的。
 
就跟莫莫和小红也存在等级制度一样，其他魔物也会有等级吧。
 
并且也有经验值和技能制度。
 
如果说我们打倒魔物可以提升等级，那反过来可以这样做也不奇怪。
 
。
 
魔物狩猎人类，提升自身的等级。
 
如同莫莫会进化一样，时间越是过得越久，产生的魔物也就愈加强大。
 
……这世界难度太高了吧，对人类完全不友善。
 
。
 
「……嗯？等一下？既然如此，人类杀害人类的情况下会怎么样啊？」
 
。
 
该不会……一样会得到经验值？
 
不经意间，脑海里浮现出最坏的预测。
 
虽然我没有尝试的勇气，但感觉可能性相当高。
 
杀死人，提升等级……。
 
。
 
不……还是别想了。
 
即便真是这样也不能这么做。
 
当我将别人视为经验值对象时，恐怕从那瞬间开始我就不是『人』。
 
我知道自己是个自我任性的家伙，可即便如此我也不想舍弃人类的思想。
 
。
 
回到原题。
 
寻找应该可以进食的食物。
 
结果我成功找到了碗装方便面和点心、调味料以及塑料瓶一类的饮用水。
 
特别是得到水让我十分开心。
 
先收纳进道具包。
 
绕到工作人员利用的后面，也把未开封的瓦楞纸箱回收。
 
有这么多应该足够了吧。
 
虽然终究只是建立在没有用在与魔物的战斗中的前提之下。
 
。
 
「明天到堤坝取水吧……」
 
。
 
如果生活用水老用在战斗上根本没完没了。
 
有必要提前准备好用于『雨』的水。
 
回收完食品之后，我走到其他区域。
 
这种设施基本上都是贩卖衣服和鞋子，不过我要找的在其他地方。
 
。
 
「……太好了，这里好像没有事」
 
。
 
抵达销售场。
 
虽然书架倒塌，商品也散落一地，但品质并没有问题。
 
可以正常看懂。
 
。
 
「『超简单，即便是新手也能入门的生存技术』吗……」
 
。
 
我翻开手中的书本，露出笑容。
 
没错，我们目前就在购物中心里面的书店。
 
如今已然无法使用手机、电脑以及网络等方便事物，所以要查询资料只能踏实地调查专门的书籍。
 
。
 
特别是医疗相关和生存技术、农业技术都是为了生存所不可缺少的知识。
 
食粮虽然还绰绰有余，但最糟糕的情况下自己还得进行农业不可。
 
虽然想要更加专业的书籍就只能到市中心的图书馆或者较大的书店，总之现在这样就足够了吧。
 
……姑且也将漫画和轻小说也回收吧。
 
如果有从容的时候想看一下。啊，有新出版的。
 
。
 
「不过已经没办法再看到续集了吧……」
 
。
 
这种世界。
 
应该不会再出版书籍了。
 
想到这件事就有点悲伤。
 
。
 
回收完书籍，再次在场内走着。
 
逛过杂货区、床具区、厨具区，也将物品都回收。
 
幸好没有像宝箱怪一样进行拟态的魔物。
 
探索结束的时候，周遭已经完全变暗了。
 
 
 
 
 
。
 
「今天就到此为止吧今……」
 
。
 
结束探索，我吐了口气。
 
『索敌』随即有了反应。
 
是魔物的气息。
 
。
 
「不是在购物中心里面，是外面吗……」
 
。
 
『危机感知』并没有多大的反应。
 
应该是哥布林或者僵尸吧？
 
往『索敌』所反应出来的方向走过去，只见那边有初次看到的魔物。
 
。
 
「……骨头啊」
 
。
 
「汪」
 
。
 
那就是俗称骷髅兵的魔物。
 
它们有着骨骼标本形的姿态，手持着棍棒和剑四处徘徊。
 
肋骨的中心部分能看到代替心脏的红色魔石。
 
虽然周遭很暗，但由于我有『暗视』因此得以清楚看见魔物的样子。
 
。
 
数量是……五只吗。
 
实力似乎没什么大不了。
 
从『危机感知』的反应来看，差不多就跟哥布林和僵尸相同层次吧。
 
。
 
机会难得，HP也恢复大半了。
 
就让我尝试一下新技能吧。
 
。
 
「莫莫，小红，狩猎了喔」
 
。
 
「汪」
 
「……（抖抖）」
 
。
 
我们保持着不会被它们察觉到，逐渐接近。
 
首先是使用『操影』。
 
『操影』跟莫莫一样是操控『影子』使其发生变化的技能。
 
影子变化成绳子和藤蔓将魔物缠绕住，从而封锁住它们的行动，我如此想象着。
 
而『影子』也就如同我想象般迫近骷髅兵它们。
 
。
 
「……!?」
 
。
 
骷髅兵都为突然束缚住自己的『影子』感到惊讶。
 
不对，虽然没有表情就是了，不过大概是在惊讶。
 
骷髅兵它们拼命地用身体抵抗，但还是无法挣脱。
 
。
 
「好厉害啊……」
 
。
 
看到莫莫使用的时候就想过了，这是比想象中还要方便的技能。
 
可以随心所欲地运用，并且汎用性很高。
 
跟其他技能的相性也不错。
 
虽然可以直接绞杀或者使用『忍术』打倒，不过还是让莫莫来吧。
 
。
 
「莫莫，你能行吗？」
 
。
 
「汪！」
 
。
 
莫莫点点头，往前迈出一步。
 
接着伸缩出『影子』，变化成锤子模样。
 
影之锤轻而易举地就将骷髅兵压死了。
 
看样子它们就跟外表一样防御力很低。
 
红色魔石留在地面上。
 
。
 
≪获得了经验值≫
 
。
 
经过高级猪头人的战斗，莫莫操控影子的技能似乎也提升等级。
 
虽然还有其他已得到的技能，但这次不会用。
 
毕竟那技能很强，太引人注目。
 
有可能会吸引到其他魔物，还是别使用吧。
 
。
 
「汪！」
 
。
 
战斗结束后，莫莫走进过来。
 
。
 
「很好很好，了不起喔ー莫莫」
 
。
 
「咕呜」
 
。
 
仔细地摸摸它并称赞着。
 
莫莫越来越值得依靠了啊。
 
。
 
与骷髅兵的战斗结束后，我们又回到场内。
 
今天就在这里过夜吧。
 
明天还得会见一之濑小姐。
 
。
 
「好，拜托你了，小红」
 
。
 
「……（抖抖）」
 
。
 
找到一定的空间，用『影子』将我和莫莫围起来。
 
并且还让小红拟态为能够与周围融为一体的姿态，并进行监视。
 
如果发生什么事小红会立刻注意到，即便受到攻击，除非是高级猪头人级别的攻击力，否则根本无法突破小红的防御。
 
安全的被窝。
 
。
 
「那么莫莫，小红，晚安」
 
。
 
「汪」
 
「……（抖抖）」
 
。
 
也许是因为疲劳吧，我立马就睡着了。
 
如此这般，第三天夜晚已经度过，太阳又升起来。
 
第四天就此开始。
 
一一一一一一一一一一
