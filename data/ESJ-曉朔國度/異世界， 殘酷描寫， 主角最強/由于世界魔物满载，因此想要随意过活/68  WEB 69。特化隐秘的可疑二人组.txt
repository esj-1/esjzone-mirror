翻译：飒君CONAN
 
 
 
WEB 69.特化隐秘的可疑二人组
 
 
一一一一一一一一一一
 
 
 
 
 
那么，在前往学校之前先筹备物资吧。
 
就到附近的便利店和超市好了。
 
。
 
「……话说回来，和人先生，我有件关于食粮的事情来着」
 
。
 
「什么事？」
 
。
 
「在超市的时候就在想了，如果想要得到大量物资不就应该到批发市场、批发商以及农业合作社比较好吧？你想想，说不定那里还有储备的大米、仍未出货的蔬菜、种子之类的……」
 
。
 
啊~的确是这样。
 
毕竟放进我的道具包也不会腐烂，所以应该还是从那边开始筹备比较好。
 
。
 
「说的也是……不过令人可耻的是我不知道仓库的地点……」
 
。
 
农业合作社是在公务所附近，不过我可不知道仓库和保管场所的详细地点。
 
要找到这些地方应该也要花上时间吧。
 
批发市场可是在这里穿过市中心后的更远处。
 
这边也是要花时间。
 
。
 
「―――『地图』」
 
。
 
一之濑小姐如此嘀咕着，随即在她眼前出现３Ｄ影像的立体性地图。
 
这是她拥有的技能『地图』。
 
似乎也是通过抽奖抽到的。
 
某种程度上可以搜索地点，就像是谷歌地图一样。
 
话说回来，虽然她说过我是开挂混º蛋，可我觉得一之濑小姐大概也是不相上下。
 
她抽奖的运气太好了。从氪金废人来看，她的人品都好到可以令他们大骂「别开玩笑了」。
 
。
 
「仓库在……啊啊，地点相当远呢」
 
。
 
地图有几个闪烁的地点标记，似乎就是农业合作社的仓库。
 
无论哪个地方都距离这里相当遥远。
 
。
 
嗯~，那就以后再过去吧。
 
除非像我一样拥有道具包，否则其他人即便到了那边也很难全部带走吧；就算真有道具包好了，如果不是像我这样拥有点数额外奖励，技能等级应该不会那么高。
 
。
 
「那我们还是按照原本的计划行事吧，之后再到那边」
 
。
 
「也是呢」
 
。
 
如此这般，我们按照顺序从附近的便利店开始搜索。
 
移动的时候是由我来背着一之濑小姐，毕竟这样比较快。
 
在背着她的时候顺便得知一件事，一之濑小姐的胸部并不是很大，根本不需要在意。
 
。
 
「……不知道为什么，总觉得心情很恶劣」
 
。
 
「那是你的错觉，请不要用指甲抓我」
 
。
 
话说回来，在表面上看来我们就是拿着枪支和菜刀，并且头戴摩托车头盔的二人组。可疑到极限的打扮。
 
如果是在日常生活中应该会成为报警案件吧，如果是我就会报警。
 
嘛，虽说小红的拟态和技能都在起到作用，应该没问题就是了。
 
。
 
话说真不愧是中心街 。
 
人类的气息跟郊外完全不能相提并论。
 
并且也有许多魔物的气息。
 
从刚才开始『索敌』一直都在产生反应。
 
。
 
「有很多人都在外面啊……」
 
。
 
「你说的是呢……」
 
。
 
虽然基本上都是寻找避难所的普通人，但人数很多。
 
嘛，他们应该没有技能吧，根本不可能察觉到我们才对。
 
。
 
「这样都能没注意到，真是令人毛骨悚然呢……」
 
。
 
「嘛，毕竟我们是特化隐秘能力呢」
 
。
 
移动的时候有『气息消除』、『无声动作』、『无味』在发挥效果，而在室内则还加上了一之濑小姐的『认知妨碍』效果。
 
『认知妨碍』是在建筑物中难以察觉到本人存在的特殊技能，不过由于现在是她触碰到我的状态，因此连我也似乎产生效果了。
 
这可是连莫莫的嗅觉都能蒙混过关的强力技能，十分值得依赖。
 
除非是高级猪头人或者黑狼这种不同寻常的例外，否则根本不用担心。
 
。
 
「啊，和人先生，那个……」
 
。
 
「嗯？」
 
。
 
我往一之濑小姐所指的方向看过去。
 
那边有魔物。
 
。
 
「……是史莱姆吗」
 
。
 
有三只史莱姆逗留在垃圾场。
 
它们还是一如往常抖动身体吞噬着垃圾。
 
这里就轮到小红出场了。
 
。
 
「小红」
 
。
 
「……（抖抖）」
 
。
 
我靠近史莱姆后，小红将拟态的一部分解除，伸出触手。
 
接着将三只史莱姆都吸收了。
 
。
 
「……就像是吸尘器一样」
 
。
 
嗯，的确。
 
真是强大的吸引力。
 
确认小红将史莱姆吸收完以后，我们迅速离开。
 
。
 
后来我们到便利店、杂货店、超市收集物资。
 
店内虽然十分乱七八糟，但还留有一定程度的物资。
 
果然道具包是稀有技能吧。
 
之后顺利收集完物资的我们就前往高中学校了。
 
。
 
。
 
。
 
 
 
 
 
过了几分钟，我们终于来到距离目的地——学校附近的地方。
 
由于没遇到什么强大的魔物，我们比起预料中的还要更快抵达。
 
这里也有相当多的人类气息。
 
果然这里集中着避难民他们。
 
。
 
「……啧，没有毁掉吗」
 
。
 
一之濑小姐虽然嘀咕着什么，我还是无视吧。
 
更重要的是从学校里面感觉到强大的气息。
 
并且还是多数。
 
大概是技能持有者吧。
 
虽然不知道等级具体有多少，不过感觉似乎能得到不错的情报。
 
。
 
「嗯……？那不就是……」
 
。
 
我躲藏在阴影里，利用『远望』窥视情形时在校门附近找到认识的面孔。
 
是西野君。
 
还有打扮夸张的女高中生……记得是叫做六花来着？
 
并且还有在购物中心看到的新妻石泽小姐，以及运动少女。
 
看样子她们似乎平安无事地活下来了，太好了。
 
西野君他们正在跟在校门看守的学生说着话。
 
。
 
究竟在聊些什么呢？
 
果然在这种距离下就连『聆听』也不能完全听到吗……。
 
谈话结束后，西野君他们走进了校内。
 
。
 
「有什么熟人吗？」
 
。
 
「啊，不是，还不到熟人的程度就是了」
 
。
 
「嗯——，这样啊。没有技能的我，果然在这种距离下是看不到的呢」
 
。
 
不不不，你要是跟拥有『远望』的我同样可以看到远处，那才真的是开挂吧。
 
总之先从守备比较薄弱的地方入侵到内部吧。
 
一旦入侵到校内，一之濑小姐的技能也会发动。
 
。
 
接下来如果能得到什么有用的情报就好了。
 
抱着这种期待，我们寻找着比较容易入侵的地方。
 
。
 
。
 
。
 
 
 
 
 
此时在另一边―――。
 
。
 
黑狼正在中心街奔驰着。
 
伤口很痛，大意了。
 
没想到那人类具有如此程度的力量。
 
远远超出自己的预料。
 
拜此所赐，导致队伍半毁，自己也身受重伤。
 
伤亡过大。
 
。
 
不过有收获了。
 
自从世界崩坏后一直在寻找的伴侣气息。
 
终于被我找到了。
 
。
 
「就是那条狗」
 
。
 
跟那人类在一起的狗。
 
从那只狗身上感觉到自己伴侣的气息。
 
。
 
「那条狗才是吾之伴侣」
 
。
 
不会错的。
 
终于遇到了。
 
。
 
「给我等着」
 
。
 
绝对会带回来。
 
不过现在不能焦虑。
 
目前应该要治好伤势，积蓄实力。
 
。
 
「首先是跟那些同伴汇合……」
 
。
 
为了以防对方追踪，它们都各自逃跑。
 
对于嗅觉优秀的它们而言，这并非是什么严重的问题。
 
立刻就可以汇合了吧。
 
它闻着气味，正意图寻找同伴的气味时，出现了几只哥布林。
 
。
 
「你们都不知天高地厚吗？」
 
。
 
虽然进行警告，但哥布林依然举起武器，以虚无缥缈的眼神瞪视着这边。
 
多么的愚蠢。
 
然而也正合我意。
 
现在有必要尽可能的恢复力量。
 
既然对方是这样的打算，那就别想抱怨什么了。
 
。
 
「死吧」
 
。
 
黑狼展开『暗』，将哥布林它们瞬间秒杀。
 
哥布林的魔石都掉落在它的脚边。
 
。
 
「我就收下了」
 
。
 
正当它为了吞下魔石而伸出头。
 
这瞬间。
 
突然，魔石从中出现『锁链』，将它的头部缠绕住。
 
。
 
「!!?」
 
。
 
什么？
 
不祥的锁链奏响出「沙拉沙拉」厌恶的金属声响。
 
突发的情况让它感到动摇。
 
。
 
「这锁链是怎么回事⁉」
 
。
 
即便想咬碎也咬不断。
 
纵使想甩落也甩不开。
 
。
 
「啊哈哈哈哈哈哈哈！哎呀—，我真是幸运啊。真没想到会抓到这种好东西。我的人品真是挺好的啦」
 
。
 
不经意间听到从背后传来刺耳的声音。
 
怎么回事，这人类？究竟是从什么地方出现的？
 
不对，比起这件事，这道锁链难不成是这家伙干的好事？
 
。
 
「咕噜噜！！」
 
。
 
「哦，你想甩开也是白费力气的喔？毕竟条件已经满足了啦ー。从今以后我就是你的主人，小狗狗」
 
。
 
条件？
 
主人？
 
什么？这人类到底在说些什么？
 
无意间力量减弱，身体内部某种厌恶的东西在游走。
 
自己的意识在逐渐消逝，有一种正体不明的感觉，仿佛像是某种不同的东西正在替换如今的意识一般。
 
。
 
「果然『魔物师』就是最棒的职业。我是完全选对了，啊哈哈哈哈哈―――……」
 
。
 
刺耳的声音逐渐变小。
 
意识也渐渐消逝。
 
不行，自己还有非做不可的事情。
 
还有必须要去迎接的人物。
 
即便它拼命地想要维持意识，可也还是没用。
 
黑狼完全失去意识。
 
。
 
「好了，隶属完毕。那么回到学校吧，再不快点那些家伙就要怀疑我了」
 
。
 
那人物对于结果感到满意，离开了此处。
 
一一一一一一一一一一
