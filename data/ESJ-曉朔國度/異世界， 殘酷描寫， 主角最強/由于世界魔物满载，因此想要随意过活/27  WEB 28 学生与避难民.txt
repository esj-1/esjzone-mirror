WEB 28 学生与避难民
 
 
一一一一一一一一一
 
　呜哇ー……。
 
　又是一群麻烦的家伙啊。
 
　我在心中叹了口气。
 
　我正躲藏在停车场的角落，观望着情形。
 
。
 
　总共有十二个逃跑过来的人吗。
 
　几乎都是中年人高年人。
 
　相对的，学生则是全部有五位。
 
　穿着上正所谓是不良学生。
 
　他们手拿着铁管和劈刀，朝着过来避难的人们施压。
 
。
 
　……百货商店里面也有人啊。
 
　至少有四个人。
 
　虽说距离有点远，以现在『索敌』的级别无法探测到整个百货商店，但在百货商店里面应该是有他们的伙伴。
 
　我使用『聆听』，听取他们的交谈内容。
 
。
 
「这里是我们先占领的，你们***！」
 
。
 
「别、别这么说，请务必让我们进去吧。至少让孩子们进去也行……」
 
。
 
「我们才没有这种余力，你们自己想点办法吧，渣渣」
 
。
 
「拜托你们了，真的拜托你们了」
 
。
 
　他们的交谈差不多一直是这种感觉。
 
　那么要怎么办呢ー。
 
　总之最好是放任不管吧。
 
　牵扯进去对我又没有好处。
 
　是说要是不小心插手进去，导致牵连到麻烦事情的话，我可敬谢不敏。
 
。
 
　应该是避难民太过喧哗吧，里面走出一位学生。
 
　穿着打扮很雅致，没有一丝邋遢感。
 
　就是这么一位氛围能令人如此信服的学生。
 
　店门处的不良学生们到感到惊讶，大概就是他们的领头吧。
 
。
 
「这场骚动是怎么回事？」
 
。
 
「西、西野，这些家伙是―――」
 
。
 
　店门处的学生们向称之为西野的学生说明事情缘由。
 
　避难民们也因为出现一位似乎能讲理的年轻人，而都纷纷露出安心的神色。
 
　听完情况后，他走到避难民的面前。
 
。
 
「……事情我已经听说了，就结论而言，我们没办法收留你们各位」
 
。
 
「不、不会吧―――」
 
。
 
　避难民的神色都浮现出绝望之情。
 
。
 
「但是，如果能够满足条件，我们会衷心欢迎你们的加入」
 
。
 
　条件……？
 
　比如说只能是可爱的女孩子？
 
　我脑海里闪过这种下流的想法，不过他说的是截然不同的事情。
 
。
 
「请问你们当中有没有人有过杀死魔物的经验？如果有的话，就可以成为我们的同伴」
 
。
 
　……什么？
 
　面对这个条件，避难民们都不知所措。
 
。
 
「……似乎是没有呢。那么从现在开始也行，请务必杀死一只魔物，当然你们所有人都要。这样一来，要我们收留各位也不是不行」
 
。
 
　虽然过来避难的人们大喊大叫着「在说什么意义不明的事情啊」，然而我很清楚。
 
　杀死魔物，这种行为意味着什么。
 
。
 
「……那些学生是技能持有者吗」
 
。
 
　虽然不知道等级是多少，但至少应该是具备与魔物战斗的力量吧。
 
　并且还选定了能够作为据点使用的百货商店，真是稳当的做法。毕竟在这里就不用为武器和食粮问题而愁。
 
。
 
「如今这个世界上只要杀死魔物便可以提升等级，得到技能和职业这种能够与魔物战斗的力量。这是我们在这两天里通过拼上性命验证出来的事实。既然你们要主张『互相帮助』，那我们也想要『回报』，名为『战力』的回报」
 
。
 
　嘿—，真意外啊。
 
　居然将情报和条件都清楚地告诉他们。
 
　与外貌相反，他们的主张不是挺正当的嘛。
 
　然而逃亡的人们似乎是无法相信他所说。
 
　比如「在说些什么乱七八糟的事情啊，这些粗人」、又比如大喊着「技能等级什么的，这又不是游戏！别给我说这些荒唐的事情！」、「你们这不就是没有在看清现实吗！」。
 
。
 
　喂喂……。
 
　是你们这边才没有在看清现实吧。
 
。
 
　不如说你们这些只会『逃跑』的家伙根本没有权力可以指责他们。即便是不良学生，他们也『与魔物战斗过，得到这个世界的情报』，比你们出色得多。
 
　在不知救援是否会来的状况下，食粮就是有限的资源。
 
　在这样的情况中无论是谁都应该会优待自己和自己亲近的人吧。
 
　何况这种情形下根本不可能带着碍手碍脚的负担存活下去。
 
　所以他们才会说出条件，并且还说达成条件后就可以成为同伴。
 
　他们的主张是十分合情合理。
 
。
 
　然而对于那些过来避难的人们来说，似乎是无法理解的事情。
 
　应该是明白再怎么大喊也是无济于事，一位大叔开始土下座了。
 
「求求你！求求你！还请务必救救我」
 
　头都磕在地上，并且还死抱着学生他们的大腿不放。
 
　面对这样的行为，就连学生们也有些退缩了。
 
。
 
「请、请放开我！纵使做这种事，我们也―――」
 
「求求你了！求求你了！求求你了！」
 
。
 
　……真的是『麻烦』啊，那帮家伙。
 
　自己完全不想主动行动，只想着让别人为自己做些什么。
 
　以为只要哭泣就能解决一切。
 
　我可绝对不想跟他们扯上关系。
 
。
 
「……总之百货商店就以后再说吧」
 
。
 
　先到药品店以及其他的店铺看看吧。
 
　我转过身，和莫莫一起离开了这里。
 
　究竟那些学生们是否能狠心做到『舍弃』呢？
 
　边听着如今还能听到的叫喊声，我边想着这件事情。
 
 
 
 
 
　。
 
 
 
 
 
　药品店就在百货商店的附近。
 
　我一走近，『索敌』就有反应了。
 
。
 
「这边也有人吗……」
 
。
 
　药品店里面有几位男生，正在物色东西。
 
 
 
 
 
从他们身上穿着的校服来看，应该是那些在百货商店里的同伴吧。
 
　我使用『潜伏』暂时观望着情形，他们拿完可以带走的东西之后就离开了。
 
　似乎是没有道具包。
 
。
 
「好，我们走吧」
 
。
 
「汪」
 
。
 
　斟酌着他们不在的时机，我和莫莫走进药品店。
 
　还留有许多东西啊。
 
　医药品的架子上也只是外面的部分是摆得乱七八糟，而里面则是还有许多东西。
 
　有感冒药和止痛药，以及包带和纱布。
 
。
 
　那么，究竟是否能够全部收纳呢。
 
　是时候轮到提升为ＬＶ９的道具包展现出效果的时刻了喔。
 
。
 
　我在脑海里默念『收纳』以后，眼前所有的医药品都收纳进去了。
 
。
 
　这是新追加的功能。
 
　不用直接用手，而是视觉上目视到的东西都进行收纳。
 
　最多可以同时收纳十件物品，效果范围最大是五米左右。
 
　真是可怕的性能。
 
　多亏于此，趁火打劫所需的时间也大幅度减少了。
 
　依次将物品收纳进道具包，仅仅几分钟，店里的商品就全部空荡荡了。
 
。
 
　等级提高，道具包本身的容量也提升了。
 
　打开目录一看，真是可怕的数目。
 
　不过收纳这么多还是没有到极限呢。
 
　毕竟机会难得，其他地方也去一趟吧。
 
。
 
　之后我还到百货商店附近的超市和饭店之类的地方，把东西都收纳进道具包里。
 
　当然，道具包大人都全部收纳进去了。
 
　真的很方便，这是神吗。
 
（颯:于是别人就惨了）
 
。
 
　下次升级的时候一定要提高道具包的级别。
 
　这样一来就是ＬＶ１０。
 
　我已经预想到我暌违已久的「保存功能」也会追加。
 
　这样一来，在这世界上的食粮问题几乎是解决了。
 
　啊啊，真令人期待。
 
　我和莫莫热衷于逛店，当回到百货商店的时候已经是夕阳西下之时。
 
 
 
 
 
。
 
 
 
 
 
「……那么，那些人怎么样了？」
 
。
 
　回来一看，已经没有人在百货商店的店门了。
 
　姑且是留有三位男学生在放哨。
 
。
 
　我看看……。
 
　再稍微接近一点，让整个百货商店都在我的『索敌』范围之内。
 
　这―――啊啊，是吗。
 
。
 
「结果还是收留了他们吗……」
 
。
 
　我从百货商店的内部感觉到相当多的人类气息。
 
　并且还是跟刚才那些人相同的气息。
 
　换句话说，他们收留了那些避难民。
 
　虽然我是没有权力说三道四，不过他们没问题吧？
 
　还是说他们打算硬是要让他们提升等级呢。
 
　嘛，怎么样都好就是了。
 
。
 
「那么……有什么地方可以入侵吗」
 
。
 
　并不是说我已经放弃百货商店里的物资了。
 
　而且我也很在意他们的等级以及拥有的情报有多少。
 
。
 
「后门的话似乎行得通吧」
 
。
 
　那么，就去看一下吧。
 
一一一一一一一一一一
