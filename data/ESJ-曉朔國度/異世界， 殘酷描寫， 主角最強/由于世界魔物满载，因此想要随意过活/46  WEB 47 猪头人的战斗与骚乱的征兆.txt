WEB 47 猪头人的战斗与骚乱的征兆
 
 
一一一一一一一一一一
 
 
 
 
 
　我躲藏起来仔细观察着猪头人。
 
　猪头人屡次三番地在环视四周。
 
。
 
　……它在寻找猎物吗？
 
　不对，以寻找猎物来说动作很不自然。
 
　感觉就像是在戒备着什么。
 
　猪头人一直在用鼻子闻着气味。
 
。
 
「！……噗咕！」
 
。
 
　接着它的目光移向我这边。
 
　难不成―――察觉到我了？
 
。
 
「噗咯喔喔喔！」
 
。
 
　猪头人举起武器，往我这边冲过来。
 
　毫无迷茫的动作。
 
　……不会错的。这家伙居然注意到我们！
 
。
 
　说起来我记得好像有听说过猪的嗅觉比起狗还要好来着？
 
　猪头人也适用于这一点吗。
 
　该死！『潜伏』系技能不光是对影狼没用，连猪头人也是吗！糟糕透顶。
 
。
 
　我在心中发着牢骚，做好觉悟。
 
　幸好『索敌』上没有其他魔物的反应。
 
　『危机感知』的反应也并没有那么大。
 
　猪头人的实力顶多就跟影狼相同，或者更强一点吧。
 
　并且对方只有一个人。
 
　如果能够冷静战斗，胜算是很大的。
 
。
 
　既然『潜伏』没有效果，继续呆在这里也是无济于事。
 
　我朝着猪头人冲过去。
 
。
 
「噗咕喔喔喔喔！」
 
。
 
　看到我的瞬间，猪头人也提高冲刺的速度。
 
　距离逐渐缩短，进入技能的有效范围内。
 
　就是现在！
 
。
 
「莫莫！」
 
。
 
「汪！」
 
。
 
　莫莫气势汹汹地从『影子』里出来。
 
　接着直接将影子延伸而出，打算束缚住猪头人。
 
。
 
「噗咕喔喔喔喔喔喔!!」
 
。
 
　然而猪头人用手上的菜刀硬是将影子切断。
 
　挺能干的。不过也足够阻碍住它的行动。
 
　这样就解决！
 
　我朝着猪头人的头上释放建筑机械。
 
。
 
「!?」
 
。
 
　猪头人为突然出现的巨大物体感到震惊。
 
　尽管它想立刻逃跑也已经来不及。
 
　有莫莫的『影子』阻拦，挡住它的去路。
 
。
 
　尽管它拼命地想要抵抗，但已经逃不了。
 
　这样就结束了。
 
　从自卫队在购物中心的战斗来看，一般的猪头人防御并不是那么高。跟『那家伙』不同。
 
　如果是这重量应该可以砸死它。
 
　我确信我赢定了。
 
。
 
「……诶？」
 
。
 
　然而这瞬间，『危机感知』敲响着警铃。
 
　猪头人将手上的菜刀『丢』向我。
 
　明显是有意的举动。
 
　菜刀以可怕的速度逼近我。
 
　糟糕……糟糕！
 
　领悟到无法避开的我，正打算利用道具包取出东西作为挡箭牌
 
。
 
「―――（抖抖）!!」
 
。
 
　然而小红比我还要更快行动。
 
　它所『拟态』的衣服膨胀起来，将飞来的菜刀给挡住。
 
　毫无动能的菜刀就这样停在了小红的身体里，接着掉落在地面上。
 
。
 
　见状，猪头人茫然若失。
 
　于是庞大的物体将那家伙砸了。
 
。
 
≪获得了经验值≫
 
。
 
　天之声告知我猪头人已经死亡。
 
　好、好危险……。真是千钧一发。
 
　心脏激烈地跳动，后背也流下冷汗。
 
。
 
「谢了小红。帮大忙了……」
 
。
 
　我跟它道谢并摸了摸之后，小红似乎很开心似的抖了抖身体。
 
。
 
「哈啊ー……」
 
。
 
　……大意了啊。
 
　到最后放松警惕了。
 
　被对方抓到这个破绽。
 
　重新让我认识到我所战斗的对手是拥有意志到真正的魔物。
 
　对方也是拼死拼命。
 
　如果不想死，那就只能挣扎到最后一刻吧。
 
。
 
　我回收建筑机械，将掉落的魔石和菜刀捡起来。
 
　放进道具包以后显示的是『猪头人的魔石（小）』和『猪头人的才对』。
 
。
 
「抱歉了莫莫，魔石这次可以让给小红吗？」
 
。
 
「汪」
 
。
 
　莫莫也点头了，于是我将魔石给了小红。
 
　作为回礼来说有些不足，不过我还是想将魔石给它。
 
　小红很开心地抖动身体，并将魔石吸收了。
 
。
 
「那么我们也该走了吧……嗯？」
 
。
 
　当想站起身的瞬间，『索敌』又有反应了。
 
　还是猪头人过来的那个方向。
 
　并且数量很多。
 
。
 
「不会是一群猪头人吧……？」
 
。
 
　使用『远望』观察『索敌』有反应的方向。
 
　于是看到一群影狼。
 
　数量很多，有十只以上。
 
　它们边闻着地面，边往这边过来。
 
。
 
　……该不会刚才的猪头人就是为了逃离这些魔物吧？
 
。
 
　如果是这样，刚才它那不自然的行动也能解释得通。
 
　魔物之间也会战斗吗？
 
　不，以后再想这件事。
 
　现在得赶紧离开这里。
 
　面对那种数量的影狼，果然还是会形势不利。
 
。
 
「可恶……只能原路返回了吗……」
 
。
 
　明明都来到这里了，居然还要走回去。
 
　我们最终勉为其难地走回刚才所来的方向。
 
 
 
 
 
。
 
 
 
 
 
　另一方面，此时在公寓的屋顶上―――。
 
。
 
　一之瀬奈津正看着邮件界面。
 
　没有回信。
 
　究竟是怎么了？
 
　正在跟魔物战斗中？
 
　还是说他单纯只是忘了回信？
 
。
 
「……该不会是邮件发太多了吧？」
 
。
 
　像几乎是初次见面的对象发送四十五封邮件，也许是做过火了。
 
　事到如今，她才终于注意到这件事。
 
。
 
「要是没有觉得我是一个麻烦的家伙就好了……」
 
。
 
　难得找到了队伍以及汪酱，并且还是成员候补。
 
　如果可以还是想搞好关系，还有就是希望能让我摸摸汪酱。
 
　应该要少发送一些邮件比较好。
 
。
 
 
 
 
 
「总之只能等了吗……」
 
。
 
　姑且还是几分钟检查一下邮件吧。
 
　在那之前就提升等级。
 
。
 
「饲养主工藤先生也意外像是好人，果然还是希望能够成为同伴啊……」
 
。
 
　在她的观念之中对于饲养主的印象并不怎么好，不过在互相通过邮件交流的期间里，似乎是多多少少有点改观。
 
　孤独的人对于善待自己的人基本上都会立马敞开心扉。
 
　虽然她并没有记住他的长相，倒是记得莫莫就是了。
 
。
 
「啊，有了……」
 
。
 
　她通过照准器发现了魔物。
 
　是哥布林。
 
　数量是一只，正是她的目标。
 
　对准准心，扣下扳机。
 
　子弹正如预料地贯穿魔物的眉间，使其毙命。
 
。
 
≪获得了经验值≫
 
。
 
「……嗯，顺利」
 
。
 
　之后她也打倒哥布林和小狼赚取经验值。
 
。
 
「这把枪果然好强啊……」
 
。
 
　她怜爱地抱着身为自己伙伴的枪身。
 
。
 
「……强大到能够弥补我贫弱的能力值呢」
 
。
 
　她的能力值全都很低。
 
　几乎都是一位数，仅有ＭＰ达到两位数。
 
　这都是她所选择的职业导致的。
 
。
 
「啊啊~，当时要是没有选择『家里蹲』就好了……」
 
。
 
　好奇心使然，她选了『家里蹲』的职业。
 
　当选择这个职业的瞬间，她的能力值都普遍低下。
 
　也许是职业的影响，即便她升级后能力值也几乎没有变化。
 
。
 
「不过要是没有选择这个技能，这把枪以及其他职业、技能就没办法得到就是了……」
 
。
 
　由于『家里蹲』职业导致能力值低下。
 
　取而代之会得到两个特殊技能。
 
。
 
「趁着事情顺利再来一发吧」
 
。
 
　她从技能栏目中选择某个技能。
 
　那技能是选择『家里蹲』时获得的两个技能的其中一个。
 
　选择完这个技能以后，眼前出现一个一米长的『箱子』。
 
　箱子是透明的，里面装着几种圆球。
 
　扭蛋机，也就是所谓的抽奖。
 
。
 
≪请问需要消耗1点ＳＰ来抽奖一次吗？≫
 
。
 
　脑海里回响着信息。
 
　她选择YES
 
　技能『抽奖』是通过消耗1点ＳＰ可以得到三次抽到各类奖品的技能。
 
　奖品的种类从日常食粮到日用品，以及职业和技能都可以得到。
 
　技能『邮件』也是通过这种抽奖得到的。
 
。
 
「那么，会出现什么呢……」
 
。
 
　转动手柄后，出来一个白色胶囊。
 
。
 
「没中吗……」
 
。
 
　白色是最普通的奖品。
 
　打开以后，变化成碗装方便面了。
 
　红色方便面和绿色方便面各三碗装满两袋。
 
　虽然有食物是很高兴，但不是现在想要的。
 
。
 
　第二次。
 
　这次是出来一个蓝色胶囊。
 
。
 
「哦，也许奖品不错呢」
 
。
 
　打开胶囊后，脑海里响起了信息。
 
。
 
≪获得了一点ＳＰ≫
 
。
 
　她在内心摆出胜利的姿势。
 
　『抽奖』也可以像这样得到ＳＰ和ＪＰ。
 
　刚才得到一点SP，就相当于这次抽奖都是免费的。
 
。
 
「最后一次……」
 
。
 
　她紧张不已地转动手柄。
 
　出现的是黑色胶囊。
 
　她展露笑容。
 
　黑色胶囊里面有很大可能会装有高稀有度的东西。
 
　她兴奋地将胶囊打开，接着脑海里响起消息。
 
。
 
≪获得了技能『身体強化』≫
 
。
 
「太好了……！」
 
。
 
　她全身表达出喜悦之色。
 
　终于得到想要的技能了。
 
　确认一下人物状态，力量和防御增加了。
 
。
 
「这样行动就更轻松了……」
 
。
 
　这次的抽奖是大成功。
 
　上次到最后都没有得到能力值上升的技能。
 
　得到的是『铅笔』、『指甲剪』、『维生素药剂』等一类可有可无的东西。
 
　因此直到今天都十分的艰苦。
 
。
 
　不过已经得到暌违已久的技能，这样情况应该有点改善吧。
 
　她开始地继续打算升级。
 
　接着在照准器上面看到在意的东西。
 
。
 
「……那是什么？」
 
。
 
　是一群猪头人。
 
　数量有十头以上，很多。
 
　并且其中还有一头明显与其他猪头人截然不同的个体。
 
　它的肌肤是红铜色，体格也相较于其他猪头人还要大上一圈。
 
。
 
「应该解决不了吧……」
 
。
 
　如果只有一头估计可以平安无事地干掉，但如果是一群就难了。
 
　而且那头红铜色猪头人的实力明显不同。
 
　自己是打不过的。
 
。
 
「它们到底要走到哪里去？」
 
。
 
　感觉不像是漫无目的地徘徊。
 
　而是正走向某处。
 
。
 
「在那边的建筑物有……」
 
。
 
　她打开地图，确认那周边的建筑物。
 
　于是她目光落在了―――。
 
。
 
「―――百货商店……？」
