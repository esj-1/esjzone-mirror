WEB 1 序章
 
 
「那么，我先告辞了」
 
。
 
　向同事打完招呼，我离开了公司。
 
　在黑心企业里就职三年。
 
　今天难得提早结束工作。
 
　话是这么说，也已经离下班时间晚了两小时了。
 
　哇，真气人。
 
（颯:世界上最令我讨厌的事情就是无偿加班）
 
。
 
　不过明天是休息日。
 
　好开心，超开心的。
 
　可以沉浸在阅读之前收藏起来的网络小说之中。
 
　我的休息方式几乎都是这样。
 
　为了以防上司紧急呼叫我，我当然也关机了。
 
　从现在起我已经兴奋得压抑不住自己。
 
。
 
　哈～，好开心。
 
　……不行啊，我居然会为休息天和提早回家这种事情感到高兴。
 
　啊啊，好讨厌。好想辞掉工作。
 
　可就算辞掉工作，也不定会找到比现在更好的就职环境吧。
 
　曾经我看到过一个实际案例，同期的一个人对于现在的公司感到厌烦，因此辞职了。不过当那个人就职时，却是更加黑心的企业。因此对于辞职我始终都感到踌躇。
 
　要是继续干下去或许就职环境会变得更好，会这么想大概是因为我是纯粹的社畜吧 。
 
。
 
　哎，随便了。
 
　总之回家吧。
 
　我驾驶着车子回家。
 
　开车回家需要三十分钟。
 
　我的家就在郊外的一栋破旧公寓里，双亲已然逝世。目前是自由过活的单身狗。
 
。
 
　外头早已昏暗无光，我得小心地开车。
 
　要是因为疲劳而放松身心，导致撞到人那可就不是开玩笑的。
 
　嘛，虽说这附近一带几乎没有车辆通行，所以基本上也没有事故就是了。
 
 
 
 
 
。
 
　啊，可是好困。
 
　糟糕，一回到家估计会直接睡着。
 
　这可不行，至少要检查一下小说是否有更新……。
 
　如此迷迷糊糊地开车是不行的吧。
 
。
 
　砰！！
 
。
 
　有种撞上什么的声音。
 
。
 
「……诶？」
 
。
 
　一时之间，我不知道究竟发生了什么。
 
　不对，一定是因为我脑袋在拒绝理解。
 
　毕竟，刚才的声音怎么想都是―――。
 
。
 
「撞、撞到了……？诶、真的假的？」
 
。
 
　是什么？我撞到什么了？
 
　是、是动物吗？狗或者猫之类的。
 
　拜托了，请务必是动物。
 
　不，如果是这样心情也会糟透了，不过还是希望不是人。
 
　人生会结束的。
 
。
 
　我急忙停下车，走了下来。
 
　在车灯的照耀下，倒在那边的是―――。
 
。
 
「……狗？不对……」
 
。
 
　就狗来说莫名觉得体型有些大。
 
　那体型可是有人类的大小喔。
 
　而且这什么？毛色也蕴含着野性，怎么看都不是日本的狗。
 
　狼？不对，它比起狼还要大。
 
　难道是外国产的品种吗？
 
　这附近有养得起这种宠物的有钱人吗？
 
　啊啊，不是撞到人真是太好了啊～。
 
　不，虽然现在这样也是问题就是了。
 
　不知道饲主看到这种情形会说些什么。
 
　也许会要求赔偿也说不定。
 
。
 
　看了一眼四周。
 
　嗯，没有半个人。
 
　……趁现在逃跑吧？
 
　完了，这就是肇事者的心境吗。
 
。
 
「―――哈……哈」
 
。
 
「呜哇！？」
 
。
 
　我吓了一跳，发出奇怪的声音。
 
　还、还活着？
 
　看向倒在地上的狗，它正微微抖动着。
 
　怎、怎么办？
 
　应该把它带到宠物医院……比较好吧？
 
　可是我要怎么解释呢。
 
。
 
　总、总而言之，看一下伤势如何吧。
 
　走近躺在地上的那条狗。
 
　呜喔，好可怕的魄力。
 
　它口里吐出鲜血，而且身体感觉已经弯曲了。
 
　……怎么看这都已经没救了……。
 
　啊，没有项圈。果然是野生的吗……？
 
。
 
「抱、抱歉！我不小心发呆了。真的抱歉……」
 
。
 
　道歉又有何用呢。
 
。
 
「―――哈、哈……」
 
。
 
　狗以颤抖的眼神看着我。
 
　应该是在责备我吧？
 
　我无意间摸着它。
 
。
 
「对不起了……真的对不起了……」
 
。
 
　我说完之后，狗就缓缓地闭上眼睛。
 
。
 
　―――死了。
 
。
 
「哈ー……」
 
。
 
　难以言喻的情感。
 
　究竟要怎么办？
 
　总之既然它是野狗，还是埋葬起来比较好吧？
 
　还是联系一下保健站―――。
 
 
 
 
 
≪―――获得了经验值。工藤和人等级提升到一级了≫
 
。
 
「……哈？」
 
。
 
　什么，刚才的声音？
 
 
 
 
 
≪―――已经确认到于Chaos・Frontier最初的讨伐≫
 
。
 
≪First・Subjugation Bonus已经颁发≫
 
。
 
≪获得了技能『早熟』≫
 
。
 
　又听到了。
 
　像是机器合成的声音。
 
。
 
「是、是谁啊？」
 
。
 
　尽管环视四周，谁也不在。
 
　幻听?不不不，开玩笑的吧。明明就听得那么清晰……。
 
　而且刚才的声音
 
。
 
「……总觉得就像是游戏通知一样……」
 
。
 
　不，怎么可能……。
 
　完了，看来我是被吓得不轻。
 
　总之现在该处理那条狗―――啊咧？
 
。
 
　没有。
 
。
 
　狗的尸体―――消失了。
 
。
 
　简直就像最初就不存在一般，狗的尸体瞬间便消失无踪。
 
。
 
「诶、啊……啊咧？」
 
。
 
　怎、怎么回事？
 
　该不会是看到幻觉了？
 
　不不不，怎么可能有这种蠢事。
 
。
 
「……嗯？」
 
。
 
　狗的尸体所在的地方。
 
　有一颗玻璃球大小的紫色石头。
 
。
 
「这什么玩意？」
 
。
 
　不存在于日本的大型野狗。
 
　在狗死亡的同时显现出游戏通知。
 
 
 
 
 
「道具掉落?……哈哈，我到底在说些什么啊」
 
。
 
　工作过头导致脑袋变奇怪了吗？
 
　对了，一定是因为忙过头导致大脑看见幻觉了。
 
　肯定是这样。
 
。
 
「打从一开始就没有什么狗，我也什么都没撞到」
 
。
 
　嗯，对，一定是这样。
 
。
 
「……回家吧。啊啊，好困……」
 
。
 
　我将紫色的石头放进口袋里，再次驾驶着汽车。
 
　果然今天还是直接睡觉吧。
 
　这样比较好。
 
 
 
 
 
。
 
　然而―――事后我才知道这并非是做梦也不是幻觉，而是毋庸置疑的现实。
 
 
 
 
 
。
 
　这时，世界究竟发生了什么呢。
 
。
 
　这时，我身上到底发生了什么呢。
 
。
 
　这时的我根本无从得知。
 
 
 
 
 
。

　不久后应该会成为吸引他人、引导命运、名为英雄的存在吧。
