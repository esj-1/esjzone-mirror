翻译：飒君CONAN
 
 
 
WEB 54 尾声 开始的结束
 
 
一一一一一一一一一
 
听到声音。
 
如果不仔细听清楚，那道小狗的鸣叫声就会听漏了。
 
。
 
『丢弃的小狗吗……』
 
。
 
那一天，我做完剩下的工作回到了破旧公寓。
 
将车子停在停车场之后，从草丛里听到一道哭泣声。
 
我以为是怎么了，走近一看，那里有一条小狗。
 
似乎还是婴儿阶段吧，它的身体小到令人惊讶。
 
。
 
『……咕呜……』
 
。
 
柔弱的哭泣声。
 
我发现这条小狗后就把它抱在怀里，让它冷静下来。
 
。
 
『没事的，别哭了』
 
。
 
这座破旧公寓禁止饲养宠物。
 
我没办法做出不负责任的事情。
 
不过毕竟是我发现的。
 
会对它动情也是没办法的事情吧。
 
。
 
『平时你就躲在这里吧，我会偷偷地给你带来吃的』
 
。
 
『咕呜ー……』
 
。
 
面对十分不安的小狗，我露出满脸的笑容。
 
像是要尽可能地让它放心。
 
。
 
『没事的，我不会舍弃你的喔』
 
。
 
『呜ー……？』
 
。
 
『所以你可以放心了喔。对了，毕竟难得，我就给你取个名字吧』
 
。
 
小狗一副不可思议的表情注视着我。
 
 
 
 
 
『我想想看，那么―――』
 
。
 
你的名字就是―――。
 
。
 
。
 
。
 
「……啊咧？」
 
。
 
我醒了过来。
 
无限延伸的湛蓝天空映入眼帘。
 
意识正慢慢地清醒。
 
。
 
啊啊，对了。
 
记得我……好像是与高级猪头人的战斗结束后就晕倒了来着？
 
。
 
「为什么会梦到那时候―――」
 
。
 
「汪！」
 
。
 
听到了叫声。
 
莫莫正在旁边以坐着的姿势注视着我。
 
。
 
「莫莫……？」
 
。
 
「汪！」
 
。
 
应该是对我醒过来而感到安心了吧，莫莫气势汹汹地用身体磨蹭着我。
 
还舔着我的脸颊。
 
。
 
「抱歉，让你担心了」
 
。
 
小红也在莫莫的旁边。
 
它慢慢地移动到我这边。
 
它们都平安无事。
 
。
 
「活下来了啊……」
 
。
 
重新涌起这股实感。
 
这次什么时候死掉都不奇怪。
 
这也意味着那头高级猪头人是多么恐怖的魔物。
 
回想起它的可怕，身体都会颤抖。
 
。
 
「啊咧？说起来―――」
 
。
 
我突然确认着自己的身体。
 
没有疼痛感，伤口……治好了？
 
衣服虽然还是破的，但伤口已经堵住了。
 
为什么？
 
ＨＰ自动回复的效果吗……？
 
还是说『有人』给我进行治疗了？
 
是谁呢……可能性最高的就是―――。
 
。
 
「一之濑小姐吗……？」
 
。
 
打开邮件界面。
 
未读邮件有80封左右。
 
噫……。
 
。
 
略微感到寒意的同时，我阅读着邮件上的内容，上面所写的内容正如我预料。
 
战斗过后她赶到这里，用她自己的技能为我治好伤势。
 
之后摸了摸莫莫，说它的毛感很不错，自己十分满足。
 
确认完我伤势治好后，就将剩下的交给莫莫和小红，自己离开了。内容大致上写着这些。
 
离开的原因很单纯，只是因为与别人直接说话会紧张，似乎就是这样。
 
……什么鬼啊这是。
 
。
 
不过帮大忙了。
 
先放一封「谢谢」的邮件。
 
立马有回信了。
 
。
 
『我还没听到您的回答，要是您死了就麻烦了』
 
。
 
对于这句话，我不禁苦笑起来。
 
这人真的是执着啊。
 
。
 
不过也的确是这样。
 
差不多也该给她答复了吧。
 
。
 
「莫莫，小红……」
 
。
 
我询问着它们。
 
。
 
「我想让一之濑小姐加入队伍，可以吧？」
 
。
 
这一次如果没有她的帮助，我就已经死了。
 
在这之前正因为她通知过我关于百货商店的事情，所以我才能注意到高级猪头人的弱点。
 
已经足够了吧。
 
是时候可以相信她了，不用再固执已见。
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
它们都像是在说可以啊。
 
谢谢你们。
 
。
 
那么就将她邀请进队伍……是说邮件上究竟要怎么申请加入队伍啊？
 
查看一下，似乎没有这种类型的项目。
 
。
 
「难不成……只能直接见面了吗？」
 
。
 
说起来莫莫当时以及小红那时候都是注视着对方才会显示出队伍申请。
 
如果对象是人类也同样要这样做的话，那就有必要跟一之濑小姐见面。
 
于是我发一封邮件告诉她这件事情。
 
回信立马来了。
 
上面是这样写的。
 
。
 
『我明白了，关于队伍成员这件事非常感谢您
 
那么决定好时间就相见吧。在那之前我也可以做好准备』
 
。
 
准备？
 
究竟要准备什么？
 
试着问了一下。
 
。
 
『那个……因为我跟别人直接见面就会非常紧张，所以我需要集中精神的时间……』
 
。
 
我不禁对这内容笑了出来。
 
真的是有趣的人。
 
。
 
总而言之，一之濑小姐说是想要一天的时间。
 
我也同意了，关闭邮件界面。
 
。
 
「接下来……」
 
。
 
我站起身，伸下懒腰。
 
总之先到购物中心里面吧。
 
我还想确认一下刚才的战斗中提升的技能和等级，以及魔石。而且我还要看看建筑里面有没有好东西。
 
这次与高级猪头人的战斗几乎花光了我所储存的物资，还得出现补充一下。
 
还有很多事情需要做。
 
。
 
「那么我们走吧」
 
。
 
「汪！」
 
「……（抖抖）」
 
。
 
 
 
 
 
我们还活着。
 
接下来也还会活着。
 
。
 
在这个―――魔物满载的世界中。
 
。
 
定下新的决意，我们再次走了起来。
 
一一一一一一一一一
 
 
作者说到这里第一章就结束了，嗯，54话等于第一章，你们没看错。
