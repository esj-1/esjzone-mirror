WEB 80 食堂的攻防
 
 
一一一一一一一一一一一一一
 
 
 
 
 
　走在校内的五十岚十香立刻注意到异状。
 
　她听到悲鸣声。
 
　往那边一望，只见食堂那边的学生们正拼命地跑向这边。
 
　他们后面有着无数魔物的身姿。
 
。
 
（……果然是在校内吗……）
 
。
 
　五十岚确信自己的推测是正确的。
 
　果然魔物师就在校内。
 
　并且就连能考虑到的最糟糕状况以及自己人所处于的现状都推测过了。
 
。
 
「会、会长！五十岚会长！」
 
。
 
　争先恐后地跑来这边的是副会长宫本。
 
　他激烈地喘着气，肩膀也剧烈的上下起伏，就这样跑向自己。
 
。
 
「哈啊哈啊，不、不好了！魔物师是学生，出现魔物发生了战斗―――」
 
。
 
「请冷静一下，魔物师是谁?伤亡的情况如何？」
 
。
 
「魔物师是……那个、葛木。葛木就是犯人……」
 
。
 
　听到这条报告，她骤然间感到惊讶，但立刻恢复冷静。
 
　五十岚在寻找魔物师的方面上还是有某种程度上的目标。
 
　那就是与自己同样是学生会的成员，在现场进行战斗的探索班和避难民。
 
　其中葛木是可能性较高的人物。
 
。
 
（原来如此……果然就是她呢）
 
。
 
　只是没有证据。
 
　虽然她是从第二天开始就参加探索班的学生，不过由于那献身般的态度而得到宫本以及其他学生们的高度评价。
 
　而积极地参加探索也就是说，在外面接触到魔物的机会也就会更多。
 
　一定是瞒过别人的耳目，偷偷凝聚战力的吧。
 
。
 
（……报告上没有半点可疑之处，只不过是她演技更胜一筹罢了……）
 
。
 
　虽然感到屈辱，但她不得不承认。
 
　自己彻底被她摆了一道。
 
　五十岚立刻调整心情，思考着目前的状况。
 
。
 
「我明白了，那么你和其他学生会成员一起进行避难诱导。这里就由我来抵挡住」
 
。
 
　幸好从食堂到校舍是条直道。
 
　只要在这里挡住它们，校内的伤亡就能压抑到最小。
 
。
 
「可、可是，让会长独自一人……」
 
「好了啦！快点走吧！」
 
「是、是的！」
 
。
 
　五十岚从犹如逃跑一般当场离开的宫本身上移开目光，注视着眼前向她迫近而来的魔物大军。
 
　让宫本他们退下并非是因为珍惜他们。
 
　而是认为他们会在战斗中碍事。
 
。
 
（心灵遭到击溃的人即便在这里也只会扯后腿）
 
。
 
　他们无非就是连挡箭牌都谈不上的肉块，还是不要呆在这里比较好。
 
　而且她的技能是善于以一敌多。
 
　可即便如此，这种数量的魔物说不定还是有点负担。
 
。
 
「呐呐，姐姐！这里有好多魔物喔！」
 
「是啊，弟弟！想要跃跃欲试了！」
 
。
 
　不经意间听到从背后传来天真无邪的声音。
 
　回头一看，那是一对年幼的双胞胎。
 
。
 
「士织、士道，你们怎么在这里？」
 
。
 
　她如此询问后，娇小的双胞胎笑着说道。
 
。
 
「「太吵了午觉睡不着（啦）！」」
 
。
 
「……还睡午觉，真是的你们……」
 
。
 
　面对即便处于这样的状况也毫无紧张感的两人，她苦笑着。
 
　这对双胞胎的名字分别是五十岚士织、五十岚士道。
 
　都是她―――五十岚十香的亲妹妹和亲弟弟。
 
（颯:改姓为五河岂不是美哉）
 
。
 
「哎，算了。无论如何你们两人在这里是令人开心的误算」
 
。
 
　五十岚的笑容更加灿烂。
 
　这两人在这里就放心多了。
 
　毕竟这两人是她的王牌。
 
　在座学校里是『最强战力』。
 
。
 
「―――士织、士道，我们驱除敌人吧」
 
「好的~」
 
「了解」
 
。
 
　三人毫不犹豫地走向魔物军团。
 
 
 
 
 
。
 
。
 
　另一边，食堂此处―――
 
。
 
　我躲藏在桌子和障碍物的阴影里进行移动。
 
　利用所有隐秘系的技能，尽可能躲过学生和魔物的耳目。
 
　嘛，所有人包括西野君和六花在内都将注意力放在眼前中，所以注意到我的可能性很低的，不过还是以防万一。
 
。
 
「吱吱！？」
 
。
 
　我使用小红所拟态的武器将哥布林的首级砍下来。
 
　尽可能一击毙命魔物们。
 
　这样一来，注意到我的可能性就极低了。
 
。
 
≪获得了经验值≫
 
。
 
　很好，首先是一只。
 
　旁边的哥布林则是为同伴突然消失一事而歪着头。
 
　受到操控的魔物们还具有意识吗？
 
　脑海里浮现出这种无所谓的疑问。
 
　然而我立刻将其抛之脑后，将注意力放在眼前。
 
。
 
≪获得了经验值≫
 
。
 
　第二只。
 
　果然没办法这么简单就提升等级吗。
 
　我现在等级是１７。
 
　上次提升等级是在早上与黑狼群体战斗的时候吧。
 
　仔细想想看，自从那场战斗过后直到现在都还没有经过半天时间。
 
　世界发生改变以后，总觉得好像过了很久。
 
。
 
「吱！？」「吱吱！」
 
「咕噜噜噜」「……嗷嗷嗷嗷嗷嗷嗷嗷嗷」
 
。
 
　食堂里剩余的魔物……大概还有三十只左右吧。
 
　其他还有一半的魔物似乎都前往校舍的方向了。
 
。
 
　剩下的学生包括西野君他们在内，总共有十多名。
 
　虽然多少是有些善战，但数量是压倒性碾压 。
 
　何况魔物之中还有猪头人、影狼、大型哥布林等之类的强大魔物。
 
　再这样下去，没过多久就会全灭吧。
 
。
 
「―――嘛，到时候再说」
 
。
 
　虽然对不起他们，但这边可没有这么从容。
 
　我只会做我能做到的事情。
 
。
 
　毕竟当前面临的是这样的状况。
 
　和伙伴一同平安无事地逃离这里。
 
　窗户和墙壁都被『暗』以蜘蛛网的形态覆盖殆尽。
 
　如果不对那家伙想点办法，我们很难逃脱这里。
 
　能看到窗户外面的样子。
 
　看来不光是这间食堂，就连校舍的墙壁都似乎遭受『暗』所侵蚀。
 
　毕竟她都说要杀死所有人，应该是没想要放过校内任何一个人吧。
 
。
 
　既然如此，只能对这只操控『暗』的魔物想点办法才能离开这里。
 
　规模这么大的暗，毫无疑问就是那只黑狼吧。
 
　然而当事人没有想要从魔物师的影子出来。
 
。
 
「那么，要做的事情只有一件」
 
。
 
　突破魔物的防线，对魔物师她做点什么。
 
　我发动新的『忍术』。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
　―――魔物师葛木对于眼前的光景感觉到违和感。
 
　好奇怪，有什么奇怪的地方。
 
。
 
（是谁……？从刚才开始就将魔物杀死的家伙？）
 
。
 
　即便仔细观察，也没能看到半点痕迹。
 
　注意到的时候，魔物已经惨遭杀害，只剩下魔石而已。
 
。
 
（……是那家伙吗？刚才对我开枪的？）
 
。
 
　不对，错了。她否定自己的推测。
 
　杀死魔物的家伙毫无疑问是近距离型。
 
　与狙击是截然不同的战斗方式。
 
　换句话说―――。
 
。
 
（至少有两个人……隐藏起气息和身姿在到处行动）
 
。
 
　究竟是谁？
 
　根据她所知道，学校里并没有人拥有这样的技能。
 
　这也就是说他们是外部的人。
 
。
 
（是最近来到这里的避难民？不对，没有像是这样的家伙。……等一下？说起来在开枪之前，相坂好像说过莫名其妙的事情啊……）
 
。
 
　她回想起六花所说。
 
　记得好像是说过邮件什么的。
 
。
 
（……那该不会是狙击的暗号吧？）
 
。
 
　那之后立马进行狙击了。
 
　如果没有『暗』的自动防御，自己这时候早就已经死掉了吧。
 
　想到这里就觉得毛骨悚然。
 
　她无意识地注视着潜伏在自己影子里的下仆。
 
　越来越觉得可以得到这家伙真是太好了。
 
。
 
（狙击我的那帮人在某处监视着我的行动。于是正因为知道我要展开行动才会先发制人。这样来考虑就解释得通了，是因为初次攻击失败，所以才没有追击过来吗？既然这样―――）
 
。
 
　她眯着眼睛。
 
　接着向魔物们下达指示。
 
。
 
「魔物们！给我优先对付西野和相坂！」
 
。
 
　魔物们往两人袭击而去。
 
　原本那两人光是对付三头猪头人就已经竭尽全能了。
 
　敌人再增加应该就难以应付了吧。
 
　如果进行狙击的家伙跟那两人有关系的话，那就绝不会见死不救。
 
。
 
（给我现身吧）
 
。
 
　是要对自己开枪呢，还是对魔物它们开枪呢。
 
　无论是哪一边都一定会有某种行动迹象。
 
　万一自己遭受狙击，也有暗的自动防御，根本死不了。
 
。
 
　随即她的猜测是对的。
 
　有开枪声。
 
　有颗子弹飞向朝着那两人过去的一只哥布林头部。
 
　紧接着是第二只、第三只，想要包围那两人的魔物逐渐被打死。
 
。
 
（好，上钩了）
 
。
 
　从死亡的哥布林身上就可以预测子弹的轨道。
 
。
 
（在那边吗）
 
。
 
　视线前方没有任何人。
 
　仅仅只有椅子和桌子。
 
　不过微微感觉到视线。
 
　绝对有人在，如此心想的她凝神端详。
 
。
 
（―――有了）
 
。
 
　如此『认知』后，她就清楚看到那边有位少女举着枪支。
 
　果然是通过某种技能隐藏起自己的存在。
 
　不过一旦认知到以后，效果就会变得微弱吧。
 
 
 
 
 
（……喂喂，这不就是一之濑吗……）
 
 
 
 
 
看到她，魔物师不禁惊讶起来。
 
　她是一年前遭受欺凌而不愿上学，最终退学的同班同学。
 
　这样的人为什么会在这里，而且又为何拿着枪支呢，各种疑问涌现而出，不过她将其抛之脑后。
 
。
 
（嘛，怎么样都好吧，总之给我去死！）
 
。
 
　向魔物们下达命令。
 
　立刻有三只影狼往她那边过去。
 
。
 
（成为下仆的魔物们与我『共有』眼界，它们也应该能清楚看到她的存在）
 
。
 
　而且影狼还是行动力比较优秀的魔物。
 
　这种距离完全可以在开枪打死它们之前接近她。
 
　说不定会死掉一只、两只，但剩下的一只就能够确实地咬到她的喉咙。
 
。
 
（好，死吧！）
 
。
 
　她究竟会露出什么样的表情呢?
 
　她想象着原同班同学因害怕而胆怯的表情，接着望着那边恐怖―――。
 
。
 
（……啊？什么嘛，这表情？）
 
。
 
　一之濑的神色上完全没有任何害怕。
 
　她仅仅是若无其事地朝着过来的影狼们开枪。
 
。
 
（喂喂，她还没有理解自己的现状吗？）
 
。
 
　一只影狼被她杀死了。
 
　然而还有。
 
　剩下的两只影狼正接近她。
 
　即便如此，她的表情依然毫无变化。
 
。
 
（真不爽……）
 
。
 
　这种表情是干嘛，这种从容是什么。
 
　为什么处于这种状况中还是这样的神色。
 
　无法理解。
 
　还是说背地里还有什么手段？
 
。
 
　如此想着之后，影狼它们的头上出现了『自动贩卖机』。
 
。
 
「…………哈？」
 
。
 
　响起一阵声音。
 
　突然出现的自动贩卖机将影狼它们直接压在地面上。
 
。
 
「什么……哈啊!?」
 
。
 
　什么鬼？刚才发生什么事了？
 
　自动贩卖机？为什么有自动贩卖机？
 
　究竟是怎么回事？
 
　她陷入混乱之中。
 
　这也是决定性的破绽。
 
。
 
「―――别动」
 
。
 
　不经意间，她听到后面传来的声音。
 
　回头一看，那是一位将兜帽戴得紧紧的男人。
 
。
 
「……哈？」
 
。
 
　兜帽戴得太紧，看不到他的长相。
 
　然而看到这男人的瞬间，她全身感到害怕。
 
。
 
（―――怎么回事，这家伙……!?）
 
。
 
　不妙。
 
　这男人很不妙，她的本能在如此告知。
 
。
 
「我应该说过别动了」
 
。
 
　男人手上是一把猪头人使用的巨大菜刀。
 
　其刀刃正架在自己的脖子上。
 
　刀刃反射着银白色的光芒，映照出自己的面孔。
 
。
 
「……我已经预测到你要让我们现身就会盯上那两个人，包括一之濑小姐不会对此视而不见。所以我反过来利用这一点，让你的注意力全放在那边，再像这样接近你想办法让你破绽」
 
。
 
　面临的敌意和杀气。
 
　她确信了。
 
　刚刚在这里将魔物杀死的迷之人物就是这家伙。
 
。
 
（喂喂，搞什么鬼。我光是警惕着五十岚和西野……这不是有更加不妙的家伙在吗）
 
。
 
　正如字面意思上的层次不同。
 
　居然被这样的敌人接近到这里。
 
。
 
「那么，魔物师。如果你不想死就乖乖听我的话照做吧?」
