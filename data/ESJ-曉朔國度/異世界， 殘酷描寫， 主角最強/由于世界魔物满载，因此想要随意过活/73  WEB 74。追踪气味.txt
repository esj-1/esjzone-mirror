WEB 74.追踪气味
 
 
我们追着逃亡的黑狼而来到市区中。
 
追踪全靠的是莫莫的嗅觉，毕竟如果不拉近到一定程度上的距离，『索敌』就不会有反应。
 
而且如今就连隔着一段距离也能感觉到的『厌恶感』也消失不见。
 
是因为黑狼只有一只吗，还是说已经很虚弱了呢，总之威胁若不超过一定的水准，那种感觉似乎就不会出现。
 
反过来也是说现在我们有很高的可能性能够解决掉黑狼。
 
。
 
我们跟在用鼻子进行追踪的莫莫后面。
 
一之濑小姐则是由我背着。
 
其实我想坐摩托车的，不过由于市区的街道上到处都有障碍物，根本没办法正常驶行。
 
虽说只要提升『骑乘』的等级应该就行得通了，但没有的东西再强求也是无济于事。
 
。
 
总之现在只能跑着追上去。
 
紧接着，『索敌』有反应了。
 
是魔物的气息。
 
。
 
已经追上了吗？
 
不对……错了。
 
这是其他魔物的气息。
 
。
 
往那边看过去，对面的车道线上不远处有哥布林。
 
三只吗……它们没有注意到这边，而是漫不经心地走在道路上。
 
虽然我想要经验值，但现在没时间管其他家伙……。
 
我无视它们继续前进。
 
跑了一阵子，来到一座公园。
 
遮掩物很少，周围的景色也能清楚看见。
 
。
 
「汪」
 
。
 
莫莫停下脚步。
 
看样子『气味』是到这里就消失了。
 
。
 
我左顾右盼地望着周遭。
 
没有黑狼的身姿。
 
『索敌』也没有反应。
 
。
 
「……没有呢」
 
。
 
「是呢……」
 
。
 
我再次仔细端详着周围，但还是没看到黑狼。
 
。
 
「咕呜……？」
 
。
 
莫莫也闻了闻，歪着头。
 
怎么回事……？
 
气味居然突然中断了……？
 
。
 
是某种技能吗？
 
不对，等一下……之前也发生过这种事吧。
 
对了，记得是在初次遇到黑狼的时候。
 
那家伙潜伏在『暗』里，观望这边的情形。
 
当时也是直到它出现为止都没有任何气息。
 
或许就跟莫莫躲在『影子』里就可以消除气息一样，那家伙说不定也同样可以做到这一点。
 
如果是这样就棘手了……。
 
这边完全没有适用的攻击手段，那就只能等待对方出现了。
 
。
 
……怎么办？
 
姑且放弃追踪黑狼，按照原本的预定前往市场和农业仓库吗？
 
正当我考虑着这些关于今后的目标时。
 
。
 
「！―――汪！汪汪！」
 
。
 
「噢，怎么了，莫莫？」
 
。
 
在公园到处乱跑，并寻找着气味的莫莫突然跑到我们这边来。
 
相当慌张的样子。
 
究竟是怎么了？
 
。
 
「汪！汪汪！汪——！」
 
。
 
「诶？你说什么？这是真的吗，莫莫？」
 
。
 
「汪」
 
。
 
莫莫点着头。
 
真的假的……可这又是怎么回事？
 
。
 
「怎么了，和人先生？……是说为什么你听得懂小莫莫想要说的意思？」
 
。
 
「诶，一般来说都听得懂吧」
 
。
 
我和莫莫是以心传心。
 
我肯定会知道莫莫想要说的事情。
 
不，严肃地来说，应该是莫莫拥有将自己的想法传达给他人的技能。
 
总而言之，我隐约能理解莫莫所说的意思。
 
。
 
「……什么是「一般」来着？」
 
。
 
一之濑小姐露出十分无法接受的表情。
 
不过这件事还是先放到一边吧。
 
。
 
「呃……那小莫莫说了什么呢？」
 
。
 
「它说闻到黑狼的气味了，虽然很微弱」
 
。
 
「！……这是真的吗？」
 
。
 
「是啊，不过有件在意的事情。从莫莫所说的内容中，似乎还混杂着其他『人类』和多种『魔物』的气味……」
 
。
 
「诶？」
 
。
 
正确地来讲，应该是魔物和黑狼的气味都依附在那位『人类』身上。
 
莫莫的嗅觉真厉害，居然连这种事情都可以知道。
 
。
 
「然后这股气味似乎一直延伸到公园前面」
 
。
 
「诶，可是这样……」
 
。
 
面对我所说，一之濑小姐也深入思考起来。
 
没错，这样一来就会产生疑点。
 
为何黑狼和其他魔物的气味会在那位人物身上。
 
。
 
「莫莫说魔物的气味中有哥布林、猪头人以及至今没遇到的品种」
 
。
 
「汪」
 
。
 
这句话使得一之濑小姐的表情愈发险峻。
 
。
 
「诶，可是我们来到这里的时候，魔物就已经不见了吧？那该不会是那人将黑狼和其他魔物打倒了吧？」
 
。
 
「如果是这样，黑狼的气息和气味会在这里结束也的确是解释得通……」
 
。
 
黑狼已经相当虚弱，如果是等级比较高并且拥有强大技能的人物，说不定就打倒它了。
 
。
 
「……可这样的话，这里应该有战斗的痕迹吧?」
 
。
 
「啊……」
 
。
 
一之濑小姐似乎也注意到了。
 
没错，这座公园没有半点战斗留下的迹象。
 
如果那人是在这里将黑狼和魔物打倒，从而导致这些气味都沾到身上，那这座公园即便更加破烂不堪也不足为奇。
 
。
 
「那就是躲在了某处，没有被魔物它们察觉到？」
 
。
 
「于是魔物它们离开以后，那人也逃跑了吗?这样一来，魔物的气味应该是不同方向才对」
 
。
 
根据莫莫所说，人类和魔物的气息全都在同个方向。
 
那么，这些魔物究竟消失到哪里去了？
 
为什么我的『索敌』没有反应？
 
。
 
黑狼的气味在这里中断。
 
仅留下气味的多数魔物。
 
以及身上有这些气味的迷之人物。
 
这究竟意味着什么呢。
 
脑海里浮现出最糟糕的想象。
 
。
 
「……莫莫，你能够追踪那人和魔物的气味吗？」
 
。
 
「汪」
 
。
 
莫莫强而有力地点点头。
 
我希望这是杞人忧天，是我想多了。
 
不过如果真是这样，事情就麻烦了。
 
确认一下总比没有做来得好。
 
。
 
「我们走吧」
 
。
 
我把我的想法告知以后，一之濑小姐也点点头。
 
莫莫根据气味再次奔跑起来。
 
我们也紧跟后头。
 
跑出公园，跑过道路，前往气味传来的方向。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
―――然后过了几分钟。
 
我们抵达这里。
 
不对，正确地来讲应该是『回到了』这里。
 
毕竟这里就是我们刚才所在的地方。
 
。
 
「这里不就是……？」
 
。
 
一之濑小姐发出惊讶的声音。
 
我也是同样的感受。
 
。
 
「……这究竟是怎么回事啊……」
 
。
 
气味传来的地点。
 
便是―――学校。
 
一一一一一一一一一
