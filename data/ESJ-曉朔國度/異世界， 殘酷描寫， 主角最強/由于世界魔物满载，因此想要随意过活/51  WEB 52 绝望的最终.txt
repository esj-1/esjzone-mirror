WEB 52 绝望的最终
 
 
一一一一一一一一一一
 
 
 
 
 
　……为什么我会遇到这种事啊？
 
。
 
　偶然间脑子里想到的就是这样的疑问。
 
　拼死地逃跑，经历过可怕到死的经验，并且还遇到这种怪物。
 
　我有做过什么足以把自己逼入这种状况的坏事吗？
 
　毫无道理可言啊，世上真的是。
 
。
 
「哈啊……哈哈、哈哈哈……」
 
。
 
　真不可思议。
 
　明明是无法发笑的状况，我却笑了出来。
 
　眼泪溢出，膝盖颤抖。
 
。
 
　高级猪头人又向我走近一步。
 
　这点距离大概就是我离死亡的时间。
 
。
 
　不经意间，我和高级猪头人四目相对。
 
。
 
　那家伙嗤笑着。
 
　怎么，杀死我就这么高兴吗。
 
　别开玩笑了。
 
　我才不想死。
 
　不想死、不想死、不想死！
 
。
 
「……！别、别过来啊啊啊啊啊啊啊啊啊！」
 
。
 
　我从道具包里面胡乱地丢出物体。
 
　建筑机械、巨大岩石、沙子、废车、冰箱、洗衣机。饮用水。辣椒水气球、碗具、水果。什么都有。
 
。
 
「咯……」
 
。
 
　高级猪头人像是感到厌烦似的挥开。
 
　简直就像是在挥落虫子一般，犹如踢飞小石头一样。
 
　别开玩笑了。
 
　为什么光凭这样的动作，岩石就击碎了？
 
　为什么废车会像足球一样飞上天？
 
　它避开辣椒水气球和饮用水，用脚践踏掉落在地上的水果。
 
。
 
　一切都太荒唐了。
 
　与那庞大身躯相配的力量，不相配的速度。
 
　全然是压倒性，根本不在同个层次。
 
　回过神时，那家伙已经就在我眼前。
 
。
 
「咯……」
 
。
 
　高级猪头人满脸笑容満面。
 
　简直就像是在期待着『接下来你还会怎么做？』这种眼神。
 
　咻！响起某种声音。
 
。
 
「咕……!?」
 
。
 
　被打飞之后，我才注意到那是它挥手的动作。
 
　光是这样我就已经飞出几米了。
 
　在地面上翻滚了几下。
 
。
 
「呜……好、痛……」
 
。
 
　好痛。
 
　如果不是小红给我缓冲一下冲击，现在估计已经骨折了吧。
 
　久违感到的『痛楚』。
 
　说起来在跟魔物战斗的时候我只害怕过，并没有受伤过。
 
　光是这一下就有这么痛。
 
　那如果骨折或者砍伤究竟会有多痛呢。
 
　哈、哈哈哈……。
 
。
 
「救、救救我……」
 
。
 
　回过神时，我已经说出求救的言语了。
 
。
 
「谁、谁都行……救救我……」
 
。
 
　眼泪滴落下来。
 
　好害怕。
 
　身体完全动弹不得。
 
。
 
　魔物的恐怖。
 
　高级猪头人的压倒性实力，足够让我感到绝望了。
 
。
 
「有人吗啊啊啊啊啊啊啊啊！就没有人在吗啊啊啊啊啊啊啊啊！再这样下去我会死的！求求你们了！谁都行！救救我吧啊啊啊啊啊啊啊啊啊啊！」
 
。
 
　我拼命地大喊。
 
　不顾一切，难堪地大喊着。
 
。
 
　回应是―――无。
 
。
 
　仅仅是一片静谧。
 
　静寂满溢在购物中心的停车场。
 
　谁都没有回应我。
 
　谁都―――。
 
。
 
「―――汪！」
 
。
 
「……诶？」
 
。
 
　仔细一看，莫莫从我影子里出来了。
 
　像是陪伴在我身边，站在前面直视着高级猪头人。
 
。
 
「咕噜噜……！汪汪！」
 
。
 
　莫莫朝着高级猪头人大叫。
 
　你在干什么，莫莫……？
 
　你不害怕吗？
 
。
 
　……不，错了。
 
　仔细一看，莫莫的身体正发着抖。
 
　莫莫也是害怕。
 
　不过还是在拼命地压抑住害怕的心情。
 
。
 
「住、住手吧，莫莫！已经够了，你就自己逃跑――好痛!?」
 
。
 
　莫莫咬着我向它伸出的手。
 
　干、干嘛啊，莫莫！
 
。
 
「……汪」
 
。
 
　接着莫莫舔着我的脸颊。
 
。
 
「莫莫……？」
 
。
 
「咕呜……」
 
。
 
　随即发出一道感到寂寞的叫声，面向前方。
 
　你到底要干什么啊，莫莫？
 
　停下来吧。
 
　逃跑啊。
 
。
 
「噜噜噜噜……嘎咕噜噜噜噜噜!!」
 
。
 
　莫莫奔跑着。
 
　背向着我，往高级猪头人冲过去。
 
。
 
「ゴァ……ァァァアアアアアアオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオオッッ!!」
 
。
 
　高级猪头人满脸笑容地迎敌莫莫。
 
　莫莫的下方延伸出『影子』。
 
　影子变化成长枪，袭向高级猪头人。
 
。
 
「嘎啊啊啊啊啊啊啊！」
 
。
 
　然而不管用。
 
　刺到身体的长枪仅仅是伤到皮肤而已。
 
　高级猪头人挥开长枪，往地面一踹，逼近莫莫。
 
。
 
「ゴアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアッッ！」
 
。
 
　震撼大地的咆哮声。
 
　多亏了小红膨胀身体保护了我。
 
　即便利用『影子』防御，莫莫也被振飞了。
 
　它倒在地面上。
 
。
 
「呜ー……」
 
。
 
　即便如此，它依然忍耐着痛楚站起来。
 
　在地面上疾驰。
 
　住手吧，停手啊……。
 
。
 
「嘎噜噜噜噜！」
 
。
 
　咬住高级猪头人的手腕。
 
　它全力咬上，然而牙齿依然穿不透。
 
　高级猪头人无论怎么甩着手也甩不开莫莫，它烦躁似的盯着莫莫看。
 
　它用另外一只手缓慢地举起菜刀。
 
 
 
 
 
　―――住手。
 
　要是被那种东西砍到，莫莫―――会死。
 
。
 
「咯啊啊啊啊啊啊啊!!」
 
。
 
　刀刃迫近莫莫。
 
 
 
 
 
「给我住手啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊!!」
 
。
 
　我在莫莫和高级猪头人之间释放几块砖头，形成一道小『墙壁』。
 
　利用『墙壁』硬是让莫莫和高级猪头人分开。
 
。
 
　回过神来，我已经忍耐着痛楚跑了出去。
 
　在莫莫摔在地面上之前我就已经跳过去抱住它，直接滚落在地面上。
 
。
 
「混º蛋！你想死吗，莫莫！」
 
。
 
　我对着抱在怀里的莫莫怒吼。
 
。
 
「咕呜……」
 
。
 
　明明被怒吼了，但莫莫似乎很高兴似的舔着我脸颊。
 
　简直就像是相信我会去救它一样。
 
。
 
　什么啊……。
 
　为什么你可以这么相信我？
 
　为什么你要为我这么努力？
 
　明明都怕到身体颤抖不停。
 
　说不定还会死掉。
 
。
 
「究竟是为了什么啊，莫莫……？」
 
。
 
「……汪！」
 
。
 
　莫莫像是在说“这是当然的吧”一样叫了一声。
 
　是吗，是这样啊……。
 
　对于莫莫而言，这些都是理所当然的事情吗。
 
。
 
「啊啊，是吗……我明白了」
 
。
 
　我把莫莫慢慢地放在地面上。
 
　站起身，从道具包里面取出『猪头人的菜刀』。
 
　接着全力瞪向高级猪头人。
 
　做好觉悟。
 
。
 
≪达到一定的熟练度≫
 
≪恐怖耐性从ＬＶ6升级到ＬＶ7≫
 
。
 
≪达到一定的熟练度≫
 
≪恐怖耐性从ＬＶ7升级到ＬＶ8≫
 
。
 
　临近死亡的恐怖。
 
　那又怎么了？
 
　跨越过去吧。
 
。
 
　如果这时我还不振作起来，莫莫就会死。
 
　我也会死，小红也会死。
 
。
 
　既然如此，挣扎吧。
 
　直到最后一刻都要尽量活下来。
 
。
 
　心脏激烈地跳动着，反复地进行短促的呼吸。
 
。
 
　思考吧。
 
　打破这种状况的方法。
 
。
 
　思考吧。
 
　怎么样才能活下来。
 
。
 
　思考吧。
 
　怎么样才能打倒这家伙。
 
。
 
　看着吧。
 
　看着那家伙——高级猪头人。
 
。
 
　回想一下。
 
　回想起以往的战斗，以往的行动。
 
　其中的一切。
 
。
 
≪达到一定的熟练度≫
 
≪观察从ＬＶ7升级到ＬＶ8≫
 
。
 
≪达到一定的熟练度≫
 
≪观察从ＬＶ8升级到ＬＶ9≫
 
。
 
　在购物中心遇见它。
 
　接着逃跑。
 
　以往的所有行动。
 
　我的技能、等级、道具包、莫莫、小红。
 
　自卫队的战斗、枪、视线、哥布林、洗衣机、相性、影狼、小狼、宝箱怪、僵尸、魔石、远嗥、百货商店、火灾、西野君、女高中生、一之濑小姐、狙击、高层公寓、邮件、捉迷藏、障碍物、咆哮、举动、视线、违和感―――全部都回想起来。
 
。
 
　扑通、扑通，心脏开始输送血液。
 
　好热。
 
　渴望着氧气，调整呼吸。
 
　全身犹如燃烧般一样热。
 
　害怕、紧张、兴奋等激烈心情都毫不留情地灼烧着我的大脑。
 
。
 
　思考吧……思考吧……！
 
。
 
　想想看、想想看、想想看。
 
　想想看、想想看、想想看、想想看。
 
　想想看、想想看、想想看、想想看、想想看、想想看、想想看、想想看、想想看、想想看、想想看、想想看。
 
。
 
≪满足一定的条件≫
 
≪获得了技能『集中』≫
 
。
 
≪达到一定的熟练度≫
 
≪集中从ＬＶ1升级到ＬＶ2≫
 
。
 
≪达到一定的熟练度≫
 
≪集中从ＬＶ2升级到ＬＶ3≫
 
。
 
「―――啊」
 
。
 
　于是想到了。
 
　回想起一切的记忆―――不经意间想起某个『违和感』。
 
　对了，说起来那家伙为什么要这样做……。
 
。
 
「莫莫……小红……」
 
。
 
　我笔直地注视着眼前的敌人、
 
。
 
「―――要活下来喔，绝对」
 
。
 
「汪！」
 
「……！（抖抖）」
 
。
 
　同时并做好觉悟。
 
　能看到渺小的希望之光。
 
　称之为可能性的细微的胜利之线。
 
　我会将它掌握在手中。
 
。
 
　开始吧―――我们的垂死挣扎。
