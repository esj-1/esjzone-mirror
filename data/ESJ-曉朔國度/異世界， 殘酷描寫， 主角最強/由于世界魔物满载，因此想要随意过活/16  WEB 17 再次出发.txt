柴犬　Ｌｖ２
 
 
 
 
 
。
 
　抗压耐性、恐怖耐性、敌意感知、危机感知的等级都提升了。
 
　虽然昨天还不是这样子，但像这样一看技能等级还真是提高了许多啊。
 
　虽然面对着死亡，不过还是有一定程度的回报。
 
　虽然完全不合算就是了。
 
。
 
　话说回来，好像有说过可以获得新的技能。
 
　点击一下技能栏目试试看。
 
　于是在『初期可获得的技能一览』的项目下新增加一条『可获得的追加技能一览』。
 
　点击以后，显示出新到技能。
 
。
 
『可获得的追加技能一览』
 
逃跑、防御本能
 
。
 
　『逃跑』和『防御本能』吗……。
 
　天之声说的是『满足一定条件』什么的。
 
。
 
「一定条件……」
 
。
 
　也就是说在现实中我的行动以及状况，亦或是愿望之类的会产生出新的技能吗？
 
　获得技能的状况与技能名字实在是巧合过头。
 
　换句话说，只要面对各式各样的状况，就有可能得到许多的技能。
 
　虽然我并不想冒险行事，但有值得一试的价值。
 
。
 
　不过在说那件事之前，现在可以获得的新技能只有两个。
 
　我点击『逃跑』技能。
 
。
 
≪请问需要消耗两点ＳＰ学习技能『逃跑』吗？≫
 
。
 
　啊咧？两点？
 
　不是一点吗？
 
　为了确认一下，我在初期可获得的技能上面点击看看。看样子初期技能是需要一个点数，而追加技能这边则是需要两个点数。
 
。
 
　两个点数吗……。
 
　这两个技能都挺有魅力，得到了也不会有所损失。
 
　另一个技能还是等下次升级后再学习比较好。
 
　总之先得到『逃跑』。
 
。
 
≪请问需要消耗两点ＳＰ学习技能『逃跑』吗？≫
 
。
 
　YES。
 
。
 
≪获得了技能『逃跑』≫
 
。
 
　好，这样在逃离魔物的时候会有加成才对。
 
　虽然这个技能也是必须处于那种状况中才能进行验证的，不过技能效果大概就是这样吧。
 
。
 
　不如说获得的技能即便不消耗SP也可以通过职业等级的提高、熟练度的提高，相应地提升技能等级。所以首先是获得许多技能，之后再通过熟练度来提升级别比较好。
 
　这部分SP的开支也能节省掉。
 
　先行投资是很重要的事情啦。除了股票，那玩意绝对是坑人的。
 
　下次升级的时候还是包括初期技能在内都学习掉比较好。
 
。
 
「嗯？」
 
。
 
　偶然地看向莫莫那边，它保持坐着的姿势盯着某一点看着。
 
　它视线的前方什么都没有，仅仅只是一道墙壁。
 
　接着慢慢地怕打着地面。
 
　它在干嘛？
 
。
 
「……莫莫？你怎么了？」
 
。
 
　我向它搭话以后，它葛然看着我。
 
。
 
「汪！」
 
。
 
　莫莫精神饱满地回应我。
 
。
 
「有什么在意的事情吗？」
 
。
 
「汪！汪！」
 
。
 
　十分开心一样，它用身体磨蹭着我。
 
　嗯~？到底是怎么了？
 
　这种想要让我夸奖它的举动。
 
　以及刚才的迷之行动，难道说……。
 
。
 
「……呐，莫莫，你该不会也是获得了某种『技能』吧？」
 
。
 
「汪！」
 
。
 
　莫莫像是回答说正确一般，激烈地摇摇尾巴。
 
　真的假的，虽然我只是在开玩笑，不过真的得到某种技能了？
 
　果然莫莫也能听到那道像是『天之声』的声音吗？
 
。
 
「真、真的吗？究竟是什么样的技能啊？」
 
。
 
「汪！」
 
。
 
　莫莫在我身边咕噜咕噜地转圈，离开点距离跳了起来。
 
　接着一副「怎么样啊」的表情看着我。
 
。
 
「……那个，这是什么意思？」
 
。
 
「汪！」
 
。
 
　它再一次重演动作。
 
　诶？什么？这是什么技能？
 
　来回跳动的技能吗？
 
　回转跳跃……难道说是那个超有名的汪酱必杀技―――。
 
。
 
「汪！汪呜！」
 
。
 
　激动地否定我。
 
　看来不是。
 
　那到底是什么技能啊？
 
　虽然莫莫重复多次同样的动作，但结果我还是搞不懂。
 
　由于没能让我明白，莫莫十分失落。
 
。
 
「…………咕呜…」
 
。
 
　抱歉了，莫莫。
 
　之后我摸着它摸了一会儿，总算是让它回复情绪了。
 
 
 
 
 
。
 
 
 
 
 
「那么我们出发吧」
 
。
 
「汪」
 
。
 
　用来作为障碍物的家具都全部收纳完毕，我和莫莫来到外面。
 
　雨已经停了。
 
　朝阳好耀眼。
 
。
 
「到处浮现的烟气也已经消失了……」
 
。
 
　多亏昨天下的雨，把很多火都浇灭了吧。
 
　远处的光景一目了然。
 
　而视线的前方也能看到昨天的购物中心。
 
。
 
「那头高级猪头人还在那里吗？」
 
。
 
　或者有可能已经为了狩猎人类而移动了。
 
　我得快点提升等级了。
 
。
 
「……嗯？」
 
。
 
　『敌意感知』立马有反应了。
 
 
 
 
 
不过感觉并不是很明显。
 
　比起昨天感受到的『厌恶感』，这边只是『稍微注意一下就好』的程度。
 
。
 
「不是在这楼……是一楼吗」
 
。
 
　也许是级别提高的影响，能够感知到『敌意』的范围变广泛了。
 
　我使用『隐秘行动』、『轻步』迅速下楼。
 
　顺便一提，是以抱住莫莫的状态。
 
　毕竟这样下楼比较快。
 
。
 
「汪呜」
 
。
 
　被我抱着的莫莫似乎很舒服的样子。
 
　住手，别用这种眼神看我。
 
　我会很难离开你的吧！
 
。
 
 
 
 
 
　抵达一楼。
 
　我狠下心把莫莫放下来，咕呜呜……。
 
。
 
　敌意感知所反应的地方就在稍微前面一点。
 
　往那边一看，停车场附近有一只魔物。
 
。
 
「僵尸吗……」
 
。
 
　一只僵尸在停车场附近徘徊着。
 
　没看到其他魔物的身影。
 
　它就是目标了。
 
。
 
　我从道具包里面拿出哑铃，一口气接近它。
 
　敲碎它的头部。
 
。
 
≪获得了经验值≫
 
。
 
　僵尸消失了，留下魔石。
 
　很好，不错的前兆。
 
　总之先以僵尸以及哥布林为主要攻击对象吧。
 
　当然，前进的方向是跟那座购物中心是完全相反。抱着安全第一的念头开干吧。
