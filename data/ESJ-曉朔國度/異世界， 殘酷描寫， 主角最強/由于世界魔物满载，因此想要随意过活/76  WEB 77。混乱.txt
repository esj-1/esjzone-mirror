WEB 77.混乱
 
 
一一一一一一一一一一一一一
 
 
 
 
 
西野和六花正在校内的食堂。
 
为的是与探索班的成员见面。
 
他们坐在椅子上，等待着成员的到来。
 
其他学生似乎很忙碌似的来回奔走，在这种状况中像这样无所事事地等待别人也是颇为急不可耐。
 
。
 
「还没来吗ー」
 
。
 
六花晃动着双脚，很无聊似的嘀咕道。
 
。
 
「再稍微等一下应该就来了吧，在那之前我们只要等着就行」
 
。
 
「好——……」
 
。
 
西野对于她那没有干劲的态度苦笑着。
 
之后在等待其他成员的期间，他们聊着不着边际的闲话。
 
如此这般，门口那边走过来几名学生。
 
看到最前面的人物，西野在内心一脸嫌弃。
 
。
 
「……哼，居然没有迟到，以不良学生来说还真是值得钦佩的心态」
 
。
 
宫本开口就是招人厌的言语，对此西野在内心苦笑着。
 
为什么这家伙总是会特意引起别人的不和。
 
不对，还是说他是故意挑拨，等待着别人对自己动手呢？
 
如果是这样就不能中招了，即便这是多么容易看透的挑拨。
 
。
 
「那还真是谢谢您的称赞。然后呢，你为什么会在这里？我听说的是我们要与探索班会面来着？」
 
。
 
「啊啊，是啊。不过计划变更了，你们要和我们一起在学校周围寻找可疑人物。而我则是这支队伍的指挥长，仅此而已」
 
。
 
西野对于宫本的说明歪着头。
 
。
 
「……寻找可疑人物？怎么回事？」
 
。
 
「会长认为本次魔物袭击是有人所为，所以说要我们查到那位幕后黑手」
 
。
 
「你说什么……？」
 
。
 
这番话使得西野睁大眼睛。
 
应该是没听到详细的情形吧，呆在后方的成员微微有点喧哗。
 
。
 
「哼，一副意想不到的表情啊，低能的不良学生。五十岚会长说过本次魔物袭击有可疑之处，于是察觉到暗地里有操控魔物的人」
 
。
 
「嘿……原来如此。话说，如果就如刚才所说，你不也是没有察觉到这件事？就跟我们这些『低能的不良学生』一样」
 
。
 
硬是挑拨地说完以后，宫本瞬间涨红了脸。
 
。
 
「吵、吵死了！闭嘴！我肯定早就察觉到了吧！只、只不过没有说出来而已！」
 
。
 
看样子他似乎可以瞧不起人，但却忍受不了自己被人瞧不起。真是没有度量的男人，西野在内心中感到厌烦。
 
他这种态度似乎愈加触怒了宫本。
 
只见宫本焦躁地用手指抵起眼镜。
 
。
 
「总而言之！你们被选为寻找那位魔物师的成员。……原本我光是想象到与你们这些不良学生共同行动都会感到不悦，然而五十岚会长已经命令过我，让我务必把你和相坂加入成员之中，所以我也不能断然拒绝。哼，你们这些不良学生可是得到与我们一同为那位会长效力的荣誉，给我哭着感谢吧！」
 
。
 
「……小西，我可以揍这家伙吗？是说我要揍了，把他的脸揍变形」
 
。
 
面对宫本高高在上的措辞，六花逐渐忍耐到极限，她紧握拳头站起来。身上所释放的杀气足够使得宫本胆怯了。
 
他发出「噫」的一声难堪的悲鸣。
 
。
 
「住手吧，六花」
 
。
 
「为什么要阻止我，小西？」
 
。
 
「现在就算揍这家伙也毫无意义。……而且这只会给五十岚会长带来麻烦而已」
 
。
 
「……虽然我才更加搞不懂你这样说的意思，这种事怎么样都无所谓吧。是说为什么在这时候提前那女人的名字？」
 
。
 
六花对于莫名提前那位学生会长的西野感到不可思议，与此同时也先放下拳头。
 
。
 
「哼、哼。真的是，多么野蛮啊……。总之我们走吧！」
 
。
 
明显是虚张声势的宫本连忙原路返回。
 
其他成员也紧跟其后，西野又思考着。
 
。
 
（操控魔物的人物吗……真棘手……）
 
。
 
并且水平还是达到可以操控那么强大的魔物。
 
十分具有威胁。
 
的确是有必要迅速查找出来。
 
为了会长。
 
。
 
（没错，无论发生什么都要守护那位会长……）
 
。
 
因为她是非常重要、非常重要的人。
 
…………等一下？为什么我会觉得非常重要？
 
怎么说呢？总觉得静不下心。
 
为什么自己会这么珍惜那位会长啊？
 
。
 
「怎么了，西野君？」
 
。
 
被人搭话，西野的思考也回到了现实。
 
往那边一看，身旁有位穿着学校指定运动衫的女学生。
 
她是在购物中心遇到的少女——葛木。
 
。
 
「是葛木同学啊，我没问题，什么事也没有」
 
。
 
「是吗？总觉得你的表情很不好」
 
。
 
「哈哈，谢谢你担心我。……话说回来，葛木同学也是探索班的？」
 
。
 
西野硬是岔开话题。
 
。
 
「啊，嗯。虽然我的等级很低，但现在是这样的状况，我也想帮到大家的忙。刚才真对不起了，给你们两人带来不好的印象……。宫本同学也不是什么坏人，只是似乎很讨厌不良学生……」
 
。
 
「没关系，我也没怎么在意」
 
。
 
西野说完以后，葛木露出像是安心似的表情。
 
。
 
「啊，葛木也是一起的啊，请多多指教咯ー」
 
。
 
六花挥着手与葛木身体亲密接触，虽然她一瞬间有点胆怯，但立刻恢复成笑容。
 
。
 
「嗯、嗯。相坂同学也请多多指教」
 
。
 
如此说着，两人握着手。
 
西野也微笑地注视着这俩人。
 
。
 
（没错，现在首先是要寻找操控魔物的人物）
 
。
 
如此想着，西野将刚才的疑问抛到脑后。
 
。
 
「好，那我们也走吧」
 
。
 
「也是呢ー，要是聊太久，那个眼镜又要说什么坏话了」
 
。
 
六花模仿起用手指抵起眼镜的动作说完之后，他们也像是同意似的互相欢笑着。
 
紧接着。
 
。
 
「……嗯？」
 
。
 
骤然间，六花歪着头。
 
。
 
「怎么了，六花？」
 
。
 
「没事，感觉刚才好像听到奇怪的声音，似乎是『收到邮件』」
 
。
 
「邮件？」
 
。
 
「嗯」
 
。
 
六花心想究竟是怎么回事，姑且打开人物状态面板。
 
毕竟是跟升级时同样的声音，她觉得该不会是有什么情报在上面。
 
接着正如她所预料，界面发生变化了。
 
。
 
「诶、这是什么？……邮件信息？未读……一封？」
 
。
 
初次看到的界面。
 
其中有个显示为未读的部分正闪烁着。
 
她慢慢地点击那部分，界面又发生变化。
 
。
 
「寄信人……一之濑　奈津。诶、奈津？诶、啊、诶？」
 
。
 
看到寄信人名字的瞬间，六花更加混乱。
 
。
 
「喂，六花，怎么了？」
 
。
 
就连西野那觉得诧异的声音都没有传到她耳里。
 
她的注意力全都在眼前的邮件界面上。
 
随后映入眼帘的是主题。
 
。
 
「主题是……快逃？」
 
。
 
接着最后就是本文。
 
。
 
在那上面单单只是写着一句话。
 
 
 
 
 
。
 
。
 
 
 
 
 
―――现在立刻从那家伙身边离开。
 
 
 
 
 
。
 
「……诶？」
 
。
 
随着一道微弱的开枪声，身旁的葛木飞了出去。
 
一一一一一一一一一
