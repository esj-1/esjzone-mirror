WEB 22 决出胜负与视线的来源
 
 
一一一一一一一一一
 
　被注意到了？
 
　一瞬间我这么想着，但似乎是搞错了。
 
　看守的哥布林在叫唤它们看向外面。
 
。
 
　……怎么了？
 
　往那边一看，外面有几只哥布林。
 
。
 
　……完了，还有同伴吗？
 
　我和莫莫加强警戒。
 
。
 
「吱吱——！！吱！吱吱！」
 
。
 
　然而看守的那位哥布林一直在向外面的家伙威吓着，并且还将东西丢出去。
 
　……不是同伴吗？
 
　难道说哥布林也有所谓的群体和团体之类的？
 
。
 
　仔细想想看也是。
 
　毕竟人类和动物都会形成群体，都会互相吵架。
 
　所以哥布林会有这种情况发生也不是奇怪的事情。
 
　也就是说哥布林也不全是团结一致吗。
 
。
 
「吱吱——！吱—吱——吱！！」
 
。
 
　看守的哥布林无数次吼叫着，然而那些哥布林依然还是不肯离开。
 
　也许是感到不耐烦，吃完苹果的大型哥布林站起身。
 
　它慢吞吞得走到店门，大声咆哮。
 
。
 
「嘎嘎——啊啊啊啊！！」
 
。
 
　一瞬间我提防着类似于那头高级猪头人一样的冲击力，但是杞人忧天了。
 
　大型哥布林真的就只是在『大叫』而已。
 
。
 
　它没有像是那头高级猪头人的『大叫技能』吗。
 
　……正合我意。
 
。
 
　大型哥布林叫完以后，外面那些哥布林们才勉为其难地离开了。
 
　而看守的哥布林则是无数次向大型哥布林低头。
 
　完全呈现出阶级差距了ー……是说不对啊，才不是这样！
 
　为什么我要悠闲地看着啊。
 
　这可是机会！
 
　三只哥布林和大型哥布林已经分开来！
 
。
 
「莫莫！」
 
。
 
「……汪！」
 
。
 
　我给莫莫发出信号，行动起来。
 
。
 
「收纳！」
 
。
 
　首先我将把我遮掩住的前后『橱柜』都收纳到道具包里。
 
　我已经验证过即便是这种东西也可以收纳进道具包里。
 
　而橱柜上的东西都停留在空中，掉在地面上发出杂乱的声音。
 
　果然我只要想『只』收纳橱柜就会变成这样吗。
 
。
 
「吱!?」
 
「吱吱!?」
 
。
 
　哥布林们面对突然发出的声音而大叫着，目光都往那边望去。
 
　于是看到我这个人类，哥布林们更加惊愕。
 
　在它们看来，就是柜橱突然消失不见，而我出现了。
 
　就连大型哥布林都甚至惊讶到露出一张很傻的表情。
 
　很好，你就继续给我惊讶到僵硬不动吧！
 
。
 
　我立刻取出收纳进去的『橱柜』。
 
　为了作为隔开大型哥布林和哥布林它们的『墙壁』。
 
　在某些场所里，道具包的优势就是可以像这样创造出利于自己的地形。
 
　虽然花了几秒钟，但这样一来哥布林它们就完全分开了。
 
　一口气决出胜负吧！
 
。
 
「莫莫！那边就交给你了！」
 
。
 
「汪！」
 
。
 
　我和莫莫分开行动。
 
　我是跑向蔬菜水果区那边呆立不动的哥布林们。
 
　而莫莫则是奔向看守的哥布林那边。
 
。
 
　我动作利落地取出菜刀，往眼前的哥布林刺过去。
 
　面对我提升到三位数的敏捷速度，哥布林甚至都反应不过来。
 
。
 
「吱……!?」
 
。
 
　它倒下了。
 
　我没有空闲去确认它的生死。
 
　剩下两只……。
 
　我往哥布林头上取出『洗衣机』和『冰箱』。
 
　那两只哥布林连逃跑时间都没有，直接被电器家具压在底下。
 
　在最初的战斗中就已经验证过，普通的哥布林变成这样就不能动弹了。
 
　这样就封锁住这些家伙的行动。
 
　再来就是―――。
 
。
 
「―――吱吱吖吖吖吖吖吖吖吖吖吖！！」
 
。
 
　刹那间―――大型哥布林突破橱柜现身了。
 
　表情已然不是刚才那傻呆呆的样子。
 
　而是一副自己的基地***乱的魔物性的表情。
 
。
 
「抱歉了―――我可没打算跟你正面硬碰硬啦！」
 
。
 
　我从道具包里面取出『那个』，将其抛向大型哥布林。
 
　想当然尔，大型哥布林会用手甩开，进而避免攻击。
 
。
 
　然而，骤然间『那个』就发出啪地一声，并随之破裂。
 
。
 
　随着散发出独特的气味，里面的液体也溅洒在大型哥布林的颜面上。
 
　大型哥布林一时之间蹙起眉头，立刻想要擦拭掉。
 
　然而―――
 
。
 
「―――吖？～～～～～!!?吖吖吖吖吖吖吖吖吖吖吖‼??」
 
。
 
　这瞬间，大型哥布林发出不成喊叫的悲鸣。
 
　它痛苦地呻吟着，双手按住自己的脸。
 
。
 
　那个就是所谓的『水气球』。
 
　里面混杂进红辣椒粉。
 
。
 
　刚才吃完午饭之后，我和莫莫在商量的时候制作出来的。
 
　跟装水不同，要装进一些辣椒粉实在是很麻烦。不过多亏『技巧』提升了，因此能够正如我所想象中制作出来了。
 
　这可是我的自信作，你是否喜欢呢？
 
。
 
「～～!!吖吖吖!嘎吖吖吖吖吖!!」
 
。
 
　也许是太痛苦了，大型哥布林总想要擦掉辣椒粉，但却反而是自作自受。
 
　毕竟辣椒粉进到眼睛里真的是很痛苦的。
 
 
 
 
 
「抱歉了，我可不会手下留情……！」
 
。
 
　虽然大型哥布林按着脸呻吟着，但也绝不能轻易接近。
 
　于是我这次对准它头上取出了『自动贩卖机』。
 
　这是之前在大街上走的时候借来的东西。
 
　自从看到某本轻小说里面某位调酒师风格的男人将这玩意丢出去以后，我一直就想要这么做。
 
　自动贩卖机的功能可不单单只是滋润他人的喉咙。
 
 
 
 
 
没错，自动贩卖机也是可以砸死人的凶器。
 
　利用自动贩卖机的边角部分对准它的头顶―――我放。
 
。
 
　咚锵！响起一阵沉重声色。
 
。
 
　大脑被自动贩卖机砸到，大型哥布林趴在了地面上。
 
。
 
「アアアアアアアアアアアア!!!アアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアアッッ!!」
 
。
 
　它发出悲鸣。
 
　头脑有一半被砸惨了。……好血腥。不过它还没有死。
 
。
 
「……再来一遍」
 
。
 
　我压抑住涌起的想呕吐的冲动，收纳了自动贩卖机。
 
　正当我想要再次释放的瞬间，大型哥布林想要站起来。
 
。
 
「吖嘎……吱―――!!?」
 
。
 
　然而它的双手双脚都被『影子』束缚住。
 
。
 
「干得好，莫莫」
 
。
 
　仔细一看，莫莫已经打倒看守的哥布林了。
 
　它往这边过来，而『影子』的束缚力也愈加强大。
 
　这样大型哥布林就没办法避开了。
 
　尽管它拼命抵抗，但大脑已经砸坏一半了。
 
　应该是没办法正常施力吧。
 
。
 
　我再次向大型哥布林的头上取出『自动贩卖机』。
 
　咔嚓咔嚓。
 
　重复两次这种工程之后，大型哥布林完全死亡了。
 
　脚边滚落一块蓝色魔石。
 
。
 
　剩下的就是收拾残局。
 
　我和莫莫将剩下的那些无法动弹的哥布林杀死。
 
。
 
≪获得了经验值≫
 
≪经验值达到一定水平≫
 
≪工藤和人从ＬＶ６升级到ＬＶ７了≫
 
。
 
　天之声告知我升级了。
 
。
 
　好、很好、很好……！
 
　做到了、做到了喔！
 
。
 
「干掉了……赢了喔，莫莫！是我们的胜利！」
 
。
 
「汪！」
 
。
 
　七只哥布林，一只上级种的大型哥布林。
 
　考虑到以往的战斗，这已经是绰绰有余的战果了。
 
。
 
　虽然沉浸在胜利的余兴是好啦，但还是先把点数花掉吧。
 
　而且我也想将这里的食物都全部收纳了。
 
。
 
「莫莫，总之我们先往里面去」
 
。
 
「汪！」
 
。
 
　如此这般，在商业合作社的战斗结果以大成功闭幕了。
 
　然而，乐在其中的我并没有注意到。
 
　刚才的战斗。
 
　有一位『观望到的人』―――。
 
 
 
 
 
。
 
 
 
 
 
　那位人物就在距离商业合作社大约百米左右的公寓楼顶。
 
。
 
「好厉害……那种数量都能打赢……」
 
。
 
　利用照准器的那双眼瞳，一直都在注视着商业合作社发生的战斗。
 
　遗憾的是有许多地方都是死角，因而看不到。不过倒是清楚看见那条柴犬使用『影子』将哥布林瞬间秒杀，以及那位男性利用自动贩卖机砸死哥布林的上级种。
 
。
 
「毫无疑问是等级持有者……而且还拥有稀有技能」
 
 
 
 
 
　然而比什么都值得一提的刚才在那场战斗开始之前。
 
　那位男性注意到自己的『视线』。
 
　否，正确地来讲应该是感觉到违和感的程度。即便如此，在这种距离上光是有所感觉就已经值得惊讶了。
 
　回想起当时的视线，身体颤抖了。
 
　不过那阵颤抖并不是因为感到害怕。
 
　而是欢喜的颤抖。
 
。
 
「除了我以外还有那样的人……」
 
。
 
　―――积极地狩猎魔物的人。
 
。
 
　如此嘀咕完以后，『她』当场离开了。
 
　并且在其肩膀上还扛着一把长度几乎等同于身高的来福枪。
 
一一一一一一一一一
