WEB 48 破灭的步伐声
 
 
一一一一一一一一一
 
 
 
 
 
　为了逃离那一群影狼，我们按原路返回了。
 
　明明都已经拉开这么长的距离了，还要走回去实在不甘心，不过性命是无可取代的。
 
　影狼和猪头人一样是嗅觉优秀。
 
　有可能还会通过当场遗留下来的气味从而追踪我，所以现在还不能放松警惕。
 
。
 
「总觉得我和影狼真是有缘分了啊……」
 
。
 
　主要是在负面的意义上。
 
　在百货商店的时候也是，不知为何总冲着我来。
 
　我都忍不住胡思乱想，是不是当时撞死了影狼，导致吸引到它们的仇恨了。
 
。
 
「总而言之，我们稍微休息一下吧」
 
。
 
「汪」
 
。
 
　我依靠在自动贩卖机上面坐了下来。
 
　目前来看，『索敌』和『危机感知』都没有反应，所以应该是没问题的，我希望是这样。
 
　口渴了，我从道具包里面取出运动饮料。
 
　莫莫似乎也想喝的样子，于是我也倒点在碗里。
 
　当然，倒的是狗专用的饮用水。
 
　滋润着喉咙，我将刚才与猪头人战斗中获得了菜刀拿出来。
 
。
 
「『猪头人的菜刀』……吗」
 
。
 
　这武器就是字面上的意思。
 
　刀长80公分吧。
 
　既没有生锈，锋利度也似乎很不错。
 
。
 
　试着挥一下看看。
 
　嗯，这样的重量应该是没问题了。
 
　比起以前的菜刀，还是用这把吧。
 
　并不是说就因为是魔物使用的武器，所以我就挑剔地不想要用。
 
。
 
　话是这么说，既然我的主要战斗方式是道具包和奇袭，那几乎就没有剑刃对拼的机会了吧。
 
　不如说要是这样就麻烦了。
 
　即便如此，有不错的武器总比没有要好。
 
　反正『剑术』和『要害突击』在某种程度上还能弥补上水平。
 
。
 
　说起来西野君的团队里也有位女高中生使用这种武器啊。
 
　虽然那边是劈刀。
 
。
 
「啊，姑且看一下邮件吧……」
 
。
 
　打开邮件界面，正如预料『未读』有很多。
 
　……回信就回最后一封好了。
 
。
 
「嗯……？」
 
。
 
　我粗略地看了一下邮件，直到看到最新的邮件才发现到里面写着令人的内容。
 
。
 
「看到一群猪头人……？」
 
。
 
　上面写着「看到一群猪头人了，请小心一点」。
 
　是在几分钟之前受到这封邮件。
 
　也就是在我逃离影狼的时候吧。
 
。
 
　……猪头人群体。
 
　我最先想到的就是在购物中心的那些家伙。
 
　继续看了下去，有提到『红铜色的猪头人』。
 
　不会错的，就是那家伙。
 
　该不会是出来寻找猎物了吧？
 
　真是棘手。
 
　并且发现到它的地方就离这样不远。
 
　就在我昨天跟大型哥布林战斗的附近。
 
。
 
　我往那边集中意识，的确有股『厌恶感』。
 
　就跟当时在便利店面向购物中心的时候一样。
 
。
 
　看样子，一之濑的情报是准确的。
 
　这次得老实地感谢她了。
 
　先回一封道谢的邮件。
 
。
 
　……仅过了几十秒就有回信了。
 
　所以说你这也太快了吧！
 
　不如说一之濑小姐。
 
　既然有时间回我信息，你还不如赶紧逃跑不就好了。
 
。
 
「嗯~……」
 
。
 
　话说回来，这样麻烦了。
 
　总觉得这附近的魔物在增加。
 
　有必要警惕的是影狼群体和猪头人群体。
 
　由于这两支魔物群体，导致能够通行的路线相当有限。
 
　有必要从最初的路线大幅度迂回过去了啊。
 
　嘛，这也是没办法。
 
　虽然很花时间，但安全才是最重要的。
 
。
 
　……嗯？等一下啊。
 
　说起来那附近距离百货商店也很近吧。
 
。
 
　……西野君他们没事吧？
 
　嘛，虽然我感觉他们应该能顺利逃跑。
 
。
 
　那么，我该走哪条线路呢……。
 
 
 
 
 
。
 
。
 
。
 
 
 
 
 
　另一方面，此时在百货商店―――。
 
。
 
　西野向回归的柴田打听究竟发生了什么事。
 
。
 
　回来的柴田遍体鳞伤，还陷入轻微的恐慌症。
 
　做好应急的治疗措施后，斟酌着他取回冷静的时机询问他发生了什么事。
 
。
 
　柴田渐渐地诉说道。
 
　增加探索范围，到达那座购物中心。
 
　在那边遇到实力不同寻常的猪头人。
 
　以及除了自己以外其他人都死亡。
 
　这些事情他都毫无保留地说了出来。
 
。
 
「是吗……」
 
。
 
　听完柴田所说后，西野仰头看着天花板如此嘀咕。
 
。
 
「事情我已经清楚了，真亏你能逃回来啊，柴田……」
 
。
 
　他把手搭在柴田的肩膀上，似乎在慰劳。
 
。
 
「……西野，我太差劲了。丢下朋友不管，自己一个人逃回来……可恶！可恶啊！」
 
。
 
　柴田悔恨似的紧握拳头。
 
　应该是在宣泄无处释放的情感吧。
 
　虽然他平时态度恶劣、性情暴躁，但他是真的为同伴着想。
 
。
 
「啊啊，我能体会你的心情。不过柴田，如果连你也死了，我们根本不知道这样的情报。你的行动是正确的，并不是值得羞耻的事情」
 
。
 
「……」
 
。
 
　面对这番话，柴田不禁移开目光。
 
。
 
「不能接受吗？那我就换种说法吧。如果你真觉得悔恨，那就尽可能想着活下去。如果不这样做，那些人到底是为了什么而死?而你又是为了什么而抛弃他们？如果你还想为你的行动寻求意义，如果真心想要回报他们，那你自己就要一直努力活下去。……明白了吧？」
 
。
 
「……知道了」
 
。
 
　面对西野这番无可反驳的安慰，柴田只能点点头。
 
 
 
 
 
他似乎还不能接受，不过之后只能交给本人整理心情了。
 
。
 
（……如果不能振作起来，到时候就再说吧）
 
。
 
　西野在心中如此思忖着，并没有表露在脸上。
 
。
 
「然后呢小西，今后要怎么办？」
 
。
 
　看准谈话结束的时机，辣妹女高中生六花说道。
 
　面对她在这种时候都没有紧张感的语调，西野苦笑着。
 
。
 
「……是呢，既然有这么强大的魔物在，那就只能到其他的地方了吧」
 
。
 
「其他的地方是指？」
 
。
 
「像学校、市政府这一带吧。虽说这些地方都离这里很远，不过设施应该比起这里还要齐全才对。大概也有其他的避难民吧」
 
。
 
「让他们收留我们吗?不过对方估计也没有这种从容了吧？」
 
。
 
「那就跟他们交涉。总之决定好路线，做好出发准备后立马动身吧」
 
。
 
　西野摊开地图。
 
　以现在的人数可以走过的路线。
 
　并且要怎么走才能尽可能保证安全。
 
　他目光落在地图上，思考着接下来的事情―――这瞬间。
 
。
 
「喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔喔!!」
 
。
 
　听到魔物的咆哮。
 
　犹如震撼着整座建筑物般的猛烈咆哮。
 
。
 
「敌袭！敌袭！」
 
「是魔物！出现了一群猪头人！」
 
。
 
　门口那边的声音传达过来。
 
。
 
「……看样子是让对方抢先一步了啊」
 
。
 
「……似乎是这样呢」
 
。
 
　两人举起武器，立马赶往门口。
 
一一一一一一一一一
