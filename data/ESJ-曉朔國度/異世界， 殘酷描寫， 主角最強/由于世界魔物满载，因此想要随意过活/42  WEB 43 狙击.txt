WEB 43 狙击
 
 
一一一一一一一一一
 
 
 
 
 
　赶紧用掉升级时得到的点数吧。
 
　这次提升的职业也是『暗杀者』。
 
　提升到ＬＶ８，剩下５点数保留起来。
 
　顺利的话再升两级就可以提升到ＬＶ１０。
 
　这样一来估计会转职为下个阶段的上级职业。
 
　『暗杀者』的上级职业究竟会是什么呢？
 
　……虽然我只有可怕的预感就是了。
 
　嘛，这件事还是到时候再想好了。
 
。
 
　接下来是技能这边，『危机感知』提升到ＬＶ８，『敌意感知』提升到ＬＶ６。
 
　这样就剩下0点数。
 
。
 
　虽然技能我是使用了，但熟练度不怎么提升啊。
 
　感觉也该差不多提升『剑术』和『要害突击』比较好了。
 
　只能脚踏实地地打倒魔物了吗。
 
。
 
　那么就走吧。
 
　啊，说起来这附近还有商业合作商铺来着。
 
　顺便也把那边的物资带走吧。
 
。
 
　―――走了几分钟。
 
　抵达商业合作商铺。
 
　『索敌』有反应了。
 
　不是魔物，是人类。
 
　看来是有人先到了。
 
。
 
　我躲藏在附近窥视着情况，是学生们在店里。
 
　那应该……是西野君的团伙吧。
 
　总共是四个人数，其中一个是眼镜君。
 
　他们满脸笑容地将食材装进背包或者购物推车里面。
 
　应该是终于找到食粮了吧，相当高兴的样子。
 
。
 
「太棒了，有这么多吃的食物」
 
「啊哈哈，可乐超好喝的！虽然不是冰的就是了！哈哈哈哈哈哈」
 
「喂，马铃薯也吃点吧！反正全部都要带回去啦」
 
「你、你们……不要再吃了，快点拿回去吧……」
 
「啊？大野，你这家伙究竟是在命令谁啊？」
 
「对对对。西野也真是没有眼光，找个更有用的家伙不就好了吗。啊，你要是告状就杀了你。啊哈哈哈哈」
 
「好、好痛。别这样，别打我肩膀啊……！」
 
「也就只是碰碰你而已，别总搞得这么大的反应，你这家伙」
 
「没错。啊，大野，马铃薯你拿着」
 
。
 
　呜哇ー……感觉就像是典型的霸凌者与被霸凌者之间的谈话。
 
　眼镜君好可怜，虽然我不会帮你。
 
　一个人在默默地装着食粮，还被他们指使。
 
。
 
　话说东西都装在推车上真的没问题吗？
 
　那玩意在推的时候就没有声音？
 
　虽然我感觉遭到魔物袭击的概率变高了……。
 
。
 
　嘛，这也不是我该想的事情。
 
　总之在这里等一阵子吧。
 
　过几分钟之后，他们也应该会离开了。
 
　然后我再回收物资就好了。
 
 
 
 
 
。
 
　接着几分钟过后―――。
 
　他们终于装好东西，正准备离开时，『索敌』有反应了。
 
。
 
「嗯……？」
 
。
 
　不远处有股气息……不是人啊。
 
　是魔物。
 
　我稍微探出身子利用『远望』确认一下，发现有两只小狼。
 
　它们正逐渐往这边缓慢接近过来。
 
。
 
　偏偏是这群家伙吗。
 
　又是我吗?又是来我这边吗？虽然我是这么想的，但看来似乎不是。
 
　小狼它们貌似是盯上了店里的学生们。
 
　看也不看我这边一眼，直接往商业合作商铺过去。
 
。
 
「喂、喂！你们看！」「不会吧……那不是魔物吗」
 
「那群家伙不就是昨天战斗过的吧？」
 
。
 
　眼镜君他们终于注意到小狼它们。
 
　相当慌张的样子。
 
　就没有人拥有『索敌』系的技能吗？
 
　真亏你们这样也能出来探索啊。
 
　是说你们也该预料到魔物会袭击吧，根本用不着这么慌张。
 
　西野君，你这是搞错人选了吧？
 
。
 
「你、你们还在干什么啊！快点逃跑吧！」
 
。
 
　学生们都纷纷拿起武器，只有眼镜君一个人在唱反调。
 
　他很宝贵似的背着背包，并且还大声地叫喊逃跑吧。
 
。
 
「哈？大野，你是在害怕吗？敌人就只有两只喔？这不是很轻松吗？」
 
「说的对！昨天也升级了，这次可不回输给它们了啦」
 
「可、可它们有可能会像昨天一样唤来同伴吧？这、这样的话，凭我们根本就打不过……」
 
。
 
　眼镜君似乎在警惕小狼会像昨天一样呼唤同伴。
 
　这是正确的。
 
　它们那种举动，连我都有了大麻烦。被换来的魔物个体不知为何总来我这边。
 
　然而，其他的学生似乎不会这么想。
 
。
 
「哈！那么只要在那种事发生之前打倒它们不就好了！」
 
「对啊！毕竟机会这么难得，也顺便提升等级吧。总不能让六花那个笨蛋一直在抢尽风头吧！」
 
「呜啊啊啊啊啊啊啊！给我去死吧啊啊啊！」
 
。
 
　他们往小狼那边冲过去。
 
　在呼唤同伴之前打倒就好了吗。
 
　呜哇——……这是什么无脑理论。
 
。
 
　是说这样根本行不通吧。
 
　这些家伙真的有在看清现实？
 
　明明昨晚都经历过那种地狱了，为什么还能说出这种话？
 
　还是说他们的等级都已经高到这么自信了？
 
。
 
「呜啊啊啊啊啊啊！去死啊啊啊啊啊啊!!」
 
「嘎咕噜噜噜噜噜!!」
 
。
 
　小狼与学生们开始战斗。
 
　眼镜君没有参与其中，所以是三打二吗。
 
　毕竟数量上比较有利，说不定还真的能秒杀小狼……办不到了
 
　他们完全就是苦战中，不如说被小狼压制着。
 
。
 
　嗯，单纯只是没有看清现实而已。
 
。
 
　无论怎么看，他们的等级都不高。
 
　技能也都没什么了不起的吧？
 
　真服了他们这样也能说大话。
 
　西野君没有跟他们说过食粮和安全是最重要的吗？
 
。
 
　看到他们的样子，眼镜君的脸色越来越苍白。
 
 
 
 
 
他连忙背紧背包。
 
。
 
「……！抱、抱歉！」
 
。
 
「啊、喂！大野，你这家伙!?」
 
「逃跑了吗，喂！」
 
。
 
　眼镜君丢下同伴逃跑了。
 
　他的身影也逐渐越走越小。
 
　好快的逃跑速度啊……。
 
。
 
　嘛，不过这也是正确的判断吧。
 
　毕竟无论怎么样，他们都听不进眼镜君说的话。
 
　还有就是他们大概打不过了。
 
　既然这样，还不如自己拿着食粮尽可能逃跑比较好。
 
。
 
　嗯，他并没有做错。
 
　至少我就会这样做。
 
　那么，他们要怎么办呢？
 
。
 
「哈啊……哈啊……可恶！」
 
「喂，该怎么办啊……？再这样下去……」
 
「吵死了！事到如今只能硬着头皮上了吧！」
 
。
 
　学生们的伤口越来越多，逐渐感到疲惫。
 
　这样大概会死吧。
 
。
 
「呜啊啊啊啊啊啊！」
 
。
 
　一位学生向小狼发起突击。
 
　然而小狼轻松躲开，并且打算就这样直接咬住他的脖子。
 
　啊，这人死定了吧。
 
。
 
　当我这么想着的瞬间，听到一声「砰」的微弱开枪声。
 
。
 
「……诶？」
 
。
 
　骤然间，小狼的头部炸开了。
 
。
 
「哈……？」
 
。
 
　学生露出一副不知道发生什么事的茫然表情。
 
　我也一样。
 
。
 
　怎么了？刚才发生什么事了？
 
。
 
　我又再次听到一道「砰」的微弱开枪声。
 
　于是，另外一只小狼的头部也炸开了。
 
。
 
　……该不会刚才的是狙击吧？
 
　可是到底是从哪里开枪的？
 
　那位眼镜君的技能？
 
　不对，大概不是。
 
　声音传来的方向跟他逃跑的方向完全不一样。
 
　地点是―――。
 
。
 
　我往声音传来的方向看过去。
 
　距离这里百米以上有座高层公寓。
 
　公寓的屋顶上似乎闪着不自然的光芒。
 
　在那里吗。
 
。
 
「喂、喂！虽然不知道怎么了，但我们还是趁现在逃跑吧！」
 
「啊、啊啊！」
 
「噫噫！」
 
。
 
　学生们都拼死地逃跑。
 
　比起他们那不堪的样子，我的注意力早就集中在对面的高层公寓上。
 
。
 
　是在那边射击的？
 
　这样一来，这距离可是相当远喔？
 
　枪？……在这个日本？
 
　不对，是某种技能吗？
 
。
 
　正当我思考着种种疑问时，天之声响了起来。
 
。
 
≪收到了信息≫
 
。
 
　……哈？
 
。
