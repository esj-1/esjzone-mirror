翻译：飒君CONAN
 
 
 
WEB 36 即便知道也有着无法抵抗的魅力
 
 
一一一一一一一一一一
 
　我注视着处于十字路口中心处的宝箱。
 
　好可疑……实在太可疑了。
 
　那怎么看都是陷阱吧？
 
　与其说是陷阱，不如说该不会是魔物吧?比如宝箱怪之类的。
 
。
 
　可是『敌意感知』和『危机感知』、『索敌』都没有反应啊。
 
　不过那玩意又这么引人注目……。
 
　『没有反应』，反而令人觉得可疑。
 
　也有可能是『妨碍感知的技能』。
 
　在有『鉴定妨碍』这种技能的时点上，就可以预料到存在妨碍系的技能。
 
。
 
「可是也有可能是真的宝箱啊……」
 
。
 
　如果是真的，里面会有什么东西啊？
 
　宝物？魔石？还是武器？
 
　要说在意也的确是蛮在意的。
 
。
 
「就稍微试一下吧」
 
。
 
　我走到在十字路口看来是死角的位置，躲藏起来。
 
　总之丢块石头看看。
 
　砸中了。
 
　没有反应。
 
。
 
「莫莫」
 
。
 
「汪」
 
。
 
　我让莫莫伸长『影子』，试试看能不能打开。
 
　于是当莫莫的『影子』触碰到宝箱的瞬间―――
 
。
 
 
 
 
 
「呷丫丫丫丫丫丫丫丫丫!!」
 
。
 
　宝箱打开，从中冒出一个寄居蟹一样的魔物。
 
　比起宝箱，很明显是魔物的体型更大。
 
　你究竟是怎么钻进去的啊？
 
　给我遵守物理法则啊。
 
。
 
「话说果然是魔物么……」
 
。
 
　不是宝箱，而是宝箱怪……。
 
　该死，把我以为或许是真正的宝箱这种期待心还给我。
 
。
 
　当宝箱怪展现姿态的瞬间，『敌意感知』、『索敌』都出现反应了。
 
　果然是拥有『妨碍系』的技能。
 
　不过从宝箱出来以后似乎就没有效果了。
 
。
 
「呷丫丫丫―――丫丫…………丫？」
 
。
 
　宝箱怪注意到自己周围没有猎物之后，做出一种类似于「啊咧？」的举动。
 
　这幅样子莫名觉得可爱。
 
　不过魔物就是魔物。
 
　我不会手下留情。
 
　宝箱怪所在的地方正是我道具包的效果范围。
 
。
 
「―――砸死吧」
 
。
 
　我把刚才在施工现场借走的建筑机械朝着宝箱怪释放。
 
　咕吱，响起一阵低沉声。
 
。
 
≪获得了经验值≫
 
。
 
　听到了天之声。
 
　宝箱怪似乎因为刚才那一击就死翘翘了。
 
　意外没什么了不起的。
 
（颯:我他º妈在RPG游戏里被宝箱怪虐几次了）
 
。
 
　我迅速回收建筑机械。
 
　宝箱怪所在的位置有一颗棕色的魔石。
 
　放进道具包里面一看，显示的是『宝箱怪的魔石（极小）』。
 
　从威胁性来看就跟哥布林和小狼差不多吗。
 
　嘛，毕竟是那么显然的陷阱。
 
 
 
 
 
　不过要是还有除了『拟态』成宝箱以外宝箱怪就麻烦了。
 
（颯:日本那边对于宝箱怪是有两种解释的。
 
。
 
其一，宝箱怪原本外形就是类似于宝箱
 
其二，是某种怪物变化成其他样子，也称拟态，有些游戏里还会变成道具）
 
。
 
　如果『拟态』成随处可见的日用品―――衣橱或者垃圾桶之类的，危险度会瞬间提升吧。
 
　多注意一下比较好吧。
 
。
 
「下次要提升『感知系』的技能吗……」
 
。
 
　如果『索敌』和『危机感知』的级别提升了，也许就能看穿宝箱怪的拟态也说不定。
 
　比起『肉体強化』或者『剑术』，说不定还是提升这方面的技能比较好。
 
　我应该要慎重考虑一下技能点数的分配。
 
　嘛，过去的事情再后悔也是无济于事。
 
　只要继续赚取技能点数就行。
 
。
 
「那我们走吧，莫莫」
 
。
 
「汪！」
 
。
 
　我们再次以百货商店为目标，踏出了步伐。
 
 
 
 
 
。
 
 
 
 
 
　―――舞台转到百货商店这边。
 
。
 
　百货商店的休息室。
 
　里面集中了以西野为首的学生们，他们正在商讨关于今后的目标。
 
　桌子上摊开着城市地图，地图各处都做好了标记。
 
。
 
「那么柴田和大野就分别在周围进行搜索吧。虽然确保食粮是最优先事项，但切勿不要乱来，听清楚了吧？」
 
。
 
「好」
 
「我、我知道了」
 
。
 
　外表像是不良学生的柴田，以及戴着眼镜丝毫很担心的大野都各自回应。
 
　而他们后面还有几位学生。
 
。
 
「还有，我们还要寻找其他的避难所。如果有比这里还要更安全的地方，那就确保路线移动到那边。如果有其他避难民已经占据了，那就跟他们交涉一下……说不定当中还有你们的家人，大家努力吧」
 
。
 
　西野边指着地图，边向大家说明。
 
　他所分配的地区以及路线都合情合理，因此大家都可以接受。
 
　学生们点点头。
 
。
 
「那么你们出发吧，就拜托大家了喔」
 
。
 
「「「「好的！」」」
 
。
 
　探索组到学生们精神奕奕地回应道，接着离开了房间。
 
。
 
「剩下的成员就跟以往一样守护这里，也帮帮昨天刚升级的那些避难民们。轮流的顺序就由你们自己决定，不过不要太逼迫别人」
 
。
 
「「「了解」」」
 
。
 
　回应以后，剩下的成员也都离开房间。
 
　剩下的就只有两个人，西野和绑着侧马尾发型的女高中生辣妹而已。
 
。
 
「大家都走了吗……。好，那我和六花就到那些还没有升级的家伙那边，可以吧？」
 
。
 
「诶~不会吧？超麻烦的说……」
 
。
 
　称之为六花的少女露骨地展现出不满的表情。
 
。
 
 
 
 
 
「别这么说嘛……在我们这些人里面就你是最强的，考虑到技能的相性，我和你一组才是最好的。而且理由我在昨天就说过了吧?现在尽可能想要多点战力」
 
。
 
「嗯~虽然我是知道小西想要表达的意思啦 ……」
 
。
 
　名叫六花的女高中生将手挽在脑后，撅起嘴。
 
　小西似乎是西野的爱称。
 
。
 
「老实说像那种大叔就算硬是要他升级，我也觉得只会碍手碍脚而已啦。特别是那个秃头的老头，现在都还在叫唤根本不可能杀死魔物什么的。真的烦死了」
 
。
 
「……」
 
。
 
「还不如干脆干掉他，伪装成事故一样不是更好吗？这样我们的经验值也―――」
 
。
 
「六花！」
 
。
 
　受到西野的怒斥，六花吓了一跳。
 
　她用手按住自己的嘴，很过意不去似的移开目光口。
 
。
 
「……对不起，忘了刚才那句话……」
 
。
 
「不要再说这种话了，只有这件事绝对不能让大家知道。如果一旦有人知道，肯定会出现跨过这条线的家伙。只有这点是……绝对不行」
 
。
 
「……嗯，我知道了」
 
。
 
　西野和六花走出休息室
 
　他们的表情似乎有些僵硬着。
 
一一一一一一一一一
