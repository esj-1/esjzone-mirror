搬運
移居之前，加奈子提出了自己成为维达新种族的建议。那一瞬间的她没有考虑，只是遵从强烈的感情，就这样直觉地说了。
。
但是这个发言是过去的加奈子狡猾的意志，巧妙地使用了能力的结果。
。
在罗德科尔特的神域中知道了轮回转生系统。所属于谁，有什么结果，亦自由地阅览了罗德科尔特和其御使的亚乱们的思想。
。
当然，这对于为了向班达鲁汇合为目的的加奈子来说很麻烦。实际上，正因如此，犯了一个和阿沙兹相同的错误。
。
为了摆脱与自己的眼睛和耳朵相连的间谍照相机和麦克风的情况下，只能从罗德科尔特的轮回转生系统中脱离。
。
为此，将成为某一维达的新种族，进入维达式轮回转生系统，就是加奈子所想到的最确切的方法。
。
也有其他的方法，种族可以直接被引导到班达鲁处。但是，不知道见面后会否马上被引导，亦不知道要用多长的时间。
。
但是，关于轮回转生系统的记忆从神域出来转生至《拉姆达》的时候就被取消了。所以没有相关的情报，不仅是物品，连肉体都没有的状态下，传达是不可能的。
。
因此，加奈子使用了由罗德科尔特给予的能力【维纳斯】。
。
【维纳斯】的能力，其他的转生者都是以为是使他人入迷的能力，实际上是可以复制自己和他人的记忆和感情，亦能包含自己的对象上贴上复制的能力。
。
而且，在努力和经验下的结果，加奈子成功地添加了一些条件。
例如，事先准备好能力，在与某个特定的人物直接见面时，使其潜伏的能力发动。
。
加奈子在神域期间，发现罗德科尔特的注意力从自己脱离时，把能力施给自己。和班达鲁直接见面时，对「维达的新种族化」的思想会活化【维纳斯】的能力。
。
但在这一点上，自己是否真的希望维达新种族化是一个赌注。
。
加上条件的方法，随着数量和复杂度的增加，【维纳斯】的成功率本身也会下降，即使成功也可能无法获得之前设定的想法。
。
而且根据当时的情况，也有可能无法感受到初时的想法。
。
但是，只要相信过去的加奈子是自己的话，相信这个感情是由【维纳斯】产生的便可以。
而现在的加奈子，成功地取得了胜利。
。
「啊，也就是说，能成为新种族的话，就可以移民了吗？？」
。
不过，没有考虑到维达的新种族化的理由。
。
「不，你能告诉我为什么你要成为维达的新种族吗？」
「加奈子好像不能说明理由」
。
在惊吓和困惑的时候，班达鲁交替地凝视着大古和梅丽莎，然后点了点头开始了说明。
。
「我要向你们要求维达的新种族化，是因为没有退路。阿德帝国就不必说，而在奥尔巴选王国的几个公爵领上，维达新种族的容忍度都很狭窄。特别是吸血鬼、魔人族、鬼人、古尔更被认为不是人类而是魔物。」
「原来如此……在变成那样的种族后，不管发生什么事都只能为国家效力了。很有意义啊。」
。
『大古！比起你所想的，班达鲁君临的国家很宽广，而且很舒适！而且完全没有局促的感觉，我瓦尔基里可以保证！』
。
「我不认为你说谎……国家？」
。
班达鲁的说明使大古的脸抽搐着，但瓦尔基里向大古这样保证。然而大古不知道班达鲁的影响力已经超越了塔洛斯海姆，所以他露出了一脸蠢相。
。
班达鲁无视了这一点继续进行谈话。
。
「还有其他理由的……要说的话是成为维达民族之后吧。那么，想得怎么样？如果真的決定了成为维达新种族的话，会事前进行简单的指导，之后也会进行各种各样的跟进。」
。
种族改变是指生态会发生很大的变化。在吸血鬼的情况下，如果成为深渊种，即使沐浴阳光也能满不在乎，外形，瞳孔的颜色和牙齿的変化也会压低至最低的変化。
。
但即便如此，也会伴随终身的吸血冲动，而且对吸血鬼用的阿蒂法特、内梅西斯贝尔等的武器会终身威胁。吸血鬼化时年龄就会停止增长，但长生不老只不过是个人的诅咒而已。
。
……另外，与原邪神派的爱莲奧诺拉和马伊鲁兹可能会发生各种各样的问题，比如与父母的原种、贵族种、其他吸血鬼的人际关系等，以及因为是新人，所以在吸血鬼社会的地位也会低下。
。
还有，除了吸血鬼以外的种族、魔人族、鬼人、古尔、拉米亚和阿刺克涅等的情况下，身姿会大幅改变。
和以前的人生会有很大的对比。所以不会用轻松的心情来推荐转生的种族。
。
「吸血鬼啊。嗯，也不是那么坏的事吧……」
「是啊，有点」
。
除了自己提议的加奈子，梅丽莎和大古还在犹豫着。
。
「在我的余生中，永远都是青少年的外表，真是有点抗拒呢。被小看的情况，以后也会继续下去。」
。
「如果知道要被这样要求的话，那一开始就要转生为成人的身体了。」
。
人种和精灵，种族是不同的，大约一年前转生成十几岁的少年少女的肉体，加奈子们三人还是年幼的身体。
。
在这个《拉姆达》世界上很多国家和地区都是十五岁就当作成人了，所以不论是对冒险者的入职，无论是酒巴还是赌場，什至进出欢乐街都是自由的。
。
但是，在现代日本，20岁的青年在社会上只是半社会人仕，会被当作年轻人对待。在《拉姆达》，很多场合也不被认为是已经独立的情况很多。
。
这样理解之后，所以大古感觉上『在外表上被小看了』。
。
在梅丽莎的情况下，外表看起来是个问题。
。
「……问题是外貌吗？」
。
『很马马虎虎的理解啊，平常没注意到的心理影响很快就会浮现。首先基本的外表和长生不老是没法改変的』
。
『即使是邪神派的原种吸血鬼，似乎都要等到吸血鬼的候选人在某种程度上成长。那是爱莲奥诺拉说的』
。
对加奈子和无奈的班达鲁，瞳和杰克这样说道。
。
「嗯……确实，比起现在的身体，在前世二十多岁左右的时候，体格是更好的……但是，精灵要等到那时要花几十年的时间，因为原本肉体的种族就是不同的，也可能不会成长到同样的体格，十几岁的样子也可能会持续下去。」
。
然后，也许是受到了大古语言的影响，甚至连加奈子都开始烦恼了。
。
『班达鲁，差不多该对我考虑的plan B进行说明吧？』
。
「plan B？阎王，有那个吗？」
。
『啊，实际上不是另一种选择，而是在维达新种族化前需要的时间下进行微调。
维达新种族化的仪式，根据种族的不同需要花费时间也不同。我们收到你们的信后马上就来了，所以会根据你们所选择的种族要暂时等待。』
。
而且，也有进行仪式的种族方面的问题。
。
如果加奈子成为维达新种族，她们选择的种族的人就会是她们的父母。因为在表面上的新种族有养子或婚姻的含意。所以若没有得到同意，就不能随意地进行改変种族。
。
原抵抗组织的队长，《解放的姫骑士》伊丽斯的魔人化拜托了哥德文时是紧急事态，她和班达鲁结成了一年以上的同盟，所以他说「我保证她的人格」就答应了。
。
在加奈子的情况下，与雷吉恩在前世合作了几年以上的活动，应该说进行了非法活动……。
（保证人格，怎么看都不能保证。）
。
作为临时的同盟者，比【克洛诺斯】（村上）和【人偶】更可靠，但在这种情况下，很难推荐。
。
「也就是说接受维达新种族化的仪式是前提，在这个仪式之前观察的时间是plan B。
我明白了。先用那个plan B吧」
。
加奈子这样说后，这次大古和梅丽莎也点头了。在确认了这一点后，班达鲁从怀里取出了三个布袋。
。
「那么，在这一段时间内就一起生活吧。在被隔离的生活环境中，听我的话，烹饪后吃饭，练级，没有什么困难的。但因为有特别原因，所以请戴上袋子」
。
对呆住了眼神的加奈子，班达鲁促使她们把布袋戴在头上。
之后，加奈子回顾当时的事情。
。
「虽然已经知道，但是和《地球》的时候完全是另一个人。不只是外表，而是内在。
如果从当时开始就是这样的人的话，就会留在学校里的人的记忆中了」
。
被布袋遮挡视觉的加奈子，被雷吉恩的杰克【转移】到不知名的地方。
。
那里是草原一样的地方，有明显是赶造的白色小屋，附近有口井，不知道为何有像迷宫入口似的楼梯。
。
于是，加奈子们和每天不知从哪里【转移】来的班达鲁生活，教授维达的教义、维达派观点看的历史课，潜入迷宫进行训练，吃着班达鲁烹饪的菜，就这样过了一周左右的生活。
。
「班达鲁老师，真的是什么plan B吗？过着比村的旅馆更好的生活，一边讲解简单的课，和很多不太强的魔物战斗，而且每天都有好吃的东西」
。
小屋的制作若是和《地球》和《origin》相比的话，可以说是普通的。但如果是干净而且有入浴设施的话，在这世界的旅馆里也是挺高级的。
。
「没那么奢侈，真的是plan B啊。但现在你们要老实地喝这个谜的特制果汁。但现在不能说材料是什么。」
。
「……这是果汁？虽然很好喝但是颜色是红色的是什么？」
。
「你们两人不要管外观好吗？好久未尝的甜味啊，还有油炸食品和味增汤哦！哎呀，真是奢侈的东西啊！」
。
「确实，饭是很豪华的。在这个世界里生活了一年以上，才知道《地球》和《origin》的生活很舒适，到现在我才明白。
一边这样说着，大古把抹上了奶油的汉堡放在嘴里。
。
在《拉姆达》的许多国家和地区，虽然也知道包括吃饭在内的各种生活，但现实是比前世自己生存的国家更恶劣，所以对大古的说法点头认同。
。
「你们是用大人的身体转生的，是否可以设计膳食的各种菜式？而且在这个世界上，预算的重要性和《地球》《origin》也相同，只要有钱，也可以过和以前差不多的生活。是了，你知不知道米粉酱的材料？」
。
只要有钱的话，在《拉姆达》也可以住在宽广的宅邸，雇佣优秀的佣人，也可以备齐代替家电的魔术道具，过着舒适的生活。
。
特别是吃饭。有很多不存在的食物和调味料，而且能使用高级食材的魔物的材料来料理，那是在《地球》和《origin》绝对不能吃到的极上美味。
。
有钱使得鬼推磨，所以在异世界大致的问题也可以解决。
。
对于这样主张的班达鲁，大古们都露出了微妙的表情。
。
「嗯，是啊。如果有钱的话」
「但没下工夫也没办法」
「如果是以我们的情况，就不能那么显眼了」
。
大古们脱离了瞄准班达鲁的穆拉姆他们的集团。因此，为了不引人注目，冒险者公会的等级不能上升，控制了委托的领取和魔物素材的出售量。
。
因为职业改变是从镇上移动到其他镇上进行的，所以实力是与A级匹敌，但在公会的实际等级是D级。
「D级冒险者最多，所以容易接到从镇上移动到其他镇上的商人委托做护卫。以我的年纪，D级是有点快，但不是完全没有」
。
「不过，在收入方面就如您所知那样。虽然也有用自己的力量狩猎美味的魔物的手段……但年轻的D级冒险者三人在大魔境的深处、高难度的迷宫出入的话就太显眼了。嗯，但免强来说我也算是大叔啦」
。
「而且我们在短时间内从镇上移动到其他镇上来改变据点，所以几乎没有办法办到你所说的事。」
。
三个人都有相应的知识和技术。虽然在《地球》中是普通的学生，但在《origin》中接受过军事训练，之后更与《第八引导》汇合而进行了非法活动。
。
但是，资金也是在有限的基础上动用，无法使其增长。
。
「想做蛋黄酱的话，醋和油还好说，新鲜的鸡蛋很贵啊。而且，万一我们做的东西被谁留意到而传播开去的话，就会有危险了……贝武夫主义者，可能还有其他危险的人。」
。
「贝武夫主义者？」
。
第一次听到的话，班达鲁回过头来，加奈子先说「我也不太清楚」。
。
「从阿尔达过激派再分枝的集团，惩罚使用堕落的勇者寨卡特的遗产的人作为使命。
也就是说，在我们看来，是杀了使用异世界知识和技术的人们的家伙。当然，在历史上被认定为异端，当时的阿尔达神殿也被肃清了，现在已经不存在。」
。
「那又是怎么说呢？」
。
「不管怎么说，活动太过过激了。只因为栽培了大米就把整个村子烧掉，贵族在领地内的遗迹中偶然发现了勇者的遗物，然后就会被暗杀，发明了未知魔术的便利道具的贤者和他的弟子和家人们都会被杀。」
。
「……那些家伙是狂犬吗？特别是最后的，和异世界的知识和技术似乎没有关系。」
。
虽说是以前的事，但班达鲁不由得听到呆了。只是这样听，就真的不知道是否与寨卡特的遗产有关系。或许只是为了妨碍这个世界独自的进步。
。
如果冷静分析的话，那些贝武夫主义者们也不会正确地了解到《寨卡特》的遗物，也就是不识认对异世界的知识和技术。尽管如此，在开始活动的时候，是以保留传承为基础而判断的吧，但是随着时代的交替，活动也越来越精密，在反复进行过激行动的过程中，不能停止了吧。
。
「当时的阿尔达神殿好像也觉得太过分了。因此，当时的主要成员被认定为异端而被肃清。
如果在被这样肃清前，神会降些神托，让他们停下来就好了」
。
「虽然不想为阿尔达辩护，但是已经远离信仰的信徒，说不定声音就够不到了。
但是，那现在不是不在了吗，贝武夫主义者」
。
「作为组织已经在几万年前消失了，但据说现在也还可能存在。像童话中的妖精一样，只不过是个坏妖精而已。」
。
看来，现在好像没有被确认。在班达鲁想问为什么他们要警戒之前，梅丽莎开口了。
。
「因为我们是以移居你的国家为目的而行动的。为了慎重起见，我们会警戒，而且在奥尔巴选王国的生活是暂时的，所以现在也不能认真地进行调查。」
「所以我们只好忍耐了一段时间。直到能移居前。我想如果从一开始就决定在奧尔巴选王国生活的话，我就会去调查了。」
。
好像是这样的理由。但如果是阿沙兹或村上的话，可能会详细调查一下……本来阿尔达势力的神的信徒就是警戒的对象，所以现在不详细调查也可以吧。
。
在生活的同时谈论这样的世界，大古突然有一种奇怪的感觉，能力值突然上升了。
。
「这个，这是！？力量增强了……这是我隐藏的才能吗！？」
「呵呵，看来你好像被什么觉醒了呢」
「不是那样，大概这是plan B的效果」
。
「是的，你们只是被我引导了而已」
。
plan B的真面目。那是「一起度过并引导」的说话，为了边界山脉外的人们想出來的办法。
能力值的上升是【引导：冥魔创道】的效果。
。
但是，加奈子是精灵，並不是人种，所以并不希望死亡，所以【冥魔创道引诱】没有效果。另外，也没有想到「如果做了这个就绝对能引导」这样确实的手段，做了一些随着时间的推移以某种方式引导，班达鲁的血液是原材料的血药伪装成了果汁给她们喝了。
。
「原来如此……即使迷宫也只是背景而已，这也是其中的一环。」
。
「不，那是为了慎重起见而建的，那是我不会告诉你我在哪里的转移物。」
。
「哦，我明白了。」
。
「那么请重新振作，请确认状态。罗德科尔特的加护和幸运消失了吗？」
罗德科尔特的加护和幸运，是班达鲁以外的转生者们在《origin》的时候开始得到的。
。
但是，【罗德科尔特的加护】並没有显示在雷吉恩的见沼瞳状态上。
。
从那里开始，班达鲁等人想起了假设：「如果从罗德科尔特的影响下脱离的话，就不会显示加护和幸运了吧」。
。
所以，试着先尝试一个星期左右。
。
「【状态】……啊，消失了」
「我也是。顺便说一下，【目标雷达】也消失了。但是【宙斯盾】完好无缺」
「我也还剩下【维纳斯】呢。因为很方便所以很好」
。
三人的加护和幸运都消失了，顺便说一下，【目标雷达】这一亿以上的死属性魔力感知也消失了。
。
「这样我们就可以移民了……啊……也许是我的错觉，你看起来很耀眼……」
「是的，太好了。不是只有我一个人这样感觉到」
「大古，梅丽莎，这是一时的症状。不久就会习惯了」
。
作为引导的结果，他们似乎感受到班达鲁的魅力。
。
「到现在为止没有表情所以没注意到……真是个美少年啊」
。
「……在你面前的是在《地球》中的同学，是和你同一年龄的人。请冷静下来，原偶像」

在加奈子被血迷的时候，家里的门和激烈的声音一起被踢破了。
实际上，这个家的草原是在班达鲁的地下工作室创造的实验用迷宫的内部。可以更安全地作为居住空间和娱乐设施来活用，是为了在指定阶层不产生魔物的实验创造的东西。
。
附近有古尔们进行训练所使用的迷宫入口，入口看起来像是伪装往下层的楼梯。
。
是的，加奈子已经移居到塔洛斯海姆了。
。。。。。
班达鲁引导的效果，已经从罗德科尔特分离出来的加奈子，之后再次提供了其他转生者的信息后，为了成为塔洛斯海姆的国民，正在被真正的指导着……接受了法律、不死者和其他种族的交流方法等的讲座后，在接受维达新种族化的仪式前，被分配到雷吉恩的部下。
。
「因为也有监视的意思，所以我明白了……但到底你们是怎样的状态呢？」
还是一如既往地把兜帽长胞穿上的雷吉恩，大古觉得很奇怪。
。
「依然很奇怪，一次听到多个声音……可以确认吗？」
。
『是啊，从现在开始你就是我们的部下了。』
『我们一直这个样子真是太拘束了！』
『模仿有骨架的人的动作，对现在的我来说是不太纯熟。』
。
『那么，从现在开始看看我们本来的形状吧。想想在《origin》的恐怖电影和游戏中出现的怪物，很有意思的。但如果不能忍受的话就放弃吧』
。
『为了慎重起见，注视着班达鲁后背吧。放心，我准备了各种药品。』
。
每次被雷吉恩的人格发出声音的时候，加奈子的脸色就渐渐变差了。特别是突然回头看的梅丽莎，发现了真相的她被班达鲁看到了，她的面色已经变得苍白如灰。
。
班达鲁必须要保护的对像就像雷吉恩，所以从现在开始就必须习惯。
。
『最后要说的话，就算是悲鸣而害怕，我们也不会在意』
。
「不，我觉得我能忍住的 ――」
。
大古抽搐的悲鳴声Miji Miji尖叫同时扩大，掩盖了其他的声音。
。
但意外的是，加奈子立刻从冲击中恢复，梅丽莎和大古由班达鲁的【精神侵蚀】技能恢复，似乎不需要班达鲁分泌的药剂。
。。。。。。
就这样暂时花了一段时间的班达鲁，但事态却有各种各样的发展。
。
他们的引导后不久，班达鲁从定期来访的《暴虐之岚》的约会场所留下的留言，首次知道阿德帝国的政变。
。
《法命神》阿尔达在历史上也是如此，积极地干涉人类社会。而且，在时间点上只能认为那是为了对付班达鲁他们的行动。
。
但只是让皇帝退位，在其他表面上还没有出现什么奇怪的行动。
。
另一方面，班达鲁现在在做什么的话，就在刚迎来新年的冬天海边寻找能渡海的船。
。
「我为了阿尔达做了好事吧。魔王军残党的恶神和原种吸血鬼都被我消灭了，他们猜不到是我这个大冷门干的吧？」
。
「但是，班达鲁被阿尔达教的伟大人们夸奖也不会高兴吧？」
。
「啊，也是这样」
换算成人种的话是年龄七岁的帕碧娜，和十岁半左右的班达鲁相比，即使无视身体的大小，也看似比班达鲁更年长。
。
帕碧娜和班达鲁，从深渊噩梦马的萨姆货台上露出了脸，看着外面的景色。
。
『哎呀，海即使在冬天也很温暖』
「因为边界山脉内部的海，是在巴恩加亚大陆的南边」
『陛下，那艘船不是很好吗？船头的装饰很帅哦』
「……利维亚公主，可惜那艘船只有前半。」
。
在南面蓝色的大海里，受到白浪冲击的岩礁、有一艘附有藤壶和贝类的沉船上，有『アン゛アーー！』大喊大叫的不死者船员们。
。
「嗯，但是无论是哪艘遇难船也是大同小异的状态。但如果有尺寸合适的遇难船的后半的话，就用不死化紧贴吧」
。
《暴虐之岚》在留言中留下了去魔大陸的海图，所以班达鲁他们正在寻找适合的船。
